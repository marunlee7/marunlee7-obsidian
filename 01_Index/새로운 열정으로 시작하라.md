# 새로운 열정으로 시작하라

## 책소개

한근태 저자의 재출간작!
일과 일상에 새로운 열정을 불어넣을 자기계발 지침서
‘매일 오전 10시, 어떤 마음가짐으로 일을 하고 있는가?'
‘메일 오후 10시, 무슨 생각을 하면서 하루를 정리하고 있는가?’
이 책은 쳇바퀴처럼 똑같이 반복되는 현대인들의 일과 일상에 새로운 열정을 불어넣어주는 지침서로서 일에 대한 가치와 '나 자신'을 잃지 않고 일을 잘하는 방법에 대한 저자의 일침과 노하우를 담고 있다.

## 📕 도서목차

오전 10시, 직장에서의 나
일에 대하여 / 일은 축복이다 / 일이 너무 재미있어요 / 원주민인가 전문가인가 / 일의 심리학 / 주인처럼 일하라 / 지식이 열정을 만든다 / 일 근육을 키워라 / 일 근육 만들기 1 / 일 근육 만들기 2

오전 10시, 내 것 만들기
내가 생각하는 커리어개발 / 커리어를 관리하라 / 자존심이 밥 먹여 준다 / 부정적 낙관주의자가 되라 / 영향력의 원에 집중하라 / 앞장 서서 일하라 / 공격이 최선의 수비다(주도적으로 일하기) / 시간은 목숨이다 / 어제의 나와 경쟁하라

오후 10시, 직장 밖에서의 나
일이 주는 의미 / 자신을 점검하라 / 늘 마지막을 생각하라 / 독립을 꿈꾸는가 / 함부로 그만두지 마라 / 절실해야 한다 / 준비가 끝나면 떠나라

오후 10시, 내 것 만들기
색깔을 분명히 하라 / 이때 움직여라 / Catch me if you can / 책으로 증명하라 / 걱정대신 고민하라 / 걱정하지 말고 화를 내라 / 실패하기 위한 방법

하루 두 번의 10시, 온전히 내 것으로 만들기
누구나 처음엔 초보다 / 지금은 알 수 없다 / 시간은 목숨이다 / 욕심을 부려라 / 관심이 시작이다 / 드러커의 일하는 방식 / 현장에 답이 있다 / 공격이 최선의 수비이다(효과적으로 일하기)
