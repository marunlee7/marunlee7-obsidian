## 📕 MOC
```dataview
TABLE WITHOUT ID
file.link as "제목",
choice(checkbox, "❗", "") AS "❗",
dateformat(created, "yyyy-MM-dd") as "시작일",
dateformat(modified, "yyyy-MM-dd") as "종료일"
FROM "" 
WHERE contains(file.tags, "포엠/금요일에읽는가족의시")
and !contains(file.folder, "90-Settings")
and !contains(file.folder, "10-프로젝트")
and !contains(file.folder, "20-에어리어")
and !contains(file.folder, "40-아카이브")
AND !contains(file.tags, "moc") 
AND !contains(file.tags, "toc")
sort file.name desc
//sort file.mtime desc
//sort file.name asc
sort file.name desc
```

## 📕 소개
:: 남들의 시선으로 바라보는 책읽기
도서관에서 이 책을 선택한 것은 작가가 엄선한 시에 작가의 생각이 있어서이다.
남들이 바라보는 시각이 궁금한 내게 딱 막는 책이었다.
늘 이렇게 남들의 시선이 궁금해지는 책들에 관심이 간다.
추후 책 선택도 또한 같을 것 같은 느낌.
