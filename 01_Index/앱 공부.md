# 📚 T01. 앱 공부 (Master Index)



## 📌 개요 (Overview)

- **목적:** 이 T01 파일은 **앱 공부**과 관련된 모든 T02 노트들을 집계하고 관리하는 마스터 허브입니다.
- **상태:** 이 프로젝트는 현재 **`planned`** 상태입니다.


말씀하신 방식대로 하면 **앱 중심 학습 구조**가 이렇게 압축·정리돼요:

1. **인덱스 (t1)**
    - 학습할 앱 전체 목록, VS Code, Obsidian, Cursor 등
    - 앱별 t2-t6로 연결되는 중심 포인트
2. **리스트 (t2)**
    - 앱별 학습 단계, 핵심 기능, 큰 그림
    - 예: VS Code → 기본 설정, 확장 프로그램, 터미널 연동, 백업 등
3. **개념 (t3)**
    - 앱 관련 용어·기능·핵심 개념 한 페이지에 요약
    - 군더더기 없이 핵심 + 예시 포함
    - 예: 터미널, Git push, Workspace, Snippet
4. **성찰 (t4)**
    - 학습하면서 느낀 점, 개선 아이디어, 나만의 팁
    -d 앱을 직접 다뤄보며 적는 실습 중심
5. **문장 수집 (t5)**
    - 문서·공식 가이드·포럼에서 배운 문장, 명령어, 예시 코드 수집
    - 필요 시 참조용, 바로 검색 가능
6. **개인 데이터 (t6)**
    - 내가 만든 설정, 단축키, 템플릿, 실험 결과 기록
    - 복습·실습 재활용 용도
