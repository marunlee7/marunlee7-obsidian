# Obsidian Interface Menu - Cheat Sheet
## Obsidian Interface Menu
![[Obsidian-Interface-Menu-SS.png]]
 
>[!info] Legend
>1. Explorer contains the Folder and Search shortcuts and you have shortcuts to create a New Note, New Folder, Change Sort Order, Collapse
>2. New Note and New Folders are accessed here
>3. The Ribbon Menu including shortcuts to Comamands are located here
>4. The Left Pane which can be expanded and collapsed
>5. Vaults can be managed here, Open or Create New Vaults
>6. This is where you can access Help Resources for Obsidian
>7. The Obsidian Settings Menu
>8. Top Bar shows you open Notes
>9. Active Note or File Editor Space
>10. The Tab Bar with a link to the File Menu
>11. Backlinks, Outgoing Links, Tags and Outline usually shown here
>12. File Menu can be accessed via the three dots here
>13. The Right Pane which can be expanded and collapsed
>14. Status Bar, shows Backlink, Editor Status, Word Count
