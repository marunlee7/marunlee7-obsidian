## 📘 요약

시각화 매니아들에게 필수 플러그인! 이런 분들은 꼭 보세요~!  
\- 메모에 단계나 과정이 있고, 이런 과정을 비주얼하게 관리하고 싶은 경우  
\- 프로젝트 관리를 아예 옵시디안을 활용해서 기록하고 관리하고 싶은 경우  

### 옵시디안 칸반 보드 플러그인

- 프로젝트 관리 및 작업 흐름을 시작적으로 구성하는 데 유용
- 도요타 자동차 생산 라인에서 유래했다.
- 메모의 진행 단계를 관리하는 데 사용할 수 있다.

### 유튜브 영상 제작 진행관리 예시
아이디어 구상부터 촬영, 편집, 발행까지의 단계를 
칸반 보드로 관리하는 방법.
- 원고작성중
	- 사용자 템플릿 공유하기
	- 사용자 템플릿 만들기
- 스크립트 완료
	- 옵시디안 태그 바꾸기
	- 최근 메모 이동 방법 3가지
- 촬영 완료
- 편집 완료
	- 옵시디언 칸반보드 
- 발행 대기
	- 본문 이동 링크 넣기
	- 옵시디안 데이 플레너

### 프로젝트 진행 단계 예시
프로젝트 진행 단계를 
기획, 집필, 제작, 완료 등으로 나누어 관리하는 방법을 제시한다.

![[작가의 방 칸반보드 프로젝트 진행 관리 예시.png]]

칸반 보드는 복잡한 일을 단순화하여 한눈에 파악하고 
동기를 부여하는 생산성 도구로 활용될 수 있다.


### 칸반 보드 설정 및 추가 기능
카드에 날짜를 추가하여 프로젝트의 시간 관리를 할 수 있다.
완료된 작업은 삭제 대신 보관(아카이브)할 수 있다.
