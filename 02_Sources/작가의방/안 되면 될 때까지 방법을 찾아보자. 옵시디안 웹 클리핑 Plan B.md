# 안 되면 될 때까지 방법을 찾아보자. 옵시디안 웹 클리핑 Plan B

![](https://www.youtube.com/watch?v=PucX0AvxwkM)  

작가님들 어서 오세요, 여기는 작가의 방입니다. 반론의 여지 없이 정보의 많은 부분을 인터넷에서 얻고 있는 시대입니다. 옵시디안과 경쟁하고 있는 많은 메모 앱들은 인터넷에서 찾은 자료롤 스크렙하는 툴을 기본적으로 제공하는 추세인데도 불구하고, 옵시디안은 공식 웹 클리퍼가 없습니다. 그래서 예전에 제가 소개시켜드린 방법이 있었는데, 여러 분들께서 윈도 환경에서는 잘 작동하지 않는다는 이야기를 해주셨습니다. 그래서 이번에는 좀 다른 방식인 구글 크롬에 익스텐션을 활용한 방법을 소개해드립니다. 윈도든 맥이든 크롬이 작동한다면 이 방법으로 이제 웹의 무한한 정보를 쉽게 옵시디안 메모로 가져올 수 있습니다. 바로 시작하겠습니다.
