# 현재 선택 영역 추출 사용법

제목으로 쓸 글을 복사하거나 생각한 후 
문단을 드래그해서 선택하고
단축키를 누른다.

아래 파일은 [[기본만 충실해도 생산성 200% 옵시디안 메모 기본 잔기술 2편]]에서
현재 선택 영역 추출(Cmd + Ctrl + N)로 
새롭게 만들어진 페이지다.

원문에는 이 페이지가 임베딩 되어있다. 
확인방법 : 아래 [링크된 언급]에서 원문 확인 가능

> [!abstract]+ abstract 현재 선택 영역 추출 → 텍스트를 새로운 파일로 추출
> 1. 병합할 파일 선택
> 2. 또는 새로운 파일명
> 3. 설정에서 [[tp2 note x]] 빈 페이지를 이용해야 병합할 때 꼬이지 않는다.
> 	- 즉 프론트매터 및  본문 빈 페이지로 설정
> 4. 새로운 페이지 생성시 : [[tp2 note column]] 사용
> 	1. 개인적으로 병합보다는 새로운 페이지이다. 
> 	2. 또는 독후감, 한자어휘들을 중첩해서 모아갈때는 병합도 괜찮음. 
> 5. 사용법
> 	1. 공감가는 글을 적고
> 	2. 별도 페이지로 만들기
> 	3. 병합아닌 새로운 페이지 제목 설정
> 	4. 원문에 적었던 글들이 이동
>
