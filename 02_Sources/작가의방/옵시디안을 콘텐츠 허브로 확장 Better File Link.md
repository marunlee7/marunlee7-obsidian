# 옵시디안을 콘텐츠 허브로 확장하는 추천 플러그인

![](https://www.youtube.com/watch?v=4I-5xyuyYjI)

오늘은 심플하지만 저에게 있어 아주 유용한 플러그인을 하나 소개해드리려고 합니다. 옵시디안을 사용하다 보면 단순한 메모장으로서뿐만 아니라 제2의 두뇌, 내 모든 자료의 허브로까지 사용 영역이 자연스럽게 넓혀지기 마련이죠. 특히, 메모뿐 아니라 엑셀, 워드, 심지어 아래아 한글 문서들까지도 옵시디안 메모와 연결해서 통합적으로 관리하고 활용하고 싶을 때가 있는데요, 오늘 소개시켜드릴 이 간단한 플러그인만 있으면 가능해집니다. 워낙 단순한 플러그인이라 이번 영상에서는 소개와 함께 제가 사용하고 있는 예시도 담았습니다. 바로 시작하겠습니다.

## 要約 ✍︎

⚙️ 옵시디안 플러그인: Better File Link 소개
작가의 방 채널에서 옵시디안을 콘텐츠 허브로 확장하는 플러그인, Better File Link를 소개합니다. 이 플러그인을 통해 **옵시디안이 단순한 메모장을 넘어 제2의 뇌**처럼 활용될 수 있다고 설명합니다. Better File Link는 엑셀, 워드, HWP 등 다양한 파일들을 옵시디안 노트와 통합 관리할 수 있게 해줍니다. **설치 및 사용법이 매우 간단**하여 초보자도 쉽게 활용할 수 있습니다. 

🛠️ 설치 및 설정 방법
설치는 다른 플러그인과 마찬가지로 설정 -> 커뮤니티 플러그인에서 Better File Link를 검색하여 설치합니다. 설치 후 활성화 버튼을 누르고, **옵션 설정은 기본값으로 유지해도 무방**합니다. 

🔗 Better File Link의 작동 원리
이 플러그인은 ==외부 파일의 위치 정보(주소)만을 가져와 노트에 연결하는 방식(outbound link)을 사용==합니다. **실제 파일은 원래 위치에 그대로 유지**되며, 링크를 클릭하면 해당 파일이 열립니다. 옵시디안 자체적으로는 PDF나 비디오 파일만 첨부 가능했지만, Better File Link를 사용하면 **컴퓨터에서 사용 가능한 모든 디지털 파일을 연결**할 수 있어 활용도가 매우 높습니다. 

📝 사용 방법 및 장점
Command + P 팔레트에서 Better File Link를 실행 후 파일 선택을 통해 연결할 파일을 선택합니다. **복잡한 설정 없이 간단하게 링크를 생성**할 수 있습니다. 옵시디안에서 지원하지 않는 파일 형식도 링크할 수 있으며, 링크 클릭 시 해당 프로그램으로 바로 열립니다. 또한, **파일 링크 정보만 가져오기 때문에 옵시디안 저장소 용량을 소비하지 않습니다.** 
단축키 : Ctrl + Shift + F

📚 활용 예시
책을 읽고 독서 노트를 작성할 때, **관련 마인드 맵 파일을 링크**하여 노트와 함께 관리할 수 있습니다. 기존에는 파일 탐색기에서 마인드 맵 파일을 찾아야 했지만, 이제는 링크 클릭 한 번으로 쉽게 접근할 수 있습니다. 복잡한 계산이 필요한 엑셀 파일도 노트와 연결하여 함께 관리할 수 있습니다. **센서스 관련 노트에 엑셀 파일을 링크**하여 모든 자료를 한 곳에서 관리하는 것이 가능합니다. Windows 환경에서도 문제없이 작동하며, 한글 파일(HWP)도 잘 지원됩니다. **파일 형식에 구애받지 않고 모든 지식을 옵시디안 노트와 연결**할 수 있다는 점이 핵심입니다. 

🚀 결론: 옵시디안의 확장
Better File Link 플러그인을 통해 기존의 데이터 관리 시스템을 유지하면서 옵시디안 노트와 통합할 수 있습니다. **옵시디안이 단순한 노트를 넘어 제2의 뇌로 확장**되는 경험을 할 수 있습니다. 개인의 사용 환경과 목적에 따라 무한한 활용이 가능하며, **데이터 관리 효율성을 극대화**할 수 있다는 점에서 매우 유용한 플러그인입니다.
