# 옵시디안 웹 클리핑 논란 종결 최고의 공식 웹 클리퍼

![](https://www.youtube.com/watch?v=VKIuI1_58_w)  

작가님들 어서 오세요. 여기는 작가의 방입니다. 이제까지 옵시디안에 웹에서 찾은 정보를 손쉽게 스크렙하는 방법에 대해서 소개해드린 적이 있습니다. 이제까지 옵시디안에 공식적으로 스크렙 기능을 제공지 않다보니, 이용자들이 이런 기능을 추가했고, 그러다 보니 기능이나 안정성에서 문제가 좀 있었습니다. 그래서 제가 모든 작가님들이 만족할 만한 방법을 끝까지 찾아보겠다고 약속까지 드렸는데요, 드디어 오늘 그 약속을 지킬 수 있게 되었습니다. 무려 '공식' 솔루션이고요, 옵시디안을 사용하신다면 고민할 필요 없이 설치해서 사용하셔야 할 것 같습니다. 바로 시작해보겠습니다.

## 要約 ✍︎

🎉 옵시디안 공식 웹 클리퍼 등장!
옵시디안에서 공식 웹 클리핑 도구를 드디어 출시했습니다! 기존의 스크랩 방식들의 문제점을 해결하고, **옵시디안 사용자라면 고민 없이 사용할 만한** 편리한 기능입니다. 

💡 웹 클리퍼란 무엇일까요?
웹 클리퍼는 웹페이지를 스크랩하여 옵시디안 메모로 바로 옮기는 기능입니다. HTML 웹페이지를 마크다운 형태로 변환하고, **원하는 폴더에 바로 저장**할 수 있어 매우 편리합니다. 이번에 나온 옵시디안 웹 클리퍼는 웹 클리핑뿐만 아니라, **웹에서 특정 부분만 하이라이트**하고, 하이라이트한 부분만 메모로 추출하는 기능도 제공합니다. 

⚙️ 설치 및 기본 설정 방법
옵시디안 공식 사이트의 다운로드 메뉴에서 옵시디안 클리퍼를 설치할 수 있습니다. 크롬, 사파리, 파이어폭스 등 **대부분의 웹 브라우저를 지원**합니다. 설정은 크롬 확장 프로그램에서 바로 해야 합니다. 보관소 설정, 단축키 설정 등을 할 수 있으며, 특히 ==**단축키는 세 가지==밖에 없으니 외워두는 것이 좋습니다**. 

✨ ==템플릿 설정의 중요성==
스크랩할 때 어떤 템플릿을 이용해서 메모를 만드느냐가 중요합니다. 
기본 템플릿을 사용하거나, **사용자 정의 템플릿을 만들어** 원하는 형태로 스크랩할 수 있습니다. 
템플릿 속성을 설정하여 URL, 작성일 등의 메타데이터를 자동으로 가져올 수 있습니다. 
**자신에게 필요한 정보만** 템플릿에 추가하여 효율적인 메모 작성이 가능합니다. 

🖍️ 하이라이트 기능 활용
웹페이지에서 원하는 부분을 하이라이트하고, 하이라이트된 부분만 추출하여 메모로 만들 수 있습니다. 하이라이트 스타일을 설정하여 **나만의 강조 스타일**을 적용할 수도 있습니다. 하이라이터가 있으면, 이전에 하이라이트했던 부분이 페이지를 다시 방문했을 때 그대로 남아있어, **마치 책에 밑줄을 그어놓은 것 같은 효과**를 줍니다. 

🎬 ==트리거 기능으로 특정 페이지에 특정 템플릿 적용==
트리거 기능을 활용하면, 
특정 페이지에서 스크랩할 때 자동으로 특정 템플릿이 적용되도록 설정할 수 있습니다. 
예를 들어, **유튜브 페이지에서 스크랩할 때 유튜브 템플릿이 자동으로 적용**되도록 할 수 있습니다. 

👍 옵시디안 웹 클리퍼, 클리퍼 끝판왕?
옵시디안 웹 클리퍼는 매우 간단하고 사용하기 쉬우며, 다양한 기능을 제공합니다. 
템플릿과 트리거 기능을 조합하여 **무한한 응용**이 가능합니다. 
아직 초기 버전이라 부족한 부분도 있지만, 앞으로 기능이 더욱 좋아질 것으로 기대됩니다. 
이제 더 이상 클리핑 문제로 고민할 필요 없이, **옵시디안 웹 클리퍼로 완벽하게 해결**하세요!
