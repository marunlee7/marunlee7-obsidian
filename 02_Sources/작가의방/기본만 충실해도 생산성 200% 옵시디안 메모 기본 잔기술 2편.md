	- 개인적으로 플러그인을 최소화하려는 사용자에게 유용합니다. 
- 사용법
	- 템플릿 폴더에 파일 생성 : [[inst time stamp]]

	- Cmd + P : inst time stamp 불러오기 : {{date:YYYY-MM-DD}} {{time}} :

📝 노트 작성기를 활용한 문헌 노트 생성
- 책을 읽으면서 ==하이라이트한 부분을 문헌 노트로 만들 때==, 노트작성기를 활용하면 효율적입니다.
- 텍스트 리팩토링을 통해 선택한 텍스트를 새로운 파일로 추출하고,
- 원본 파일에는 링크를 남길 수 있습니다. 

- 사용법
	- 설정에서 "노트 작성기"를 활성화한 후, 
	- 텍스트를 선택하고 마우스 오른쪽 클릭하여 "현재 선택 범위 추출"을 선택
	- 새로운 파일의 제목을 입력하면 해당 내용이 새로운 노트로 생성됩니다.
	- 단축키를 활용하면 더욱 편리하게 사용할 수 있습니다. 
	- 마우스 우클릭 대신 단축키를 활용하여 생산성을 높일 수 있습니다. 
	- 단축키 설정에서 "==현재 선택 영역 추출=="에 원하는 
	- 단축키 (예: ==Command + Ctrl + N==)를 지정합니다. 
	- 이렇게 하면 더욱 빠르고 효율적으로 ==문헌 노트를 생성==할 수 있습니다.
