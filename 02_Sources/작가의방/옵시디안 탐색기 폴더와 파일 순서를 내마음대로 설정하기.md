## 要約 ✍︎

👋 옵시디안 탐색기 폴더 정렬, 왜 마음대로 안 될까?
많은 옵시디안 사용자분들이 파일 탐색기에서 폴더나 파일 순서를 **자신만의 논리적인 기준으로 정렬하고 싶어합니다**. 기본적으로 옵시디안은 알파벳순, 역순, 업데이트/작성일 기준으로만 정렬을 제공하기 때문에, **사용자 정의 정렬에 대한 요구가 꾸준히 있었습니다**. 기존의 정렬 방식은 폴더의 용도나 중요도와 상관없이 나열되어 **사용자 워크플로우에 맞지 않는 경우가 많습니다**. 예를 들어, '인박스', '리터러처 노트', '연구 노트' 순으로 폴더를 정렬하고 싶어도 기본 기능으로는 불가능했습니다. 

🛠️ ==Custom File Explorer Sorting 플러그인== 설치 및 설정
이 문제를 해결하기 위해 'Custom File Explorer Sorting'이라는 플러그인을 사용할 수 있습니다. 설정 > 커뮤니티 플러그인에서 해당 플러그인을 찾아 설치하고 활성화합니다. **플러그인 활성화 후 추가 설정은 필수가 아닙니다**. 플러그인이 작동하려면 **`sortspec.md`라는 특정 이름의 파일이 볼트 내에 존재해야 합니다**. 이 파일 안에 정렬 규칙을 정의하는 코드를 작성해야 합니다. 

📝 `sortspec.md` 파일 설정 및 정렬 규칙 정의
`sortspec.md` 파일을 생성한 후, 아래 영상 설명란에 제공된 코드를 복사하여 붙여넣습니다. **이 코드는 메타데이터 문법을 사용하여 정렬 명령어를 정의합니다**. 코드 내에서 `root`를 기준으로 정렬할 폴더 또는 파일을 지정할 수 있습니다. 예를 들어, `인박스`, `리터러처 노트`, `영구 노트` 순으로 정렬하려면 해당 순서대로 코드를 수정합니다. **원하는 순서대로 폴더명을 나열하면 됩니다**. 설정을 완료한 후, 파일 탐색기에 생성된 아이콘을 클릭하면 **정렬이 즉시 적용됩니다**. `sortspec.md` 파일의 위치는 어디든 상관없습니다. 

🔄 정렬 끄기 및 원래대로 복구
언제든지 정렬을 끄고 원래대로 되돌릴 수 있습니다. 파일 탐색기의 아이콘을 다시 클릭하면 **정렬이 비활성화되고 기본 정렬 방식으로 돌아갑니다**. 정렬을 다시 활성화하려면 아이콘을 다시 클릭하면 됩니다. **이 플러그인은 필요에 따라 쉽게 켜고 끌 수 있습니다**. 

📌 팁: 폴더명 앞에 숫자 붙여서 정렬하는 방법
개인적으로는 폴더명 앞에 숫자를 붙여서 정렬하는 방법을 선호합니다. 예를 들어, '0 인박스', '1 리터러처 노트'와 같이 숫자를 붙이면 **중요도나 논리적 순서에 따라 폴더를 정렬할 수 있습니다**. 태그나 링크를 활용하여 메모를 정리하는 방법도 있습니다. **다양한 방법을 시도해보고 자신에게 가장 적합한 방법을 찾는 것이 중요합니다**.
