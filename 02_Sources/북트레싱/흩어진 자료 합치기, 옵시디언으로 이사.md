# 흩어진 자료 합치기, 옵시디언으로 이사

![](https://www.youtube.com/watch?v=WdYz5kpeR2c)  

기존에 사용하시던 여러 노트앱에서 옵시디언으로 노트를 옮기는 방법을 다룬 영상입니다.  
사용 중인 혹은 사용했던 프로그램에서 Export 하여 옵시디언에 Import 하면, 링크나 테이블, 마크다운 등도 그대로 가져오게 되어 원본에 최대한 가깝게 자료를 유지할 수 있습니다.  

옵시디언의 Importer 플러그인 사용해 노션(Notion), 업노트(UpNote), 원노트(OneNote), 에버노트(EverNote), 애플 메모 앱 (Apple Memo), 베어앱 (Bear App) 등에서 파일을 가져올 수 있습니다.  

지식 자료들을 옵시디언 한 곳에서 통합 관리하여 자료 간의 연결을 통해 생산성을 향상 시킬 수 있습니다 :)
