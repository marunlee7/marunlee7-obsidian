# 2. 옵시디언 핵심 기능

- **양방향 링크 (Bidirectional Linking):** 형태로 노트를 연결하면, 두 노트 모두에서 서로를 참조하는 링크를 자동으로 생성하여 지식 간의 관계를 명확히 보여줍니다.

- **그래프 뷰 (Graph View):** 볼트 내의 모든 노트와 그 연결 관계를 시각적으로 보여주는 기능입니다. 지식 구조를 한눈에 파악하고 숨겨진 통찰을 발견하는 데 유용합니다.

- **마크다운 기반:** 단순한 텍스트 파일($\text{.md}$)을 사용하므로, 플랫폼 독립적이며 장기적인 보존이 용이합니다.
