# 3. 옵시디언 플러그인

- **강력한 확장성:** 옵시디언은 코어 기능 외에도 사용자들이 개발한 **커뮤니티 플러그인**을 통해 기능을 무한대로 확장할 수 있습니다.
    
- **주요 플러그인 예시:**
	Dataview: 노트를 데이터베이스처럼 쿼리하여 목록이나 표로 보여주는 기능 (데이터 시각화/관리)
	Templater: 노트 생성 및 반복 작업을 자동화하는 고급 템플릿 기능
	Tasks: 마크다운 체크리스트를 기반으로 할 일을 추적하고 관리하는 기능

## 플러그인
