# 4. 옵시디언 효과적인 사용전략

- **단축키 습관화:** 자주 사용하는 명령(명령 팔레트, 노트 이동, 사이드바 토글 등)에 단축키를 설정하여 작업 속도를 높입니다.
    
- **템플릿 활용:** 일일 기록, 프로젝트 계획, 독서 노트 등 반복적인 형식의 노트는 템플릿을 만들어 일관성을 유지하고 시간을 절약합니다.
    
- **지식 연결에 집중:** 단순한 자료 보관을 넘어, 양방향 링크와 태그를 사용하여 서로 다른 주제 간의 관계를 적극적으로 연결하고 탐색합니다.
    
- **최적화된 환경 구축:** 자신의 워크플로우에 맞는 소수의 필수 플러그인과 테마를 선택하여 과부하 없이 효율적인 개인 맞춤형 도구를 구축합니다.
