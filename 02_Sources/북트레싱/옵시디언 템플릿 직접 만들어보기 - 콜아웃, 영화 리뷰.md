# 옵시디언 템플릿 직접 만들어보기, 콜아웃 기능, 영화 리뷰 예시

![](https://www.youtube.com/watch?v=Q2-aHQKEOHs)  

이번 영상은 Templater 플러그인 2편의 영상 중 두 번째 영상입니다.  
1편에서 알아본 템플레이터 플러그인 명령어를 사용하여 영화 리뷰 템플릿을 직접 만들어보고, 플러그인을 활용해 더욱 쉽게 템플릿을 사용하는 방법에 대해 알아봤습니다.  

1편에서 언급하지 않았던, split 라는 명령어와 콜아웃 기능에 대해서 추가로 살펴봤습니다.  

다음 영상에서는 지금까지 배운 것들을 응용하여 데일리 노트 작성에 대해 알아보겠습니다.
