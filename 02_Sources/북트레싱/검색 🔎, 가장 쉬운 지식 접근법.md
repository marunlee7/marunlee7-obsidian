# 검색 🔎, 가장 쉬운 지식 접근법

![](https://www.youtube.com/watch?v=KZkBdaJpQ-c)  

## 📘 요약

지식을 수집하는 것 만큼 지식에의 접근도 중요합니다.  
==검색의 기본 기능부터 정규 표현식을 사용한 디테일한 설정==을 통해 
내가 찾고자 하는 노트를 정확하게 찾을 수 있습니다.  

## ⏰ 타임스탬프
[00:00](https://www.youtube.com/watch?v=KZkBdaJpQ-c) 영상 설명
옵시디언에서 노트가 쌓일수록 검색 기능의 중요성이 커집니다. 
**제목, 경로, 태그, 프론트매터, 노트 내용까지 검색 가능**하며, 
이번 영상에서는 기본적인 검색 방법부터 정규 표현식을 이용한 고급 검색까지 다룹니다. 
	
[00:56](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=56s) 검색 단축키
**Cmd + F는 현재 노트 내에서 검색**, Cmd + Shift + F는 볼트 전체에서 검색하는 단축키입니다. 
노트 내 검색 시 상단에 검색창이 나타나며, '독서'와 같은 단어를 입력하여 찾을 수 있습니다. 
노트 내 검색 결과는 한 번에 볼 수 없고, 
단어 변경 기능도 기본적으로는 제공되지 않습니다.
**Global search and replace 플러그인을 사용하면 볼트 전체에서 단어 변경이 가능**하지만, 
불편함이 있어 기본 기능에 포함되기를 기대하고 있습니다. 

[01:19](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=79s) 노트 내 검색
볼트 전체 검색은 탐색기 하단의 검색 버튼 또는 Shift + Cmd + F 단축키로 실행합니다. 
검색 옵션은 6가지가 있으며, 검색창에 'note'를 입력하면 관련 결과를 확인할 수 있습니다.
**검색 결과는 복사하거나 북마크에 추가**할 수 있습니다. 
대소문자 구분 옵션을 통해 검색 결과를 조절할 수 있습니다. 
**AA 아이콘을 누르면 대소문자를 구분하여 검색**하며, 끄면 구분 없이 검색합니다.
설정 아이콘을 통해 검색 결과 표시 방식을 변경할 수 있습니다. 
검색 결과 축소, 앞뒤로 표시, 검색 단어 표시 등의 설정을 통해 
검색 결과를 원하는 방식으로 볼 수 있습니다.
**Cmd 키를 누른 상태로 노트를 클릭하면 내용 미리보기가 가능**하며,
미리보기 창이 팝업되지 않는 경우 코어 플러그인에서 페이지 미리보기를 활성화해야 합니다. 
검색된 노트는 제목, 업데이트 날짜, 작성일 순서 등으로 정렬할 수 있습니다. 

[02:16](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=136s) 볼트(Vault) 내 검색- 기본
[03:47](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=227s) 검색 결과 보기 설정
[04:48](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=288s) 검색 결과 설명

[05:41](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=341s) 검색 창 기본 사용법
검색창에서 띄어쓰기는 AND 조건으로 작동합니다. 
**"옵시디언 마크다운"처럼 쌍따옴표(\")로 묶으면 해당 구문을 정확히 포함하는 노트를 검색**합니다. OR 조건을 사용하려면 '옵시디언 OR 마크다운'과 같이 입력해야 합니다. 
==OR은 반드시 대문자로 작성해야 합니다. ==
괄호()를 사용하여 검색 조건을 그룹화할 수 있습니다. 
예를 들어, '(옵시디언 OR 마크다운) 독서'는 옵시디언 또는 마크다운이 포함되고
'독서'도 포함하는 노트를 검색합니다. 

[07:42](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=462s) 검색 옵션 설명
'파일 경로 일치'를 선택하면 특정 폴더 내의 노트를 검색할 수 있습니다.
'파일 이름과 일치'는 파일 제목을 기준으로 검색합니다. 
'태그' 옵션을 사용하면 특정 태그가 포함된 노트를 검색할 수 있으며, 
**AND나 OR 조건을 함께 사용**할 수 있습니다. 
'Line' 옵션은 한 줄에 특정 키워드가 모두 포함된 노트를 검색합니다.
'Section' 옵션은 특정 제목 아래에 키워드가 포함된 노트를 검색합니다. 
'Property' 옵션은 프론트매터의 Key와 Value를 이용하여 검색합니다.
Property 검색을 태그 검색에 활용하는 것을 추천합니다.
기존 태그 검색은 정확한 태그명을 입력해야 하지만, 
**Property 검색을 사용하면 유사한 태그까지 검색**할 수 있습니다.


[11:06](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=666s) 태그 확장 검색
[13:50](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=830s) 검색 결과 본문 임베딩
Tag Wrangler 플러그인을 사용하여 하위 태그를 만들거나, 
Property 검색을 통해 유사한 태그를 검색할 수 있습니다. 
옵시디언의 버그로 인해 'tags'가 아닌 'tag'로 검색해야 하는 경우가 있습니다.
코드 블록 내에 query를 사용하여 검색 결과를 노트에 임베딩할 수 있습니다. 
`[tag: 도서]` 또는 '옵시디언 마크다운'과 같은 검색어를 코드 블록에 입력하면 
해당 검색 결과가 노트에 표시됩니다. 

[14:32](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=872s) 정규 표현식 (Regex) 활용
[15:53](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=953s) Regex- 패턴 검색 (숫자)
[17:02](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=1022s) Regex- 패턴 검색 (한글)
[18:25](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=1105s) Regex- 패턴 응용 검색
[19:03](https://www.youtube.com/watch?v=KZkBdaJpQ-c&t=1143s) Regex- 패턴 검색 (영어)
정규 표현식을 사용하려면 검색창에 슬래시(/)를 입력합니다. 
'^마크다운'은 '마크다운'으로 시작하는 단어를 검색하고, 
'설정$'은 '설정'으로 끝나는 단어를 검색합니다. 

특정 패턴을 검색할 수도 있습니다. 
예를 들어, '/개/'는 모든 '개'라는 글자를 검색하고, 
'/\\d개/'는 '개' 앞에 숫자가 있는 부분을 검색합니다. 
'[가-힣]력'은 '력'으로 끝나는 한글 단어를 검색합니다.
'회복[가-힣]'을 검색하면 '회복' 뒤에 한 글자가 오는 단어를 찾을 수 있습니다.
'time[a-z]*p'는 'time'과 'p' 사이에 여러 개의 영문자가 있는 단어를 검색합니다.
