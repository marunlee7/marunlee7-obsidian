# 나만 몰랐던 기본 기능- 파일 복구, 레이아웃 설정, 마크다운 입력

![](https://www.youtube.com/watch?v=jU7nwO_ZBCc)  

옵시디언에서 노트를 작성하다 보면 기존에 작성했던 내용으로 되돌리고 싶은 경우가 있습니다. 되돌리기(Undo) 기능으로 어느 정도는 가능하지만 복구가 되지 않는 경우가 있습니다.  

이번 영상에서는 기존의 버전을 복구할 수 있는 스냅샷 기능에 대해 알아보고 예전 옵시디언 설정에 대한 설명 영상에서 다루지 못했던 [[Properties]] ([[규칙 - 프론트매터]]) 숨기기, 마크다운 작성법, 레이아웃 저장하기 등의 내용에 대해 다뤄보았습니다.  

CSS 스니펫을 적용하는 부분을 제외한 모든 내용은 옵시디언에서 기본적으로 제공하는 기능으로 사용이 가능합니다.  

옵시디언 사용에 많은 도움이 되길 바랍니다.  

⏰ 타임스탬프  

- 00:00 영상 설명  
- 00:45 **스냅샷 기능 (파일 복구)**  
- 04:05 명령어 팔레트 즐겨찾기 추가  
- 05:44 페이지 미리보기 설정  
- 06:59 리본 (내비게이션 바)  
- 08:01 Properties (프론트매터) 보기  
- 09:36 **Properties 숨기기 CSS 스니펫**
- 10:35 괄호, 마크다운 페어링  
- 11:54 **괄호, 마크다운 입력방법 2** → 텍스트 드래그후 마크다운 넣기
- 12:51 **레이아웃 저장, 관리**  
- 15:21 내용 정리
