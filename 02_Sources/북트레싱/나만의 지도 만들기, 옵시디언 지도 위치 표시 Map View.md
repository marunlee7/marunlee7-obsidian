# 나만의 지도 만들기, 옵시디언 지도 위치 표시 Map View

![](https://www.youtube.com/watch?v=yLwZk86LoCk)  

옵시디언에서 위치를 기록하도록 도와주는 Map View 플러그인에 대한 설명 영상입니다.  

영상은 총 두 편으로 나눠 업로드 예정이고, 본 영상에서는 Map View 플러그인의 기본적인 기능과 기본적인 기능을 활용한 예시도 포함되어 있으니 많은 시청 부탁드리겠습니다.  

노트에 위치를 포함하게 되면 맛집 리스트, 부동산 임장 노트, 지역별 고객 관리, 여행 기록 등 다양하게 활용이 가능합니다.
