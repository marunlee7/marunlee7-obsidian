# 🧑‍💻 코딩 공부 기록, 옵시디언에 깔끔하게 정리하기

![](https://www.youtube.com/watch?v=bRbaiTbngUc)  

옵시디언의 사용 목적은 사용하시는 분들의 필요에 따라 다양합니다.  
이번 영상에서는 코딩 공부를 학습하시는 분들에게 많은 도움이 될 영상입니다.  

총 3개의 플러그인에 대한 설명과 사용법을 알아보았습니다.  
1\. Excute Code- 코드 블록 실행  
2\. Code Styler- 코드 블록 스타일링 (제목, 아이콘, 하이라이트 등)  
3\. Vscode editorVS Code처럼 옵시디언에서 직접 소스 코드 파일을 편집 가능  

다양한 옵시디언의 활용 방법에 대해 고민해보는 시간이 되었으면 합니다.
