# Whisper(위스퍼)로 음성 메모 파일 텍스트로 변환하기

![](https://www.youtube.com/watch?v=a2s6gV9SSwc)  

대화나 회의 내용을 녹음한 파일을 음성 파일 그대로만 보관한다면 그저 '저장'의 기능에 그치지 않습니다. 이 음성 파일을 텍스트로 변환하게 되면 더욱 손쉽게 정보에 접근하고, 그 내용이 파악하기 쉬어 활용도가 높아집니다.  

이번 영상은 옵시디언에 텍스트로 변환하는 내용에 대한 영상입니다.  
음성 녹음을 통한 메모는 순간 떠오르는 아이디어를 저장하는데도 굉장히 유용한 도구로 생산성을 한층 높여줍니다.  

음성을 텍스트로 변환하는데 whisper라는 모델이 자주 사용되는데, chatGPT를 만든 OpenAI라는 회사에서 제작한 모델로 무료로도 사용이 가능하지만 옵시디언에서는 적은 금액으로 유료로 사용하시는 것이 훨씬 더 쉽고 편리하다고 생각합니다.

## 要約 ✍︎

📝 음성 메모의 중요성
음성 메모는 스마트폰으로 간편하게 기록할 수 있어 회의 내용 기록이나 아이디어 메모에 유용합니다. **음성 파일만으로는 내용을 파악하기 어렵지만, 텍스트로 변환하면 검색이 용이하고 내용을 한눈에 볼 수 있습니다.** **순간적으로 떠오르는 아이디어를 기록하는 것은 생산성 향상에 매우 중요합니다.** 음성 메모는 복잡한 생각을 정리하지 않고 자연스럽게 기록할 수 있으며, 휘발되기 쉬운 아이디어를 즉시 기록하는 데 효과적입니다. 

💽 옵시디언(Obsidian)에 음성 파일 임베딩하기
음성 파일을 옵시디언으로 드래그 앤 드롭하면 자동으로 임베딩됩니다. **옵시디언 내에서 재생, 음량 조절, 재생 속도 조절이 가능합니다.** 하지만 직접 들으면서 타이핑하는 것은 번거롭습니다. 

🗣️ Whisper 플러그인 소개 및 설치
Whisper는 음성을 텍스트로 변환해주는 모델로, **이미 여러 프로그램에서 널리 사용되어 성능이 검증되었습니다.** OpenAI에서 개발되었으며, 옵시디언에서 편리하게 사용하기 위해 Whisper 플러그인을 설치합니다. 옵시디언 설정에서 커뮤니티 플러그인 탐색을 통해 Whisper를 검색하여 설치하고 활성화합니다. 옵션에서 OpenAI API 키를 설정해야 합니다. 

🔑 OpenAI API 키 발급 및 설정
OpenAI API 키는 openai.com에서 발급받을 수 있습니다. **기존 계정으로 로그인하거나 새로 가입할 수 있습니다.** API 페이지에서 Billing을 통해 크레딧을 확인하고, 필요시 결제 정보를 등록하여 크레딧을 충전할 수 있습니다. API 키를 발급받기 위해서는 휴대폰 번호 인증이 필요합니다. 인증 후 API 키를 생성하여 옵시디언 Whisper 플러그인 옵션에 붙여넣습니다. 플러그인 옵션에서 언어를 한국어(ko)로 설정합니다. Recording folder와 Transcription folder는 기본 설정대로 유지합니다. 

💰 Whisper 사용 비용 및 제한 사항
Whisper 사용 비용은 **약 8원/분**이며, 월 10분씩 사용 시 약 2,500원 정도입니다. 크레딧은 음성 파일 길이에 따라 정확히 차감됩니다. Whisper 플러그인은 25MB 이하의 파일만 지원합니다. **아이폰으로 1시간 녹음 시 30MB가 넘을 수 있으므로, 파일 분할이 필요할 수 있습니다.** 지원하는 파일 확장자는 .mp3, .m4a, .wav, .webm 등입니다. 

⚙️ Whisper 플러그인 사용법
Cmd + P를 눌러 Whisper 명령어를 실행하고, "Upload audio file"을 선택하여 텍스트로 변환할 파일을 선택합니다. **텍스트 변환 결과는 매우 정확합니다.** Whisper 플러그인은 직접 녹음 기능도 제공합니다. Cmd + P를 눌러 "Start recording"을 선택하면 녹음이 시작되고, 다시 한번 누르면 녹음이 종료되며 텍스트로 변환된 내용과 음성 파일이 새로운 노트에 저장됩니다. 녹음 파일과 텍스트 파일은 볼트 최상단에 저장되며, 설정에서 저장 위치를 변경할 수 있습니다. 

✨ 생산성 향상을 위한 메모 습관
메모는 생산성과 직접적인 연관이 있으므로, **언제 어디서나 메모할 수 있는 환경을 갖추는 것이 중요합니다.**
