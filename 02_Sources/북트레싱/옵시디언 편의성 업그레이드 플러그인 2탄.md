# 옵시디언 편의성 업그레이드 플러그인 2탄

![](https://www.youtube.com/watch?v=O8iZRtHYcxQ)  

지난 [[🏷️ 옵시디언|옵시디언 플러그인]] 7가지를 다뤘던 1편에 이어서 
두 번째 영상으로 이번에는 8가지 플러그인을 담았습니다.  
;플러그인을 위주로 다뤘으니 많은 시청 바랍니다.  

## ⏰ 타임스탬프
[00:00](https://www.youtube.com/watch?v=O8iZRtHYcxQ) 영상 설명
[00:31](https://www.youtube.com/watch?v=O8iZRtHYcxQ&t=31s) ==커서 위치 기억== (Remember cursor position)
	노트를 다시 열 때 커서가 항상 맨 앞으로 이동하는 불편함을 해결해줍니다. 
	**'Remember Cursor Position' 플러그인을 설치**하면, 
	노트를 닫았다가 다시 열어도 마지막 커서 위치를 기억하여 이어서 작성할 수 있습니다. 

[01:39](https://www.youtube.com/watch?v=O8iZRtHYcxQ&t=99s) 불필요한 파일 정리 (File cleaner redux)
	Vault 내에 불필요하게 쌓이는 파일들을 관리하는 데 유용합니다. 
	**'File Cleaner Redux' 플러그인을 사용**하면 
	==내용이 없는 노트나 사용되지 않는 이미지 파일 등을 쉽게 찾아 삭제할 수 있습니다==. 
	기본적으로 삭제된 파일은 시스템 휴지통으로 이동되지만, 
	`.trash` 폴더로 이동하도록 설정할 수 있습니다. 
	또한, 특정 폴더나 파일 확장자를 지정하여 삭제 대상에서 제외할 수도 있습니다. 
	예를 들어, 이미지 파일(.jpg, .png)만 검색하도록 설정할 수 있습니다. 
	Front matter가 포함된 노트를 제외하거나, 특정 폴더만 포함하도록 설정하는 것도 가능합니다.
	**이 플러그인을 사용하면 Vault를 깔끔하게 유지**할 수 있습니다. 
	Obsidian에서는 읽기 모드와 편집 모드 두 가지 뷰 모드를 제공합니다. 
	편집 모드 안에는 라이브 프리뷰 모드와 소스 뷰 모드가 있습니다. 
	
[06:06](https://www.youtube.com/watch?v=O8iZRtHYcxQ&t=366s) 노트별 뷰 모드 개별 설정 (Force note view mode)
	**'Force' 플러그인을 사용**하면 각 노트별로 기본 뷰 모드를 설정할 수 있습니다. 예를 들어, 기본 설정은 편집 모드이지만 특정 노트는 읽기 모드로 열리도록 설정할 수 있습니다. Front matter를 활용하여 뷰 모드를 설정할 수도 있습니다. `ObsidianUIMode` 속성을 사용하여 라이브 프리뷰 또는 소스 뷰 모드를 지정할 수 있습니다. 
	
[09:20](https://www.youtube.com/watch?v=O8iZRtHYcxQ&t=560s) 슬래시(/)로 명령어 입력 (Slash commander)
	**'Slash Commander' 플러그인**은 편집기 창에서 
	슬래시(/)를 입력하여 명령어를 실행할 수 있게 해줍니다. 
	테이블 생성, 콜아웃 삽입, 템플릿 적용 등을 간편하게 수행할 수 있습니다. 
	명령어 팔레트에서 자주 사용하는 기능을 등록하여 슬래시 명령어로 사용할 수 있습니다. 
	예를 들어, 날짜 템플릿을 등록하여 `/date` 명령어로 현재 날짜를 쉽게 삽입할 수 있습니다. 
	템플릿을 활용하여 자주 사용하는 테이블 템플릿 등을 등록하고, 
	`/Table 1`과 같이 명령어를 지정하여 간편하게 템플릿을 삽입할 수 있습니다. 
	
[12:29](https://www.youtube.com/watch?v=O8iZRtHYcxQ&t=749s) 태그 관리 플러그인 (Tag wrangler)
	**'Tag Wrangler' 플러그인**을 사용하면 태그 이름을 변경하거나, 
	하위 태그를 쉽게 생성하여 기존 태그를 체계적으로 관리할 수 있습니다. 
	태그 이름을 변경하면 해당 태그가 사용된 모든 파일에서 자동으로 업데이트됩니다. 
	또한, 태그를 드래그 앤 드롭하여 하위 태그로 쉽게 이동시킬 수 있습니다. 
	
[14:16](https://www.youtube.com/watch?v=O8iZRtHYcxQ&t=856s) 리본, 탭바 등 아이콘 생성 (Commander)
	**'Commander' 플러그인**은 리본, 탭 바, 상태 표시줄, 탐색기 등에
	 아이콘을 추가하여 명령을 쉽게 실행할 수 있도록 해줍니다. 
	 리본 메뉴에 폴더 이동 기능을 등록하고, 아이콘과 색상을 변경할 수 있습니다.
	  탭 바에는 핀 토글 기능을 추가하여 노트를 쉽게 고정할 수 있습니다. 

[16:10](https://www.youtube.com/watch?v=O8iZRtHYcxQ&t=970s) 리본 원클릭 노트 열기 (Pinned note)
	**'Pinned Note' 플러그인**을 사용하면 특정 노트를 리본 메뉴에 고정하여 쉽게 접근할 수 있습니다. lucide.dev 웹사이트에서 아이콘을 찾아 플러그인에 적용할 수 있습니다. 
	플러그인 아이콘을 클릭하면 해당 노트가 바로 열립니다. 
	
[17:44](https://www.youtube.com/watch?v=O8iZRtHYcxQ&t=1064s) 웹페이지 스크랩 (Slurp) 22:57 내용 정리
	**'Slurp' 플러그인**을 사용하면 웹 페이지를 Obsidian으로 스크랩할 수 있습니다. URL을 입력하고 Slurp 버튼을 클릭하면 웹 페이지의 내용을 노트로 저장할 수 있습니다. 웹 페이지가 자동으로 스크랩되며, 이미지와 코드 블록도 잘 캡처되는 것을 확인할 수 있습니다. 다만, 네이버 블로그와 같은 일부 사이트는 스크랩이 허용되지 않을 수 있습니다.
