# 터미널 명령으로 파일명 일괄 치환하는 간단한 스크립트



## 작업 내용 상세하게

맥북 터미널에서 파일명 일괄 치환을 위한 간단한 스크립트는 `for` 루프와 `sed`를 조합하거나 `rename` 명령어를 사용하는 방법입니다. 먼저 대상 폴더로 이동(`cd /path/to/folder`)한 후 아래 명령어를 실행하세요. 백업을 위해 원본 파일을 복사해두는 것을 권장합니다.[](https://kyull-it.tistory.com/209)​

## 간단 Sed 기반 치환 스크립트

특정 문자열(예: "門戶" → "문호")을 바꾸는 경우:

text

`for file in *; do mv "$file" "$(echo "$file" | sed 's/門戶/문호/g')"; done`

- `*`는 모든 파일을 대상으로 하며, `sed 's/찾을문자열/바꿀문자열/g'`로 치환(대소문자 구분).[](https://blog.naver.com/holyruby/40117762477)​
    
- 괄호 위치 변경처럼 복잡한 경우: `sed 's/\(0300 \)\(門戶\)(문호)/\1문호(\2)/g'` (정규식 사용).[](https://dondonelectronics.tistory.com/21)​
    

## Rename 명령어 사용 (추천, 더 강력)

macOS에 기본 없으므로 Homebrew 설치 후: `brew install rename`.

text

`rename 's/門戶/문호/g' *`

- `-n` 옵션으로 테스트: `rename -n 's/門戶/문호/g' *` (실제 변경 전 미리보기).[](https://www.infracody.com/2023/08/linux-file-directory-rename-mv-rename.html)​
    
- 여러 파일 패턴: `rename 's/門戶(문호)/문호(門戶)/g' *0300*` (사용자 예시 패턴 적용).[](https://www.infracody.com/2023/08/linux-file-directory-rename-mv-rename.html)​
    

이 명령어들은 안전하게 작동하며, 100개 파일도 순식간에 처리합니다. 변경 전 `-n`으로 확인하고 실행하세요.[](https://kyull-it.tistory.com/209)​
