# 📙 원길1-193 숙숙적독 叔淑寂督

- [[叔]]  
	- 아재비 숙
		- 손 위(上)로 아버지보다 작은(小) 또(又) 다른 작은아버지나 아저씨니 **작은아버지 숙. 아저씨 숙**
	- ==[[叔姪]]==
		- 숙질
			- 아저씨와 조카.
			- 수양대군이 단종을 폐위시킨 사건은 **( )姪** 간에 벌어진 비극이다
			- 두 사람은 먼 촌수의 **( )姪**간이었다.
				- 叔 아저씨 숙 姪 조카 질
	- [[伯仲叔季]]
		- 백중숙계
			- 맏이 : 맏 백(伯)
			- 둘째 : 버금 중(仲)
			- 셋째 : 작은아버지 숙(叔)
			- 끝 : 끝 계(季)
	- [[叔母]]
	- [[叔父]]
	- [[堂叔]]
- [[淑]] 
	- 맑을 숙
		- 물(氵)처럼 아저씨(叔) 성품이 맑으니 **맑을 숙**
		- 원래는 氵로만 자란 콩[叔 - 콩 숙(菽)의 축약형]
		- 나물은 맑고 깨끗하다는 데서 맑을 숙(淑)입니다.
	- 淑女
	- 淑明
	- 貞淑 
	- 窈窕淑女
- [[寂]] 
	- 고요할 적
		- 집(宀)이 아저씨(叔)만 있는 듯 고요하니 **고요할 적**
	- 寂寞
	- 孤寂
	- 靜寂
	- 閑寂
- [[督]]  
	- 감독할 독
		- 아저씨(叔)가 보고(目) 감독하니 **감독할 독**

	- 督勵
