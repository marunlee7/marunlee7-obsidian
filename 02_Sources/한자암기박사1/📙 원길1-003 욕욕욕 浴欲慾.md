# 📙 원길1-003 욕욕욕 浴欲慾

- [[浴]]  
	- 목욕할 욕
		- 물(氵)이 흐르는 골짜기(谷)에서 목욕하니 **목욕할 욕**
	- 沐浴
		- 목욕
	- 浴客
		- 욕객
	- 浴室
		- 욕실
	- 海水浴
		- 해수욕
- [[欲]] 
	- 하고자할 욕
		- 골짜기(谷)처럼 크게 하품(欠)하며 잠자기를 바라니 **바랄 욕**
			- 欠 하품 흠, 모자랄 흠, 이지러질 결, 빠질 결
	- 意欲
		- 의욕
	- 慾求
		- 욕구
	- 欲求不滿
		- 욕구불만
	- 欲速不達 
		- 욕속부달
			- 빨리하려고 하면 도리어 이루지 못함
				- 欲 하고자 할 욕 速 빠를 속 不 아닐 부 達 통달할 달
- [[慾]] 
	- 욕심 욕
		- 바라는(欲) 마음(心)이 많으면 욕심이니 **욕심 욕**
	- 慾望
		- 욕망
	- 物慾
		- 물욕
	- [[貪慾]]
		- 탐욕
- 學優登仕  
	- 학우등사
		- 배운 것이 넉넉하면 벼슬에 오를 수 있음.
			- 學 배울 학 優 넉넉할 우 登 오를 등 仕 섬길 사
