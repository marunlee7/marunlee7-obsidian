# 📙 원길1-164 제방방 帝旁傍

- [[帝]] 
	- 임금 제
		- 머리 부분(亠)을 받치고 덮어(冖) 수건(巾) 같은 면류관을 쓴 제왕이니 **제왕 제**
	- [[帝王]]
	- [[帝國]]
	- [[日帝]]
	- [[皇帝]]
- [[旁]] 
	- 곁 방, 두루 방, 넓을 방
		- 서(立) 있는 방향(方)의 곁을 두루 넓게 살피니 **곁 방, 두루 방, 넓을 방**
		- 두루 : 빠짐없이 골고루
	- [[旁求]]
	- [[旁通]]
- [[傍]] 
	- ==곁 방==
		- 사람(人)이 두루(旁) 마음 써야 하는 곁이니 **곁 방**
			- 가까운 곁일수록 조금만 잘못해도 서운해하니 가까울수록 더욱 조심하여 신경써야 하지요. 
	- [[傍系]]
	- [[傍觀]]
	- [[傍聽]]
	- [[傍聽客]]
	- [[傍若無人]]
- 天時 不如地利 地利 不如人和   
	- 천시 불여지리 지리 불여인화
		- 하늘이 준 절호의 기회는 땅을 잘 이용함만 못하고
		- 지리는 식구끼리 화목함만 못하다.
		- 和睦이 제일 중요함을 강조한 말
		- A 不如 B = A보다 B가 낫다로 해석
