# 📙 원길1-292 거신호와 巨臣互瓦

- [[巨]] 
	- 클 거
		- ㄷ자 형의 큰 자를 손에 든 모양을 본떠서 **클 거**
	- [[巨金]]
		- 거금
	- [[巨物]]
		- 거물
	- [[巨富]]
		- 거부
	- [[巨人]]
		- 거인
- [[臣]] 
	- 신하 신
		- 임금 앞에 엎드려 눈을 크게 뜬 신하를 본떠서 **신하 신**
	- [[臣道]]
		- 신도
	- [[奸臣]]
		- 간신
	- [[功臣]]
		- 공신
	- [[忠臣]]
		- 충신
	- [[亂臣賊子]]
		- 난신적자
			- 나라를 어지럽게 하는 신하(臣下)와 어버이를 해(害)치는 자식(子息) 또는 불충(不忠)한 무리.
			- 亂 어지러울 란(난) 臣 신하 신 賊 도둑 적 子 아들 자
- [[互]] 
	- 서로 호
		- 새끼줄이 서로 번갈아 꼬이는 모양을 본떠서 **서로 호**
	- [[互相]]
		- 호상
			- = 相互
			- 서로서로.
				- 互 서로 호 相 서로 상
	- [[互換]]
		- ==호환==
			- 서로 맞바꿈.
			- 두통약, 치통약, 생리통약은 서로 互換이 된다
				- 互 서로 호 換 바꿀 환
	- [[互角之勢]]
		- 호각지세
			- 서로 뿔을 대고 있는 형세
			- 역량이 서로 비슷비슷한 위세
- [[瓦]] 
	- 기와 와
		- 지붕에 엇갈리게 겹쳐 놓은 기와를 본떠서 **기와 와**
		- 기와처럼 구워서 만든 질그릇이나 실패니 **질그릇 와, 실패 와**
	- [[瓦解]]
		- 와해
			- 잘 맞추어 놓은 지붕의 기와가 풀어진 것처럼 어떤 일이나 모임이 허물어진다는 뜻이지요.
			- 瓦 기와 와 解 해부할 해, 풀 해
	- [[瓦屋]]
		- 와옥
	- [[弄瓦之慶]]
		- 농와지경
			- 실패를 가지고 노는 경사
			- 딸을 낳은 경사를 말함.
			- `동` 弄瓦之喜 농와지희
			- `반` 弄璋之慶 농장지경
			- 옛날 중국에서 딸을 낳으면 실패(瓦) 장난감을 주고 아들을 낳으면 구슬을 준 고사에서 유래
			- 弄 희롱할 롱 瓦 기와 와 之 慶 경사 경 喜 기쁠 희
