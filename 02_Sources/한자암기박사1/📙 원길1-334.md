# 📙 원길1-334

- **진진진**  
	- [[眞]] 참 진
		- [[鎭]] 진압할 진
		- [[愼]] 삼갈 신
- [[眞]] 
	- 참 진
		- 비수(匕)처럼 눈(目)뜨고 감추어진(乚) 것을 나누고(八) 파헤쳐 보아도 참되니 **참 진**
		- 비수(匕)를 눈(目)앞에 놓고 숨김(乚)없이 여덟(八) 명이 참되게 살기를 약속하니 **참 진**
			- `약` 真 - 많은(十) 눈(目)이 쳐다봐도 하나(一)같이 팔(八)방에 통하도록 참되니 **참 진**
			- 乚 감출 혜, 덮을 혜 = 匸
	- 眞價
		- 진가
	- 眞假
		- 진가
	- 眞善美
		- 진선미
	- 寫眞
		- 사진
	- 眞面目
		- 진면목
	- 眞金不鍍
		- 진금부도
			- 진짜 금은 도금하지 아니함
			- 참으로 유능한 사람은 겉치레하지 않음을 비유하여 이르는 말.
			- 眞 참 진 金 쇠 금 不 아니 불 鍍 도금할 도
	- 純眞無垢
		- 순진무구
			- 순수하고 참된 그대로 때 묻지 않음.
				- 純 순수할 순 眞 참 진 無 없을 무 垢 때 구
- [[鎭]] 
	- 진압할 진 ← 누를 진
		- 쇠(金)처럼 무거운 것으로 참(眞)되게 눌러 진압하니 **누를 진, 진압할 진**
	- 鎭靜
		- 진정
			- 시끄럽고 요란(擾亂)한 일이나 상태(狀態)를 조용하게 가라앉히는 것.
				- 鎭 진압할 진 靜 고요할 정
	- 鎭痛劑
		- 진통제
			- 중추신경(中樞神經)에 작용(作用)하여 아픔을 멎게 하는 약제(藥劑).
				- 鎭 진압할 진 痛 아플 통 劑 약제 제
	- 鎭魂
		- 진혼
			- 망혼(亡魂)을 가라앉힘.
				- 鎭 진압할 진 魂 넋 혼
	- [[鎭火]]
		- 진화
			- 화재(火災)가 꺼짐.
				- 鎭 진압할 진 火 불 화
- [[愼]]  
	- 삼갈 신
		- 마음(忄)까지 참(眞)되게 하려고 삼가니 **삼갈 신**
	- 謹愼
		- ==근신==
			- 언행(言行)을 삼가고 조심(操心)함.
				- 謹 삼갈 근 愼 삼갈 신
	- 愼獨 
		- 신독
			- 홀로 있을 때에도 道理에 어그러짐이 없도록 삼감.
				- 愼 삼갈 신 獨 홀로 독
	- 愼慮
		- 신려
			- 愼重하게 思慮함.
				- 愼 삼갈 신 慮 생각할 려(여)
	- 愼重
		- 신중
			- 매우 조심(操心)스러움.
				- 愼 삼갈 신 重 무거울 중
		- ↔ 輕率
			- 경솔
				- 말이나 행동(行動)이 조심성(操心性) 없이 가벼움.
					- 輕 가벼울 경 率 거느릴 솔
	- 恪愼
		- 각신
			- 조심(操心)하고 삼감.
				- 恪 삼갈 각 愼 삼갈 신
- 待己秋霜 接人春風  
	- 대기추상 접인춘풍
		- 자기를 대할 때는 가을의 서리처럼 (엄하게)
		- 사람을 대접할 때는 봄바람처럼 (부드럽게)
