# 📙 원길1-052 묘답답답 苗畓沓踏

- [[苗]]
	- 모 묘, 싹 묘
		- 풀(艹)처럼 밭(田)에 나는 싹이니 **싹 묘**
		- `유` [[笛]] 피리 적
	- [[苗木]]
	    - 묘목
	        - 심기 위해 재배하는 어린 나무.
	        - 조림이나 원예 등에 쓰임.
	        - 苗 싹 묘, 木 나무 목
	- [[苗板]]
	    - 묘판
	        - 주로 모종이나 묘목을 기르는 판, 즉 묘상(육묘판).
	        - 묘종을 시육·양성하는 장소.
	        - 苗 싹 묘, 板 판 판
	- [[育苗]]
	    - 육묘
	        - 씨앗을 틔워 묘종을 키움.
	        - 모종을 키우는 과정 전체를 뜻함.
	        - 育 기를 육, 苗 싹 묘
	- [[種苗]]
	    - 종묘
	        - 재배·번식용 씨앗 또는 묘종(모종).
	        - 농업·원예에서 번식을 위해 쓰는 씨나 어린 식물.
	        - 種 씨 종, 苗 싹 묘
- [[畓]] 
	- 논 답
		- 물(水)을 밭(田)에 넣어 만든 논이니 **논 답**
	- [[田畓]]
	    - 전답
	        - 밭과 논 전체를 아울러 이르는 말.
	        - 토지의 기본 구분.
	        - 田 밭 전, 畓 논 답
	- [[宗畓]]
	    - 종답
	        - '종가 소유 논' 또는 대대로 내려오는 주된 논.
	        - 전통, 가문의 중심 논을 지칭함.
	        - 宗 마루 종, 畓 논 답
	- [[門前沃畓]]
	    - 문전옥답
	        - 집 앞에 펼쳐진 기름진 논.
	        - 부와 번영을 상징.
	        - 門 문 문, 前 앞 전, 沃 기름질 옥, 畓 논 답
- [[沓]] 
	- 겹칠 답 → 활발할 답
		- 물(水)이 햇(日)볕에 증발하여 활발하게 합하니 **활발할 답, 합할 답**
	- [[沓雜]]
	    - 답잡
	        - 어지럽게 겹쳐서 혼란스러움.
	        - 복잡하고 어수선함.
	        - 沓 겹칠 답, 雜 섞일 잡
	- [[紛沓]]
	    - 분답
	        - 여러 것이 뒤섞여 어지러움.
	        - 혼란하고 질서가 없다.
	        - 紛 어지러울 분, 沓 겹칠 답
- [[踏]] 
	- 밟을 답
		- 발(足)을 활발하게(沓) 움직여 밟으니 **밟을 답**
	- [[踏步]]
	    - 답보
	        - 제자리걸음.
	        - 진전 없이 그대로 머무는 상태.
	        - 踏 밟을 답, 步 걸을 보
	- [[踏査]]
	    - 답사
	        - 현장을 직접 밟아 살펴봄.
	        - 직접 조사하거나 현장 확인함.
	        - 踏 밟을 답, 査 조사할 사
	- [[踏襲]]
	    - 답습
	        - 이전의 방법이나 관례를 그대로 따름.
	        - 낡은 것을 그대로 본받는 것.
	        - 踏 밟을 답, 襲 엄습할 습
	- [[踏破]]
	    - 답파
	        - 걸어서 다 돌아다님.
	        - 길이나 산을 밟아 지나감, 정복함.
	        - 踏 밟을 답, 破 깨뜨릴 파
