# 📙 원길1-098

- [[耂]]  
	- 늙을 로 엄
		- 늙을 로(老)가 부수로 쓰일 때의 모습으로 흙(土)에 지팡이(丿)를 짚으며 걸어야 할 정도로 늙으니 **늙을 로** 엄
		- 엄 : 글자의 위와 왼쪽을 덮는 부수 이름
- [[老]] 
	- 늙을 로
		- 흙(土)에 지팡이(丿)를 비수(匕)처럼 꽃으며 걸어야 할 정도로 늙으니 **늙을 로**
			- 丿 삐침 별 → 여기서는 지팡이로
			- 匕 비수 비, 숟가락 비
			- 비수 : 날카롭고 짧은 칼
	- [[老益壯]]
		- 노익장
			- 나이는 들었으나 氣力은 더욱 좋아짐.
			- 老 늙을 로(노) 益 더할 익 壯 장할 장
	- [[敬老]]
		- 경로
	- [[元老]]
		- 원로
	- [[30-리소스/Dict/Hword2/한자암기박사/偕老]]
		- 해로
	- [[百戰老將]]
		- 백전노장
		- 수많은 싸움을 치른 노련(老鍊)한 장수(將帥).
		- 온갖 어려운 일을 많이 겪은 노련(老鍊)한 사람.
		- 百 일백 백 戰 싸움 전 老 늙을 로(노) 將 장수 장
- [[孝]] 
	- 효도 효
		- 늙은(耂) 부모를 아들(子)이 받드는 효도니 **효도 효**
	- [[孝道]]
		- 효도
	- [[孝誠]]
		- 효성
	- [[孝悌忠信]]
		- 효제충신
			- 어버이에 대한 孝道, 兄弟끼리의 友愛, 임금에 대한 忠誠과 벗 사이의 믿음는 말.
			- 孝 효도 효 悌 공손할 제 忠 충성 충 信 믿을 신
- [[者]] 
	- 놈 자 → 것 자
		- 노인(耂)이 낮추어 말하는(白) 놈이나 것이니 **놈 자, 것 자**
			- 白 흰 백, 말할 백, 깨끗할 백, 아뢸 백
			- 어른이 사람을 지칭할 때 : ==이놈, 저놈==
			- 사물을 지칭할 때 : ==이 것, 저것==
	- [[强者]]
		- 강자
	- [[讀者]]
		- 독자
	- [[仁者無敵]]
		- 인자무적
- [[考]] 
	- 생각할 고
		- 노인(耂)처럼 크게(丂) 살피고 생각하니 **살필 고, 생각할 고**
			- [[丂]] : [[巧]] 공교할 교 변형 → 여기서는 大 변형으로 풀이
	- [[考慮]]
		- 고려
			- 깊이 생각하여 헤아림.
				- 考 생각할 고 慮 생각할 려(여)
	- [[考察]]
		- 고찰
			- 잘 생각해서 살핌.
				- 考 생각할 고 察 살필 찰
	- [[再考]]
		- 재고
			- 어떤 일이나 문제(問題) 따위에 대하여 다시 생각함.
				- 再 두 재 考 생각할 고
	- [[熟考]]
		- 숙고
			- 곰곰이 잘 생각함.
				- 熟 익을 숙 考 생각할 고
	- [[深思熟考]]
		- 심사숙고
			- 深 깊을 심 思 생각 사 熟 익을 숙 考 생각할 고
