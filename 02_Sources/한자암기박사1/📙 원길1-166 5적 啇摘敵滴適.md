# 📙 원길1-166 5적 啇摘敵滴適

- [[啇]]  
	- 밑동 적, 뿌리 적
		- 머리 부분(亠)을 받친(八) 성(冂) 모양으로 오래된(古) 밑동이나 뿌리니 **밑동 적, 뿌리 적**
		- `유` 商 장사 상
		- 밑동 : 나무줄기의 밑부분으로, 사물의 제일 중요한 부분을 말함.
- [[摘]] 
	- 딸[手收] 적
		- 손( 扌)으로 과일의 밑동(啇)을 따니 **딸 적**
	- 摘果
		- 적과
			- 과실(果實)을 솎아냄.
				- 摘 딸 적 果 실과 과
	- 摘讀 
		- 적독
			- 띄엄띄엄 가려서 읽음.
				- 摘 딸 적 讀 읽을 독
	- 摘發
		- 적발
			- 숨겨져 있는 일이나 드러나지 아니한 것을 들추어냄.
				- 摘 딸 적 發 필 발
	- 指摘
		- 지적
			- 꼭 집어서 가리킴
			- 指 가리킬 지 摘 딸 적
- [[敵]]  
	- 대적할 적 → 원수 적
		- 뿌리(啇), 즉 근본까지 치며(攵) 달려드는 원수니 **원수 적**
	- 匹敵
		- ==필적==
			- 能力, 勢力이 서로 어슷비슷함. 맞섬.
			- 걸맞아서 견줄 만함.
				- 匹 짝 필 敵 대적할 적
	- 敵國
		- 적국
			- 상대(相對)가 되어 싸우는 나라.
				- 敵 대적할 적 國 나라 국
	- 敵軍
		- 적군
			- 적국(敵國)의 군사(軍士).
				- 敵 대적할 적 軍 군사 군
	- 對敵
		- 대적
			- 적(敵)과 마주 대함.
				- 對 대할 대 敵 대적할 적
	- 宿敵
		- 숙적
			- 오래전부터의 원수(怨讐).
				- 宿 잘 숙 | 별자리 수 敵 대적할 적
- [[滴]] 
	- 물방울 적
		- 물(氵)중 밑동(啇)으로 떨어지는 물방울이니 **물방울 적**
	- 滴露
		- 적로
			- 방울지어 떨어지는 이슬.
				- 滴 물방울 적 露 이슬 로(노)
	- 滴水
		- 적수
			- 떨어지는 물방울
	- 餘滴欄 
		- 여적란
			- 新聞이나 雜誌 따위에서 餘錄이나 gossip 等을 실으려고 마련한 紙面.
				- 餘 남을 여 滴 물방울 적 欄 난간 란(난)
	- 硯滴
		- 연적
			- 벼룻물을 담는 그릇.
				- 硯 벼루 연 滴 물방울 적
- [[適]]  
	- 맞을 적 → 갈 적
		- 뿌리(啇)가 알맞은 곳으로 뻗어가듯(辶) 알맞게 가니 **알맞을 적, 갈 적**
	- ==悠悠自適==
		- 유유자적
			- 餘裕가 있어 閑暇롭고 걱정이 없는 模樣
			- 俗世를 떠나 아무 속박(束縛) 없이 조용하고 便安하게 삶.
				- 悠 멀 유 自 스스로 자 適 맞을 적
			- 자유로우며 편안하게 자기 하고 싶은 대로 마음 편히 지내는 상태나 그러한 태도를 뜻한다.
			- 유의어 : 유연자적(悠然自適)
			- 다만 일반적으로 유유자적은 가진 돈이 많아서 자기 하고 싶은 대로 마음껏 산다는 뜻이 아니다. 돈 같은 것에 얽매이지 않고 돈을 많이 벌 생각도 없으며, 적당히 소박하게 살면서 그것에 만족하고 앞으로의 삶을 느긋하게 천천히 살아간다는 의미로 안빈낙도 뜻과도 적절하게 대응된다. 
	- 適當
		- 적당
			- 事物의 程度･水準･狀態 等이 지나치거나 모자람이 없이 또는 잘 어울려 마땅함.
			- 알맞음.
			- 要領껏 엇비슷하게 하거나 말썽만 없을 程度로 大綱大綱 하는 狀態에 있음.
				- 適 맞을 적 當 마땅 당
	- 適性
		- 적성
			- 어떤 일에 알맞은 性質이나 適應 能力.
			- 또는 그와 같은 素質이나 性格.
				- 適 맞을 적 性 성품 성
