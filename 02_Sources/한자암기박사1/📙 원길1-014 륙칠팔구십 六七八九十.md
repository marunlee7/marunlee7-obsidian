# 📙 원길1-014 륙칠팔구십 六七八九十

- 六  
	- 여섯 륙
		- 머리(亠) 아래 나눠지는(八) 방향이 동서남북 상하의 여섯이니 **여섯 륙**
	- 六旬
		- 육순
	- 死六臣
		- 사육신
	- 五臟六腑
		- 오장육부
- 七 
	- 일곱 칠
		- 하늘(亠)의 북두칠성 모양을 본떠서 **일곱 칠**
	- 七夕
		- 칠석
	- 七旬
		- 칠순
	- 七顚八起
		- 칠전팔기
			- 七 일곱 칠 顚 엎드러질/이마 전 八 여덟 팔 起 일어날 기
- 八 
	- 여덟 팔, 나눌 팔
		- 두 손을 네 손가락씩 위로 편 모양에서 **여덟 팔**
		- 양쪽에서 잡아당겨 나누니 **나눌 팔**
	- 八角亭
		- 팔각정
	- 八達
		- 팔달
	- 十中八九
		- 십중팔구
- 九 
	- 아홉 구 → 많을 구
		- 열 십, 많을 십의 가로줄을 구부려 하나가 모자란 **아홉 구**
		- 아홉은 한 자리 숫자 중에서 제일 크고 많으니 **클 구, 많을 구**
	- [[十九孔炭]]
		- 십구공탄
			- 十 열 십 九 아홉 구 孔 구멍 공 炭 숯 탄
	- [[九牛一毛]]
		- 구우일모
			- 아홉 마리의 소 가운데 박힌 하나의 털
			- 매우 많은 것 가운데 극(極)히 적은 수(數)를 이르는 말.
- 十 
	- 열 십 → 많을 십
		- 일(一)에서 하나(丨)를 그어 한 묶음인 열을 나타내어 **열 십**
		- 전체를 열로 보아 열이면 많다는 데서 **많을 십**
	- 十戒
		- 십계
	- 十進法
		- 십진법
	- 十匙一飯
		- 십시일반
			- 밥 열 술이 한 그릇이 된다.
			- 여러 사람이 조금씩 힘을 합(合)하면 한 사람을 돕기 쉬움을 이르는 말.
				- 十 열 십 匙 숟가락 시 一 한 일 飯 밥 반
