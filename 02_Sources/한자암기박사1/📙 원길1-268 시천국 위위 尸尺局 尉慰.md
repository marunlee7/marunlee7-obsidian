# 📙 원길1-268 시천국 위위 尸尺局 尉慰

- [[尸]] 
	- 주검 시 → 몸 시
		- 사람이 누워 있는 몸을 본떠서 **주검 시, 몸 시**
		- `유` 戶 문 호, 집 호
		- 사람이나 집과 관련된 글자에 부수로도 쓰입니다.
- [[尺]] 
	- 자 척
		- 몸(尸)을 구부리고(乀) 길이를 재는 자니 **자 척**
	- [[尺度]]
	    - 척도
	        - 측정하거나 평가하는 기준 또는 단위.
	        - 尺 자 척, 度 법도 도
	- [[越尺]]
	    - 월척
	        - ‘한 자(尺)’를 넘는 큰 물고기, 또는 기준치를 넘음.
	        - 越 넘을 월, 尺 자 척
	- [[咫尺]]
	    - 지척
	        - 아주 가까운 거리, 바로 곁.
	        - 咫 자 지, 尺 자 척
	- [[吾鼻三尺]]
	    - 오비삼척
	        - “내 코가 석 자도 안 된다.”는 뜻으로, 내 사정이 급해 남을 도울 겨를이 없음.
	        - 吾 나 오, 鼻 코 비, 三 셋 삼, 尺 자 척
- [[局]]   
	- 판[形局] 국, 형편(상황) 국, 부분 국
		- 자(尺)로 재어 말(口)로 구역을 확정지은 판이나 관청이니 **판 국, 관청 국**
		- 어떤 판 같은 상황이니 **상황 국**
	- [[大局]]
	    - 대국 ← ==판 국==
	        - 전체 상황, 큰 판을 의미함.
	        - 중요한 문제나 상황에서 전체의 흐름이나 큰 틀을 보는 시각.
	        - 예시: 협상할 때는 작은 이익보다 대국(大局)을 봐야 한다.
	        - 출전 : 정치, 경제, 사회 전반에서 널리 사용됨.
	        - 大 클 대, 局 판 국
	- [[政局]]
	    - 정국 ← ==형편 국==
	        - 정치적 상황이나 국면.
	        - 정부나 정치권의 현재 상태나 변화 양상.
	        - 예시: 최근 정국(政局)이 불안정해졌다.
	        - 출전 : 정치학 및 언론 용어.
	        - 政 정사 정, 局 판 국
	- [[局地戰]]
	    - 국지전 ← ==부분 국==
	        - 한정된 지역 내에서 이루어지는 전투.
	        - 전면전이 아닌 좁은 지역에서 벌어지는 군사 작전.
	        - 예시: 국지전(局地戰)은 전쟁에서 국부적 충돌을 의미한다.
	        - 출전 : 군사 용어.
	        - 局 판 국, 地 땅 지, 戰 싸움 전
	- [[難局]]
	    - ==난국
	        - 어려운 형세나 국면.
	        - 難 어려울 난, 局 판 국
	- [[局面]]
	    - 국면
	        - 어떤 일이 전개되어 가는 형세나 단계.
	        - 局 판 국, 面 얼굴 면
	- [[局長]]
	    - 국장
	        - 기관이나 부서, ‘국’(局)을 책임지는 사람(장).
	        - 局 판 국, 長 길 장
	        - [[局長]]
	- [[局部]]
	    - 국부
	        - 전체 중 일부분, 부분.
	        - 局 판 국, 部 떼 부
	- [[局限]]
	    - 국한
	        - 범위(한계)를 정하거나 한정함.
	        - 局 판 국, 限 한할 한
- [[尉]] 
	- 벼슬 위
		- 주검(尸)을 보아도(示) 두려워하지 않고 법도(寸)에 따라 처리하는 벼슬이니 **벼슬 위**
	- [[尉官]]
	    - 위관
	        - ‘군의 하위 장교’(소위, 중위, 대위의 통칭).
	        - 尉 벼슬 위, 官 벼슬 관
	- [[尉級]]
	    - 위급
	        - 위관급(尉官級), 즉 소위·중위·대위의 계급 범주.
	        - 尉 벼슬 위, 級 등급 급
	- [[大尉]]
	    - 대위
	        - 군대 계급 중 위관의 제일 위.
	        - 大 큰 대, 尉 벼슬 위
- [[慰]] 
	- 위로할 위
		- 벼슬아치(尉)가 마음(心)으로 위로하니 **위로할 위**
	- [[慰勞]]
	    - 위로
	        - 괴로움이나 슬픔을 달래고 격려함.
	        - 慰 위로할 위, 勞 일할 로
	- [[慰靈祭]]
	    - 위령제
	        - 죽은 이의 넋을 위로하는 제사.
	        - 慰 위로할 위, 靈 신령 령, 祭 제사 제
	- [[慰問]]
	    - 위문
	        - 위로하고 안부를 물음.
	        - 慰 위로할 위, 問 물을 문
	- [[慰安]]
	    - 위안
	        - 위로하고 편안하게 함.
	        - 慰 위로할 위, 安 편안 안
	- [[慰撫]]
	    - 위무
	        - 위로하고 어루만져 다독임.
	        - 慰 위로할 위, 撫 어루만질 무
