# 📙 원길1-207 당당(탕)혜혜혜 唐糖彗慧惠

- [[唐]] 
	- 당나라/당황할 당 → 갑자기 당
		- 집(广)에서라도 손(彐)에 회초리(丨) 들고 입(口)으로 갑자기 소리치면 황당하니 **갑자기 당, 황당할 당**
		- 갑자기 세력을 떨쳤던 당나라니 **당나라 당**
		- 갑자기 : 미처 생각할 겨를도 없이 급히
		- 丨뚫을 곤 → 회초리로 봄
	- [[荒唐]]
		- 황당
			- 말이나 행동 따위가 참되지 않고 터무니없다.
			- 荒 거칠 황 唐 당나라/당황할 당
	- 唐突
		- 당돌
			- 올차서 꺼리는 마음이 없음.
				- 唐 당나라 당 突 갑자기 돌
	- 唐慌
		- 당황
			- = 唐惶 당황
			- 놀라거나 다급(多急)하여 어찌할 바를 모름.
				- 唐 당나라/당황할 당 慌 어리둥절할 황 惶 두려울 황
	- 唐根
		- 당근
			- 미나릿과의 일년초 또는 이년초
				- 唐 당나라 당 根 뿌리 근
- [[糖]] 
	- 엿 당
		- 쌀(米)밥에 엿기름을 넣으면 갑자기(唐) 바뀌어 되는 사탕이니 **사탕 당, 탕**
	- 糖度
		- 당도
	- 糖分
		- 당분
	- 糖水肉
		- 탕수육
	- 雪糖
		- 설탕
			- 雪 눈 설 糖 엿 당
- [[彗]] 
	- 살별 혜 → 비/혜성 혜
		- 풀 무성한 가지 두 개(丰丰)를 손(彐)으로 묶어 만든 비니 **비 혜**
		- 빗자루 모양으로 꼬리를 끌며 날아가는 혜성이니 **혜성 혜**
			- 彗星 혜성
				- ㉠ 빛나는 긴 꼬리를 끌고 태양을 도는 별. 꼬리별.
				- ㉡ (밤하늘에 혜성이 갑자기 나타나듯) 어떤 분야에서 갑자기 뛰어나 드러남을 비유한 말.
			- 丰 풀 무성할 봉, 예쁠 봉, 풍성할 풍
			- 살별 = 혜성 = 꼬리별
	- 彗掃 
		- 혜소
			- 비로 깨끗이 淸掃함.
				- 彗 비 혜 掃 쓸 소
	- [[解消]]
		- 해소
			- 어떤 狀態나 關係를 풀어 없앰.
				- 解 풀 해 消 사라질 소
- [[慧]] 
	- 슬기로울 혜 → 밝을/지혜 혜
		- 잡념을 비(彗)로 쓸어버린 마음(心)처럼 밝은 지혜니 **밝을 혜, 지혜 헤**
	- [[慧巧]]
		- 혜교
			- 영리(怜悧ㆍ伶俐)한 슬기와 기묘(奇妙)한 기교(技巧).
				- 慧 슬기로울 혜 巧 공교할 교
	- [[慧眼]] 
		- 혜안
			- 사물을 밝게 보는 총명한
				- 慧 슬기로울 혜 眼 눈 안
	- [[智慧]] 
		- 지혜
			- 슬기
				- 智 슬기 지 慧 슬기로울 혜
- [[惠]] 
	- 은혜 혜
		- 언행을 삼가고(叀) 어진 마음(心)을 베푸는 은혜니 **은혜 혜**
		- 叀 - 車에 점(丶)찍는 일은 삼가니 **삼갈 전** - 실제 쓰이는 글자는 아닙니다.
	- [[恩惠]]
		- 은혜
			- 자연(自然)이나 남에게서 받는 고마운 혜택(惠澤).
				- 恩 은혜 은 惠 은혜 혜
	- [[惠澤]]
		- 혜택
			- 은혜(恩惠)와 덕택(德澤).
			- 자연(自然)이나 문명(文明)이나 단체(團體) 등(等)이 사람에게 베푸는 이로움이나 이익(利益).
				- 惠 은혜 혜 澤 못 택
	- [[施惠]]
		- 시혜
			- 은혜(恩惠)를 베풂. 또는 그 은혜(恩惠).
				- 施 베풀 시 惠 은혜 혜
	- [[不費之惠]]
		- 불비지혜
			- 自己에게는 害가 될 것이 없어도 남에게는 利益이 될 만 하게 베풀어 주는 恩惠.
				- 不 아닐 불 費 쓸 비 之 갈 지 惠 은혜 혜
