# 📙 원길1-041 곤수온 인인은 困囚溫 因姻恩

- [[困]]  
	- 곤할 곤
		- 에워싸인(囗) 나무(木)는 자라기가 곤란하니 **곤할 곤**
	- [[困難]]
		- 곤란
			- 사정이 몹시 딱하고 어려움.
	- 困境
		- 곤경
			- 곤란(困難)한 경우(境遇).
			- `유의어` 難境(난경) 曲境(곡경)
			- 困 곤할 곤 境 지경 경
	- 貧困
		- 빈곤
	- 疲困
		- 피곤
- [[囚]] 
	- 가둘 수
		- 에워싸여(囗) 갇힌 사람(人)이면 죄인이니 **죄인 수**
	- [[囚衣]]
		- 수의
	- [[死刑囚]]
		- 사형수
			- 사형(死刑) 선고(宣告)를 받은 죄수(罪囚).
			- 死 죽을 사 刑 형벌 형 囚 가둘 수
	- [[良心囚]]
		- 양심수
	- [[罪囚]]
		- 죄수
- [[溫]]   
	- 따뜻할 온
		- 물(氵)을 죄인(囚)에게도 그릇(皿)으로 떠 주는 마음이 따뜻하니 **따뜻할 온**
		- 따뜻하도록 여러 번 반복하여 익히니 **익힐 온**
		- → 무엇이나 여려 번 문지르면 따뜻해지듯이 따뜻하도록 여러 번 반복하여 익힌다는 데서 **익힐 온**
			- `약` 温 물(氵)이 해(日)가 비친 그릇(皿)에 있으면 따뜻하니 **따뜻할 온**
	- ==溫和==
		- 온화
			- 날씨가 맑고 따뜻하며 바람이 부드러움.
				- 溫 따뜻할 온 和 화할 화
	- [[溫氣]]
		- 온기
	- [[溫情]]
		- 온정
	- [[溫故知新]]
		- 온고지신
- [[因]] 
	- 인할 인 → 말미암을 인
		- 에워싼(囗) 큰(大) 울타리에 말미암아 의지하나 **말미암을 인, 의지할 인**
		- → 사회가 안정되지 않았던 옛날에는 크고 튼튼한 울타리에 많이 의지하였겠지요
	- [[因緣]]
		- 인연
			- 사람들 사이에 맺어지는 관계(關係).
			- 어떤 사물(事物)과 관계(關係)되는 연줄(緣-).
			- 일의 내력(來歷) 또는 이유(理由).
			- 인(因)과 연(緣)을 아울러 이르는 말. 인(因)은 결과(結果)를 만드는 직접적(直接的)인 힘이고, 연(緣)은 그를 돕는 외적(外的)이고 간접적(間接的)인 힘이다.
			- 원인(原因)이 되는 결과(結果)의 과정(過程).
				- 因 인할 인 緣 인연 연
	- [[因習]]
		- 인습
			- 이전(以前)부터 전(傳)하여 몸에 젖은 풍습(風習)..
				- 因 인할 인 習 익힐 습
	- [[原因]]
		- 원인
			- 어떤 사물(事物)이나 상태(狀態)를 변화(變化)시키거나 일으키게 하는 근본(根本)이 된 일이나 사건(事件).
				- 原 언덕 원 因 인할 인
	- [[敗因]]
		- 패인
		- 싸움에 지거나 일에 실패(失敗)한 원인(原因).
			- 敗 패할 패 因 인할 인
- [[姻]]  
	- 혼인 인
		- 여자(女)가 의지할(因) 남자에게 시집가니 **시집갈 인**
	- [[婚姻]]
		- ==혼인==
			- 장가들고 시집(媤-)가는 일.
				- 婚 혼인할 혼 姻 혼인 인
	- [[姻戚]]
		- 인척
	- [[姻親]]
		- 인친
			- 혼인(婚姻)으로 맺어진 관계(關係). 또는 혼인(婚姻) 관계(關係)로 척분(戚分)이 있는 사람.
- [[恩]] 
	- 은혜 은
		- 의지하도록(因) 마음(心) 써주는 은혜니 **은혜 은**
	- [[恩惠]]
		- 은혜 : 베풀어 주는 혜택
	- [[恩功]]
		- 은공
	- [[恩德]]
		- 은덕
			- 은혜(恩惠)와 덕(德). 또는 은혜(恩惠)로운 덕(德).
	- [[恩人]]
		- 은인
