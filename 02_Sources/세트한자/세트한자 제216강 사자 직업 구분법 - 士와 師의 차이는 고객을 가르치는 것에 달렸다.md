# 세트한자 제216강 사자 직업 구분법 - 士와 師의 차이는 고객을 가르치는 것에 달렸다

![](https://www.youtube.com/watch?v=w9tilzqn8s0)  

접미사인 사자를 사용하는 직업군의 한자 사용이 다릅니다. 다른 사자를 사용하는 이유를 밝혀봅니다.  
士와 師는 같은 범주의 직업군인데, 士는 문제의 결과만 고객에게 알려주지만 師의 직업군은 고객에게 문제의 원인과 과정을 가려쳐 줍니다. 師의 의미대로 고객에게 스승이 되어 가르치는 직업군에 師를 접미사로 사용합니다.
