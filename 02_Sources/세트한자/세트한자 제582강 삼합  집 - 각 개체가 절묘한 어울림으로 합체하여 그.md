# 세트한자 제582강 삼합 집 - 각 개체가 절묘한 어울림으로 합체하여 그

![](https://www.youtube.com/watch?v=lEVgFZA54mk)  

각 개체의 힘으로는 그저 평범한 능력치이지만 절묘한 어울림으로 합체되면 특별한 능력치를 발휘하는 시너지 효과를 보여준다. 이런 어울림을 의미하는 亼을 중심으로 하는 한자들을 세트한자로 해석합니다.
