# 세트한자 제595강 나무 목 - 木이 공동체을 의미하는 경우의 한자모음

![](https://www.youtube.com/watch?v=oxCMXC4KXHE)  

공동체는 사람의 삶을 유지 발전시키는데 결정적인 역할을 합니다. 어떤 나라에 살고 있는가, 어떤 직장에 다니고 있는가, 어떤 친구들과 어울리는가, 어떤 가정에서 살고 있는가, 한 사람의 삶을 표현하는 지표이기도 합니다. 그런 공동체를 의미하는 한자 부수인 木으로 구성된 한자들을 세트한자로 해석합니다.
