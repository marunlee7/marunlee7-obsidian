# 세트한자 제764강 호피무늬 호 - 호랑이의 겉모습을 의미하는 호피

![](https://www.youtube.com/watch?v=G0FhFjcnrHw)  

예전에 호랑이는 사람이 직접 대면할 수 없는 공포의 대상이었습니다. 산 중의 왕이요. 호걸이었던 호랑이를 직접 대면한다는 것은 곧 죽음을 의미했기에 알고는 있지만 직접 보지 못하여 없는 것이나 마찮가지 였습니다. 그렇게 실제로는 있지만 없는 것이나 마찮가지란 의미의 虍를 중심으로 하는 한자들을 세트한자로 해석합니다.
