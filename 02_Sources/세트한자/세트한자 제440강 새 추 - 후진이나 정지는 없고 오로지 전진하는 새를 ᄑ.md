# 세트한자 제440강 새 추 - 후진이나 정지는 없고 오로지 전진하는 새를 ᄑ

![](https://www.youtube.com/watch?v=8BWa1yjc3DY)  

하늘을 날아가는 새는 정지하지 않습니다. 후진도 없지요. 오로지 전진하는 새가 하늘을 날아가는 모습은 사람들에게 오래된 동경의 대상입니다. 그런 새의 습성을 표현하는 한자들을 세트한자로 해석합니다.
