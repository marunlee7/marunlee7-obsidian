# 세트한자 제448강 아홉 구 - 최선은 아직 완전함이 아니다

![](https://www.youtube.com/watch?v=N66GdaDr5HY)  

최선의 노력에도 불구하고 아직 성공한 것은 아닙니다. 하나의 행운이 주어진다면 하나의 은혜가 주어진다면 최선의 노력에 영광이 있을 겁니다. 최선의 노력을 의미하는 아홉에 대한 표현들을 세트한자로 해석합니다.
