# 세트한자 제161강 자기 기(己) - 나를 표현하는 여러 방법

![](https://www.youtube.com/watch?v=doMTMxdLJx8)  

나를 가장 잘 표현하는 한자가 '자기 기' 자 입니다. 맹세를 할 때에도, 부탁을 할 때에도, 명을 받을 때에도 몸을 구부리면서 겸손의 자체를 취하는데, 이를 표현하는 한자가 '자기 기' 입니다. 각 분야에서 스스로를 표현하는 세트한자 입니다.
