# 세트한자 제667강 새 추 - 하늘을 나는 새가 부럽기도 하고 질투도 나는데

![](https://www.youtube.com/watch?v=q0YdvFb5x1A)  

공간이동이 자유로운 새를 보면서 부럽기도 하여 선망을 표현하지만 한편으로는 질투도 나서 무시하거나 얕잡아 표현하기고 합니다. 선망과 질투의 대상이 되는 새를 隹로 표현하며 관련한자들을 세트한자로 해석합니다.
