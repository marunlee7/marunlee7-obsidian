# 壯 장할 장

![](https://www.youtube.com/watch?v=AmpPuaXkxrQ)  

공동체에는 열정과 신념을 가진 사람도 중요하지만 전체를 리드하는 힘센 사람도 필요합니다. 그들이 위기에서 공동체를 구해내고 더 좋은 사회로 안내하기 때문입니다. 우람하고 센 사람을 의미하는 壯을 중심으로 하는 한자들을 세트한자로 해석합니다.
