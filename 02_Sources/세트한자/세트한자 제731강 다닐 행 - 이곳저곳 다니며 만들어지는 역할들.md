# 세트한자 제731강 다닐 행 - 이곳저곳 다니며 만들어지는 역할들

![](https://www.youtube.com/watch?v=kQXqLJVrV5A)  

사람이 활동을 하면 이곳저곳을 다니게 됩니다. 그렇게 다니면 여러가지 역할을 하게 되는데 그럴 때 만들어지는 상황들을 설명하는 한자들이 있습니다. 다니는 것을 의미하는 行을 중심으로 하는 한자들을 세트한자로 해석합니다.
