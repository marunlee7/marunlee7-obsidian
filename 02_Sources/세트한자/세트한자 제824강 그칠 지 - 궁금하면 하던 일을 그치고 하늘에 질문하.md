# 세트한자 제824강 그칠 지 - 궁금하면 하던 일을 그치고 하늘에 질문하

![](https://www.youtube.com/watch?v=H9PRWUyOwJo)  

왜 그럴까 어떻게 그럴수가 있을까 많은 궁금증이 생깁니다. 그럴 때 하던 일을 멈추고 하늘을 바라보며 질문을 합니다. 왜 그럴까요? 하던 일을 멈추고 하늘을 바라보는 상황이 止입니다. 止를 중심으로 하는 한자들을 세트한자로 해석합니다.
