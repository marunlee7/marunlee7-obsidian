# 세트한자 제888강 대답할 유 관련 세트한자 / 공동체에 화합과 조화를 안겨주는 대답

![](https://www.youtube.com/watch?v=GkGNmyhGlG4)  

질문에 대답하는 것은 예의이고 도리입니다. 최선의 대답을 한다면 공동체의 화합되고 발전의 토대가 되기도 합니다. 화합의 대답을 의미하는 兪와 관련된 한자들을 세트한자로 해석합니다.

	- 대답할 유
- [[揄]]
	- 끌어낼 유
- [[鍮]]
	- 놋쇠 유
- [[諭]]
	- 깨달을 유
- [[楡]]
	- 느릅나무 유
- [[癒]]
	- 치유할 유
