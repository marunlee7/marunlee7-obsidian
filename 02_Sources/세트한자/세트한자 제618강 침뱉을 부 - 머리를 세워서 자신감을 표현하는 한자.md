# 세트한자 제618강 침뱉을 부 - 머리를 세워서 자신감을 표현하는 한자

![](https://www.youtube.com/watch?v=FsmMQMM48AE)  

가슴을 펴고 머리를 세우면 자신감이 올라갑니다. 그런 자신감으로 대하는 세상은 좀 더 너그럽고 능력치를 올릴수 있게 됩니다. 그런 자신감을 표현하는 咅를 중심으로 하는 한자들을 세트한자로 해석합니다.
