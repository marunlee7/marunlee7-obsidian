# 세트한자 제492강 진흙 근 - 흙에서 왔으니 흙으로 가리라

![](https://www.youtube.com/watch?v=Gzd8HPn8wks)  

진흙은 흙 중에서도 진짜 흙이라 할 것입니다. 고운 가루에 붉은 빛을 띠면 찰진 흙이어서 여러 모로 쓸모가 많지요. 그런 흙으로 만들어진 한자들을 세트한자로 해석합니다.
