# 세트한자 제213강 감언이설 - 달콤한 말의 속임수 감언이설

![](https://www.youtube.com/watch?v=W16a-uiroU4)  

명령이나 당부의 말씀에 달콤함을 덫붙이면 사기가 되고 속임수가 됩니다.

그말에 속아서 패가망신하는 사람들도 많습니다. 

명령이나 당부를 설명하는 사람이 감하거나 더해서는 않됩니다. 

감언이설은 죄악입니다.
