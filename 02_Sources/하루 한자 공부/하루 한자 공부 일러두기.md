# 일러두기

자주 사용하는 한자([[漢字]]) 위주로 선정([[選定]])했다.
하루에 한 자([[字]])씩 공부할 수 있도록 날짜 순으로 배열했다.
계절([[季節]]), 기념일([[紀念日]]), 국경일([[國境日]])에는 가급적 그와 관련된 한자를 소개했다.
초창기([[初創期]]) 글꼴을 제시하여 본뜻([[本뜻]])과 파생([[派生]])된 뜻을 설명했다.
해당 한자로 조합([[組合]])된 단어([[單語]]), 숙어([[熟語]]), 성어([[成語]])를 예로 들었다.
해당 한자와 관련된 중국([[中國]]) 문화([[文化]])를 종종 언급([[言及]])했다.
해설([[解說]]) 용어([[用語]])는 편의([[便宜]])상 흔한 명칭([[名稱]])을 사용했다.
현대([[現代]]) 중국어([[中國語]])는 한글 독음으로 표시했다.
