# 천자문 05강 운등치우 노결위상 (雲騰致雨 露結爲霜)

![](https://www.youtube.com/watch?v=wnCMQGfYF6I)  

## 4자 풀이
- [[운등치우(雲騰致雨)]] : 구름이 날아올라 비를 이루고
- [[노결위상(露結爲霜)]] : 이슬이 맺히고 서리가 서리도다.

- ## 雲騰致雨 
- [[雲]]
	- 구름 운
		- [[云]](이를 운, 말할 운, **어조사 운**) ; 말할 때 입김이 나가다. 
			- 하늘(一)과 땅(一)의 기운이 만나면 구름 모양(厶)이 나온다
- [[騰]]
	- 오를 등
		- 舟(배주달 월) + 关(웃을 소/**틈 진**)
		- [[朕]](나 짐) + [[馬]](말 마)
		- 설문해자 : 등개마야(騰[[犗]]馬也)
	- [[勝]]
		- 이길 승, 뛰어날 승
			- 배의 틈진부분을 메꿔 힘써 막아 이기다
			- 누구와의 싸움과 자기와의 싸움에서 이기니 뛰어나다
			- [[명승지(名勝地)]] - 뛰어날 승으로 쓰임
	- [[滕]]
		- 물 솟을 등
			- 틈진 곳으로 물이 들어오다
	- [[藤]]
		- 등나무 등
			- 넝쿨이 특징이라 艹를 넣음
		- 葛藤
			- 갈등

- [[致]]
	- 이를 치
		- 정확하게 맞춰(攵) 이르렀다.(至) 
	- 致死量
		- 치사량

- [[雨]]
	- 비 우

- ## 露結爲霜
- [[露]]
	- 이슬 로, 드러나다 로
		- 설문해자 : 윤택야(潤澤也)
			- [[윤택(潤澤)]] : 살림이 풍부(豐富)함.
		- 밤사이에 내리는 이슬은 풀에 맺혀있다가 곁에 가게되면 묻게 된다.
		- [[노숙자(露宿者)]] 

	- [[各]]
		- 각각 각
		- [[恪]](정성 각)
		- [[閣]](큰집 각)
		- [[客]](손 객)
		- [[額]](이마 액)
		- [[路]](길 로)
		- [[賂]](뇌물 뢰)
		- [[鷺]](백로 로)


- [[結]](맺을 결)
	- 설문해자 : 체야(締也)
		- [[締]](맺을 체) : 임금의 면류관에 구슬을 실로 맺어주다
	- 結은 이슬이 [[締]]처럼 몽글몽글 맺는다는 느낌

	- [[吉]](길할 길) : 선비가 하는 말은 길하다.
	- [[士]](선비 사) : 선비는 하나를 알려주면 열을 하는 사람
	- [[喆]](밝을 철, 쌍길 철)
	- [[桔]](도라지 길) : 생명에 영향을 주는 도라지는 나무로.

- [[爲]](할 위, 삼을 위, 행할 위, 위할 위, 지을 위, 될 위
	- 설문해자: 모후야(母猴也)
	- 원숭이가 머릴 긁적대는 모습(尸겹친)과 행위(손으로 긁다)
	- [[이위(以爲)]] : **무슨 이유로~**
	- [[40. PARA/43. Rereferencess/Hword2/최성엽 실용한자/위인자자(爲人子者)|위인자자(爲人子者)]]

	- [[僞]](거짓 위)
		- 아직은 거짓이다 

- [[霜]](서리 상)
	- 相(서로 상, 모양 상, 재상 상)
	- [[재상(宰相)]]
