# 한자이야기 1784 돈대... 무슨 뜻인가?

![](https://www.youtube.com/watch?v=IauA_yZqjFA)  

- [[墩臺]]
    - 돈대
        - 성곽 시설의 하나로, 성벽 위에 쌓아 올려 망루나 포루 역할을 하는 누대(樓臺)
        - 형태는 원형과 방형이 있으며, 내부에 여러 단의 마루가 있고 외부를 향한 작은 안혈(눈구멍)이 있어 대포나 총, 화살을 쏠 수 있도록 설계됨
        - 대표적 예로 조선시대 강화도와 수원성에 설치된 돈대가 있음
        - 예시: 강화도에 설치된 돈대는 병자호란 당시 방어에 중요한 역할을 했다.
        - 墩 흙 더미 돈, 臺 누각 대
