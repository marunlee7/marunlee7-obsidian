# 한자이야기 1653 '퇴고'의 유래... 스님은 달 아래에서 문을 두드려야 할까, 밀어야 할까?

![](https://www.youtube.com/watch?v=M8023dXlIYg)  

- [[推敲]]
    - 퇴고
        - 글을 쓸 때 문장이나 표현을 여러 번 다듬고 수정하는 과정을 뜻한다.
        - 중국 당나라 시인 가도가 시를 짓던 중 ‘미느냐(推)’와 ‘두드리냐(敲)’를 고민하다 친구 한유의 조언을 받아 ‘두드리다(敲)’로 고쳐 완성했다는 고사에서 유래한다.
        - 예시: 그는 논문을 발표하기 전에 여러 번 퇴고 과정을 거쳤다.
        - 推 밀 퇴, 敲 두드릴 고
