# 📙 원길1-189 봉(풍)4봉 丰夆峯峰蜂逢

- [[丰]] 
	- 풀 무성할 봉 → 예쁠 봉, 풍성할 풍
		- 풀이 무성한 모양에서 **풀 무성할 봉**
		- 풀이 무성하면 예쁘니 **예쁠 봉**
		- 재물이 삼(三)대까지 이어질(丨) 정도로 풍성하니 **풍성할 풍**
- [[夆]]  
	- ==이끌 봉, 만날 봉==
		- 뒤져오더라도(夊) 예쁜(丰) 것을 이끌어 만나니 **이끌 봉, 만날 봉**
		- 夊 천천히 걸을 쇠, 뒤져 올 치
- [[峰]] 
	- 멧부리 봉 → 산 봉우리 봉
		- 산(山)등성이가 만나(夆) 이루어진 산봉우리니 **산봉우리 봉**
		- = 峯 봉우리 봉
	- [[雪鋒]]
	    - 설봉
	        - 눈이 쌓인 산봉우리, 눈 덮인 봉우리
	        - 雪 눈 설, 鋒 봉우리 봉
	- [[連峰]]
	    - 연봉
	        - 여러 산봉우리가 연이어 있음, 산봉우리들이 이어진 산맥
	        - 連 이을 연, 峰 봉우리 봉
	- [[雲峰]]
	    - 운봉
	        - 구름이 낀 산봉우리, 구름처럼 높이 솟은 봉우리
	        - 雲 구름 운, 峰 봉우리 봉
	- [[最高峰]]
	    - 최고봉
	        - 가장 높은 산봉우리
	        - 最 가장 최, 高 높을 고, 峰 봉우리 봉
- [[蜂]] 
	- 벌 봉
		- 벌레(虫) 중 만나(夆) 무리지어 사는 벌이니 **벌 봉**
			- 벌은 여왕을 중심으로 수만 마리가 모여 살지요.
			- 虫 벌레 충(蟲)의 속자와 부수
	- [[蜂起]]
	    - 봉기
	        - 벌떼처럼 떼 지어 세차게 일어남, 또는 사람들이 곳곳에서 떼지어 일어나는 것
	        - 蜂 벌 봉, 起 일어날 기
	- [[分蜂]]
	    - 분봉
	        - 벌집에서 일부 벌 무리가 분리되어 새로운 집단을 이루는 것
	        - 分 나눌 분, 蜂 벌 봉
	        - [[分蜂]]
	- [[養蜂]]
	    - 양봉
	        - 꿀벌을 기르는 일
	        - 養 기를 양, 蜂 벌 봉
- [[逢]] 
	- 만날 봉
		- 이끌고(夆) 가서(辶) 만나니 **만날 봉**
	- [[逢變]]
	    - 봉변
	        - 뜻밖의 변이나 망신스러운 일을 당함
	        - 逢 만날 봉, 變 변할 변
	- [[逢別]]
	    - 봉별
	        - 만남과 이별
	        - 逢 만날 봉, 別 나눌 별
	- [[逢着]]
	    - 봉착
	        - 만나서 부닥침, 만남
	        - 逢 만날 봉, 着 붙을 착
	- [[相逢]]
	    - 상봉
	        - 서로 만남
	        - 相 서로 상, 逢 만날 봉
	- [[絶處逢生]]
	    - 절처봉생
	        - 극도로 궁박한 끝에 살길이 생김
	        - 絶 끊을 절, 處 곳 처, 逢 만날 봉, 生 날 생
