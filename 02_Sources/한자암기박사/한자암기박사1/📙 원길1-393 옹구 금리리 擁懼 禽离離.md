# 📙 원길1-393 옹구 금리리 擁懼 禽离離

- [[擁]]    
	- ==낄 옹== → 안을 옹
		- 손( 扌)으로 머리(亠)까지 작은(幺) 새(隹)처럼 안으니 **안을 옹**
- [[懼]]  
	- ==두려워할 구==
		- 마음(忄)이 두 눈(目目) 두리번 거리는 새(隹)처럼 두려워하니 **두려워할 구**
	- 疑懼心
		- 의구심
			- 의심(疑心)하고 두려워하는 마음.
				- 疑 의심할 의 懼 두려워할 구 心 마음 심
	- 悚懼
		- 송구
			- 두려워서 마음이 몹시 거북함.
				- 悚 두려울 송 懼 두려워할 구
- [[禽]] 
	- 새 금 → 날짐승 금
		- 그물(人)로 씌워 잡는 짐승(离)은 날짐승이니 **날짐승 금**
			- 离 도깨비/산신 리 → 짐승 리, 떠날 리
			- 人 사람 인 → 여기서는 그물로 봄
			- 총이 없없던 옛날에는 날짐승을 그물로 씌워 잡았지요.
	- [[禽獸]]
		- 금수
			- 날짐승과 길짐승
			- 모든 짐승을 이르는 말.
				- 禽 새 금 獸 짐승 수
	- [[禽獸魚蟲]]
		- 금수어충
			- 날짐승과 길짐승과 물고기와 벌레
			- 모든 동물(動物)을 이르는 말
				- 禽 새 금 獸 짐승 수 魚 물고기 어 蟲 벌레 충
	- [[禽獸行]]
		- 금수행
			- 금수(禽獸)같은 행동(行動).
				- 禽 새 금 獸 짐승 수 行 다닐 행
- [[离]] 
	- 도깨비/산신 리 → 짐승 리, 떠날 리
		- 머리 부분(亠)에 베인(乂) 듯 입 벌린 모습(凵)의 짐승이 사사로이(厶) 성(冂) 같은 발자국을 남기고 떠나니 **짐승 리, 떠날 리**
		- 㐫(흉할 흉) + 禸(발자국 유)
- [[離]]      
	- 떠날 리 → 헤어질 리
		- 짐승(离)이나 새(隹)처럼 기약 없이 헤어지니 **헤어질 리**
			- 隹(새 추) + 离 도깨비/산신 리 → 짐승 리, 떠날 리
			- 부수 : 隹 (새추, 8획)
	- ==[[離間]]==
		- 이간
			- 둘 사이를 갈라놓고 이간질하는 것.
				- 離 떠날 이 間 사이 간
	- 相逢 ↔ ( )別
		- ==[[相峯]]==
			- 상봉
				- 서로 만남.
					- 相 서로 상 逢 만날 봉
		- ==[[離別]]==
			- 이별
				- 서로 갈려 떼어짐. 헤어짐
					- 離 떠날 리 別 나눌 별
	- ==[[距離]]==
		- 거리
			- 점과 점 사이를 잇는 직선(直線)의 길이.
			- 이것과 저것의 서로 떨어진 사이의 멀고 가까운 정도(程度).
			- 서울에서 부산까지 ==距( )==는 약 400km가 된다
			- 距 상거할 거 離 떠날 리
	- [[離散]]
		- 이산
			- 헤어져 흩어짐.
				- 離 떠날 리(이) 散 흩을 산
	- [[會者定離]]
		- 회자정리
			- 만난 자(者)는 반드시 헤어짐.
			- 모든 것이 무상(無常)함을 나타내는 말이다.
				- 會 모일 회 者 놈 자 定 정할 정 離 떠날 리(이)
	- [[離隔]]
		- 이격
			- 사이가 벌어짐, 또는 사이를 벌려 놓음
			- 離 떠날 리 隔 사이 뜰 격
