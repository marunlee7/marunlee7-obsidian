# 📙 원길1-211

- **원원완난**   
	- [[爰]] 이에 원
		- [[援]] 도울 원
		- [[緩]] 느릴 완
		- [[暖]] 따뜻할 난
- [[爰]] 
	- 이에 원
		- 손(爫)으로 한(一) 명의 벗(友)을 이에 끌어당기니 **이에 원, 끌 원, 당길 원**
			- 爫 손톱 조 → 여기서는 손으로 봄
	- 爰書
		- 원서
			- 죄인(罪人)이 진술(陳述)한 죄상(罪狀)을 적은 서류(書類).
				- 爰 이에 원 書 글 서
	- 爰居爰處 
		- 원거원처
			- 어리저리 옮겨 삶
				- 爰 이에 원 居 살 거 處 곳 처
- [[援]]   
	- 도울 원
		- 손( 扌)으로 당겨(爰) 도우니 **도울 원**
			- 爰 이에 원, 끌 원, 당길 원
				- 손( 爫)으로 한(一) 명의 벗(友)을 **이에 끌어당기다**
	- [127] 支援 - ( ) 어떤 일에 뜻을 두고 구성원이 되기를 바람.
		- ==支援==
			- 지원
				- 지지(支持)하여 도움.
					- 支 지탱할 지 援 도울 원
		- ==志願==
			- 지원
				- 뜻이 있어 지망(志望)함.
					- 志 뜻 지 願 원할 원
	- 援軍
		- 원군
			- 전투(戰鬪)에서 자기편(自己便)을 도와주는 군대(軍隊).
				- 援 도울 원 軍 군사 군
	- 援用
		- 원용
			- 자기(自己) 이익(利益)을 위(爲)해 어떤 사실(事實)을 딴 데서 끌어다가 주장(主張)하거나 또는 거부(拒否)하는 일.
				- 援 도울 원 用 쓸 용
	- 援助
		- 원조
			- 물품(物品)이나 돈 따위로 도와줌.
				- 援 도울 원 助 도울 조
	- 援護
		- 원호
			- 구원(救援)하여 보호(保護)함.
				- 援 도울 원 護 도울 호
	- 救援
		- 구원
			- 어려움이나 위험(危險)에 빠진 사람을 구(救)하여 줌.
				- 救 구원할 구 援 도울 원
- [[緩]]  
	- 느릴 완
		- 실(糸)을 당기면(爰) 늘어나 느슨하니 **느슨할 완**
		- 느슨하게 행동하여 느리니 **느릴 완**
	- 緩急
		- 완급
			- 緩 느릴 완 急 급할 급
	- 緩慢
		- ==완만==
			- ↔ 急激 급격
			- 행동(行動)이 느릿느릿함.
			- (경사(傾斜)가)급하지 않음.
				- 緩 느릴 완 慢 거만할 만
	- 緩衝
		- 완충
			- 충돌(衝突)을 완화(緩和)시킴.
				- 緩 느릴 완 衝 찌를 충
	- 緩和
		- 완화
	- 緩行
		- 완행
- [[暖]] 
	- 따뜻할 난
		- 햇(日)빛을 당기면(爰) 따뜻하니 **따뜻할 난**
			- `유` 煖 더울 난
			- 暖 따뜻할 난 : 햇볕으로 인하여 따뜻하다는 말
			- 煖 더울 난 : 불 화(火)가 들어가 불로 인하여 따뜻하다는 말
	- 暖氣
		- 난기
			- 따뜻한 기운.
				- 暖 따뜻할 난 氣 기운 기
	- 暖帶
		- 난대
			- 열대(熱帶)와 온대(溫帶)의 중간(中間) 지대(地帶).
				- 暖 따뜻할 난 帶 띠 대
	- 暖流
		- 난류
			- 暖 따뜻할 난 流 흐를 류(유)
	- 煖房
		- 난방
			- 실내(室內)의 온도(溫度)를 높여 따뜻하게 하는 일.
				- 暖 따뜻할 난 房 방 방
	- 和風暖陽
		- 화풍난양
			- 솔솔 부는 화창(和暢)한 바람과 따스한 햇볕
			- 따뜻한 봄 날씨를 이르는 말.
				- 和 화할 화 風 바람 풍 暖 따뜻할 난 陽 볕 양
