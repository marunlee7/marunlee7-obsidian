	- 칠 복

		- 攴 - 점(卜) 칠 때 오른손(又)에 회초리를 들고 툭툭 치면서 점친다는 데서 **칠 복**
		- 乂 벨 예, 다스릴 예, 어질 예 - 여기서는 엇가리는 모습으로 봄

- [[散]]     
	- 흩을 산
		- 풀(艹)이 난 땅(一)에 고기(月)를 놓고 치면(攵) 여러 조각으로 흩어지니 **흩어질 산**
	- [122] 解散 ↔ ( )
		- ==[[解散]]==
			- 해산
				- 모였던 사람들이 흩어짐. 흩어지게 함. ‘흩어짐’으로 순화(醇化).
					- 解 풀 해 散 흩을 산
		- ==集合==
			- 집합
				- 한 곳으로 모음. 또는 한 곳으로 모임.
					- 集 모을 집 合 합할 합
	- [[霧散]]
		- 무산
			- ==안개가 걷히듯 흩어져 없어짐==
				- 霧 안개 무 散 흩을 산
	- [[擴散]]
		- ==확산==
			- 퍼져 흩어짐.
			- 여러 백화점에서 알뜰 구매가 **擴散**되고 있다.
				- 擴 넓힐 확 散 흩어질 산
	- ==[[散步]]== 
		- 산보
	        - 가볍게 천천히 거닐며 걷는 행위.
	        - 특별한 목적 없이 기분 전환이나 운동, 자연 감상을 위해 여유롭게 걷는 것을 뜻함.
	        - 예시: 저녁에 공원에서 산보를 하다.
	        - 散 흩을 산, 步 걸음 보
	- ==[[散華]]==
		- 산화
			- 꽃잎을 흩뿌림
			- 주로 장례식에서 고인의 명복을 기원하는 행위.
			- 수많은 젊은이들이 조국을 위해 장렬히 **산화**하였다.
				- 散 흩을 산 華 빛날 화
	- [[散髮]]
		- 산발
			- 머리를 풀어 엉클어뜨림
				- 散 흩을 산 髮 터럭 발
	- [[散發]]
	    - 산발
	        - 흩어질 산(散), 발생할 발(發)로, 어떤 일이 한꺼번에 나타나지 않고 여기저기 흩어져서 발생함을 뜻함.
	        - 머리를 풀어헤치거나 풀어진 머리를 산발(散髮)이라고도 함.
	        - 예시: 열 개의 안타를 쳤으나 산발(散發)이었기에 집중 안타를 친 상대팀에 패배하였다.
	        - 散 흩을 산, 發 발생할 발
	- [[散在]]
