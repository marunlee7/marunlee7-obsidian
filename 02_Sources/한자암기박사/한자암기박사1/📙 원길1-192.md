# 📙 원길1-192 반반판판판 反返叛板版販

- [[反]]  
	- 거꾸로 반, 뒤집을 반, 되돌릴 반, 되풀이할 반(→반복)
		- 가린 것(厂)을 손(又)으로 거꾸로 뒤집으니 **거꾸로 반, 뒤집을 반**
		- 厂 굴 바위 엄 → 여기서는 덮어 가린 것
	- [[反對]]
	    - 반대
	        - 의견이나 행동이 서로 맞서거나 일치하지 않음.
	        - 어떤 주장, 의견, 계획 등에 찬성하지 않고 반박하거나 저항하는 것.
	        - 예시: 그는 그 제안에 반대(反對) 의견을 냈다.
	        - 출전 : 일상어, 정치, 사회 용어.
	        - 反 돌이킬 반, 對 대할 대
	        - [[反對_반대_의견이나 행동이 서로 맞서거나 일치하지 않는 상태. 反 돌이킬 반, 對 대할 대]]
	- [[反抗]]
	    - 반항
	        - 권위나 명령에 저항하며 맞서는 태도.
	        - 억압이나 불합리한 상황에 대해 맞서 싸우거나 거부하는 행동.
	        - 예시: 청소년기의 반항(反抗)은 성장 과정의 일부이다.
	        - 출전 : 심리학, 사회학 용어.
	        - 反 돌이킬 반, 抗 겨룰 항
	        - [[反抗_반항_권위에 저항하며 맞서는 태도. 反 돌이킬 반, 抗 겨룰 항]]
	- [[違反]]
	    - 위반
	        - 법률, 규칙 따위를 어기거나 어긋나는 행위.
	        - 정해진 규범이나 명령에 반하여 행동하는 것.
	        - 예시: 교통법규 위반(違反)으로 벌금을 냈다.
	        - 출전 : 법률 및 행정 용어.
	        - 違 어긋날 위, 反 돌이킬 반
	        - [[違反_위반_법률이나 규칙을 어기는 행위. 違 어긋날 위, 反 돌이킬 반]]
	- [[反擊]]
	    - 반격
	        - 공격을 받은 쪽에서 되받아 공격함.
	        - 군사나 스포츠 등에서 상대의 공격에 맞서 다시 공격하는 것.
	        - 예시: 적의 공격에 즉각 반격(反擊)했다.
	        - 출전 : 군사, 스포츠 용어.
	        - 反 돌이킬 반, 擊 칠 격
	        - [[反擊_반격_받은 공격을 다시 공격함. 反 돌이킬 반, 擊 칠 격]]
	- [[贊反]]
	    - 찬반
	        - 어떤 주장에 대해 찬성과 반대 양쪽이 존재함.
	        - 의견 대립이나 토론에서 흔히 쓰임.
	        - 예시: 회의에서 찬반(贊反) 양론이 갈렸다.
	        - 출전 : 토론, 회의 용어.
	        - 贊 도울 찬, 反 돌이킬 반
	        - [[贊反_찬반_찬성과 반대 양론. 贊 도울 찬, 反 돌이킬 반]]
	- [[反復]]
	    - 반복
	        - 같은 행동이나 과정이 여러 번 이어지는 것.
	        - 어떤 과정을 되풀이하는 행위.
	        - 예시: 성공을 위해서는 반복(反復) 학습이 필요하다.
	        - 출전 : 일상, 교육용어.
