# 📙 원길2-411 시시식 어어 是匙湜 御禦

- [[是]] 
	- 이[斯]/옳을 시
		- 해(日)처럼 밝고 바르면(疋) 옳으니 **옳을 시**
		- 해(日)처럼 밝게 바로(疋) 이것이라며 가리키니 **이 시, be동사 시**
			- 疋 바를 정(正) 변형
			- 是에는 영어의 be동사처럼 ~이다 라는 뜻도 있습니다.
			- 斯 이 사 → 신령이 나타나 그(其) 도끼(斤)가 **이것**이냐?
- [[匙]] 
	- 숟가락 시
		- 옳게(是) 비수(匕)처럼 찔러 먹는 숟가락이니 **숟가락 시**
	- [[匙箸]]
	    - 시저
	        - 숟가락과 젓가락이라는 뜻으로, 음식물을 먹는 데 쓰는 도구.
	        - 예시: 우리는 식사할 때 숟가락과 젓가락을 사용한다.
	        - 匙 숟가락 시, 箸 젓가락 자
	- [[十匙一飯]]
	    - 십시일반
	        - 열 사람이 한 숟가락씩 음식을 보태면 한 그릇의 밥이 됨, 여러 사람이 힘을 합하면 어려운 일을 이룰 수 있음.
	        - 예시: 십시일반의 마음으로 어려운 이웃을 돕자.
	        - 十 열 십, 匙 숟가락 시, 一 한 일, 飯 밥 반
- [[湜]] 
	- 물맑을 식
		- 물(氵)의 옳은(是)  모습은 맑으니 **물 맑을 식**
		- 오염되지 않은 원래 물은 맑지요.
- [[御]] 
	- 거느릴 어 → 말몰/다스릴/임금 어
		- 가다가(彳) 정오(午)쯤 되면 그쳐(止) 무릎 꿇고(卩) 쉬게 하며 말을 몰고 다스리니 **말 몰 어, 다스릴 어**
		- 백성을 다스리는 임금이니 **임금 어**
			- 午 낮 오, 말 오, 일곱째 지지 오
			- 彳 조금 걸을 척
- [[禦]] 
	- 막을 어
		- 다스려(御) 보이지(示) 않게 막으니 **막을 어**
	- [[禦敵]]
	    - 어적
	        - 적의 침공을 막음.
	        - 예시: 적의 공격을 어적으로 막았다.
	        - 禦 막을 어, 敵 대적 적
	- [[防禦]]
	    - 방어
	        - 적의 공격이나 위험으로부터 스스로를 지킴.
	        - 예시: 성곽은 도시를 방어하는 중요한 시설이다.
	        - 防 막을 방, 禦 막을 어
	- [[攻擊]]
	    - 공격
	        - 힘을 써서 적을 쳐 공격함.
	        - 예시: 공격을 받아 방어에 실패했다.
	        - 攻 칠 공, 擊 칠 격
	- [[禦冬]]
	    - 어동
	        - 겨울을 대비하거나 막음.
	        - 예시: 어동 준비를 철저히 하다.
	        - 禦 막을 어, 冬 겨울 동
