# 📙 원길2-413 행(항)연함구 行衍銜衢

- [[行]] 
	- 다닐 행 | 항렬 항
		- 사람이 다니며 행하는 사거리를 본떠서 **다닐 행, 행할 행**
		- (친척의 이름에서 돌려) 다니며 쓰는 항렬이니 **항렬 항**
- [[衍]] 
	- 넓을 연
		- 물(氵)이 물건 속으로 스미어 가면(行) 불어나 퍼지고 넓으니 **퍼질 연, 넓을 연**
	- [[衍文]]
		- 연문
			- 본문에 덧붙여진 군더더기 글자나 문장
			- ‘연문(衍文)’은 원래의 본문에 불필요하게 덧붙여진 글자나 문장을 의미합니다. 주로 고전 문헌의 판본을 비교할 때, 후대에 첨가된 부분을 가리킬 때 사용됩니다.
			- 衍 넘칠 연, 文 글월 문]]
	- [[衍義]]
		- 연의
			- ‘연의(衍義)’는 본문의 의미를 넓혀서 해석하거나 풀이하는 것을 말합니다. 
			- 원문에 없는 의미를 덧붙여 해석할 때 쓰입니다.
	- [[敷衍]]  
		- 부연  
			- ==자세히 설명하거나 내용을 덧붙여 늘어놓음.  ==
			- 또는 대충 얼버무리며 넘김.
			- = [[敷演]](부연) 
			- 敷 펼 부 衍 넓을 연
- [[銜]] 
	- 재갈 함
		- 말을 부릴(行) 때 쇠(金)로 만들어 물리는 재갈이니 **재갈 함**
		- 일을 행할(行) 때 쇠(金)로 새겨 주는 직함이니 **직함 함**
		- 재갈 - 말을 부리기 위하여 아가리에 가로로 물리는 가느다란 막대, 소리를 내지 못하도록 사람의 입에 물리는 물건.
	- [[職銜]]
		- 직함
			- 벼슬의 이름
			- 직책이나 직무의 이름
	- [[銜勒]]  
	 - 함륵  
	  - 말의 입에 물리는 재갈과 고삐.  
	   - 銜 머금을 함, 勒 굴레 륵
	- [[銜泣]]  
	 - 함읍  
	  - 울음을 억누르고 삼킴, 또는 억지로 눈물을 참음.  
	   - 銜 머금을 함, 泣 울 읍
	- [[銜字]]  
	 - 함자  
	  - 이름(특히 본명), 또는 문서에 적는 이름.  
	   - 銜 머금을 함, 字 글자 자
	- [[名銜]]  
	 - 명함  
	  - 이름과 직함, 또는 직위를 나타내는 호칭.  
	   - 名 이름 명, 銜 머금을 함
- [[衢]] 
	- 네거리 구
		- 두 눈(目)을 새(隹)처럼 크게 뜨고 조심히 가야(行)할 사거리니 **사거리 구**
	- [[衢街]]
		- 구가
			- 큰길과 거리, 사통오달(四通五達)로 통하는 넓은 거리.
			- 衢 네거리 구, 街 거리 가
	- [[衢路]]
		 - 구로
		  - 여러 갈래로 통하는 큰길, 네거리.
		  - 衢 네거리 구, 路 길 로
	- [[康衢煙月]]
		- 강구연월
			- 번화한 큰길거리에서 달빛이 연무(煙霧)에 은은하게 비치는 모습.
			- 태평한 세상의 평화롭고 여유로운 풍경을 뜻하는 사자성어.
			   - 康 편안할 강, 衢 네거리 구, 煙 연기 연, 月 달 월
			- 고대 중국 열자(列子) ‘중니편(仲尼篇)’에 나오는 말로, 요임금 치세의 태평성대를 상징함.
