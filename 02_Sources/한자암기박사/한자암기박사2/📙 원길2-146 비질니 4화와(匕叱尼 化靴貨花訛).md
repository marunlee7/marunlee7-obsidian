# 📙 원길2-146

- [[匕]] 
	- 비수 비
		- 비수를 본떠서 **비수 비**
		- 비수처럼 입에 찔러 먹는 숟가락이니 **숟가락 비**
- [[叱]] 
	- 꾸짖을 질
		- 입(口)으로 비수(匕)처럼 날카롭게 꾸짖으니 **꾸짖을 질**
	- [[叱責]](질책) : 꾸짖어 나무람.
	- [[叱咤]](질타) : 큰 소리로 꾸짖음.
	- [[叱正]](질정) : 꾸짖어 바로잡음.
- [[尼]] 
	- 여승 니
		- 몸(尸)의 머리털을 비수(匕)로 깍은 여승이니 **여승 니**
		- 尸 주검 시, 몸 시
	- [[尼僧]] 이승 : 승려(僧侶)가 된 여자(女子).
	- [[僧尼]] 승니 : 중과 여승. 비구(僧)와 비구니((尼).
	- [[比丘尼]] 비구니 : 불교에서 여승을 이르는 말.
	- [[30-리소스/Dict/Hword3/釋迦牟尼]] 석가모니 : 불교의 창시자인 석가족의 성자. 본명은 고타마 싯다르타.
- [[化]] 
	- 될 화 → 가르칠 화
		- 사람(人)이 비수(匕)같은 마음을 품고 일하면 안 되는 일도 되고 변화하니 **될 화, 변화할 화**
		- 되도록 가르치니 **가르칠 화**
- [[靴]] 
	- 신[履, 鞋] 화
		- 가죽(革)을 변화시켜(化) 만든 가죽신이니 **가죽신 화**
		- 革 가죽 혁, 고칠 혁
	- [[靴工]](화공)
	- [[軍靴]](군화)
	- [[長靴]](장화)
- [[貨]] 
	- 재물 화
		- 변하여(化) 돈(貝)이 되는 재물이나 물품이니 **재물 화, 물품 화**
- [[花]] 
	- 꽃 화
		- 풀(艹)의 일부가 변하여(化) 피는 꽃이니 **꽃 화**
	- 枯木生花
		- 고목생화
			- 마루 나무에 꽃이 핌
			- 곤궁한 사람의 형세가 피거나 늙은 사람이 자식을 봄을 비유하여 이르는 말
- [[訛]] 
	- 그릇될 와
		- 말(言)이 사실과 달리 변하여(化) 그릇되니 **그릇될 와**
	- [[訛謬]](와류)
	- [[訛言]](와언)
	- [[訛傳]]
		- 와전
			- 事實과 다르게 傳함.
				- 訛 그릇될 와 傳 전할 전
