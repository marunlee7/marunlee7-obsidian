# 📙 원길2-120 조부부유 爪孚孵乳

- [[爪]] 
	- 손톱 조
		- 손톱 모양을 본떠서 **손톱 조**
			- 부수 : 爫 모습으로 짧습니다.
- [[孚]] 
	- 믿을 부 → 알깔 부, 기를 부
		- 새가 발톱(爫)으로 알(子)을 품어 굴리며 알 까게 알 속의 새끼를 기르니 **알 깔 부, 기를 부**
		- 爫 손톱 조
		- 子 아들 자, 첫째 지지 자, 자네 자, 접미사 자 → 여기서는 알로 봄
- [[孵]] 
	- 알깔 부
		- 알(卵)을 품어 기르면(孚) 알이 까니 **알깔 부**
	- [[孵卵]]
		- 부란
			- (물고기나 날짐승의)알을 까거나 깨는 일.
			- 孵 알 깔 부 卵 알 란(난)
	- [[孵化]]
		- 부화
			- 동물의 알 속에서 새끼가 껍질을 깨고 밖으로 나옴. 또는 그렇게 되게 함.
			- 孵 알 깔 부 化 될 화
- [[乳]] 
	- 젖 유
		- 기를(孚) 때 꼭지(乙)로 먹이는 젖이니 **젖 유**
			- 乚 새 을, 둘째 천간 을, 둘째 을, 굽을 을(乙)이 부수로 쓰일 때의 모습 → 여기서는 꼭지로 봄
