# 📙 원길2-202 주주 팽팽팽 尌廚 彭澎膨

- [[尌]]  
	- 세울 주
		- 좋게(吉) 받쳐(䒑) 법도(寸)에 맞게 세우니 **세울 주**
		- 寸(마디 촌) + 壴(악기 이름 주)
		- 吉 길할 길, 상서로울 길
		- 길하다 - 일이 좋거나 상서롭다.
	- 壴 악기 이름 주
		- 尌 세울 수
		- 鼓 북 고
- [[廚]]  
	- 부엌 주
		- 집(广)에 조리대를 세워(尌) 요리하도록 만든 **부엌**
			- 广(집 엄) + 尌(세울 수)
			- 寸(마디 촌) + 壴(악기 이름 주)
	- [[廚房]]
		- 주방
			- 음식(飮食)을 만들거나 차리는 방(房).
				- 廚 부엌 주 房 방 방
	- [[鼎廚]]
		- 정주 → 정지
	- [[庖廚]]
		- 포주 → 푸주
- [[彭]] 
	- 성(姓) 팽 → **세찰 팽, 많을 팽**
		- 악기(壴)를 좋게(吉) 받쳐(䒑)놓고 머리카락(彡) 휘날리며 세차게 많이 연주하다.
			- 彡(터럭 삼) + 壴(악기 이름 주)
	- [[彭湃]]
		- 팽배
			- 어떤 氣勢나 思潮 따위가 거세게 일어남.
				- 彭 성(姓) 팽  湃 물결칠 배
- [[澎]] 
	- 물소리 팽
		- 물이 북을 쳐서 머리카락이 비스듬히 서게 되면: 포효하거나 튀김.
		- 氵(삼수변 수) + 彭(나라 이름 팽)
	- [[澎排]]
		- 팽배
			- 물결이 서로 부딪쳐 솟구침
			- 어떤 사조나 기운 따위가 맹렬한 기세로 일어남.
			- = 彭湃 팽배
				- 澎 물소리 팽 湃 물결칠 배
- [[膨]] 
	- 불을 팽
		- 몸(月)이 많이(彭) 부푸니 **부풀 팽**
	- [[膨膨]]
	    - 팽팽
	        - 부풀어 오르거나 팽창하는 모양, 또는 그런 상태를 나타내는 말.
	        - 膨 부풀 팽, 膨 부풀 팽
	- [[膨滿]]
	    - 팽만
	        - 내부에 가득 차서 부풀어 오른 상태. 의학적으로는 복부 팽만(腹部膨滿) 등으로 많이 쓰임[1](https://www.koreascience.kr/article/JAKO202517643201847.pdf).
	        - 膨 부풀 팽, 滿 찰 만
	- [[膨壓]]
	    - 팽압
	        - 내부에서 바깥으로 미치는 팽창의 압력.
	        - 膨 부풀 팽, 壓 누를 압
	- [[膨脹]]
	    - 팽창
	        - 부피나 범위가 늘어나거나 커짐. 물리적, 경제적, 사회적 의미 모두에 쓰임.
	        - 膨 부풀 팽, 脹 부를 창
