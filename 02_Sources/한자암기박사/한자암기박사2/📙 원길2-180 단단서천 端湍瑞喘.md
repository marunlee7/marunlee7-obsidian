# 📙 원길2-180 단단서천 端湍瑞喘

- [[端]] 
	- 끝 단 → 바를, 실마리 단
		- 서(立) 있는 곳이 산(山)으로 이어진(而) 끝이니 **끝 단**
		- 일의 끝에 서면 마음이나 옷차림을 바르게 하여 찾는 실마리니 **바를 단, 실마리 단**
- [[湍]] 
	- 여울 단
		- 물(氵)이 산(山)으로 이어진(而) 좁을 곳을 흐르는 여울이니 **여울 단**
		- 여울 - 강이나 바닥의 바닥이 앝거나 폭이 좁아 물살이 세차게 흐르는 곳
	- [[湍水]]
	    - 단수
	        - 빠르고 급(急)히 흐르는 물, 급류.
	            - 湍 급할 단 水 물 수
	- [[急湍]]
	    - 급단
	        - 매우 빠르고 급(急)히 흐르는 여울, 급류.
	            - 急 급할 급 湍 여울 단
	- [[激湍]]
	    - 격단
	        - 매우 급(急)히 흐르는 여울.
	            - 激 격할 격 湍 여울 단
- [[瑞]] 
	- 상서 서
		- 구슬(王)로 된 산(山)이 이어진(而)듯 상서로우니  **상서로울 서**
	- [[瑞光]]
		- 서:광
			- 상서(祥瑞)로운 빛.
				- 瑞 상서 서  光 빛 광
	- [[瑞氣]]
		- 서기
			- 상서(祥瑞)로운 기운(氣運).
				- 瑞 상서 서  氣 기운 기
	- [[瑞夢]]
		- 서몽
			- 상서로운 꿈
				- 瑞 상서 서  夢 꿈 몽
- [[喘]] 
	- 숨찰 천
		- 입(口)이 산(山)으로 이어진(而) 곳을 오를 때처럼 숨이 가빠 헐떡거리니 **헐떡거릴 천**
	- [[喘急]]
	    - 천급
	        - 숨이 몹시 가쁘고 헐떡이는 증상.
	            - 喘 숨찰 천 急 급할 급
	- [[喘息]]
	    - 천식
	        - 숨이 차서 헐떡이며 거칠게 숨을 쉬는 증상 또는 질환.
	            - 喘 숨찰 천 息 쉴 식
	- [[咳喘]]
	    - 해천
	        - 기침(咳)과 숨이 가쁜 증상이 함께 나타나는 상태.
	            - 咳 기침 해 喘 숨찰 천
