# 📙 원길2-111 반판 反阪

- 反 
	- 돌이킬/돌아올 반 → 거꾸로/뒤집을 반
		- 가린 것(厂)을 손(又)으로 거꾸로 뒤집으니 **거꾸로 반, 뒤집을 반**
			- 厂 굴 바위 엄 → 여기서는 덮어 가린 것
	- 反對
		- 반대
			- 두 사물(事物)이 맞서 있는 상태(狀態).
				- 反 돌이킬 반 對 대할 대
	- 反抗
		- 반항
			- 상대방(相對方)에 반대(反對)하여 대들음.
				- 反 돌이킬 반 抗 겨룰 항
	- 違反
		- 위반
			- 법령(法令)･명령(命令)･약속(約束) 등(等)을 어기거나 지키지 않는 것
				- 違 어긋날 위 反 돌이킬 반
	- 反擊
		- 반격
			- 쳐들어오는 적(敵)을 되받아 공격(攻擊)함.
				- 反 돌이킬 반 擊 칠 격
	- 贊反
		- 찬반
			- 찬성(贊成)과 반대(反對).
			- 투표에 대한 贊( )이 대등했다
				- 贊 도울 찬 反 돌이킬 반
	- 反復 
		- 반복
			- 같은 일을 되풀이함.
			- `me` 反 처음으로 돌와서 復 다시 하다.
				- 反 돌이킬 반 復 회복할 복 | 다시 부
- 阪 
	- 언덕 판 → 비탈 판
		- 언덕(阝)이 뒤집어질듯(反) 경사진 비탈이니 **비탈 판** (=坂)
			- 阪 - 네이버사전 등록 어휘 6개
			- 坂 - 흙(土)이 거꾸로(反) 선 듯한 비탈이니 **언덕 판 → 비탈 판** 
	- 坂路
		- 판로
			- 비탈진 언덕의 길.
				- 坂 언덕 판 路 길 로(노)
	- 九折坂
		- 구절판
			- 구절판찬합(九折坂饌盒)에 담아 먹는 우리나라 固有의 飮食.
			- 둘레의 여덟 칸에 각각(各各) 여덟 가지 음식(飮食)을 담고, 가운데 둥근 칸에는 밀전병(-煎餠)을 담아 두어, 둘레의 음식(飮食)을 골고루 조금씩 전병(煎餠)에 싸서 먹는다.
				- 九 아홉 구 折 꺾을 절 坂 언덕 판
	
- **암1 192, 340**   
		- 返 거꾸로(反) 가며(辶) 돌이키니  **돌이킬 반**
		- 叛 반(半)씩 나누어도 뒤집으며(反) 배반하니 **배반할 반**
		- 板 나무(木)를 톱을 켜면 반대(反)쪽으로 벌어지면서 생기는 널조각이니 **널조각 판**
		- 版 나무 조각(片)에 글자를 새겨 뒤집어(反) 인쇄하는 판목이니 **인쇄할 판, 판목 판**
		- 販 재물(貝)을 거꾸로(反) 주며 팔고 장사하니 **팔 판, 장사할 판**
		- 飯 먹을(𩙿) 때 혀로 이리저리 뒤집으며(反) 씹어 먹는 밥이니 **밥 반**
