# 📙 원길2-402 종종 예준선 從慫睿濬璿

- [[從]]  
	- 좇을 종
		- 걸어서(彳) 두 사람(从)이 점(卜)치는 사람(人)을 좇아 따르니 **좇을 종, 따를 정**
		- `약` 従 - 걸어서(彳) 이쪽저쪽(八)으로 아래(下)까지 사람(人)을 좇아 따르니 **좇을 종, 따를 종**
		- 从 - 사람(人) 사람(人)을 좇아 따르니 **좇을 종, 따를 종**
		- 彳 조금 걸을 척
- [[慫]] 
	- 권할 종 → 놀랄 종, 두려울 종
		- 따를(從) 마음(心)이 들도록 권하니 **권할 종**
		- 갑자기 따를(從) 마음(心)이 정도로 놀랍고 두려우니 **놀랄 종, 두려울 종**
	- [[慫慂]]
	    - 종용
	        - 잘 설명하고 달래어 따르도록 권함.
	        - 예시: 그는 아랫사람들을 慫慂하여 일하게 했다.
	        - 慫 권할 종, 慂 권할 용
	- [[從容]]
	    - 종용
	        - 침착하고 느긋함, 차분함.
	        - 예시: 그는 항상 從容한 태도를 유지한다.
	        - 從 좇을 종, 容 얼굴 용
	- [[慫搖]]
	    - 종요
	        - 권하거나 달래며 움직임, 부추김.
	        - 예시: 그는 慫搖하여 마음을 돌려놓았다.
	        - 慫 권할 종, 搖 흔들 요
- [[睿]] 
	- 슬기 예
		- 점(卜)같은 미신을 덮어(冖)버리고 하나(一)같이 파고(八) 파(八) 눈(目)으로 직접 보는 밝은 슬기니 **밝을 예, 슬기 예(=[[叡]])**
		- 천자나 성인에 관한 사물에 붙여 썻는데, 우리나라에서는 왕세자에 대한 경칭으로도 쓰였습니다. 
	- [[睿德]]
	    - 예덕
	        - 깊고 밝은 덕.
	        - 예시: 그는 睿德로운 지도자다.
	        - 睿 밝을 예, 德 덕 덕
	- [[睿智]]
	    - 예지
	        - 깊고 밝은 지혜.
	        - 예시: 睿智한 결정이었다.
	        - 睿 밝을 예, 智 총명할 지
- [[濬]] 
	- 깊을 준
		- 물(氵)이 맑아 속까지 들여다보이게 밝고(睿) 깊으니 **깊을 준**
	- [[濬潭]]
	    - 준담
	        - 깊고 맑은 못.
	        - 예시: 濬潭에서 낚시를 했다.
	        - 濬 깊을 준, 潭 못 담
	- [[濬源]]
	    - 준원
	        - 근원이나 시작점이 깊음.
	        - 예시: 우리의 전통은 濬源이 깊다.
	        - 濬 깊을 준, 源 근원 원
	- [[濬哲]]
	    - 준철
	        - 깊은 지혜.
	        - 예시: 그는 濬哲한 학자다.
	        - 濬 깊을 준, 哲 밝을 철
	        - [[濬哲]]
- [[璿]] 
	- 구슬 선
		- 옥(王)빛이 밝은(睿) 구슬이니 **구슬 선**
	- [[璿源大鄕]]
	    - 선원대향
	        - 조선 왕조 왕실의 관향(貫鄕)인 전주(全州)를 말함.
	        - ‘璿(선)’은 아름다운 옥을 뜻하며, ‘源(원)’은 근원, ‘大(대)’는 크다는 뜻, ‘鄕(향)’은 고을을 의미함.
	        - 즉, 고귀한 옥 같은 뿌리를 가진 큰 고을이라는 의미이다.
	        - 예시: 璿源大鄕은 조선 왕실의 시조가 나온 고향이다.
	        - 璿 옥돌 선, 源 근원 원, 大 클 대, 鄕 고을 향
