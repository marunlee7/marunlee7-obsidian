# 📙 원길2-270 봉(풍)해할할헌 계(결･글･설)끽결 丰害割轄憲契喫潔

- [[丰]] 
	- 풀 무성할 봉 → 예쁠 봉, 풍성할 풍, ==어지러울 봉==
		- 풀이 무성한 모양에서 **풀 무성할 봉**
		- 풀이 무성하면 예쁘니 **예쁠 봉**
		- 재물이 삼(三)대까지 이어질(丨) 정도로 풍성하니 **풍성할 풍**
- [[害]] 
	- 해할 해 → 방해할 해
		- 집(宀)에서 ==어지럽게==(丰) 말하며(口) 해치니 **해칠 해, 방해할 해**
- [[割]] 
	- 벨 할, 나눌 할
		- 해(害) 되는 것을 칼(刂)로 **베어 나누다**
- [[轄]] 
	- 다스릴 할
		- 차(車)가 다니는데 방해(害)되지 않도록 다스리니 **다스릴 할**
	- [[管轄]]
	    - 관할
	        - 권한(權限)에 의(依)하여 지배(支配)함.
	        - 행정, 사법, 군사 등에서 일정한 지역이나 사무를 맡아 다스림.
	        - [[管]] 대롱 관, 주관할 관, 轄 다스릴 할
	- [[直轄]]
	    - 직할
	        - 윗 기관이 직접 지배하거나 관리함.
	        - 중앙 정부가 직접 관할하는 지역이나 기관.
	        - 直 곧을 직, 轄 다스릴 할
	- [[總轄]]
	    - 총할
	        - 여러 가지 일을 한데 묶어 모두 관장함.
	        - 總 거느릴 총, 轄 다스릴 할
	- [[統轄]]
	    - 통할
	        - 전체를 한데 묶어 거느리며 다스림.
	        - 統 거느릴 통, 轄 다스릴 할
- [[憲]] 
	- 법 헌
		- 집(宀)이나 나라의 어지러운(丰) 일을 법망(罒)으로 다스리기 위해 마음(心)을 다해 만든 법이니 **법 헌**
		- 法網 법망 : 법의 그물로, 범죄자에 대한 제재를 물고기에 대한 그물로 비유하여 이르는 말
			- 法 법 법 網 그물 망
- [[契]] 
	- 맺을 계
		- 어지럽지(丰) 않도록 칼(刀)로 크게(大) 새겨 확실하게 맺으려고 애쓰니 **맺을 계, 애쓸 결**
		- 어지럽게(丰) 칼(刀)로 크게(大) 싸우는 부족이름이니 **부족이름 글**
		- 契丹 글단 : 거란 - 5세기 중엽부터 내몽골에서 살던 유목 민족
- [[喫]] 
	- 먹을 끽 → 마실 끽
		- 입(口)과 맺듯이(契) **마시거나 먹다**
			- 口(입 구) + 契(맺을 계)
	- [[喫煙]]
	    - 끽연
	        - 담배를 피우는 것.
	        - 흡연(吸煙)과 같은 뜻으로 사용됨.
	        - 喫 먹을 끽, 煙 연기 연
	        - 흡연(吸煙) - 吸 마실 흡, 煙 연기 연
	- [[滿喫]]
	    - 만끽
	        - 음식(飮食)을 마음껏 먹고 마시는 것.
	        - 어떤 것을 충분히 즐김.
	        - 滿 찰 만, 喫 먹을 끽
	- [[喫茶]]
	    - 끽다
	        - 차를 마심.
	        - 喫 먹을 끽, 茶 차 다
- [[潔]] 
	- 깨끗할 결
		- 물(氵)로 어지럽게(丰) 더러워진 칼(刀)과 실(糸)을 씻어 깨끗하니 **깨끗할 결**
