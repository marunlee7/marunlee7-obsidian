# 📙 원길2-245 유유욱 회회 포(보)포 有宥郁 灰恢 布怖

- [[有]] 
	- 있을 유, 가질 유
		- 많이(𠂇) 고기(月)을 가지고 있으니 **가질 유, 있을 유**
		- 𠂇 - 열 십, 많을 십(十) 변형
- [[宥]] 
	- 너그러울 유
		- (죄 지은 자를 방명하여) 집(宀)에 가 있게(有) 용서하니 **용서할 유**
		- 유의어 : [[寬]](너그러울 관)
	- [[宥恕]]
	    - 유서
	        - **寛大한 마음으로 죄나 잘못을 용서하는 것**, 또는 대범하게 눈감아 주는 행위.
	        - 상대방의 잘못이나 실수를 **크게 탓하지 않고 넓은 아량으로 받아들이는 태도**를 의미함.
	        - 예시: 피해자가 가해자를 유서(宥恕)했다.
	        - 宥 너그러울 유, 恕 용서할 서
- [[郁]] 
	- 성할 욱
		- 자원을 많이 가진(有) 고을(阝)처럼 번성하니 **번성할 욱**
	- [[郁郁]]
	    - 욱욱
	        - 향기가 짙게 풍기는 모양, 또는 기운이 가득 차 있는 모습.
	        - 분위기나 감정이 깊고 진하게 퍼져 있는 상태를 나타냄.
	        - 예시: 꽃밭에서 욱욱(郁郁)한 향기가 퍼졌다.
	        - 郁 향기 욱, 郁 향기 욱
	- [[馥郁]]
	    - 복욱
	        - 매우 짙고 그윽하게 풍기는 향기.
	        - 주로 꽃이나 술, 차 등에서 깊고 진한 향이 날 때 사용.
	        - 예시: 찻잔에서 복욱(馥郁)한 향이 피어올랐다.
	        - 馥 향기 복, 郁 향기 욱
	- [[郁郁靑靑]]
	    - 욱욱청청
	        - 나무나 풀이 무성하게 자라 짙은 푸르름이 가득한 모습.
	        - 생기가 넘치고 싱싱한 자연의 상태를 표현함.
	        - 예시: 산에는 욱욱청청(郁郁靑靑)한 숲이 펼쳐져 있었다.
	        - 郁 향기 욱, 郁 향기 욱, 靑 푸를 청, 靑 푸를 청
- [[灰]] 
	- 재 회
		- 많이(𠂇) 불(火)에 타고 남은 재니 **재 회**
- [[恢]] 
	- 넓을 회
		- 마음(忄)에 온갖 욕망이 재(灰)처럼 사그라지면 넓고 큰마음이 되니 **넓을 회, 클 회**
		- ==마음속의 온갖 사물을 구분하고 집착하고 그래서 고통스러워하는 번뇌의 불, 욕망의 불이 꺼진 상태를 불교에서 열반이라 하지요. ==
	- [[恢宏]]
	    - 회굉
	        - 마음이나 기상이 넓고 크며, 도량이 큼.
	        - 웅장하고 당당한 분위기나 태도를 나타냄.
	        - 예시: 회굉(恢宏)한 기상으로 어려움을 극복했다.
	        - 恢 넓을 회, 宏 클 굉
	- [[恢弘]]
	    - 회홍
	        - 도량이 넓고 크며, 포용력이 있음.
	        - 뜻이나 규모가 크고 넓은 상태를 의미함.
	        - 예시: 회홍(恢弘)한 안목으로 세상을 바라보다.
	        - 恢 넓을 회, 弘 클 홍
	- [[恢廓]]
	    - 회곽
	        - 매우 넓고 큼, 또는 그런 상태.
	        - 주로 공간이나 마음이 탁 트여 있음을 표현함.
	        - 예시: 회곽(恢廓)한 들판이 펼쳐져 있다.
	        - 恢 넓을 회, 廓 넓을 곽
	- [[恢恢]]
	    - 회회
	        - 넓고 넓음, 끝이 없이 큼.
	        - 주로 하늘이나 도량, 또는 어떤 범위가 한없이 넓을 때 사용.
	        - 예시: 회회(恢恢)한 하늘 아래 모든 것이 평등하다.
	        - 恢 넓을 회, 恢 넓을 회
- [[布]] 
	- 베/펼 포(:) | 보시 보
		- 많이(十) 수건(巾)처럼 베를 펴니 **베 포, 펼 포**
		- 불교에서 펴 베푸는 보시니 **보시 보**
			- 𠂇 - 열 십, 많을 십(十) 변형
- [[怖]] 
	- 두려워할 포
		- 마음(忄)을 여러모로 펴며(布) 두려워하니 **두려워할 포**
	- [[怖苦]]
		- 포고
			- 괴로움을 두려워함. 
			- 怖 두려워할 포, 苦 쓸 고
	- [[怖畏]]
		- 포외
			- 두려워하고 무서워함. 
			- 怖 두려워할 포, 畏 두려워할 외
	- [[怖慄]]
		- 포율
			- 몹시 두려워 떨림.
			- 怖 두려워할 포, 慄 떨릴 율
	- [[恐怖]]
		- 공포
			- 크게 두렵고 무서움. 
			- 恐 두려울 공, 怖 두려워할 포
