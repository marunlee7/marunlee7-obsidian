# 📙 원길2-143 4향 鄕響嚮饗

- [[鄕]]
	- 시골 향
		- 어린(乡) 시절 흰(白) 쌀밥을 숟가락(匕)으로 먹으며 살던 시골 고을(阝)이 고향이니 **시골 향, 고향 향**
		- `약` 郷 = 어린(乡) 시절 멈추어(艮) 살던 시골 고을(阝)이 고향이니 **시골향, 고향 향**
		- `유` 卿 벼슬 경
		- 乡 작을 요, 어릴 요(幺) 변형
		- 匕 비수 비, 숟가락 비
		- 먹을 것이 귀한던 옛날에 좋은 음식이었던 흰 쌀밥을 먹던 고을을 고향이라고 했네요. 이처럼 한자가 만들어지던 옛날을 생각해 봄도 한자 이해에 큰 도움이 됩니다.
- [[響]] 
	- 울릴 향
		- 시골(鄕)에서 소리(音)치면 산이 울리니 **울릴 향**
- [[嚮]]  
	- 길잡을 향 → 향할 향
		- 마음은 항상 고향(鄕)으로 향하니(向) **향할 향**
		- 向 향할 향
		- 객지에 나가면 마음은 항상 고향으로 향하지요.
	- [[嚮導]]
	- [[嚮導官]]
	- [[嚮往]]
- [[饗]] 
	- 잔치할 향
		- 고향(鄕)에서 음식(食)을 차려 놓고 잔치하거나 제사지내니 **잔치 향, 제사지낼 향**
		- 食 밥 식, 먹을 식, 밥 사
	- [[饗禮]]
	- [[饗宴]]
	- [[饗應]]
	- [[歆饗]]
