# 📙 원길2-368 효박상 爻駁爽

- [[爻]] 
	- 점쾌 효  → 사귈 효, 본받을 효, 수효 효
		- 육효가 서로 엇갈린 점괘를 본떠서 **점괘 효**
		- 서로 교차하여 사귀며 좋은 점을 볻받으니 **사귈 효, 본받을 효, 수효 효**
			- 六爻 육효 : 周易의 괘를 이루는 6개의 가로 그은 획.
			- 周易 : 유학 경전의 하나
- [[駁]] 
	- 논박할 박 ← 얼룩말 박
		- 말(馬) 중 점괘(爻) 같은 무늬가 있는 얼룩말이니 **얼룩말 박**
		- 얼룩말처럼 보통 말과 다른 점을 논박하니 **논박할 박**
	- [[論駁]]
	    - 논박
	        - 다른 사람의 주장에 반대하여 말함.
	        - 예시: 그는 조리 있게 상대방을 논박했다.
	        - 論 논할 논, 駁 논박할 박
	- [[駁馬]]
	    - 박마
	        - 얼룩이 있는 말.
	        - 예시: 박마가 들판을 달렸다.
	        - 駁 얼룩말 박, 馬 말 마
	- [[面駁]]
	    - 면박
	        - 얼굴 앞에서 꾸짖고 책망함.
	        - 예시: 상관에게 면박을 당했다.
	        - 面 얼굴 면, 駁 논박할 박
	- [[反駁]]
	    - 반박
	        - 상대의 주장이나 의견에 맞서 되받아 말함.
	        - 예시: 그는 재빨리 반박했다.
	        - 反 돌이킬 반, 駁 논박할 박
	- [[甲論乙駁]]
	    - 갑론을박
	        - 갑이 논하고 을이 반박한다 → 여러 사람이 서로 자기 주장만 내세우고 논쟁함.
	        - 예시: 갑론을박 끝에 결론에 이르지 못했다.
	        - 甲 첫째 갑, 論 논할 론, 乙 둘째 을, 駁 논박할 박
- [[爽]] 
	- 시원할 상
		- 마음 큰(大) 사람과 사귀고(爻) 사귄(爻)듯 시원시원하니 **시원할 상**
	- [[爽達]]
	    - 상달
	        - 막힘이 없이 시원하고 소통됨.
	        - 예시: 그는 성격이 상달하다.
	        - 爽 시원할 상, 達 통달할 달
	- [[爽明]]
	    - 상명
	        - 밝고 또렷함.
	        - 예시: 그의 논리는 상명하다.
	        - 爽 시원할 상, 明 밝을 명
	- [[爽秋]]
	    - 상추
	        - 하늘이 높고 날씨가 맑은 가을.
	        - 예시: 상추의 계절이 왔다.
	        - 爽 시원할 상, 秋 가을 추
	- [[爽快]]
	    - 상쾌
	        - 시원하고 쾌적함.
	        - 예시: 아침 공기가 상쾌하다.
	        - 爽 시원할 상, 快 즐거울 쾌
