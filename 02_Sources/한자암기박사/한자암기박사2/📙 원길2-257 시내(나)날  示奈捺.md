# 📙 원길2-257 시내(나)날 示奈捺

- [[示]] 
	- 보일 시
		- 하늘 땅(二)에 작은(小) 기미가 보이니 **보일 시**
		- 이렇게 기미를 보이는 신이니 **신 시**
			- `유` 矢 화살 시
			- 부수로 쓰일 때는 **보일 시, 신 시 변(礻)**
			- 옷 의(衣)가 부수로 쓰일 때의 모습인 **옷 의 변(衤)**
- [[奈]] 
	- 어찌 내
		- 자기 잘못이 커(大) 보이니(示) 어찌할까에서 **어찌 내・나**
- [[捺]] 
	- 누를 날
		- 손( 扌)으로 어찌(奈)할까를 생각하며 누르니 **누를 날**
		- [[奈]] - 자기 잘못이 커(大) 보이니(示) 어찌할까에서 **어찌 내・나**
	- [[捺印]]
	    - 날인
	        - 도장을 찍음.
	        - 문서 등에 공식적으로 도장을 찍어 효력을 발생시키는 행위.
	        - 직인(직장의 공식 도장)도 임의로 날인할 수 있음.
	        - 捺 누를 날, 印 도장 인
	- [[記名捺印]]
	    - 기명날인
	        - 자기 이름을 쓰고 도장을 찍는 것.
	        - 다른 사람이 이름을 대신 써줘도 무방하나, 도장은 반드시 날인해야 함.
	        - 예시: 계약서에는 기명날인이 필요하다.
	        - 記 기록할 기, 名 이름 명, 捺 누를 날, 印 도장 인
	- [[署名捺印]]
	    - 서명날인
	        - 자필로 이름을 쓰고 도장을 찍는 것.
	        - 반드시 본인이 직접 서명해야 하며, 대리로는 불가.
	        - 예시: 서명날인이 완료된 문서만 효력이 있다.
	        - 署 마을 서, 名 이름 명, 捺 누를 날, 印 도장 인
	- [[捺染]]
	    - 날염
	        - 천이나 직물에 무늬나 그림을 찍어내는 염색 방법.
	        - 예시: 날염 티셔츠는 다양한 디자인이 많다.
	        - 捺 누를 날, 染 물들일 염
