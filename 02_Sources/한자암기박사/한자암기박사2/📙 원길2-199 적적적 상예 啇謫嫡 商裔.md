# 📙 원길2-199 적적적 상예 啇謫嫡 商裔

- [[啇]]   
	- 밑동 적, 뿌리 적
		- 머리 부분(亠)을 받친(八) 성(冂) 모양으로 오래된(古) 밑동이나 뿌리니 **밑동 적, 뿌리 적**
		- `유` 商 장사 상
		- 밑동 : 나무줄기의 밑부분으로, 사물의 제일 중요한 부분을 말함.
- [[謫]] 
	- 귀양갈  적
		- 말(言)로 뿌리(啇), 즉 근본까지 들춰 가며 꾸짖고 귀양 보내니 **귀양 보낼 적**
		- 서로 삼가야 할 말도 있는데 뿌리까지 들춰 가며 꾸짖고 귀양 보낸다는 글자.
	- [[謫降]]
	    - 적강
	        - 죄를 지어 관직에서 강등되거나 먼 지방으로 좌천됨을 의미함.
	        - 謫 꾸짖을 적, 降 내릴 강
	- [[謫居]]
	    - 적거
	        - 죄를 지어 유배되어 머무르는 것, 또는 귀양살이.
	        - 謫 꾸짖을 적, 居 살 거
	- [[謫仙]]
	    - 적선
	        - 하늘에서 죄를 받아 인간 세상에 내려온 신선. 주로 시인 이백(李白)을 일컫는 별칭으로도 쓰임[1](http://jst.re.kr/upload/pdf/RC00013676.pdf).
	        - 謫 꾸짖을 적, 仙 신선 선
	- [[謫所]]
	    - 적소
	        - 유배형을 받은 사람이 머무르는 곳, 즉 귀양지나 유배지.
	        - 謫 꾸짖을 적, 所 바 소
- [[嫡]] 
	- 정실 적 → 본마누라 적
		- 여자(女) 중 밑동(啇)으로 삼는 본마누라니 **본마누라 적**
	- [[嫡嗣]]
	    - 적사
	        - 정실(正室)이 낳은 자식 중에서 대(代)를 이을 후계자, 즉 본처의 아들이자 가계를 잇는 후사(後嗣)
	        - 嫡 정실 적, 嗣 이을 사
	- [[嫡庶]]
	    - 적서
	        - 정실(嫡)과 첩(庶), 즉 본처의 자식과 첩의 자식을 아울러 이르는 말. 또는 그 구분.
	        - 嫡 정실 적, 庶 여러 서
	- [[嫡孫]]
	    - 적손
	        - 정실이 낳은 아들의 아들. 즉, 대를 잇는 본처 소생의 손자[1](https://m.cafe.daum.net/hahaca/Rhz5/945?listURI=%2Fhahaca%2FRhz5)[2](https://encykorea.aks.ac.kr/Article/E0025749).
	        - 嫡 정실 적, 孫 손자 손
	- [[嫡子]]
	    - 적자
	        - 정실이 낳은 아들. 가계와 제사를 잇는 데 우선권을 갖는 자식[1](https://m.cafe.daum.net/hahaca/Rhz5/945?listURI=%2Fhahaca%2FRhz5)[2](https://encykorea.aks.ac.kr/Article/E0025749).
	        - 嫡 정실 적, 子 아들 자
- [[商]] 
	- 장사 상 → 헤아릴 상
		- 머리(亠)에 물건을 이고(八) 다니며 성(冂) 안에서 사람(儿)이 말하며(口) 장사하니 **장사할 상**
		- 장사하듯 이익을 헤아리니 **헤아릴 상**
- [[裔]] 
	- 옷자락 예 → 후손 예
		- 옷(衣)에서 성(冂)처럼 사람(儿)을 둘러싼(囗) 옷자락이니 **옷자락 예**
		- 옷자락처럼 이어지는 후손이니 **후손 예**
	- [[裔孫]]
	    - 예손
	        - 먼 후손, 방계의 자손 또는 여러 대를 지난 후손을 의미함.
	        - 裔 후손 예, 孫 손자 손
	- [[裔胄]]
	    - 예주
	        - 먼 후손이나 후예, 또는 대를 이어 내려온 자손을 의미함. '胄'는 주로 맏손자, 장손의 의미로도 쓰임[1](https://solpee.tistory.com/15384733).
	        - 裔 후손 예, 胄 맏손자 주
	- [[後裔]]
	    - 후예
	        - 어떤 사람이나 집안, 민족의 먼 후손. 주로 여러 대를 지난 뒤의 자손을 의미함
	        - 後 뒤 후, 裔 후손 예
