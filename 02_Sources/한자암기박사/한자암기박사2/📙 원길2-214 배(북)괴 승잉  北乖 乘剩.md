# 📙 원길2-214 배(북)괴 승잉 北乖 乘剩
 :: 剩 제외하고 중복

- [[北]] 
	- 북녘 북 | 달아날 배 → 등질 배
		- 두 사람이 등지고 달아나는 모습에서 **등질 배, 달아날 배**
		- 항상 남쪽을 향하여 앉는 임금의 등진 북쪽이니 **북쪽 북**
		- `유` 比 견줄 비, 나란할 비
		- 임금은 어느 장소에서나 그곳의 북쪽에서 남쪽을 향하고 앉았답니다.
- [[乖]] 
	- 어그러질 괴
		- 많이(千) 등져(北) 어긋나니 **어긋날 괴**
		- 어긋나다 - 잘 맞물려 있는 물체가 틀어져서 맞지 아니하다.
- [[乘]] 
	- 탈 승 → 곱할 승
		- 두 발을 어긋나게(乖) 디디며 사람(人)이 타니 **탈 승**
		- 타는 수레는 세는 단위나 어긋나게 곱하는 뜻으로 쓰여 **대 승, 곱할 승**
		- [[乖]] - 많이(千) 등져(北) 어긋나니 **어긋날 괴**
		- 나무에 오르거나 차를 탈 때는 두 발을 어긋나게 디디며 타지요.
		- `약` 乗 - 많은(千) 풀(艹)이 땅(一)을 뚫고(八) 올라오듯 올라타니 **탈 승**
- [[剩]] 
	- 남을 잉
		- 다 타고(乘) 칼(刂)만 남으니 **남을 잉**
	- [[剩餘]]
		- 잉여
			- 다 쓰고 난 나머지.
				- 剩 남을 잉 餘 남을 여
	- [[剩員]]
		- 잉원
			- 남아돌아가는 인원(人員).
			- 나머지의 인원(人員).
				- 剩 남을 잉  員 인원 원
	- [[過剩]]
		- 과잉
			- 지나침
			- 예정(豫定)한 수량(數量)이나 필요(必要)한 수량(數量)보다 많음.
				- 過 지날 과  剩 남을 잉
