# 📙 원길2-168 중충관(천･곶)환 中沖串患

- [[中]] 
	- 가운데 중
		- 사물(口)의 가운데를 뚫어(丨) 맞히니 **가운데 중, 맞힐 중**
- [[沖]] 
	- 화(和)할 충 → 빌 충, 어릴 충
		- 물(氵) 가운데(中) 섞이면 화하니 **화할 충**
		- 또 물(氵)과 위에 언 얼음 가운데(中)처럼 비니 **빌 충** 
		- 또 아직 생각이 비게 덜 성숙하여 어리니 **어릴 충**
		- 약자 - 얼음(冫) 가운데(中) 끼이면 화해도 가운데가 비니 '화할 충, 빌 충, 또 아직 생각이 비게 덜 성숙하여 어리니 '어릴 층' 
		- 화(和)하다 - 무엇을 타거나 섞다. 날씨나 바람, 마음 따위가 온화하다.
		+ 화(和)하다 - 다른 상태가 되다. 여기서는 和, 化 뜻 둘 다 통함.
	- [[沖虛]]
		- 충허
		- 텅빔. 아무 거리낌이 없음.
		- 沖 빌 충 虛 빌 허
	- [[沖積]]
		- 충적
			- 흙과 모래가 흐르는 물에 운반(運搬)되어 쌓임.
			- 沖 빌 충 積 쌓을 적
	- [[沖年]]
		- 충년
			- 열 살 안팎의 어린 나이.
	- [[相沖]]
		- 상충
			- 어울리지 않고 서로 마주침.
			- 방위(方位)ㆍ일진(日辰)ㆍ시(時) 등(等)이 서로 맞질림.
			- 相 서로 상 沖 빌 충
- [[串]] 
	- 꿸 관 | 땅이름 곶
		- (고대에 화폐로 사용되었던) 조개를 꿰어 놓은 꿰미니 **꿸 관, 꿰미 천**
		- 또 무엇을 꿰어 놓은 꼬챙이처럼 바다 쪽으로 길게 뻗은 땅 이름이니 **땅 이름 곶**
		+ 곶 - 바다로 좁고 길게 뻗은 반도보다 작은 육지.
	+ [[串枾]]
		+ 관시
	+ [[長山串]]
		+ 장산곶
		+ 황해도 장연군의 반도 남쪽 끝 지역. 해수욕장이 있으며, 대청도(大靑島), 소청도, 백령도 등이 바라보인다.
	+ [[虎尾串]]
		+ 호미곶
- [[患]] 
	- 근심 환
		- 가운데(中) 가운데(中)의 마음(心)에 있는 근심이니 **근심 환**
		- 적당히 잊어버려야 하는데 잊지 못하고 가운데 가운데 있으면 근심이지요.
