# 📙 원길2-428 5부 박박박 尃傅賻簿敷 搏縛膊

- [[甫]] 
	- ==클 보 → 넓을 보==
		- 많이(十) 쓰이도록(用) 점(丶)까지 찍어가며 만들어 크고 넓으니 **클 보, 넓을 보**
			- 술보, 졸보, 울음보처럼 사람의 별명에 쓰이기도 합니다.
			- 尃 펼 부・포 - 넓게(甫) 법도(寸)에 맞게 펴니 **펼 부・포**
			- 專 오로지 전, 마음대로 할 전 = 叀(삼갈 전) + 寸(마디 촌)
- [[尃]]  
	- ==펼 부, 두루 알릴 부==
		- 널리(甫) 마디마디(寸) 펴 두루 알리니 **펼 부, 두루 알릴 부**
			- `유` [[專]] 오로지 전, 마음대로 할 전 
			- [[甫]] 클 보, 넓을 보
			- 寸 마디 촌, 법도 촌
- [[傅]] 
	- 스승 부
		- 사람(人) 중 두루 펴(尃) 가르치는 스승이니 **스승 부**
			- 유 : 傳 전할 전, 이야기 전
			- 참 : 師 스승 사, 전문가 사, 군사 사
- [[傅育]]
    - 부육
        - 가르치고 기름, 교육하고 양육함
        - 예시: 그는 자녀를 부육하는 데 힘썼다.
        - 傅 가르칠 부, 育 기를 육
- [[師傅]]
    - 사부
        - 스승, 기술이나 도를 가르치는 사람
        - 예시: 그는 훌륭한 사부를 만났다.
        - 師 스승 사, 傅 가르칠 부
- [[賻]] 
	- 부의 부
		- 상가에 돈(貝)을 펴(専) 주는 부의니 **부의 부**
		- 賻儀(부의) : 초상짖ㅂ에 부조로  내는 돈이나 물품. 또는 그 일. 
		- 儀 거동의, 법도 의
	- [[賜賻]]
		- 사부
			- 임금이 특명(特命)으로 신하(臣下)에게 부의(賻儀)를 내려 주던 일
	- [[致賻]]
		- 치부
			- 임금이 특명(特命)으로 신하(臣下)에게 부의(賻儀)를 내려 주던 일
	- [[賻儀金]]
		- 부의금
			- 부의(賻儀)로 보내는 돈.
		- [[弔意金]]
			- 조의금
				- 남의 죽음을 슬퍼하는 뜻으로 내는 돈.
				- 弔 조상할 조 意 뜻 의 金 쇠 금
- [[簿]] 
	- 문서 부 → 장부 부
		- 종이가 없던 옛날 대(竹) 조각을 물(氵)처럼 넓게 펴서(尃) 글을 적은 장부니 **장부 부**
		- 帳簿(장부)
- [[敷]] 
	- 펼 부
		- 크게(尃) 어떤 방향(方)으로 쳐서(攵) 펴고 베푸니 **펼 부, 베풀 부**
		- 약자 : 尃
	- [[敷設]]
		- 부설
			- (다리나 철도(鐵道) 또는 기뢰(機雷) 따위를)설치(設置)함. 깔아 놓음. 펴놓음.
	- [[敷衍]]･[[敷演]]
		- 부연
			- 이해(理解)하기 쉽도록 설명(說明)을 덧붙여 자세(仔細ㆍ子細)히 말함.
				- 敷 펼 부 [[衍]] 넓을 연 [[演]] 펼 연
	- [[敷地]]
		- 부지
			- 건축물(建築物)이나 도로(道路)에 쓰이는 땅.
			- 敷 펼 부 地 땅 지
	- [[高水敷地]]
		- 고수부지
			- 큰물이 날 대만 물에 잠기는 하천 언저리 터. 둔치. 강턱
			- 高 높을 고, 水 물 수, 敷 펼 부, 地 땅 지
- [[搏]] 
	- 두드릴 박 → 칠 박
		- 손( 扌)을 펴(尃) 치니 **칠 박**
	- [[搏動]]
	    - 박동
		    - 심장이나 혈관 등이 규칙적으로 뛰는 운동
		    - 예시: 심장의 박동이 정상이다.
		    - 搏 칠 박, 動 움직일 동
	- [[搏殺]]
	    - 박살
		    - 세차게 쳐서 부숨
		    - 예시: 적군을 박살냈다.
		    - 搏 칠 박, 殺 죽일 살
	- [[脈搏]]
	    - 맥박
		    - 심장에서 발생한 혈류의 규칙적인 파동
		    - 예시: 의사는 환자의 맥박을 재었다.
		    - 脈 맥 맥, 搏 칠 박
	- [[龍虎相搏]]
	    - 용호상박
		    - 용과 호랑이가 서로 싸운다는 뜻으로, 강력한 두 세력이 맞서 싸울 때를 비유
		    - 예시: 두 기업은 용호상박의 경쟁을 벌이고 있다.
		    - 龍 용 룡, 虎 범 호, 相 서로 상, 搏 칠 박
- [[縛]] 
	- 얽을 박 → 묶을 박
		- 실(糸)을 펴(尃) 묶으니 **묶을 박**
	- [[結縛]]
		- 결박
			- 몸이나 손 따위를 움직이지 못하도록 동이어 묶음.
			- 結 맺을 결 縛 묶을 박
	- [[束縛]]
		- 속박
			- 어떤 행위(行爲)나 권리(權利)의 행사(行使)를 자유(自由)로이 하지 못하도록 강압적(強壓的)으로 얽어매거나 제한(制限)함
			- 束 묶을 속 縛 묶을 박
	- [[捕縛]]
		- 포박
			- 잡아 묶음.
			- 捕 잡을 포 縛 묶을 박
	- [[自繩自縛]]
		- 자승자박
			- ‘자기(自己)의 줄로 자기(自己) 몸을 옭아 묶는다.’는 뜻으로,
			- 자기(自己)가 한 말과 행동(行動)에 자기(自己) 자신(自身)이 옭혀 곤란(困難)하게 됨을 비유적(比喩的)으로 이르는 말.
			- 自 스스로 자 繩 노끈 승 縛 묶을 박
- [[膊]] 
	- 팔뚝 박 → 포 박
		- 몸(月)에서 잘 펴(尃) 팔뚝이니 **팔뚝 박**
		- 고기(月)를 펴(尃) 말린 포니 **포 박**
		- 포 - 말린 고기
	- [[上膊]]
		- 상박
			- 위팔. 어깨에서 팔꿈치까지의 부분(部分).
			- 上 윗 상 膊 팔뚝 박
	- [[下膊]]
		- 하박
			- 아래팔. 팔꿈치부터 손목까지의 부분(部分).
			- 下 아래 하 膊 팔뚝 박
	- [[上膊筋]]
		- 상박근
			- 위팔에 있는 근육(筋肉)을 통틀어 이르는 말.
			- 上 윗 상 膊 팔뚝 박 筋 힘줄 근
