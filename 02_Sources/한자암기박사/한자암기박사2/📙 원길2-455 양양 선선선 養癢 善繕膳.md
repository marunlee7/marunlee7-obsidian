# 📙 원길2-455 양양 선선선 養癢 善繕膳

- [[養]] 
	- 기를 양
		- 양(羊)처럼 먹여(食) 기르니 **기를 양**
- [[癢]] 
	- 가려울 양
		- 병(疒)이 기른(養)것처럼 여기저기 퍼지며 가려우니 **가려울 양**
	- [[搔痒]]
	    - 소양
	        - 가려운 곳을 긁는다는 뜻이다.
	        - 긁다(搔)와 가렵다(痒)의 결합어로, 문자 그대로의 의미와 비유적으로 ‘일이 뜻대로 잘 풀림’을 의미하기도 한다.
	        - 예시: 일이 소양처럼 순조롭게 진행되었다.
	        - 搔 긁을 소, 痒 가려울 양
	- [[搔癢]]
	    - 소양 (동음이의어)
	        - 痒 대신 癢을 사용한 표기로, 동일한 뜻이다.
	        - 긁고 가려움을 의미한다.
	        - 搔 긁을 소, 癢 가려울 양
	- [[隔靴搔癢]]
	    - 격화소양
	        - 신을 신고 가려운 곳을 긁는다는 뜻으로, 일을 애쓰나 실제 문제 핵심을 찌르지 못하는 답답한 상황을 나타낸다.
	        - 隔 사이 뜰 격, 靴 가죽 신 화, 搔 긁을 소, 癢 가려울 양
	- [[隔靴搔瘍]]
	    - 격화소양
	        - 신발을 신고 발바닥의 종기를 긁는다는 뜻으로, 격화소양과 의미가 비슷하지만 ‘瘍(종기 앙)’을 사용해 더 깊은 답답함이나 문제의 본질 해결 실패를 뜻한다.
	        - 隔 사이 뜰 격, 靴 가죽 신 화, 搔 긁을 소, 瘍 종기 앙
- [[善]] 
	- 착할 선 → 좋을/잘할 선
		- 양(羊)처럼 풀(䒑)만 입(口)으로 먹는 짐승은 순하고 착하니 **착할 선**
		- 착하면 좋고 잘하니 **좋을 선, 잘할 선**
		- 초식동물은 대부분 순하고, 사람도 초식을 좋아하면 성질이 온순해진다지요.
- [[繕]] 
	- 기울 선
		- 실(糸)로 좋게(善) 기우니 **기울 선**
	- [[繕補]]
	    - 선보
	        - 구멍이나 닳은 곳을 꿰매거나 고쳐서 보완하는 행위이다.
	        - 예시: 옷을 선보하다.
	        - 繕 수선할 선, 補 기울 보
	- [[繕寫]]
	    - 선사
	        - 글을 깨끗이 다듬어 베껴 쓰거나 자세히 써 나가는 것을 뜻한다.
	        - 예시: 문서를 선사하다.
	        - 繕 수선할 선, 寫 베낄 사
	- [[修繕]]
	    - 수선
	        - 망가진 곳을 고쳐 다시 쓸 수 있게 하는 것을 뜻한다.
	        - 주로 건축물이나 기계 등을 수리하는 데 쓰인다.
	        - 예시: 집을 수선하다.
	        - 修 닦을 수, 繕 수선할 선
	- [[營繕]]
	    - 영선
	        - 경영하며 동시에 시설이나 건물을 관리하고 수선하는 것을 의미한다.
	        - 예시: 공공기관에서 시설의 영선을 담당한다.
	        - 營 경영할 영, 繕 수선할 선
- [[膳]] 
	- 선물/반찬 선
		- 고기(月)로 먹기 좋게(善) 만든 반찬이니 **반찬 선**
		- 반찬처럼 관계를 좋도록 해주는 선물도 뜻하여 **선물 선**
	- [[膳物]]
	    - 선물
	        - 남에게 주는 음식이나 물품을 뜻하며, 주로 진수성찬 같은 고급 음식이나 정성스러운 음식을 의미한다.
	        - 예시: 임금이 신하들에게 膳物을 하사하였다.
	        - 膳 음식 선, 物 물건 물
	- [[膳賜]]
	    - 선사
	        - 임금이나 윗사람이 신하들에게 음식을 하사하는 것을 뜻한다.
	        - 예시: 임금이 신하에게 膳賜를 내렸다.
	        - 膳 음식 선, 賜 줄 사
