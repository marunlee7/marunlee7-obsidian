# 📙 원길2-167 5형 形刑邢荊型

- [[形]] 
	- 모양 형
		- 우물(井)에 머리털(彡)이 비친 모습이니 **모습 형**
			- 彡 터럭 삼
			- 거울이 없었던 옛날에는 우물에 자기의 모습을 비추어 보기도 했지요.
- [[刑]] 
	- 형벌 형
		- 우물(井) 같은 형틀에 매어 칼(刂)로 집행하는 형벌이니 **형벌 형**
			- `유` 刊 새길 간, 책 펴낼 간
- [[邢]] 
	- 성(姓) 형 → 형나라 형
		- 우물틀(井)처럼 싸인 고을(阝)에 세운 형나라니 **형나라 형, 성씨 형**
		- 阝고을 읍 방
		- 邢나라 - 중국 주대에 있었던 나라
- [[荊]] 
	- 가시 형
		- 풀(艹) 중 형벌(形) 하듯이 찌르는 가시니 **가시 형**
	- [[荊冠]]
		- 형관
		- 그리스도(Kristos)가 십자가(十字架)에 못박힐 때 그리스도(Kristos)의 머리에 씌웠던 가시로 만든 관(冠). 가시관.
	- [[荊棘]]
		- 형극
	- [[荊路]]
		- 형로
			- 가시밭길
- [[型]] 
	- 모형 형 → 틀 형
		- 우물틀(井)처럼 칼(刂)로 흙(土)을 깍아서 만든 틀이나 모형이니 **틀 형, 모형 형**
		- 형(型) : 거푸집, 꼴, 다른 것들과 구별되는 특징을 이루는 유형이나 형태
		- 거푸집 : 만들려는 물건의 모양대로 속이 비어 있어 거기에 쇠붙이를 녹여 붓도록 되어 있는 틀
	- [[大型]]
		- 대형
	- [[模型]]
		- 모형
	- [[新型]]
		- 신형
