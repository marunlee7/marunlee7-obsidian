# 📙 원길2-241 설괄괄 舌括刮

- [[舌]] 
	- 혀 설
		- 혀가 입에서 나온 모습
		- 千 일천 천 이지만 여기서는 혀의 모습
- [[括]] 
	- 묶을 괄
		- 손( 扌)으로 혀(舌)처럼 휘어잡아 묶으니 **묶을 괄**
		- 혀는 부드럽게 휘두르며 입안의 음식을 이리저리 섞지요
	- [[括弧]]
	    - 괄호
	        - 말이나 글, 숫자, 수식 등을 한데 묶어 구별하거나 설명을 덧붙일 때 사용하는 문장 부호( ( ), { }, [ ] 등)
	        - ‘괄(括)’은 묶을 괄, ‘호(弧)’는 활 호(활처럼 휜 모양)로, 활처럼 휘어진 모양으로 묶는다는 뜻[3](https://hanja-bang.tistory.com/412).
	        - 소괄호( ), 중괄호 { }, 대괄호 [ ] 등이 있음
	- [[一括]]
	    - 일괄
	        - 한데 묶어 아우름, 여러 가지를 하나로 묶음
	        - 一(한 일), 括(묶을 괄)
	- [[總括]]
	    - 총괄
	        - 전체를 한데 묶어 거느림, 모두 합쳐서 통합적으로 다룸.
	        - 總(거느릴 총), 括(묶을 괄)
	- [[包括]]
	    - 포괄
	        - 있는 대로 온통 휩쓸어 쌈, 넓게 아우름
	        - 包(쌀 포), 括(묶을 괄)
	- [[槪括]]
	    - 개괄
	        - 대충 묶어 한데 뭉뚱그려 요약함, 전체를 대강 추려서 설명함[3](https://hanja-bang.tistory.com/412).
	        - 槪(대개 개), 括(묶을 괄)
- [[刮]] 
	- 긁을 괄, 씻을 괄
		- 혀(舌)로 입 속을 긁듯이 칼(刂)로 긁고 씻다
	- [[刮目]]
		- 괄목
			- 눈을 비비고 볼 정도(程度)로 매우 놀람.
	- [[刮目相對]]
		- 괄목상대
			- 눈을 비비고 상대편(相對便)을 본다.’는 뜻으로, 남의 학식(學識)이나 재주가 놀랄 만큼 부쩍 늚을 이르는 말.
				- 刮 긁을 괄  目 눈 목  相 서로 상  對 대할 대
