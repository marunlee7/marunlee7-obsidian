# 微吟緩步

- 微吟緩步
    - 미음완보

- [[微吟緩步]]	
	- 미음완보	
		- 작을/나직할 (微) + 읊을 (吟) + 느릴 (緩) + 걸음 (步). **작은 소리로 나직이 읊조리며 천천히 걸음.**	
		- (추가 설명 및 보충 내용) 주로 **자연의 풍경을 즐기며 유유자적하게 거니는 모습**을 나타냅니다. 바쁘게 서두르지 않고, 시(詩)나 노래를 나지막이 읊으며 자연과 교감하고 **내면을 성찰하는 한가롭고 풍류적인 삶의 태도**를 의미합니다. 정극인의 가사 《상춘곡(賞春曲)》에도 나오는 표현입니다.	
		- 유의어) **소요음영(逍遙吟詠)**: 이리저리 거닐며 시를 읊음.	
		- 예시) 뒷짐을 지고 정원을 미음완보(微吟緩步)하며 사색에 잠겼다. / 현대인에게는 잠시 멈춰 미음완보(微吟緩步)할 시간이 필요하다.	
		- 微 작을 미, 吟 읊을 음, 緩 느릴 완, 步 걸음 보
