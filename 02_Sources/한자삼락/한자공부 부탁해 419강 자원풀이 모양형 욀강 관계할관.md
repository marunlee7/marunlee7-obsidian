# 한자공부 부탁해 419강 자원풀이 모양형 욀강 관계할관

![](https://www.youtube.com/watch?v=l2QeKITsIMc)

- [[形]]
	- 모양 형
		- 자원 풀이: 우물 정(井) 자 모양의 틀을 만들어 붓으로 가로, 세로 획을 그린 형상이나 글꼴의 모양에서 유래했습니다.
	- [[形勢]]
		- 형세
			- 모양 (形) + 형세 (勢).
			- 겉으로 드러난 모양이나 기세.
			- 예시) 전쟁의 형세가 불리하다.
			- 形 모양 형, 勢 기세 세
			- [[形勢_형세_겉으로 드러난 모양이나 기세_예시 전쟁의 형세가 불리하다_形 모양 형 勢 기세 세]]
	- [[形式]]
		- 형식
			- 모양 (形) + 법식 (式).
			- 겉으로 드러나는 일정한 모양이나 틀.
			- 예시) 형식에 얽매이다.
			- 形 모양 형, 式 법 식
			- [[形式_형식_겉으로 드러나는 일정한 모양이나 틀_예시 형식에 얽매이다_形 모양 형 式 법 식]]
	- [[形態]]
		- 형태
			- 모양 (形) + 모양 (態).
			- 사물의 모양이나 모습.
			- 예시) 액체 상태에서 고체 형태로 변하다.
			- 形 모양 형, 態 모양 태
			- [[形態_형태_사물의 모양이나 모습_예시 액체 상태에서 고체 형태로 변하다_形 모양 형 態 모양 태]]
	- [[形容詞]]
		- 형용사
			- 모양 (形) + 얼굴 (容) + 말씀 (詞).
			- 사물의 성질이나 상태를 나타내는 품사.
			- 예시) '아름답다', '높다' 등이 형용사이다.
			- 形 모양 형, 容 얼굴 용, 詞 말씀 사
			- [[形容詞_형용사_사물의 성질이나 상태를 나타내는 품사_예시 아름답다 높다 등이 형용사이다_形 모양 형 容 얼굴 용 詞 말씀 사]]
- [[講]]
	- 욀 강
		- 자원 풀이: 나무를 우물 정자 모양으로 엇갈리게 쌓는 것(짤 구)과 말씀 언(言)이 결합된 글자입니다. 이야기를 여러 갈래로 나누어 말한다는 뜻에서 '강의하다', '강론하다', '외우다'의 의미가 나왔습니다.
	- [[講論]]
		- 강론
			- 욀 (講) + 논할 (論).
			- 진리나 학술을 강설(講說)하며 토론함.
			- 예시) 신부님이 강론하다.
			- 講 욀 강, 論 논할 론
			- [[講論_강론_진리나 학술을 강설하며 토론함_예시 신부님이 강론하다_講 욀 강 論 논할 론]]
	- [[講義]]
		- 강의
			- 욀 (講) + 의논할 (議).
			- 지식이나 기술을 가르침.
			- 예시) 대학교에서 특강(特講)을 진행하다.
			- 講 욀 강, 議 의논할 의
			- [[講義_강의_지식이나 기술을 가르침_예시 대학교에서 특강을 진행하다_講 욀 강 議 의논할 의]]
- [[關]]
	- 관계할 관 / 빗장 관
		- 자원 풀이: 문 문(門) 안에 꿰 관(𢃄) 자가 들어간 모양입니다. 문을 실로 꿰어 닫아 놓은 것처럼 빗장을 걸어 잠그거나, 빗장을 풀고 문 안에 들어가면 알게 되는 사람들 사이의 관계를 나타냅니다.
	- [[關係]]
		- 관계
			- 빗장 (關) + 맬 (係).
			- 둘 이상의 대상이 서로 연결되거나 얽혀 있음.
			- 예시) 두 사람은 오랜 친구 관계이다.
			- 關 관계할 관, 係 맬 계
			- [[關係_관계_둘 이상의 대상이 서로 연결되거나 얽혀 있음_예시 두 사람은 오랜 친구 관계이다_關 관계할 관 係 맬 계]]
	- [[關稅]]
		- 관세
			- 빗장 (關) + 세금 (稅).
			- 나라의 관문(關門)을 통과하는 물품에 부과하는 세금.
			- 예시) 수입품에 높은 관세가 부과되었다.
			- 關 빗장 관, 稅 세금 세
			- [[關稅]]
	- [[關門]]
		- 관문
			- 빗장 (關) + 문 (門).
			- 중요한 길목이나 통로.
			- 예시) 최종 목표에 이르기 위한 관문.
			- 關 빗장 관, 門 문 문
			- [[關門_관문_중요한 길목이나 통로_예시 최종 목표에 이르기 위한 관문_關 빗장 관 門 문 문]]
	- [[關聯]]
		- 관련
			- 빗장 (關) + 이을 (連).
			- 어떤 일이나 현상이 서로 이어져 있음.
			- 예시) 이번 사건과 관련된 모든 자료를 수집했다.
			- 關 관계할 관, 連 이을 련
			- [[關聯_관련_어떤 일이나 현상이 서로 이어져 있음_예시 이번 사건과 관련된 모든 자료를 수집했다_關 관계할 관 連 이을 련]]
	- [[關心]]
		- 관심
			- 빗장 (關) + 마음 (心).
			- 어떤 대상에 마음이 끌림.
			- 예시) 대중의 관심이 집중되다.
			- 關 관계할 관, 心 마음 심
			- [[關心_관심_어떤 대상에 마음이 끌림_예시 대중의 관심이 집중되다_關 관계할 관 心 마음 심]]
	- [[相關關係]]
		- 상관관계
			- 서로 (相) + 빗장 (關) + 맬 (係).
			- 둘 이상의 현상이나 사실이 서로 관련되어 있음.
			- 예시) 두 변수 간에 뚜렷한 상관관계가 나타났다.
			- 相 서로 상, 關 관계할 관, 係 맬 계
			- [[相關關係_상관관계_둘 이상의 현상이나 사실이 서로 관련되어 있음_예시 두 변수 간에 뚜렷한 상관관계가 나타났다_相 서로 상 關 관계할 관 係 맬 계]]
	- [[關節]]
		- 관절
			- 빗장 (關) + 마디 (節).
			- 뼈와 뼈가 연결되는 마디.
			- 예시) 무릎 관절에 통증이 있다.
			- 關 빗장 관, 節 마디 절
			- [[關節_관절_뼈와 뼈가 연결되는 마디_예시 무릎 관절에 통증이 있다_關 빗장 관 節 마디 절]]
