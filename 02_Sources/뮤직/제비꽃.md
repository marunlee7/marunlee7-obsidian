# 제비꽃

내가 처음 너를 만났을 때
너는 작은 소녀였고
머리엔 제비꽃 너는 웃으며 내게 말했지
==아주 멀리 새처럼 날으고 싶어==

​내가 다시 너를 만났을 때
너는 많이 야위었고
이마엔 땀방울 너는 웃으며 내게 말했지
아주 작은 일에도 눈물이 나와

​내가 마지막 너를 보았을 때
너는 아주 평화롭고
창 너머 먼 눈길 넌 웃으며 내게 말했지
==아주 한밤중에도 깨어있고 싶어==
