# Modal Form - Daily Note Journal Template (FREE)


![[Modal-Daily-Note-Template.zip]]



## Description

이 일일 노트 템플릿을 볼트에 적용하는 방법

동영상을 따라 이 모달 양식을 일일 노트 템플릿에 적용하세요

다음 단계를 안내합니다:

1. ZIP 파일 “Modal-Daily-Note-Template.zip”의 내용 추출
2. “Modal Form - Daily Note Journal Template.md”를 템플릿 디렉터리 폴더에 추가
3. 커뮤니티 플러그인 Dataview, Templater, Periodic Notes, Calendar, Meta Bind 및 Modal Forms 설치
4. 사용 중인 플러그인에 따라 ‘주기적 일일 노트 설정’ 또는 ‘일일 노트 설정’ 구성 (다른 플러그인 사용 시 동일 적용)
5. “Import-Daily-Modal-Form-Data.txt” 열기 > 전체 텍스트 선택 > JSON 데이터 복사
6. Obsidina에서 CTRL+P > 모달 폼: 폼 관리 선택
7. 폼 가져오기 선택, 복사한 JSON 데이터 붙여넣기 후 가져오기 버튼 선택
8. 선택 사항: 퀵추가 템플릿 선택으로 트리거 가능
- 파일 이름 형식 = {{DATE:DD-MM-YY}}
- 위치로 일일 저널 폴더 선택
- 생성 후 노트 열기

---

How to Apply this Daily Note Template to your Vault?

Follow along with the video to apply this Modal Form to your Daily Note Template

This will cover the steps below:

1. Extract the Contents of the ZIP File "Modal-Daily-Note-Template.zip"
2. Add the "Modal Form - Daily Note Journal Template.md" to your Template Directory Folder
3. Install Community Plugins Dataview, Templater, Periodic Notes, Calendar, Meta Bind and Modal Forms.
4. Set your Periodic Daily Note Setting or Daily Note Settings depending on what you're using. If you have another plugin that’s ok too.
5. Open "Import-Daily-Modal-Form-Data.txt" and Select All the Text then Copy that JSON data
6. In Obsidina, CTRL+P > Select Modal Forms: Manage Forms
7. Select Import Form, Paste the JSON data you copied, then select the import button
8. This is optional but you can also trigger this with QuickAdd Template Choice
- File Name Format = {{DATE:DD-MM-YY}}
- Choose your Daily Journal Folder as Location
- Open Note once created
