# CSS Snippet - Collapse YAML Properties by Default Hover to Show (FREE)

CSS 스니펫 - 기본 호버로 YAML 속성 축소 표시
CSS Snippet - Collapse YAML Properties by Default Hover to Show

![[hideproperties.css]]

![First Image](https://storage.ko-fi.com/cdn/useruploads/display/766c770d-ba7b-4b9b-8ec0-5e4f997b6f4a_hide-properties.png)

## Description

Installation:

1) Obsidian 설정 열기 Open Obsidian Settings
2) 외관 클릭 Click Appearance
3) CSS 스니펫 활성화 Turn on CSS Snippets
4) hideproperties.css 파일을 .obsidian/snippets 폴더에 추가 Add hideproperties.css to .obsidian/snippets folder
5)  Obsidian 설정 > 편집기 선택Select Obsidian Settings > Editor
6) 문서 속성 표시 설정 Set Properties in Document to Visible

RESULT
이제 속성 정보가 기본적으로 접혀 표시됩니다. 표시하려면 마크다운 문서의 텍스트 속성 위에 마우스를 올리세요.
Properties will now be collapsed by default, to show properties simply hover over the Text Properties in your Markdown

NOTE:
기존에 사용자 정의 스니펫이 있다면 이 텍스트를 기존 스니펫에 추가하기만 하면 됩니다.
If you already have a custom snippet then just add this text to the existing snippet.

PLUGIN OPTION:
속성 접기 기능을 제공하는 ‘기본적으로 속성 접기(Fold Properties by Default)’ 플러그인도 있습니다.
There is also a plugin now for folding properties called Fold Properties by Default
