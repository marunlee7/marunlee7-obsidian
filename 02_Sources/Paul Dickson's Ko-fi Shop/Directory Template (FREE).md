# Directory Template (FREE)
![First Image](https://storage.ko-fi.com/cdn/useruploads/display/b7e63fcc-d45d-4c98-a9b0-67d44c3bd07b_directory-ss01.png)

Description

Installation:

1) “Directory Template.md” 템플릿 노트를 템플릿 폴더 또는 툴킷에 추가하세요.
    Add "Directory Template.md" Template Note to your Templates Folder or Toolkit.
2) 디렉터리 템플릿을 엽니다.
   Open Directory Template
3) 동영상을 따라 Obsidian 볼트 내 폴더에 적용하세요.
   Follow along with the video to apply this to a folder inside your Obsidian Vault.

팁: 디렉터리를 더욱 향상시키려면 데이터뷰 쿼리에 YAML 속성을 추가하세요. 노트가 해당 YAML과 일치해야 올바르게 표시됩니다.
TIP: Add YAML Properties to your Dataview Query to further enhance your Directory. Your notes need to match the corresponding YAML to display correctly.
