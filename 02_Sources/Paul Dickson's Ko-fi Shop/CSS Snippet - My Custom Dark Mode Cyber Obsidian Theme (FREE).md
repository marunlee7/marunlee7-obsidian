A CSS Snippet to use in your Obdisian Vault with the Minimal Theme as your primary theme. This Snippet uses some elements of the Cybertron Theme but alongs more flexibility changing font colours and other elements within Obsidian without having to get your hands too dirty inside a code editor.

Minimal is an open references distraction-free theme for Obsidian, by @kepano. Minimal makes it easy to customize the appearance of Obsidian, to create your ideal writing interface

Documentation: minimal.guide
Github: github.com/kepano/obsidian-minimal

Installation:

1) Open Obsidian Settings
2) Click Appearance
3) Turn on CSS Snippets
4) Add cyber-theme.css to .obsidian/snippets folder
5) Under Appearance > Click Manager and install and use the Minimal Theme
6) Install Style Settings and Minimal Theme Settings Community Plugin.

TIPS:
1) Font Type; If you don't like DM Sans then simply remove it from the Snippet and use your own System Font via the Obsidian Interface or Text Settings.
2) Free Fonts can be found via Google Fonts
3) Color Palette Ideas; coolors.co
