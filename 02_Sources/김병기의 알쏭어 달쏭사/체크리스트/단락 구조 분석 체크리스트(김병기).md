# 단락 구조 분석 체크리스트

## 15가지 항목으로 글의 구조 파악

글쓰기 훈련 초기 단계에서는 15가지 항목에 집중하여
**글의 뼈대를 정확히 파악**하는 것이 가장 중요합니다.

칼럼이나 논설문 등을 읽을 때
각 단락의 기능적 역할을 체계적으로 파악하는 데 유용한 도구입니다.

이 체크리스트를 활용하여 
각 단락이 전체 글의 논리 구조 내에서 어떤 기여를 하는지 확인해 보세요.

읽고 있는 단락의 번호를 기록하고, 
해당 단락의 주요 기능에 해당하는 항목에 체크(✔) 표시를 해보세요. 
한 단락이 여러 역할을 동시에 수행할 수도 있습니다.

이 체크리스트를 활용하여 칼럼을 단락별로 분석해 보시면, 
글쓴이의 논리 전개 방식과 의도를 더 명확하게 이해할 수 있을 것입니다.

| **단락 기능 (역할)**                | **세부 역할 및 특징**                              | **체크 (✔)** |
| ----------------------------- | ------------------------------------------- | ---------- |
| **I. 화제 제시 / 흥미 유발 (주로 도입부)** |                                             |            |
| 1. **배경 제시**                  | 글을 쓰게 된 사회적 상황, 사건, 또는 일반적인 통념 제시.          |            |
| 2. **질문/문제 제기**               | 독자의 호기심을 자극하는 질문이나 해결해야 할 문제점을 던짐.          |            |
| 3. **정의/어원 설명**               | 글의 핵심 키워드나 개념을 정의하거나 어원을 추적하여 설명.           |            |
| 4. **주제 예고/제시**               | 앞으로 글에서 다룰 주제나 필자의 주장을 간략히 언급.              |            |
| **II. 논증 및 근거 제시 (주로 본론)**    |                                             |            |
| 5. **구체적 사례 제시**              | 주장을 뒷받침하는 실제 사건, 일화, 예시 등을 설명.              |            |
| 6. **통계/연구 결과 인용**            | 객관적 자료(데이터, 수치)를 제시하여 주장의 신뢰성 확보.           |            |
| 7. **원인 분석**                  | 제시된 문제나 현상의 발생 **이유**를 분석하고 설명.             |            |
| 8. **결과 예측/분석**               | 특정 상황이 지속될 경우 발생할 **영향이나 결과**를 분석.          |            |
| 9. **대조/비교**                  | 두 개념이나 사례를 **비교**하거나 **대조**하여 주장을 강화.       |            |
| 10. **반론 및 재반박**              | 필자의 주장에 대한 **반대 의견**을 언급하고, 다시 **논파(반박)**함. |            |
| 11. **개념 심화/확장**              | 서론에서 제시한 개념을 더 깊이 있게 설명하거나 다른 분야로 확장.       |            |
| **III. 마무리 및 촉구 (주로 결론부)**    |                                             |            |
| 12. **주장 재강조/요약**             | 본론에서 제시한 주장을 다시 한번 간결하게 정리하고 강조.            |            |
| 13. **해결책/대안 제시**             | 문제점에 대한 필자만의 최종적인 **해결 방안**이나 **대안**을 제시.   |            |
| 14. **독자에게 촉구**               | 독자의 행동이나 태도 변화를 요청하거나 희망적인 전망 제시.           |            |
| 15. **여운 남기기**                | 비유나 상징적인 표현으로 글을 마무리하여 독자에게 생각할 거리를 제공.     |            |
