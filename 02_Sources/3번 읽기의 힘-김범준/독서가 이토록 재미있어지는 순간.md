# 독서가 이토록 재미있어지는 순간

## 왜 책을 반복해서 읽어야 하는 것일까요?

- 반복하면 심플해지기 때문
	- 내 삶의 방향성을 잡아가는 과정
	- 반복독서에서의  심플(Simple)이란, 더 이상 뺄 수 없는 상태.
	- 하나의 단어, 하나의 문장을 찾아내는 것만으로도 좋다.
- 다음으로 다시 활용하기 좋은 상태입니다.
	- 독서에서의 심플은 독자만족이고, 알기 쉽다이다.
	- 요소의 양이 적다가 아니다
