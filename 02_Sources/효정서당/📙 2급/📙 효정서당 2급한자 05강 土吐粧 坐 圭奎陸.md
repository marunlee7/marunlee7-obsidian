# 효정서당 2급 5강 土吐粧 坐 圭奎陸

![](https://www.youtube.com/embed/8E0YxXqd_5c)

- 土(흙 토)
	- = 한 일은 기준점, 땅
	- = 땅에 씨앗이 들어가면 토해낸다
	- = 부수이면서 땅에 관련된 한자에 모두 들어감
- 吐(토할 토): 입에 들어간 것이 다시 나오다
	- 형성자
	- 부수는 口(입 구)
- 粧(단장할 장) = 米(쌀 미) + 庄(농막 장, 전장 장)
	- = 단장(丹粧)하다, 꾸미다, 화장하다		
		- 丹粧(단장) = 얼굴을 곱게 하고 머리나 옷맵시를 매만져 꾸밈
		- 化粧(화장) = 얼굴을 곱게 꾸밈
		- 治粧(치장) = 잘 매만져서 꾸밈
		- 粧飾(장식) = 겉을 매만져 꾸밈
		- 銀粧刀(은장도) = 노리개로 차던 칼자루와 칼집을 은으로 장식(粧飾)한 작은 칼
		- 庄(농막 장, 전장 장) =  广(집 엄) + 土(흙 토)
			- = 農幕(농막) : 농사짓는데 편리하도록 논밭 근처에 간단하게 지은 집
			- = 전장(田莊) : 귀척, 고관 등의 사유지
- 坐(앉을 좌)
	- = 
	- 座(자리 좌): 자리가 정해져 있다
		- 좌석(座席)
- 圭(영토 규, 홀 규, 쌍토 규)
	- = 흙 토가 2개 나옴
	- = 많은 영토를 다스리다 영토(領土) 규
	- = 홀 : 제후들에게 나눠준 표식, 기록
		- 笏(홀 홀, 희미할 홀) 
			- = 홀은 대나무에 기록했다
	- 佳(아름다울 가)
		- = 넓은 땅을 잘 다스리는 사람을 아름다운 사람
		- = 일반백성보다 희생을 잘 하는 사람
			- 絶世佳人(절세가인) 
				- = 세상에 비할 데 없이 아름다운 여자
- 奎(별 규, 문운 맡을 규)
	- = 文運(문운) 맡을 규
	- = 별(=벼리: 중요한 리더) 은 하늘의 기운이고 인간사회에서는 리더
	- = 문장의 운수를 관장		
		- 奎章閣(규장각)
	- 街(거리 가)
	- 桂(계수나무 계) = 木 + 圭
		- = 계수나무 계
		- = 형성자
			- 月桂冠(월계관) = 경기의 우승자에게 씌우고
			- 桂冠詩人(계관시인) = 
			- 桂皮(계피) = 계수나무 껍질
	- 封(봉할 봉)
		- = 많은 영토를 공로자에게 봉해주다
		- 寸(손 촌, 마디 촌, 법도 촌, 규칙 촌)
			- = 자기 영역의 법도
			- = 자기 어느정도 그릇을 만들었을 때 땅을 하사받아서 그것을 제후로 봉해주다
	- 涯(물가 애, 살 애): 완만한 언덕이 있는 물가
		- 生涯(생애) = 살아있는 한 평생 동안			
			- 厓(언덕 애): 흙이 완만하게 세워진 언덕
			- 岸(언덕 안/한): 낭떠러지 같이 흙이 가파른 곳
	- 掛(걸 괘): 점을 친 것을 손에 걸어 둠
		- 掛鐘時計(괘종시계) = 시간마다 울리는 시계		
	- 卦(점괘 괘): 거북이 등에 불을 지펴서 등이 갈라지는 모습을 보고 점을 침, 기후에 대해 점을 침	
- 陸(뭍 륙, 육지 육)
	- 흙덩이 륙이 언덕을 이룸
	- 圥(버섯 록)
		- 土 + 儿
		- 땅에서 식물(생명)이 나오면 버섯
		- 썩은 나무에서 나옴
	- 坴(흙덩이 륙)
		- 썩은 나무가 흙덩이가 되기 일보 직전
	- 埶(심을 예) 
		- 坴(언덕 륙(육)) +(잡을 집)
		- 흙덩이를 판 다음에 큰 나무를 옮겨 심다
		- 丸(알 환)
	- 勢(기세 세, 권세 세)
		- 어느 정도 자란 나무를 옮겨 심은 것이 힘이 세다
	- 熱(더울 열) 
		- 부수는 불 화
		- 熱心(열심) = 어떤 일에 온 정성을 다하여 골똘하게 힘씀
		- 寒心(한심)
			- 寒(찰 한)
	- 藝(재주 예)
		-  藝術家(예술가)
		- 技(재주 기)
		- 材(재주 재): 일단 나무을 심어 기초를 뿌리내어
		- 術(재주 술)
			- = 생계형
			- 商術(상술)
		- 材 - 術 - 藝
	- 陵(언덕 릉, 무덤 릉)
		- 坴(언덕 륙(육))
		- 夌(언덕 릉)
		- 江陵(강릉)
	- 曉(새벽 효, 깨달을 효) 
		- 垚 + 어진사람
		- 흙덩이가 쌓인 곳에 해가 떠오르면 자연의 모든 것이 드러나서 깨닫게 
		- 垚(높을 요): 土(흙 토)가 3개
		- 堯(요임금 요)
	- 燒(불사를 소)
