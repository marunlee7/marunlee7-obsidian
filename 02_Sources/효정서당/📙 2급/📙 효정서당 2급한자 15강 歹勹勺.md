# 효정서당 2급 15강 歹, 勹, 勺 의 유형별 한자

![](https://www.youtube.com/embed/aZUEbJ9lYuk)

## 歹 歺

- [[歹]] : [[歺]](뼈 앙상할 알, 살 바른 뼈 알, 죽을 사) 

- 骨(뼈 골) : 덮다 + 뼈 앙상할 알 + ⺼
- 死(죽을 사) 
	- 歹(뼈 앙상할 알) + 匕(변화할 비)
	- 뼈만 남은 사람
- 𣦼(후빌 찬) 
	- 歺(뼈 앙상할 알) + 又(오른손 우, 또 우)
	- 오른손으로 살을 후비다(발라내다)
- [[粲]](정미 찬, 깨끗할 찬, 빛날 찬)
	- 歹(뼈 앙상할 알) + 又(오른손 우, 또 우) + 米(쌀 미)
	- 정미(精米) : 기계 따위로 벼를 찧어 입쌀을 만듦
- [[燦]](빛날 찬) 
	- 火(불 화) + 粲(정미 찬)
	- 빛나다, 찬란(燦爛)하다
	- 이름에 찬(燦)가 들어가면 학문에 밝히는것을 좋아할 수 있다
	- 찬연(燦然) : 번쩍거리어 눈부시게 빛나는 모양(模樣)
- [[睿]](슬기 예) 
	- 歺(뼈 앙살할 알) + 谷(계곡 곡) + 目(눈 목)
	- 슬기
	- 슬기롭다, 총명(聰明)하다
- [[叡]]
	- 밝을 예 
		- 又(또 우) + 睿(슬기 예) 
		- 歺 뼈 앙살할 알 의 변형
		- 谷(곡 곡) : 먼산 가까운 산 연못
		- 깊이 통하여 명백한, 슬기롭다, 총명하다
		- 깊이 사물을 보는 눈의 모양에서 밝다, 슬기롭다
		- 계곡안에 있는 광물 등 모든 것을 속까지 정확하게 눈으로 안다 
	- 叡智=睿智(예지) 
		- 뛰어난 깊은 지혜(智慧, 知慧)
		- 지혜롭고 밝은 마음과 생각
		- 智(지혜 지, 슬기 지) : 日 + 知(알 지)
			- 화살(矢)이 순식간에 구멍(口)을 통과하듯이 말(曰)을 잘한다
			- 말을 잘하려면 지식이나 지혜가 있어야 할 것
			- 智 아는 것이 많아 말함에 거침이 없다 : 지혜
			- 知 후전적 지식(배워서 알게 됐다는 의미에서 알다)
			- 智 선천적 지혜(지식이 아닌 사람이 타고난 지혜)
- [[壑]](골 학) : 土(흙 토) + 㕡(골 학)
	- 손으로 살 바른 뼈를 버리는 계곡
- 列
	- 벌릴 열
		- 刂(선칼도방 도) + 歹(살 바른 뼈 알)
		- 벌이다, 늘어
		- 서다, 순서를 매기다, 진열하다
		- 칼로 살을 바른 뼈
		- 칼로 살과 뼈 사이를 벌이다(분리하다)
		- 유의어
		- 羅(벌일 라, 그물 라)
		- 陳(베풀 진, 묵을 진)
	- 行列(행렬) : 여럿이 벌이어 줄서서 감
	- 行列(항렬) : 친족 집단안에서 세대 관계를 나타내는 서열(序列), 돌림
	- 竝列(병렬) : 나란히 벌여 섬, 竝列連結(병렬연결)
	- 直列(직렬) : 직렬 접속
	- 羅列(나열) : 평면위에 죽 벌이어 놓는 것
	- 序列(서열) : 순서를 좇아 늘어섬
	- 系列(계열) : 서로 관련이 있거나 유사한 점에서 한 갈래로 이어지는 계통이나 조직
	- 列車(열차) : 기관차에 운전 장치를 설비한 차량(車輛)
	- 陣列(진열) : 물건 따위를 보이기 위해 죽 벌려놓음
	- 配列, 排列(배열) : 일정한 차례나 간격을 죽 벌여 놓음
- 烈
	- 세찰 열, 매울 열
	- 灬(연화발 화) + 列(벌일 렬(열))
		- 기세, 사납다, 포악하다, 맹렬하다
		- 사납다(불길이 세차게 타오른다는 뜻)
		- 아름답다, 빛나다(온몸을 바쳐 희생한다는 의미에서)	
	- 열사(烈士) : 조국과 민족을 위하여 성심껏 장렬(壯烈)하게 싸운 사람
	- 열녀(烈女) : 고난, 죽음을 무릅쓰고 절개를 지키어 남의 모범이 될 만한 여자
	- 치열(熾烈) : 세력이 불길같이 맹렬(猛烈)함
	- 격렬(激烈) : 지극히 맹렬(猛烈)함
	- 맹력(猛烈) : 기세가 몹시 사납고 세참
	- 열렬(熱烈) : 흥분하거나 열중하거나 하여 태도, 행동이 걷잡을 수 없이 세참
	- 장렬(壯烈) : 씩씩하고 열렬함
	- 剛烈(강렬) : 성질이 억세고 매서움
	- 先烈(선열) : 나라를 위하여 싸우다가 죽은 열사(烈士)
	- 痛烈(통렬) : 몹시 맵고 사나움
- 裂(찢어질 열, 찢을 열) : 衣(옷 의) + 列(벌일 렬(열))
	- 뼈를 벌리듯이 옷을 벌리다 → 찢어지다
	- 찢다, 쪼개다, 분할하다
	- 균열(龜裂)
		- 거북의 등에 있는 무늬처럼 갈려져서 터지는 것
		- 친(親)한 사이에 틈이 생기는 일
		- 龜(터질 균, 거북 구(귀))
			- 거북이, 거북 껍데이
			- 터지다, 갈라지다(거북의 갈라진 등 껍데기에서 착안해)
	- 分裂(분열) : 찢어져 갈라짐, 단체(집단)이 여러 파로 갈라짐
	- 核分裂(핵분열)
	- 破裂(파열) : 깨뜨리거나 갈라져 터짐
	- 分裂症(분열증) : 調絃病(조현병)의 前 용어(用語)
		
- 例(본보기 례(예), 법식 례(예)) : 亻(사람인변 인) + 列(벌일 렬(열))
	- 법식, 규칙(사람이 지켜야할 순서)
	- 列 늘어사다, 순서를 매기다( 뼈를 분리한 후에 부위별로 늘어놓는다)
	- 사람이 벌려선 것을, 이 사람들을 본보기 삼아서 차례로 서다
	- 次例(차례)
	- 事例(사례) : 일의 전례(前例)
	- 條例(조례) : 조목을 적은 규례
	- 慣例(관례) : 늘 해 내려오는 전례(前例)
	- 比例(비례) : 두 양의 비가 다른 두 양의 비와 일정하게 됨
	- 判例(판례) : 법원에서 소송 사건을 판결한 전례
	- 例示(예시) : 본보기를 들어서 보임
	- 例年(예년) : 평상시(平常時) 의 해
	- 實例(실례) : 실제의 예
	- 例外(예외) : 일반적인 규정(規定)이나 정례에서 특수(特殊)하게 벗어나는 일
	- 用例(용례) : 쓰고 있는 예(例), 용법(用法)의 보기
	- 例事(예사) : 보통(普通)으로 흔히 있는 일
	- 凡例(범례) : 책 머리에 책의 內容, 쓰는 方法따위에 관한 참고 사항 따위를 말해 놓은 글
	- 通例(통례) : 일반적으로 통하여 쓰는 전례(前例)
	- 特例(특례) : 특별(特別)한 예
- 葬(장사지낼 장) : 시체를 들에 들고 나가 풀속에 장사지내다
	- 艹(초두머리 초) + 死(죽을 사) + 廾(받들 공)
	- 𢍈(장사지낼 장) : 廾(받들 공) + 死(죽을 사)
	- 廾(받들 공) : 艹 변형된 것으로 이해해야 한다. 
	- 茻(우거질 망) :(잡풀이) 우거지다, 무성한 풀	
	- 埋葬(매장) : 송장을 땅에 묻음. 못된 짓을 한 사람을 사회(社會)에서 용납(容納)하지 못하게 함
	- 葬禮(장례) : 장사(葬事)지내는 예절(禮節)
	- 葬事(장사) : 시체(屍體)를 묻거나 화장(火葬)하는 일
	- 殉葬(순장) : 죽은 사람과 가까웠던 사람이나 동물을 딸려 함께 묻는 일
	- 火葬(화장) : 죽은 사람을 불에 살라 장사지냄
	- 葬地(장지) : 장사하여 시체를 매장(埋葬)하는 땅
	- 移葬(이장) : 무덤을 옮김, 옮기어 장사지냄

- 包( 쌀 포) : 勹(쌀 포) + 巳(뱀 사, 자식 사)
	- 싸다, 감싸다, 둘러싸다
	- 巳 태아의 팔과 다리를 생략해 그린 것
	- 아이를 배다(자궁과 태아를 함께 그린 것 )
	- 包 태아가 태보에 들어가 있는 모습
	
	- 內包(내포) : 어쩐 성질이나 뜻을 그 속에 지님
	- 包攝(포섭) : 상대를 허용하여 받아 들임
	- 包括(포괄) : 있는 대로 온통 휩쓸어 쌈
	- 包圍(포위) : 도망(逃亡)가지 못하도록 둘러쌈
	- 包袋(포대) : 피륙, 가죽, 종이 따위로 만든 자루
	- 小包(소포) : 자그마하게 포장(包裝)한 물건
	- 包裝(포장) : 물건(物件)을 싸서 꾸림
	- 包藏(포장) 
		- 물건을 겉으로 드러나지 않게 싸서 간직함
		- 어떤 생각을 마음속에 지니어 간직함
	- 包含(포함) 
		- 속에 싸이어 있음
		- 어떤 사물이나 현상 가운데 함께 들어 있거나 함께 넣음
	- 包涵(포함) : 널리 싸 넣음
	- 包容(포용) : 도량이 넓어서 남의 잘못을 이해하여 싸덮어 줌
	- 包容力(포용력) : 마음씨가 너그러워 남의 잘못을 허용하고 이해하여 감싸주는 힘


- 胞(태보 포, 세포 포) : ⺼(육달월 월) + 包(쌀 포)
	- 胎褓(태보  : 태아를 싸고 있는 막과 태반)
	- 세포(細胞), 胞子(포자)
	- 包 태아가 자궁(태보)에 들어가 있는 모습
	- 胞 둥글게 둘러싸인 신체기관
	- 한 뱃속에서 태어난 친형제
	
	- 細胞(세포 : 생물체를 구성하는 가장 기본적인 單位
	- 同胞(동포) 
		- 같은 어머니로부터 태어난 형제, 자매
		- 한 나라 또는 한 민족에 속(屬)하는 백성(百姓)
		- 같은 겨례
	
- 砲(대포 포, 돌 쇠뇌 포) : 石(돌 석) + 包(쌀 포)
	- 대포(大砲)
	- 돌 쇠뇌(여러 개의 화살이나 돌을 잇따라 쏘는 큰 활)
	
	- 砲手(포수) : 총으로 짐승을 잡는 사냥꾼, 대포를 쏘는 구인
	- 大砲(대포) : 커다란 탄환(彈丸)을 멀리 내 쏘는 큰 화기
	- 空砲(공포) : 헛총, 실탄을 재지 않는 총(銃)
	
- 飽(배 부를 포) : 𩙿(밥식 식) + 包(쌀 포)
	- 배부르다, 속이 꽉 차다
	
	- 飽食(포식) : 배부르게 먹음
	- 飽滿(포만) : 무엇이나 그 용량에 충분(充分)히 참
	- 不飽和(불포화) : 어떤 정도에 한껏 이르지 아니함
	- 飽和狀態(포화상태) : 더할 수 없는 양에 이른 상태(狀態)
	
- 袍(두루마기 포, 도포 포) : 衤(옷의변 의) + 包(쌀 포)
	- 도포(道袍  : 통상 예복으로 입던 남자의 겉옷)
	- 두루마기(외출할 때 입는 우리나라 고유의 웃옷)
	
- 泡(거품 포) : 

- 抱(잡을 포, 안을 포) : 扌(재방변 수) + 包(쌀 포)
	- 안다, 품다, 가지다, 손으로 감싸안다, 가슴에 품는다
	- 包 태아가 태보에 들어가 있는 모습, 생기가 있다
	
	- 抱擁(포옹)
	- 抱腹(포복) : 아주 우스워서 배를 안음.
	
	- 雍(누그러질 옹, 화할 옹, 도울 옹)
		- 화하다(마을 사람들이 높이 나는 새에게 기원하여 씨를 나르다 줌)
		- 乡(邑 : 阝우부방部 ) 고을 읍을 뒤집은 것
	
	- 擁(안을 옹) : 扌 + 亠(머리 두) + 乡(시골 향) + 隹(새 추)
		- 화하다(서로 뜻이 맞아 사이 좋은 상태가 되다), 화목하다
		- 서로 화해져 포옹하다
		
		
- 勺(구기 작) 
	- 작은 잔(盞), 구기(자루가 달린 술이나 물을 푸던 국자)
	
- 酌(술잔 작, 따를 작, 헤아릴 작) : 酉(술 유, 닭 유) + 勺(구기 작)
	- 따르다, 마시다(술병에 있는 술을 국자로 퍼내는 모습)
	- 짐작하다(술병에 술이 얼마나 남아있는지, 술을마시며 상대의 의중을 파악한다)
	
	- 酬酌(수작) 
		- 술을 서로 주고 받는 것, 말을 서로 주고 받음
		- 엉큼한 속셈이나 속보이는 짓
		- 이런 와중에 좋지 않은 일이 꾀해졌기에 수작을 부린다 말이 생김
		- 酬(갚을 수, 잔돌릴 수) : 酉(닭 유) + 州(고을 주)
			- 갚다, 보답하다
			- 잔을 돌리다, 권하다, 술을 받는 것		
	- 酌婦(작부) - 손님에게 술을 따라 주는 일을 업으로 삼는 여자
	- 斟酌(짐작) - 어림쳐서 헤아림
		- 斟(헤아릴 짐, 짐작할 짐, 술 따름 짐)
		- 甚(심할 심, 깊을 심) : 달 감(甘) + 짝 필(匹)
			- 짝과 음양을 이루어 경지에 이름
		
- 的(과녁 적, 맞칠 적, 밝을 적, 접미사 적) : 白(흰 백) + 勺(구기 작)
	- 과녁, 목표, 표준, 분명하다
	- 흰 판에 작은 잔같이 생긴 부분이 과녁 [꼬불]	
	- 的當(적당) : 꼭 들어맞음
	- 目的(목적) : 이루려 하는 일, 나아가려는 방향
	- 的中(적중) : 목표에 꼭 들어맞음, 과녁에 들어 맞음
	- 標的(표적) : 목표(目標)가 되는 물건
	
- 約(맺을 약, 약속할 약) : 糹(가는 실 멱) + 勺(구기 작)
	- 묶다, 약속하다, 맺다
	- 𠣩(동자)
	- 糹실타래를 묶어 놓은 모습	
	- 百年言約(백년언약)-남편과 아내가 되어 한평생같이 지내자는 아름다운 言約(언약)
	- 約束(약속) : 언약하여 정함
	- 契約(계약) : 사람과 사람 사이의 약속(約束)
	- 節約(절약) : 아끼어 씀
	- 豫約(예약) : 어떤 것을 확보하기 이하여 미리 약속함
	- 制約(제약) : 어떤 조건을 붙여 제한(制限)함
	- 公約(공약) : 공중 앞에서 약속함
	- 要約(요약) : 말이나 문장의 요점을 잡아 추림
	- 誓約(서약) : 맹세하고 약속함
	- 條約(조약) : 조목을 세워서 약정(約定)한 언약(言約)
