	- 豫備(예비) : 미리 갖춤
	- 豫言(예언) : 앞에 올 일을 미리 말함(= 預言)
	- 豫告(예고) : 미리 일러서 알게 함
	- 豫買(예매) : 시기가 되기 전에 미리 삼
	- 豫斷(예단) : 미리 내린 판단(判斷)
		- 判(판단할 판)
		- 斷(끊을 단)
		
- 預(미리 예, 맡길 예) 
	- 予(나 여) + 頁(머리 혈) 
	- 맡기다
	
	- 預金(예금) : 金錢(금전)을 金融機關에 맡김
	- 預託(예탁) : 부탁(付託)하여 맡겨 두는 것
	- 預置(예치) : 맡겨 둠

## 亥
- 亥(돼지 해)
	- 아기돼지 뼈
	- 五行 : 10월 水
	
- 亠(머리 두, 돼지해 머리 두) 
	- 亥(돼지 해)에 많이 쓰인다하여 돼지해 머리 두
	- 돼지와는 아무 상관없음

- 咳(기침 해) : 口 + 該
	- 아기돼지가 빽빽거리는 모습
	- 百日咳(백일해) : 어린아이의 기침
	- 咳嗽(해수)
	- 咳唾(해타)
	
- 該(갖출 해) : 言 + 亥
	- 該當(해당)
	- 該博(해박)
	

- 核(씨 핵) : 木 + 亥
	- 核爆彈(핵폭탄)
	- 核武器(핵무기)
	- 核心(핵심) 
	
- 骸(뼈 해) : 骨(뼈 골) + 亥(돼지 해)
- 劾(꾸짖을 핵, 힘쓸 해) : 力(힘 력(역)) + 亥(돼지 해)
	- 彈劾(탄핵) : 죄상을 조사하여 꾸짖음

- 豖(돼지발걸음 얽을 축, 못걸을 축) : 豕(돼지 시) + 丶(점 주)
	- 돼지 잡을 때 다리를 묶어놓음
	
- 琢(쫄 탁, 다듬을 탁)
	- 돼지가 살려고 바둥거리다가 땅이 반들반들해짐
		- 切磋琢磨(절차탁마) 
		- 切(끊을 절), 磋(갈 차), 琢(쫄 탁, 다듬을 탁), 磨(갈 마)
		- 옥돌을 자르고 줄로 쓸고 끌로 쪼고 갈아 빛을 내다
		- 학문이나 인격(人格)을 갈고 닦음
