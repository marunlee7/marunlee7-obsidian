# 효정서당 2급 39강 中, 央, 史, 夬 의 유형별 한자

![](https://www.youtube.com/embed/6avZNRt4NRo)

- 中(가운데 중, 마음 중, 맞칠 중)
	- 마음이 들어가 있음
	- 뚫을 곤(상하통야) = 하늘, 땅
	- 하늘 땅, 사람, 마음
	
	- 的中(적중)
	- 中庸(중용)
	
- 仲(버금 중, 다음 중, 두번째 중)
	- 사람중에서 두번째
	- 伯父(백부) = 큰 아버지
	- 仲媒(중매)
	- 仲介(중개)
	- 伯仲叔季
		- 叔(아재비 숙)
		- 季(계절 계, 막내 계)
	
- 忠(충성 충)
	- 마음의 중심을 잡다
	- 정성을 들이다
	- 충은 신과 같다
	
- 沖(화할 충) 
- 冲(화할 충)

- 患(근심 환)
	- 두 마음을 가지고 꿰다, 근심이 생기다
	- 串(꿸 관, 튀어날올 곳, 익힐 관)
	- 憂患(우환)
	- 患者(환자)
	
- 央(가운데 앙)
	- 멀 경 + 큰 대
	
- 殃(재앙 앙) = 뼈 앙상할 알
	- 가운데가 뼈 앙상해지다, 죽음으로 가다
	- 災殃(재앙)
	- 天殃(천앙)
	
	
- 映(비칠 영)
	- 해가 정 중앙에 와있다, 비춰주다
	- 映畵(영화)
	- 映像(영상)
	
- 英(꽃부리 영, 영재 영)
	- 英才(영재)
	
- 夬(깍지 결, 터질 쾌)
- 快(상쾌할 쾌, 쾌할 쾌)
	- 快感(쾌감)
	- 快樂(쾌락)

- 缺(이지러질 결)
	- 질그릇이 터졌다
	- 缺點(결점)
	- 缺席(결석) = 자리에 흠이 나다
	- 缺損(결손)
	
- 決(결단할 결) [4급]
	- 수문을 열것이냐, 닫을 것이냐 결정하다
	- 決定(결정)
	- 決心(결심)
	- 決斷(결단)
	
- 訣(이별할 결)
	- 訣別(결별)
		- 別(다를 별)
	- 永訣(영결)
	
- 史(사기 사, 역사 사) = 中 + 攴=攵
	- 역사를 기록하는 사람
	- 중심을 그대로 쓰내려가다, 다음 후손을 치게 위해서
	
	- 史記史(사기사 : 역사를 기록하는 사람)
	- 史記(사기)
	- 歷史(역사)

- 吏(아전 리, 벼슬아치 리) 
	- 기록하는 사람에게 말단 벼슬(一 : 관)을 주다 
	- 官吏(관리)
	- 貪官汚吏(탐관오리)
	
	
- 使(부릴 사, 하여금 사)
	- 왕의 사신으로 가다, 직분을 줘서 부리다
	- 天使(천사) = 하늘의 부름으로 이치를 알리는 사람
	- 大使館(대사관)
	- 使用(사용)
	

- 更(고칠 경, 시간 경, 다시 갱)
	- 丙(남녘 병, 밝을 병, 셋째 천간 병) + 攴=攵(칠복)
	- 분명하게, 밝게(丙) 매을 손에 들고 쳐서 분명한 쪽으로 향하게 하다
	- 새롭게하다, 고치다, 다시
	- 更新(갱신 : 경신)
	- 更生(갱생)
	
	- 初更(초경)(오후 07시~오후 09시)
	- 二更(이경)(오후 09시~오후 11시)
	- 三更(삼경)(오후 11시~오전 01시)
	- 四更(사경)(오전 01시~오전 03시)
	- ​五更(오경)(오전 03시~오전 05시)
	
- 硬(굳을 경)
	- 돌이 쌓이고 쌓여 굳다
	- 硬化(경화)
	- 硬直(경직)
	
- 便(편할 편, 남편 편, 소식 편, 오줌 변, 똥 변)
	- 고쳐 다시 쓸 수 있는 사람은 편하다
	- 男便(남편)
	- 便安(편안)
	- 便器(변기)
