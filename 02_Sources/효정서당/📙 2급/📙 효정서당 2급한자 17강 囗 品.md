# 효정서당 2급 17강 囗 , 品 의 유형별 한자

![](https://www.youtube.com/embed/KsBYbiveE2U)

## 口
- 口(입 구) =입, 먹다, 말하다
- 囗(에울 위, 에워쌀 위, 큰입구몸) = 에울 = 울타리 = 울 = 우리
	
口(입 구) = 입으로 사용될 때

- 只(다만 지)
	- 다만, 오직, 겨우
	- 입의 모양(口) + 말이 끝나고 숨이 분산(八)하는 모양
	- 語助辭(어조사)로 말의 끝에 씀
	- 음을 빌어 다만의 뜻으로 씀
	
	- 아래의 약자
		- 隻(새 한 마리 척, 외짝 척)=隹(새 추) + 又(또 우)
			- 배의 數爻(수효)를 세는 單位
			
		- 祗(다만 지, 공경할 지) = 示(보일 시) + 氐(근본 저)
		- 衹(가사 기, 땅귀신 기, 마침 지) = 衤(옷의변 의) + 氏(각시 씨)
			- 가사(袈裟 : 어깨에 걸쳐 입는 승려의 옷)
			- 기지(祇支: 비구니가 입는 속옷)
			
	- 旦(아침 단) 
		- 아침 : 지평선 위로 태양이 떠오르는 모습
		- 태양이 자신의 모습을 드러내는 순간을 표현한 글자
		- 元旦(원단)
		
	- 但(다만 단) = 亻(사람인변 인) + 旦(아침 단)
		- 태양이 떠오르듯 사람이 모습을 드러내다
		- 다만, 오직,(語助辭로서)
		
		- 但書(단서) 
			- 첫머리에 但자를 붙여 그 앞에 나옴
			- 본문의 설명이나 조건, 예외 등을 밝혀 나타내는 글
		- 但只(단지) = 다만, 겨우, 오직
	
- 枳(탱자나무 지)
	- 탱자탱자 놀자
	- 새콤한 맛, 약재로 사용
	- 南橘北枳(남귤북지)
		- 남쪽 땅의 귤나무를 북쪽에 옮겨 심으면 탱자나무로 변한다
		- 사람도 그 처해있는 곳에 따라 선하게도 되고 악하게도 됨
		- 江南橘化爲枳(강남귤화위지) 
	- 橘化爲枳(귤화위지)
		- 橘(귤 귤) = 木(나무 목) + 矞(송곳질할 율)
		- 矞 = 矛(창 모) + 冏(빛날 경)

口(입 구) = 모양을 사용될 때

- 品(물거 품)
	- 그릇이 세 개
	- 평가하다(음식을 한 입씩 먹으면 평가하다)
	- 등급(평가하면 품질이나 품성에 따라 等級(등급)이 정해지므로)
	- 평가하여 등급을 정할 수 있는 物件이면 뭐든 品이라 했다
	- 品評(품평)
	- 商品券(상품권)
	
- 喿(새 떼지어 울 소, 울 조)
	- 나무 위에 새들이 떼 지어 지저귀는 모습
	
- 操(잡을 조, 지조 조) = 扌(재방변 수) + 喿(울 조)
	- 잡다, 손에 쥐다, 조심하다
	- 손으로 새를 잡는다
	- 새들은 사람의 인기척에 쉽게 날아가곤 하니 잡을 때는 행동을 조심해야 한다
	
	- 내 마음을 다잡는다, 굳게 지키다, 지조
	- 깨끗이 가지는 몸과 굳게 잡은 마음
	
	- 조종(操縱) : 마음대로 다루어 움직임, 자유로이 다룸
	- 操心 : 실수가 없도록 마음을 삼가서 경계함
	- 조작(操作) : 기계 등을 움직이어 작업함
	- 지조(志操) : 곧은 뜻과 절조(節操)
	- 조조(曹操) : 중국 후한 말기의 무장, 字는 맹덕(孟德)
	- 체조(體操) : 
	- 정조(貞操) : 여자의 깨끗하고 곧을 절개(節槪)
	- 절조(節操) : 절개와 지조
	- 조업(操業) : 공장 등에서 기계 등을 움직이어 작업을 실시함
	- 조신(操身) : 태도나 행동이 침착하여 얌전함
	- 조타(操舵) : 배의 키를 조종함
	- 조롱(操弄) : 마음대로 다루면서 데리고 놈
	- 조종사(操縱士) : 비행기를 操縱하는 사람
	
- 懆(근심할 조)

- 燥(마를 조, 건조할 조) = 火(불 화) + 喿(울 조)
	- 마르다,(물기가)없어지다
	- 乾燥(건조) = 습기(濕氣)나 물기가 없음, 마름
	- 焦燥(초조) = 애를 태워서 마음을 졸이는 모양
	- 燥急(조급) = 초조(焦燥)하고 급함
	- 유의어
	- 乾(하늘 건/마를 건, 마를 간) 
	- 晞(마를 희) 
	- 枯(마를 고) 
	- 槁(마를 고) 
	- 煊(마를 훤)
	- 반의어
	- 濕(젖을 습, 사람 이름 섭, 나라 이름 탑)
	
- 藻(말 조, 해조류 조)
	- 미역, 토씨
	- 바다에서 나는 풀, 해조류 조
	- 海藻類(해조류) = 바다에서 나는 조류(藻類)
- 區(구분할 구, 나눌 구, 지경 구) = 匚(감출 혜) + 品(물건 품)
	- 구분하다, 나누다, 구역
	- 물건을 구분해서 넣어두다
	- 선반 위에 그릇을 올려놓은 모습
	- 區分(구분) = 따로따로 갈라 나눔
	- 區域(구역) = 일정한 기준에 의하여 갈라놓은 지역(地域)이나 범위(範圍)
		- 域(지경 역) = 土(흙 토) + 或(혹 혹)
			- 지경(地境 : 땅의 가장자리, 경계)
			- 영역, 구역, 나라
			- 或(혹시 혹) = 창을 들고 성을 지키는 모습으로 혹시
	- 區別(구별) = 구역 별, 종류에 따라 갈라 놓음
	- 地區(지구) = 일정하게 구획한 하나의 범위
	- 中區(중구) = 釜山廣域市 南部에 있는 區
	- 區廳(구청) = 인구 50만 이상의 시의 행정구역의 하나인 구의 행정 사무 관청
	- 匚(상자 방) = 외형(匣 : 갑 갑, 상자 갑)
		- 상자, 모진 그릇
		- 물건을 담는 바구니를 그린 것
		- 덮개가 없는 바구니를 그린 것으로 물건을 담을 수 있는 부분이 열려있는 형태
		- 匠(장인 장)
		- 匣(상자 갑)
	- 匸(감출 혜) = 내형, 상태(區 : 구역 구)
		- 감추다, 덮다
		- 匚(상자 방) 바구니를 세워서 그린 것
		- 匸(덮을 혜) 덮개가 있는 상자의 측면을 그린 것
		- 외부에 쉽게 노출되지 않도록 물건을 넣어놓는다
		- 구분없이 쓰이는 경우가 많고 부수로 쓰이는 글들조차 혜와 무관한 경우가 대부분

- 驅(몰 구, 말몰 구) = 馬(말 마) + 區(구분할 구)
	- 짐칸이 달린 수레를 말이 끄는 듯한 모습
	- 몰다, 빨리 달리다
	
	- 驅使(구사) = 말이나 수단, 수법 따위를 능숙하게 다루거나 부리어 사용하는 것
	- 驅逐(구축) = 몰아서 내쫓음
	- 驅迫(구박) = 못 견디게 몹시 굶, 학대함
		- 驅逐艦(구축함) = 해군 함선의 한 가지, 어뢰를 주무기로 하여 적의 주력함을 공격
	- 驅蟲劑(구충제) = 몸 안의 기생충을 없애는 데 쓰는 약제
	- 先驅者(선구자) = 어떤 일이나 사상에 있어 그 시대의 다른 사람보다 앞선 사람
	
- 毆(때릴 구) = 殳(몽둥이 수) + 區(구분할 구)
	- 잘 구분해서 때리다
	
	- 毆打(구타) = 사람을 때리고 침

- 嘔(토할 구, 게울 구) = 口(입 구) + 區(구분할 구)
	- 게우다, 토하다
	- 嘔吐(구토) = 위 속의 음식물을 토함, 게움
	- 嘔逆(구역) = 속이 메스꺼워 토하고 싶은 느낌
	
- 鷗(갈매기 구) = 鳥(새 조) + 區(구분할 구)
	- 狎鷗亭(압구정) = 정자 이름
		- 狎(익숙할 압) 
		- 鷗(갈매기 구)
		- 亭(정자 정)

囗(에울 위, 에워쌀 위, 큰입구몸) = 에울 = 울타리 = 울 = 우리

- 困(곤할 곤) = 囗 + 木
	- 생명(木)을 가둬두면 시름 시름 앓아 죽는다
	- 곤하다(困 : 기운 없이 나른하다)
	- 졸리다, 지치다, 괴롭다
	- 疲困(피곤) = 몸이나 마음이 지치어 고달픔
		- 疲(피곤할 피)
	- 貧困(빈곤) = 가난하고 궁색하여 살기 어려움
	- 困難(곤란) = 어떤 일을 하는 입장, 상황이 좋지 않아 어렵거나 까다로운 상태
	- 困境(곤경) = 어렵고 딱한 형편이나 처지
	- 困窮(곤궁) = 가난하여 살림이 구차함
	- 困惑(곤혹) = 곤란한 일을 당하여 어찌할 바를 모름
- 梱(문지방 곤)
	- 문턱
- 囷(곳집 균, 팡이실 균)
- 菌(세균 균, 버섯 균) 
- 因(인할 인)
