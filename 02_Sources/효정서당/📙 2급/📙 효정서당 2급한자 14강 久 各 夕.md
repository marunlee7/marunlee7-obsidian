## 久 [00:00:08](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=8.85)

	- 영구(永久)

- 灸(뜸 구) 
	- 아파 오래 누워있게되면 온돌로 따뜻하게 하다 → 뜸하듯
	- 火(불 화) + 久(오랠 구)
	- 침구(鍼灸) : 한방에서 침질과 뜸질을 아울러 이르는 말

	- 冬(겨울 동) : 사계절중 겨울이 가장 뒤에 오다

- 夊(천천히 걸을 쇠) [00:01:50](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=01:50.91)

	- 夏(여름 하, 기우제 하)

## 各
- 各(각각 각) [00:02:35](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=02:35.58) 
	- 口(입 구) + 夂(뒤쳐져 올 치)
	- 각각, 따로
	- 여럿이 따로 도착한다하여 각각, 따로, 제각기
		- 止(그칠 지) : 가다
		- 夂(뒤쳐져 올 치, 머뭇거림) : 내리다, 이르다
	- 각각(各各) : 제각기, 따로따로

	
- 閣(큰집 각, 누각 각) [00:02:46](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=02:46.89) 
	- 門(문 문) + 各(각각 각)
	- 閣 : 문을 열고 사람이 들어오는 모습
	- 各 : 어느 한 지점으로 발이 들어오는 모습
	
- 格(격식 격, 바를 격) [00:03:07](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=03:07.00) 
	- 木(나무 목) + 各(각각 각)
	- 木 생명
	- 格 가지치기 한 나무 → 바로잡다, 고치다
	- 格 가지치기한 나무처럼 잘 다들어진 사람의 인성, 인격
	- 격식(格式), 법식(法式)
	- 주위 환경이나 사절에 어울리는 분수나 품위
	
	- 격물치지(格物致知) : 바르게 알아가다
	- 격물(格物) : 사물의 이치를 연구(硏究)
	- 격식(格式) : 格에 어울리는 法式

	
- 絡(이을 락) [00:03:54](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=03:54.17) 
	- 실은 옷감을 짜기도 하지만 **이어주기도** 한다. 
	- 糹(가는 실 멱) + 各(각각 각)
	- 잇다, 얽다, 포괄하다
	- 각자가 서로 끈처럼 연결되어 있다
	- 연락(連絡) : 서로 관련을 지음, 사정을 알림
	- 맥락(脈絡) : 사물의 이어져 있는 연관

	
- 落(떨어질 락) [00:04:17](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=04:17.63) 
	- 艹(초두머리 초) + 洛(물방울 락, 물 이름 락(낙))
	- 떨어지다, 떨어뜨리다
	- 나뭇잎이나 비가 떨어지다
	- 낙엽(落葉) : 떨어진 나뭇잎
		
- 洛(물방울 락) [00:04:29](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=04:29.60) 
	- 氵(삼수변 수) + 各(각각 각)
	- 낙양지가(洛陽紙價)
		- 낙양의 종이 값
		- 저서(著書)가 많이 팔리는 것

- 略(대략 략, 줄일 략, 빼앗을 략) [00:04:52](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=04:52.13) 
	- 田(밭 전) + 各(각각 각)
	- 각자의 논밭을 구분한다
	- 노략질하다, 약탈하다
	- 간략하게 하다
	
	- 약자(略字) : 한문의 글자의 획수가 많은 것을 쉽게 줄여서 쓰는 글자
	- 생략(省略) : 덜어서 줄임, 뺌
	- 간략(簡略) : 단출하고 복잡하지 아니함
	- 침략(侵掠) : 

	
- 客(손님 객) [00:06:15](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=06:15.50)

	- 각처에서 집으로( 宀 : 집, 집안) 온 사람
	- 손님, 나그네

	- 빈객(賓客) : 손님

- 額(이마 액, 얼마 액, 금액 액, 액자 액) [00:06:53](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=06:53.15) 
	- 頁(머리 혈) + 客(손 객)
	- 이마, 머릿수, 수효
	- 손님이 들어오면 이마부터 봐서, 얼마 금액을 살지 본다 [효정]
	- 頟(이마 액) : 머릿수(각각의 머리)
	- 액자(額子) : 그림이나 사진 따위를 끼우는 틀
	- 금액(金額) : 금전의 액수, 돈의 수효(數爻)
	- 잔액(殘額) : 나머지 금액(金額)
	- 액수(額數) : 돈의 머릿 수
	- 액면(額面) : 액면가격

- 路(길 로) [00:07:41](https://www.youtube.com/watch?v=iz4v3kqHAVA#t=07:41.77) 
	- 𧾷(발족 족) + 各(각각 각)

	- 足 성(城)을 향해 진격하는 모습
	- 오고가다, 통행이 빈번한 길, 도로
	- 저마다 각각(各) 걸어다니는 곳, 길
	
	- 도로(道路) :
	- 진로(進路) : 앞으로 나아가는 길, 나아갈 길
	- 애로(隘路) : 좁고 험한 길 
	- 노선(路線) : 여러 갈래로 갈린 길, 갈림길
	- 경로(經路) : 지나가는 길
	- 미로(迷路) : 갈피를 잡을 수 없는 길
	- 가로수(街路樹) : 가로의 미관을 더하고 보건을 위하여 길거리 양쪽에 줄지어 심은 나무
	- 항로(航路) : 배가 다니는 길 
	
- 露(이슬 로, 드러날 로)
	- 雨(비 우) + 路(길 로(노))
	- 이슬, 진액, 좋은 술
	- 24절기
	
	- 탄로(綻露) : 비밀이 드러남
	- 폭로(暴露) : 남의 비밀, 비행 따위를 파헤쳐서 남들 앞에 드러내 놓은 일
	- 노숙(露宿) : 비바람 등을 가릴 수 없는 집 밖의 장소(場所)에서 잠을 자는 것 
	- 노숙자(露宿者) : 
	- 토로(吐露) : 속마음을 죄다 드러내어서 말함
	- 발로(發露) : 말이나 글이나 행동에 드러남
	- 노골(露骨) : 숨기지 않고 있는 그대로 드러냄
	- 결로(結露) : 이슬이 맺힘

- 鷺(백로 로, 해오라기 로)
	- 해오라기(왜가릿과 새)
	- 白鷺 왜가릿과 새를 통틀어 이르는 말

## 夕

- 夕(저녁 석)
	- 초승달을 보고 만든 글자
	- 夕 가운데 점이 없는 모습, 달빛이 구름에 가려진 모습, 저녁
	- 저녁, 밤
	- 추석(秋夕) : 명절
	- 석양(夕陽) : 저녁때의 저무는 해. 노년(老年)을 비유적으로 이르는 말
	- 조석(朝夕) : 아침과 저녁
	- 칠석(七夕) : 음력 7월 7일
	- 석하(夕霞) : 해질 무렵의 안개, 저녁노을
		- 霞(노을 하) : 雨(비 우) + 叚(빌릴 가)

- 多(많을 다)
	- 夕 + 夕
	- 많다, 낫다, 겹치다
	- 多 신에게 바치는 고기를 쌓은 모양으로 물건이 많다
	- 多 밤이 거듭 쌓여서  𣦼(잔), 目(목)과 谷(곡)
	
	- 다행(多幸) : 운수가 좋음
	- 다정(多情) : 정이 많음
	- 다양(多樣) : 여러 가지 모양(模樣), 양식
		- 樣(모양 양) : 木(나무 목) + 羕(강 길 양)
		- 羕(강이 길 양) 강줄기가 양 떼처럼 길게 늘어져 있는 것
	- 파다(頗多) : 아주 많음
		- 頗(자못 파) : 頁(머리 혈) + 皮(가죽 피)
			- 자못, 꽤, 매우
			- 비뚤어지다, 편파적이다(부정적)
			- 사람의 얼굴을 찡그리고 있다는 의미에서 부정적으로 쓰인다
			- 偏頗(편파) : 치우쳐 공평하지 못함
			- 偏頗的(편파적)

- 移(옮길 이) 
	- 禾(벼 화) + 多(많을 다)
	- **많이 옮겨 심다**
	
	- 이식(移植) : 식물 따위를 옮겨 식음
	- 이사(移徙) : 집을 옮김
		- 徙(옮길 사) : 彳(조금 걸을 척) + 歨(걸음 보)
	- 추이(推移) : 일이나 형편이 차차 옮아 가거가 변해 감
	- 이동(移動) : 움직여 옮김
	- 전이(轉移) : 위치 등을 다른 곳으로 옮기는 것, 이동(移動)
	- 이전(移轉) : 사물의 소재나 주소를 다른 곳으로 옮김
	- 이첩(移牒) : 다른 곳으로 다시 알리는 것, 그 공문이나 통첩
		- 牒(편지 첩) : 片(조각 편) + 枼(나뭇잎 엽)
		
- 夜(밤 야)
	- 夕 + 亦(또 역)
	- 밤, 저녁 무렵, 한반중
	- 주야(晝夜) : 낮과 밤
	- 심야(深夜) : 깊은 밤
	- 철야(徹夜) : 어떤 일을 하느라고 잠을 자지 않고 밤을 새우는 것
	
	
- 腋(겨드랑이 액) 
	- ⺼(육달월 월) + 夜(밤 야)
	- 액취(腋臭) : 암내

- 液(액 액, 진 액) 
	- 氵(삼수변 수) + 夜(밤 야)
	- 잠을 충분히 자면 성장 호르몬이 생성된다.
	- 액체(液體)  
	- 혈액(血液) 
	- 진액(津液)

- 夢(꿈 몽)
	- 풀로 눈을 덮은 듯 눈을 감고 저녁(夕)에 잠자는데 나타나는 꿈 [꼬불]
	- 몽상(夢想) : 꿈에서까지 생각함
	- 몽환(夢幻) : 꿈과 환상, 덧없음
	
	
- 名(이름 명) 
	- 口(입 구) + 夕(저녁 석)
	- 어두운 저녁(夕) 저 멀리 오는 누군가를(口) 식별하기 위해 이름을 불러본다. 	
	- 성명(姓名)
	- 명예(名譽) : 세상에서 인정받는 좋은 이름이나 자랑
	- 서명(署名) : 자기의 이름을 문서에 써넣음
	- 명함(名銜)
	- 익명(匿名)
	
- 銘(새길 명)
	- 금속판에 이름을 새기다
	- 기록하다, 새기다, 조각하다 → 오래도록 보존하고 남긴다는 뜻
	
	- 명심(銘心) : 잊지않게 마음에 깊이 새김
	- 감명(感銘) : 깊이 느끼어 마음속에 새기어둠
	- 좌우명(座右銘)
	- 각골명심(刻骨銘心) : 뼈에 새기고 마음에 새긴다. 
	- 명신불망(銘心不忘) : 마음에 새기어 오래오래 잊지 아니함
