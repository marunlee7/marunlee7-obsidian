## 右
- 右(오른 우, 오른손 우, 도울 우)
	- 손중에 입을 도와주고 주로 쓰는 것이 오른손이니 오른쪽.
	- 우왕좌왕(右往左往) : 이리저리 왔다 갔다 함, 갈팡질팡함.
	- 필순주의: 내리는 것부터 1번이다
	
- 佑(도울 우): 
	- 오른쪽에서 도와주는 사람
	- 천우신조(天佑神助) : 하늘이 돕고 신이 도움
	
- 祐(신이 도울 우, 복 우)
	- 礻(제사 시)
	- 우복(祐福) : 하늘이 주는 복
	- 우조(祐助) :
		- 袏(신이 도울 좌)
		
- 若(어릴 약, 젊을 약, 만약 약, 같을 약, 반야 야)
	- 풀이 오른쪽(右)에 있는 것이 약초 같다
	- 풀 초 + 오른 우
	- 동전의 양면처럼 오른쪽이 풀이 있으면 왼쪽에도 같은 풀이 있다.
	- 어릴때의 풀은 병아리처럼 이뻐보이고 다 같아 보이다.
	- 茻(풀숲 망): 풀이 숲을 이루다
	- 반야: 큰 우주의 영안으로 보면 우리 인간, 삼라만상이 똑같이 어리고 예쁘게 보이고 도와주고 싶은 측은지심이 생긴다.
	- 약간(若干) : 얼마되지 않음
	- 만약(萬若) :
	- [태연자약(泰然自若)] :
	
- 惹(이끌 야)
	- 心(마음 심) + 若(같을 약)
	- 야기(惹起) : 무슨 일이나 사건 따위를 끌어 일으킴
	- 야단(惹端) : 떠들썩하게 벌어진 일
		
- 諾(허락할 낙, 대답할 낙)
	- 행동과 말이 같으니(若:같을 약) 허락하여 승락한다.
	- 말씀 언 + 어릴 약
	- 어지간한 어린아이에게는 허락하게 된다 (자동차 사달라고하면 장남감은 사준다)
	- 수락(受諾) : 요구를 받아들여 승낙함.
	- 승낙(承諾) : 청하는 바를 들어줌
		
- 許(허락할 허)
	- 허락은 오후에 한다(午: 낮 오)
	- 아침에 부탁한 말(言)을 정오(午)가 되어 허락이 떨어졌다.
	- 허락(許諾) : 청하고 바라는 바를 들어줌
	- 허다(許多) : 몹시 많음
	
- 午(낮 오)
	- 열두시를 가리키는 시계 바늘 모양으로 정오의 낮을 뜻한다
	- 오수(午睡) : 낮잠
	- 오시(午時) : 십이 시이 일곱째, 상오 11시부터 하오 1시의 사이
		
- 年(해 년, 나이 년)

	
- 有(있을 유)
	- 오른 손에 고기(月)을 잡고 있어유
	- 필순주의: 내리는 것부터 1번이다
	- 유구무언(有口無言) : 입이 있어도 할 말이 없음.
	- 유식(有識) : 학식이 있음, 아는 것이 많음
		
- 友(벗 우, 도울 우)
	- 왼 손에 또(又: 오른손 우) 손을 잡고 악수하는 벗
	- 우애(友愛) : 형제간의 정의
	- 우방(友邦) : 가까이 사귀는 나라
	- 우정(友情) : 친구 간의 정의
	- 붕우유신(朋友有信) : 친구 사이의 도리는 믿음에 있다
		- 朋(벗 붕): 스승이 있는 벗
		- 友(벗 우): 우리끼리 마음이 통해 사귀는 벗
	
- 左(왼 좌)
	- 손수에 오른손이 일(工)할 때 도와주는 왼쪽 손
	- 왼손은 공구(자)를 잡는다.
	- 왼 좌는 가로부터 쓴다
	- 左傾(좌경): 왼쪽으로 기울어짐
	- 左思右想(좌사우상): 여러가지로 사색(思索)함
	
- 佐(도울 좌)
	- 사람(亻)의 왼쪽(左)에 서서 도와주는 보좌관
	- 佐命(좌명): 임금을 도와줌.
	- 補佐(보좌)
		
## 布
- 布(펼 포, 베 포)
	- 손에 걸치고 있는 수건(巾)같은 베
	- 왼 손으로 천을 잘 펴다, 펼 포, 베 포
	- 布告(포고): 국가의 결정적 의사를 공식으로 일반에게 알림.
	- 布施(보시): 깨끗한 마음으로 재물을 사람에게 베풀어 줌
	
- 怖(두려워할 포)
	- 恐怖(공포)
	- 怖伏(포복)
	
- 希(바랄 휘)
	- 찢어진 베옷(巾)을 버리고 새옷 입기를 바란다.
	- 천에다 무늬를 내다, 구체적인 나의 바램을
	- 천에다 기록하다. 목표 지점에 다다를 확률이 높아진다
	- 생각, 기록, 말로 확인하면은 그대로 될 확률이 높아진다.
	- 希望(희망): 앞일에 대하여 기대를 가지고 바람.
	- 希願(희원):
	- 望(바랄 망)
		- 亡(망할 망)+月(달 월)+壬(곧을 정, 바를 정)
		- 콩심으면 콩 나오고 팥심으면 팥나온다
		- 이것저것 많이 하는 것 보다 목표를 정하고 목표와 관련된 것들을 한다.
		- 다른 것은 잊고(亡), 한가지는 잊지 않고
		- 달님(月)은 음=마음이나 마찬가지, 마음속에 항상 품고
		- 壬(곧을 정): 사람이 바로 서다, 바로 자라다

- 稀(드물 희)
	- 벼(禾: 벼 화)농사가 풍년이 되기를 바라(希: 바랄 희)지만 그런 때(풍년)가 드물다.
	- 稀貴(희귀): 드물어서 매우 진귀함
	- 稀少(희소): 드물어서 얼마 안 되고 적음
	- 稀少價値(희소가치)
	- 古稀(고희): 일흔 살까지 산다는 것은 옛날에는 드문 일이다 라는 뜻, 두보의 시에서
	
- 타(찢어질 타)=左+⺼
	- 고기의 왼쪽이 너덜너덜해지다
	
- 隋(수나라 수)
	- 언덕(阝) 한쪽이 허물어질려고 하다
	
- 墮(떨어질 타)
	- 언덕(阝:언덕 부) 왼쪽(左)에 붙은 몸통(⺼)같은 흙(土)이 무너져 떨어진다.
	- 언덕(阝) 한쪽이 허물어질려고 하다 + 土
	- 墮落(타락): 
		- 우리 몸이 한쪽이 헤어지듯, 무너지듯이 타락의 길로 가다
		- 잡되게 놀아 못된 구렁에 빠짐.
	- 惰(게으를 타)
	
- 隨(따를 수)
	- 언덕(阝:언덕 부)처럼 높은 사람을 왼쪽(左)에서 몸(⺼)을 굽혀 뛰어가(辶: 쉬엄쉬엄 갈 착: 먼 길)듯이 따라 다닌다.
	- 수반(隨伴) : 데리고 감.
	- 수필(隨筆) : 일정한 형식에 따르지 않고 붓 가는 대로 쓴 글.
	- 부창부수(夫唱婦隨) : 
		- 남편이 노래하면 아내가 따라하다
		- 가정에서의 부부 화합의 도리를 이르는 말

## 反
- 反(돌이킬 반, 되돌릴 반, 돌아올 반, 반대 반, 엎을 반)
	- 바위(厂: 바위 엄)를 또(又) 손으로 밀어 반대로 엎는다.
	- 물건을 오른손으로 돌이킨다
	- 반격(反擊) : 쳐들어 오는 적을 되잡아 공격함.
	- 찬반(贊反) : 찬성과 반대
		
- 返(돌이킬 반, 돌아올 반)
	- 가던(辶)길의 방향을 바꿔 반대(反)로 되돌아온다.
	- 갔던 길을 되돌아 오다
	- 辶(쉬엄쉬엄 갈 착) + 反(돌이킬 반)
	- 반납(返納) : 남에게서 꾸거나 빌린 돈, 물건을 도로 줌
	- 반환(返還) : 도로 돌려 줌
		
- 叛(배반할 반)
	- 반(半: 절반 반)의 인원이 본뜻에 반대(反: 반대 반)하여 배반했다
	- 叛逆(반역): 정부를 배반하고 나라를 어지럽게 함
	
- 半(반 반, 절반 반)
	- 사람이 둘로 나누어졌으니 절반이다.
	- 半月(반월): 반달
	- 半折(반절): 반으로 나눔

- 伴(짝 반)
	- 사람이 절반씩 합쳐졌으니 짝이 된 것이다.
	- 반려(伴侶) : 짝이 되는 친구
	- 반주(伴奏) : 노래나 기악을 연주할 때, 이를 돕기 위해 다른 악기로 보조하는 연구.

- 判(판단할 판)
	- 절반을 칼로 쪼개듯 갈라서 판단을 내린다.
	- 判斷(판단): 사물에 대해 인식한 결과를 놓고 그것인가 아닌가 혹은 옳은가 그른가를 단정함.

- 飯(밥 반)
	- 밥(食)이 뜨거워 엎었다(反) 뒤집었다하며 먹는 밥
	- 밥은 숟가락으로 퍼서 갔던 숟가락이 다시 돌아오다
	- 반주(飯酒) : 밥에 곁들어서 먹는 술
	- 반찬(飯饌) : 밥에 곁들여 먹는 여러 가지 음식
		
- 板(널조각 판)
	- 반 → 판
	- 나무를 톱으로 켜면 반대(反)쪽으로 벌어지며 생기는 널조각
	- 판자(板子) : 널판지
	- 판서(板書) : 칠판에 분필로 글을 씀
	- 간판(看板) :
	- 칠판(漆板) :
	
- 販(팔 판)
	- 장사꾼은 물건을 사서 돈(貝)벌이가 되면 반대(反)로 판다.
	- 판로(販路) : 상품이 팔리는 방면
	- 판매(販賣) : 상품을 팜
	- 판촉(販促) :
	
- 版(판목 판, 널조각 판, 인쇄할 판)
	- 조각(片: 조각 편)의 반패(反)쪽에 인쇄한다.
	- 片(조각 편)
	- 爿(조각널 장)
	- 판권(版權) : 일정한 출판물에 대하여 그 출판에 관계되는 권리
	- 판화(版畵) :
	- 목판(木版) :
	- 출판(出版) :
		
## 才
- 才(재주 재)
	- 手(손 수)
	- 扌(손수 변): 위로 치켜 올려 쓴다
	- 才(재주 재): 아래로 내려 쓴다
	- 나무 뿌리를 조금 내린 모양, 앞으로 큰 나무가 될 가능성이 높은 나무는 재주가 있다
	- 才의 부수는 手 
	- 손에는 선적적인 재주를 가지고 태어난다
	- 才色(재색): 여자의 재주와 얼굴모양
	
- 材(재목 재)
	- 나무를 재주껏 잘 기르면 재목이 된다.
	- 재료(材料) : 물건을 만드는 감, 일할거리
	- 재질(材質) :
	- 재목(材木) : 건축이나 기구를 만드는 데 재료가 되는 나무
	- 동량(棟梁) : 마룻대와 들보; 기둥이 될 만한 인물
		
- 財(재물 재)
	- 돈버는 재주가 있으면 불어나는 것은 재물
	- 약간의 뿌리를 내릴려면 재물을 모아야 한다
	- 재벌(財閥) : 금융 및 경제계에서 큰 세력을 가진 자본가의 무리
	- 재화(財貨) : 돈 또는 보배
	- 재물(財物) :
	
- 在(있을 재)
	- 재주 재가 약간 변형된 왼쪽의 모습 + 땅이 있어야 뿌리를 내릴 수 있다
	- 뿌리를 내릴 수 있는 흙이 있다
	- 有(있을 유)
	- 한 사람에게 살 흙이 있다
	- 재경(在京) : 서울에 머물러 있음.
	- 재야(在野) : 벼슬하지 아니하고 민간에 파묻혀 있음.
	- 재학(在學) 
	
- 存(있을 존)
	- 왼쪽이 두 손(廾:받들 공) + 子
	- 아가들은 어릴 때에 두손으로 보살핌을 받는다
	- 한 사람에게 아들이 있다
	- 존재(存在)의 이유는 부모님의 두 손이 있었기 때문이고, 땅(이 지구촌)에 왔기 때문에 있는 것이다.  
	- 존립(存立) : 생존하여 자립함
	- 존폐(存廢) : 보존과 폐지
