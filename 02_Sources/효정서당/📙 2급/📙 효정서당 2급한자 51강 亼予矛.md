# 51강 亼,予,矛 의 유형별 한자 (모을 집, 나 여, 창 모)

![](https://www.youtube.com/embed/sB85ExzPxhM)


- 合(합할 합)
	- 입구는 모양
	- 그릇과 두껑이 하나의 세트가 되다
	- 남자가 있으면 여자가 있듯이 서로 플러스와 마이너스가 합을 이루어주는 거예요. 
	- 남녀로 비유하면 이제 궁합이라고도 하거든요.
	- 당신이 그릇이면 나는 뚜껑이, 당신이 뚜껑이면은 나는 그릇이 되주면은 좋은 궁합
	- 그래서 합할 합
	- 宮合(궁합): 맞춰서 살아가는 것
		
- 拾(주울 습, 열 십, 바꿀 겁, 오를 섭)
	- 손을 다 합하다 → 열 십(十: 추상적, 음과 양이 합해졌다. 지사자)
		- 갖은자 열 십, 갖춘자 열 십, 열 십 → 중요한 숫자에 씀, 
		- 拾 = 十
	- 주울 습
		- 拾得(습득)
		- 收拾(수습)

- 給(줄 급, 공급할 급)
	- 실타래처럼 계속 주다
	- 습 → 급
	- 給食(급식)
	- 月給(월급)
	- 供給(공급)
	- 給油(급유)
	
- 答(대답할 답)
	- 얼굴처럼 생각해서 그리기
	- 竹: 글에 관련된 글자에 나옴
	- 문제를 쓰면 답을 쓰다
	- 그릇, 뚜껑으로 형태 기억
	
- 塔(탑 탑)
	- 준2급
	- 흙 토 + 풀 초 + 合
	- 종콩을 흙으로 탑을 쌓다
	- 荅(종콩 답): 
		- 메주콩=종콩
	- 한자는 상형이기때문에 전부 자연을 보고 디자인 한 것이라는 것을 기억할 것
	- 石塔(석탑)
	- 鐵塔(철탑)
	
- 搭(탈 탑)
	- 손을 잘잡고 탑승하다
	- 搭乘(탑승)
	- 搭乘客(탑승객)
	
- 令(명령 령, 하여금 령)
	- 亼(모일 집): 동양사상 천지인, 모였다해서 모일 집
	- 모이면 리더가 있기 마련, 대의를 위해서 소견을 화합하는데 양보하다
	- 리더의 명령을 따르다
	
- 領(거느릴 령, 옷깃 령)
	- 머리 혈+ 명령 령
	- 大統領(대통령)
	- 옷깃 령: 옷깃을 걸어야 옷의 구김이 없다. 
	- 要領(요령): 
		- 어떤 일을 잘 해내는 사람
		- 요령을 피운다, 개인의 영달만을 생각하고 일할 때

- 嶺(고개 령)
	- 2급
	- 부수: 山
	- 秋風嶺(추풍령)
	- 大關嶺(대관령)
	
- 零(내릴 령, 떨어질 령)
	- 雨(비 우): 위에서 아래로 비가 떨어지다
	- 零細民(영세민): 가난한 백성
	- 零點(영점): 
	- 득점(得點)이 없음.
	- 섭씨(攝氏)ㆍ열씨 온도계(溫度計)에서의 어는점.
	- 어떤 일의 성과(成果)가 전혀 없는 일. 제로.
	
- 冷(찰 냉)
	- 부수: 冫(얼음 빙)
	- 명령은 차갑게 내려야 한다.
	- 冷水(냉수) ↔ 溫水(온수)
	
- 命(목숨 명, 명령 령)
	- 목숨은 못먹고, 숨쉬기를 못하면 목숨이 다하다.
	- 天命之謂性(천명지위성): 
		- 중용 첫구절, 
		- 하늘의 본성을 만물에게 부여해준 것을 본성이라고 함
		- 인간에게 하늘은 부모님, 멀리는 하늘, 땅은 지신
	- 命令(명령)
		
- 予(나 여, 줄 여)
	- 1인칭 나
	- 북에다 실을 꿰어 옷감을 짜다에서 유래
	- 나로 부터 짜서어려운 사람에게 주는데 행복감이 온다.
	- 옷감을 짤때는 차례차례 정교하게 짜야한다
	- ### 비슷한 글자
	- 子(아들 자): 강보에 싸인 아기의 모습
	- 矛(창 모): 끈이 달린 창을 본뜬 
	- 孑(외로울 혈)
	
- 序(차례 서, 학교 서, 담 서)
	- 집에서 천을 짜내듯이 차례대로 정교하게 짜야한다.
	
- 野(들 야)
	- 里(마을 리)+予(나 여)
	- 마을 농부에 의해 농지가 정교하게 잘 짜여 있는 곳, 들 야
	- 다른 사람에게 줄려고 하는 것
	- 野外(야외)
	- 野黨(야당)
	
- 矛(창 모)
	- 矛盾(모순)
	- 矛戈(모과): 창(槍), 무기의 하나
	
- 柔(부드러울 유)
	- 창의 재질은 나무, 부드럽다
	- 外柔內剛(내유외강): 겉으로 보기에는 부드러우나 속은 꿋꿋하고 강함
	- 柔順(유순)
	- 柔軟(유연)
	
- 務(힘쓸 무)
	- 창으로 좋은 쪽으로 힘써서 훈련하다, 고치다(攵: 칠 복)
	- 義務(의무)
	- 勤務(근무)
	- 改(고칠 개): 스스로 고치다

- 霧(안개 무)
	- 雨(비 우): 날씨에 관련된 글자에는 雨가 들어간다
	- 비때문에 앞으로 나아가기가 힘들다, 안개 무
	- 雲霧(운무): 구름과 안개
	- 霧散(무산): 안개가 걷힘, 안개가 걷히는 것처럼 흔적(痕跡, 痕迹)없이 사라짐
- 秋夕(추석)
	- 가을 겆이를 다하고 햇과일, 松餠(송편: 소나무 송, 떡 편) 제사(茶禮:차례)를 지내다
	- 제사상
	- 棗栗梨柹(조율이시)
		- 제사(祭祀)의 제물(祭物)을 진설(陳設)할 때, 왼쪽부터 대추ㆍ밤ㆍ배ㆍ감의 차례(次例)로 차리는 격식(格式).
		- 棗(대추 조)
		- 栗(밤 률)
		- 梨(배나무 리)
		- 柹(감나무 리)
	- 紅東白西(홍동백서)
		- 붉은 과실은 동쪽에 흰 과실은 서쪽에 차리는 격식
	- 魚東肉西(어동육서): 제사상(祭祀床)을 차릴 때에 어찬(魚饌)은 동쪽에, 육찬(肉饌)은 서쪽에 놓음.
