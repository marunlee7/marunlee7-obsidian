# 효정서당 2급 03강 力

![](https://www.youtube.com/embed/Y_Tjgd8aeFQ)

2급 공인급수 3급 교육급수
## 力
- 力(힘 력)
	- 도구(농기구)를 이용해서 힘을 쓰다 → 힘 력
		- 노력(努力) : 힘을 씀
		- 능력(能力) : 일을 감당하거나 해결해 낼 수 있는 힘
		- 중력(重力) : 큰 힘
		- 매력(魅力) : 이상하게 사람의 눈이나 마음을 호리어 끄는 힘
		- 압력(壓力) : 
		- 권려(權力) : 강제로 복종시키는 힘
		- 세력(勢力) : 권력이나 기세의 힘
		- 폭력(暴力) : 난폭한 힘
		- 역량(力量) : 힘, 능력, 어떤 일을 감당하여 해낼 수 있는 힘
		- 실력(實力) : 실제의 역량
		- 체력(體力) : 몸의 힘

- 劦(힘합할 협) 
	- 한자에서 같은 글자가 세개면 많다라는 뜻 → 힘합할 협
	- 급수 외 한자
	- 동자: 協(화할 협)
	

- 協(**화할 협**, **도울 협**)
	-  많은 사람(十)이 많은 힘(劦)을 합하다 → 도울 협
		- 협동(協同) : 힘과 마음을 함께 합함.
		- 타협(妥協) : 두 편이 서로 좋도록 양보하여 협의 함
		- 협상(協商) : 여러 사람이 모여 서로 의논함
		- 협조(協助) : 힘을 보태어 서로 도움
		- 협력(協力) : 힘을 합하여 서로 도움
		- 농협(農協) : 노영협동조합
		- 협찬(協贊) : 협력하여 찬성함
		- 협의(協議) : 여러 사람이 모여 서로 의논함
		- 불협화음(不協和音) : 서로 뜻이 맞이 않아 일어나는 충돌

- 脅(으를 협, 위협할 협, 옆구리 협)
	- 내 몸에다 힘을 다 합해서 타인을 제합하다 → 으를 협
		- 脅迫 : 을러메서 핍박함. 迫(핍박할 박)
		- 위협(威脅) : 힘으로 으로 협박(脅迫)함. 威(위엄 위) 
		- 협탈(脅奪) : 을러메어 빼앗음. 奪(빼앗을 탈)
		- 협약(脅約) : 위협(威脅)으로서 이루어진 약속(約束)이나 조약(條約)
		
- 肋(갈비대 늑)
	- 내 몸에 힘을 주다 → 갈비대 늑
	- 읽기1급
		- 늑골(肋骨) : 등뼈와 가슴뼈에 붙은 쪽
		- 늑막(肋膜) : 가슴막
		- 늑막염(肋膜炎): 갈비대가 나가서 염증이 생기다
		- 늑간(肋間) : 갈빗대의 사이
		- 계륵(鷄肋) : 닭의 갈빗대
			- 먹기에는 너무 양이 적고 버리기에는 아까워 이러지도 저러지도 못하는 形便

- 筋(힘줄 근, 근력 근) 
	- 내 몸에 힘(肋)을 줄 때 결(竹: 대 죽)이 생김 → 힘줄 근
	- 읽기4급
		- 근육(筋肉) : 
		- 근력(筋力) : 筋肉의 힘
		- 철근(鐵筋) : 콘크리트 속에 박아 뼈대로 삼는 가늘고 신 쇠막대
		- 근골(筋骨) : 근육과 뼈

## 加
- 加(더할 가)
	- 입(口)으로 힘(力:응원)을 내게 해주다 → 더할 가
	- 새로운 음인 '가'가 나옴. 이런 것을 '회의자'라고 한다.
		- 가속(加速) :
		- 증가(增加) : 더하여 많아짐. 增(더할 증)
		- 추가(追加) : 나중에 더하여 보탬. 追(쫓을 추)
		- 첨가(添加) : 이미 있는 것에 덧붙이거나 보탬.
		- 가감(加減) : 더하기와 빼기
		- 참가(參加) : 어떤 모임이나 단체에 참여하거나 가입함.
		- 가공(加工) : 인공을 더함
		- 가담(加擔) : 거들어 도와줌. 擔(멜 담)
		- 가속(加速) : 속도를 더함
		- 가열(加熱) : 열을 가함
		- 설상가상(雪上加霜) : 눈 위에 또 서리가 내린다.
			- 어려운 일이 겹침
		
- 架(시렁 가, 가로지를 가, 선반 가, 서재 가)
	- 나무(木)를 덧대어(加) 주어 시렁을 만들다 → 시렁 가
	- 시렁: 선반
		- 가공(架空) : 공중(空中)에 가로 건너지름.
		- 가교(架橋) : 다리를 놓음
		- 가구(架構) : 재료를 결합하여 만든 구조물
		- 서가(書架) : 책을 얹어두는 시렁. 책꽃이
		- 고가도로(高架道路) :
		- 십자가(十字架) : 

- 賀(하례할 하, 축하할 하) 
	- 축하는 돈(貝) 들어가고, 입으로 축하하다 → 하례할 하
	- 加(더할 가) + 貝(조개 폐)
	- 하례(賀禮)) : 축하하다
		- 하례(賀禮) : 축하하는 예식
		- 축하(祝賀) : 기뻐하고 즐겁다
		- 치하(致賀) : 남의경사에 대하여 축하의 말을 하는 인사
		- 근하(謹賀) : 삼가 축하하다
		- 연하장(年賀狀)
		- 근하신년(謹賀新年) : 삼가 새해를 축하한다는 인사말
