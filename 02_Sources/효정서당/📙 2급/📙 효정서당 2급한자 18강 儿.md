## 儿

- 元(으뜸 원) = 儿(어진사람 인) + 
	- 하늘 아래, 땅위에서 어진마음을 가진게 으뜸이다
	- 하늘과 땅 사이에서 사람(儿)이 으뜸이다 [꼬불]
	- 으뜸, 시초, 근본, 우두머리
	- 칭기즈칸의 손자이자 몽골의 5대 황제인 쿠빌라이가 세운 나라 이름도 元

	- 한자는 음이 같으면 뜻이 같다고 생각할 것
	- 儿(어진사람 인) = 행동을 보면 仁을 발현하는지 볼 수 있음
	- 人(사람 인)
	- 仁(어질 인)
	- 廴(길게걷을 인) =선비, 군자의 행동거지가 우아하고 조화를 이룸
	- 惻隱之心 仁之端也(측은지심 인지단야) = 불쌍히 여기는 마음은 인의 근본임

	- 元亨利貞(원형이정)
		- 역학에서 말하는 천도의 네 가지 원리
		- 사물의 근본되는 원리
		- 만물이 처음 생겨나서 자라고 삶을 이루고 완성함
		
	- 장원(壯元) = 과거에서 갑과에 첫째로 급제(及第)함, 또는 그 사람
	- 복원(復元) = 부서지거나 없어진 사물을 원래의 모습이나 상태로 되돌려 놓는 것
	- 元來(원래) = 본디, 전부터, 원판(原板)
	- 次元(차원) = 일반적인 공간의 넓의의 정도, 사고 방식이나 행위의 수준, 레벨
	- 환원(還元) = 본디의 상태(狀態)로 되돌리는 일
	- 元旦(원단) = 정월 초하룻날 아침, 설날 아침
	- 元老(원로) = 관위, 연령, 덕망이 높은 공신
	- 원수(元帥) = 장수(將帥)의 으뜸
		- 帥(장수 수/ 거느릴 솔) = 𠂤(쌓을 퇴), 阜(언덕 부) + 巾(수건 건)
			- 巾 나무막대기에 걸려있는 천 + 언덕 위
			- 언덕위에 깃발이 꽃혀있는 모습
			- 부대마다 고유의 깃발이 있었고 그 병사들은 그 깃발을 따라 이동했다
			- 병사들을 거느리고 통솔하는 역할을 한 것
			- 帥 거느리다, 인도하다, 병사들을 이끄는 장수, 우무머리
			
	- 원소(元素) = 물건을 만들어 내는 근본이 되는것
		- 素(흴 소, 본디 소, 바탕 소) = 糸(실 사) + 垂(드리울 수)
			- 본디, 바탕, 성질
			- 사물의 가장 원초적인 속성
			- 누에고치에서 갓 뽑은 실타래를 묶는 모습
			- 뽑은 지 얼마 되지 않은 실을 표현한 것으로 가장 순수하고 원초적인 것
			- 빨아 널어 드리운 명주실이 깨끗하다에서 희다
			- 아직 물들이지 않는 흰 명주는 희다
			- 물건의 시초 → 바탕
	
- 完(완전할 완) 
	- = 누구나 으뜸인 어진사람으로 살아나갈(宀) 때에 완전하다고 할 수 있다
	- 宀(집 면) + 元(으뜸 원)
	- 元 사람의 머리를 강조해 그린 것으로 으뜸, 처음
	- 完 집짓기를 끝마쳤다는 것
	- 으뜸(元)으로 집을 지었다
	- 집을 잘 지었다는 뜻으로 만들어졌다
	- 집을 잘 지었다는 것은 공사가 마무리됐음을 뜻한다
	- 그래서 끝내다, 일을 완결 짓다, 완전하다
		- 完全(완전) = 부족이나 흠이 없음
		- 完遂(완수) = 목적을 完全히 達成함
		- 完了(완료) = 完全히 끝마침
		- 完結(완결) = 아주 完全하게 끝을 맺음
		- 完成(완성) = 어떤 사물을 完全히 이룸
		- 補完(보완) = 補充하여 온전(穩全)하게 함
		- 完走(완주) = 마지막까지 다 달람
		- 完治(완치) = 병을 완전히 고침
		- 完然(완연) = 완전함
		- 未完(미완) = 미완성
		- 完備(완비) = 완전히 갖춤
	
- 院(집 원, 병원 원) 
	- = 阝(좌부변 부) + 完(완전할 완)
	- 집, 정원, 관아, 규모가 큰 건물
	- 으뜸인, 모범이 되는 사람들이 학문을 하는 곳
	- 完 집을 온전하게 잘 지었다는 의미에서 완벽하다
	- 院 담벼락(阜)이 있는 잘 지어진 큰 집	
		- 淨院(정원) = 깨끗하고 조용한 집, 절간
		- 病院(병원) = 
		- 學院(학원)
		- 醫院(의원)
		- 中樞院(중추원)
		- 法院(법원)
		- 書院(서원)
		- 棋院(기원) = 바둑을 즐겨 두는 사람이 조직하는 단체
		- 院長(원장)
		- 硏修院(연수원)
	
- 冠(갓 관) 
	- = 冖(덮을 멱) + 元(으뜸 원) + 寸(마디 촌)
	- 갓, 관, 쓰다, 관례
	- 쓰는 것(冖 덮다, 덮개) = 머리(元) + 손(寸)
	- 어진사람으로 법도를 갖추기위해 머리에 갓을 쓰다 [효정]
	
	- 머리에 모자(관직에 오른 사람이 쓰던 감투)를 씌우는 모습
	- 관직에 있지 않더라도 결혼식에 감투를 쓴다 : 관례
	
	- 王冠(왕관) = 임금이 머리에 쓰는 冠
	- 衣冠(의관) = 옷과 갓
	- 冠詞(관사) = 명사앞에 놓이는 품사
	- 冠禮(관례) = 스무살이 되어 남자는 갓을 쓰고 여자는 쪽을 찌고 어른이 되던 예식
	- 月桂冠(월계관)
