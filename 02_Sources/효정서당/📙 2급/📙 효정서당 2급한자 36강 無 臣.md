# 효정서당 2급 36강 無, 臣의 유형별 한자🙇‍♀

![](https://www.youtube.com/embed/HK-o1s55Vv0?si=8JFo98TEmREdxgeA)

## 无

- 无(없을 무) : 無 [71]
		- 인지하지는 못하지만 하늘 어딘가에는 있다
	- 無(없을 무) : 人 + 廾(두손들 공) + 火
		- 양 손에 물건을 든 사람이 춤추는 모습 + 불
		- 불놀이야, 불이 춤춘다
		- 완전히 없음
	- 舛(어그러질 천)
		- 양발을 서로 엇갈린 모습을 그린 것
		- 어그러지다, 엇갈려 있다, 어지럽다
		- 止(발 지, 그칠 지)가 서로 엇갈려 모습
		- 舛자가 부수로 쓰이면 발, 발의 동작
			- 舛駁(천박) : 뒤섞여서 고르지 못하거나 어수선하여 바르지 못함
	- 舞(춤출 무) : 舛(어그러질 천) + 
		- 춤추다, 날아다니다
			- 鼓舞(고무) 
				□ 북을 쳐 춤을 추게함
				□ 격력하여 기세를 돋움, 부추겨 용기가 생기게 함
			- 舞踊(무용) : 춤
			- 舞臺(무대) : 노래, 춤, 연극 따위를 하기 위하여 마련된 곳
- 巨(클 거)
		- 5획
		- 巨人(거인)
	- 拒(막을 거) : 扌 + 巨(클 거)
		- 螳螂拒轍(당랑거철) 
			- 사마귀가 수레바퀴를 막는다
			- 자기 힘은 헤아리지 않고 강자에게 함부로 덤빔
	- 距(떨어질 거) 
		- 足(발 족) + 巨(클 거)
		- 거리(距離)
		- 단거리(短距離)
		- 장거리(長距離)
		- 거리감(距離感)
- 臣(신하 신) 
	- 갑골 : 눈을 아래로 뜨고 있는 모습
	- 윗사람 앞에서 아랫사람이 눈을 아래로 뜨고 있는 모습
		
	- 臤(굳을 간, 굳을 견, 어질 현) 
		- 臣(신하 신) + 又(또 우)
		- 굳다
		- 賢(어질 현)의 古字
	- 堅(굳을 견)의 同字
		- 堅固(견고)
	
	-  賢(어질 현) : 
		- 臤(어질 현, 굳을 간) + 貝(조개 패)
		- 어질다, 현명하다
		- 臤자는 신하가 일을 능히 잘 해낸다는 의미, 어질다, 현명하다
		- 사람이 어질고 착해 재물까지 나누어 줄 정도라는 의미가 반영
			- 愚問賢答(우문현답) : 어리석은 질문에 현명(賢明)한 대답(對答)
			- 賢母良妻(현모양처) : 어진 어머니이면서 또한 착한 아내
		- 賢者 < 仁者 
			- 현자는 돈을 정승처럼 번다, 쓸 때는 정승처럼 쓴다(굳게) 
			- 苛斂誅求(가렴주구) : 가혹하게 세금을 거두거나 백성의 재물을 억지로 빼앗음
			
	- 緊(요긴할 긴, 긴박할 긴, 팽팽할 긴) : 糸(가는 실 멱) + 臤(굳을 간, 어질 현)
		- 긴하다, 요긴하다 : 꼭 필요하다
		- 급하다, 팽팽(膨膨)하다
		- 줄이 굳다, 줄이 팽팽하다 → 매우 급한 상황
			- 緊張(긴장)
				□ 마음을 다잡아 정신을 바짝 차리거나 몸이 굳어질 정도로 켕기는 일
			- 緊急(긴급) : 요긴하고 급합, 일이 중대하고도 급함
			- 緊密(긴밀) : 서로의 관계가 몹시 긴하고 가까움
			- 緊縮(긴축) : 바싹 줄임, 재정상의 기초를 단단하게 하기 위해 지출을 줄임
			- 要緊(요긴) : 중요하고도 꼭 필요함
			- 緊迫(긴박) : 아주 급박함
			- 緊張感(긴장감) : 긴장할 때의 마음 상태
			
	- 臥(엎드릴 와, 누울 와) 
		- 臣(신하 신) + 人(사람 인)
		- 臥薪嘗膽(와신상담)
			- 섶에 눕고 쓸개를 씹는다
			- 원수를 갚으려고 온갖 괴로움을 참고 견딤

	- 臨(임할 임) 
		- 臣(신하 신) + 𠂉(-) + 品(물건 품)
		- 임하다(어떤 사태나 일에 직면하다)
		- 대하다, 내려다보다
			- 臨機應變(임기응변) : 그때그때 처한 뜻밖의 일을 재빨리 그 자리에서 알맞게 대처하는 일
			- 臨戰無退(임전무퇴) : 싸움에 임하여 물러섬이 없음
			- 背山臨水(배산임수) : 지세가 뒤로는 산을 등지고 앞으로는 물에 면하여 있음
			- 臨終(임종)
			- 臨迫(임박)
			- 枉臨(왕림)

	- 監(살필 감, 볼 감) : 
		- 臥(누울 와) + 皿(그릇 명)
		- 대야에 비친 자신을 바라보는 모습을 그린 것
		- 보다, 분위기 살피다, 거울로 삼다		
		- 監督(감독)
		- 監獄(감옥)
		- 監察(감찰) : 감시(監視)하고 살피는 것
		- 監査(감사) : 감독하고 검사함
		- 監事(감사) : 공공단체의 서무를 맡아보는 직책, 그 직책으 사람
		- 監視(감시) : 경계하기 위하여 미리 감독하고 살피어 봄
		- 監理(감리) : 감독(監督)하고 관리함
		
	- 유의어
		- 察(살필 찰) 
		- 審(살필 심, 빙빙 돌 반) 
		- 査(조사할 사) 
		- 檢(검사할 검) 
		- 看(볼 간) 
		- 示(보일 시, 땅귀신 기, 둘 치) 
		- 見(볼 견, 뵈올 현, 관의 천) 
		- 覽(볼 람(남)) 
		- 觀(볼 관) 
		- 閱(볼 열, 셀 열)

	- 鑑(거울 감)
		- 龜鑑(귀감)
		- 鑑溪(감계)
		
	- 濫(넘칠 람)
	
	- 覽(볼 람)
		- 遊覽(유람)
		- 觀覽(관람)
		
	- 鹽(소금 염): 監자를 응용 가마솥에서 소금이 만들어지는 과정을 지켜보는 모습
		- 監(볼 감) + 鹵(소금밭 로) 
		- 소금을 만드는 방법
			- 염전에서 바닷물을 증발시키는 방법
			- 가마솥에 끓여서 증발시키는 방법			
		- 염전(鹽田)
		- 염산(鹽酸)
		- 죽염(竹鹽)
		- 염장(鹽藏): 소금에 절이어 저장(貯藏)함
		- 천일염(天日鹽):
