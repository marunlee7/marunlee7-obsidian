# 효정서당 2급 38강 亡, 共의 유형별 한자

![](https://www.youtube.com/embed/xai0-sWBCSs)

## 亡
- 亡(달아날 망, 없을 망, 죽을 망)
	- 머리두 + 숨을 은, 감출 혜
	- 생 | 사 = 생에서 사로 숨어 들어가, 순간이동
	- 망자(亡者)
	
- 妄(망령 망)
	- 망언(妄言)
	- 망동(妄動)

- 忙(바쁠 망)
	- 마음만 바빠 마음을 온전히 쓰지 못함
	- 분망(奔忙) : 매우 바쁨.  奔(달릴 분)
	- 공사다망(公私多忙)
	
- 忘(잊을 망)
	- 마음을 잊어버리다
	- 물망초(勿忘草)
	- 망각(妄却)
	- 망년회(忘年會)
	
- 望(바랄 망)
	- 망 + 달님+곧을 정(사람이 우뚝 서있다)
	- 달님을 보고 우뚝 서서 곧은마음으로 바래다
	- 희망(希望)
	- 망루(望樓)
	
- 茫(망망할 망, 아득할 망)
	- 망막(茫漠)
	- 망망대해(茫茫大海)
	
- 罔(그물 망, 없을 망)
	- 그물로 못잡을 때는 없을 망
	- 망극(罔極)
	- 망측(罔測)
	
- 網(그물 망)

- 荒(거칠 황) 
	-  초 + 망 + 
	- 蕪(무성할 무)
	- 황야(荒野)
	- 허황(虛荒)
	- 황무지(荒蕪地)
	
- 盲(눈멀 맹, 소경 맹) 
	- 눈이 없다
	- 맹신(盲信)
	- 문맹(文盲)

## 共
- 共(함께 공)
	- 최초의 절구는 땅에 만듬
	- 공(두손 들 공) + 廾(받을 공, 두손들 공)
	
- 廾(받을 공, 두손들 공)

- 供(이바지할 공)
	- 사람이 자연스럽게 함께하다, 이바지하다
	- 공양미(供養米)
	
- 恭(공손할 공) 
	- 마음속, 본성을 잘쓰면 살아가는 데 도움이 됨.
	- 마음이 흩어지면 몸으로 바르게하고, 몸이 흩어지면 마음으로 다잡아야 한다
	- 공수배례(拱手拜禮)
	- 恭敬
	- 불공(不恭)
	
	
- 拱(두손 맞잡을 공, 깔지낄 공)
	- 공수(拱手) : 왼손을 오른손 위에 놓고 두 손을 마주 잡아, 공경의 뜻을 나타냄
	
- 洪(넓을 홍, 큰물 홍)
	- 물길이 여러 갈랫 길로 퍼지다, 넓다
	- 홍수(洪水)
	- 홍어(洪魚)
	

- 巷(거리 항)
	- 共(함께 공)자와 㔾(병부 절)
	- 사람들이 함께(共) 모여 사는 마을(邑)	
	- 항간(巷間) : 一般 사람들 사이, 庶民들 사이
	- 항설(巷說) : 동네에서 뭇 사람들이 지껄여 옮기는 말
		
- 港(항구 항)
	- 氵(삼수변 수) + 巷(거리 항)
	- 어항(漁港)
	- 귀항(歸港)
	
- 航(배 항)
	- 항공사(航空社) : 항공(航空) 운송(運送) 사업(事業)을 하는 회사(會社)

- 巽(유순할 손, 겸손할 손, 사양할 손) 
	- 갖출 손(선) = 무릎마디 절
	- 함께 모여 제사를 지네다
	- 謙遜  : 
		- 겸손(謙巽)
		- 遜(겸손할 손)
		
- 選(뽑을 선, 가릴 선)
	- 선거(選擧)

- [異](다를 이)
	- 두 손을 들어 앞을 가리다
	- 이견(異見)
	- 이방인(異邦人)
	
	
- 翼(날개 익)
	- 깃 우 = 깃털을 보고 만든 글자
	- 항우(項羽)
	- 우익(右翼)
	- 좌익(左翼)
	- 비익조(比翼鳥)
		- 암컷과 수컷이 눈과 날개가 하나씩이라서 짝으 맂ㅅ지 않으면 날지 못한다는 새
		- 남벼사이 혹은 부부애가 두터움을 이르는 말
		- 당태종과 양귀비
