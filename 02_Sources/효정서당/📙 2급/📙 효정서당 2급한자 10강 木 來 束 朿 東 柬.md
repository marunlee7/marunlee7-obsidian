# 효정서당 2급 10강 木(2) 來

![](https://www.youtube.com/embed/rGQugStPugg)

## 來
- 來(올 래, 보리 래) 
	- 从(많은 사람 종, 좇을 종) + 木(나무 목)
	- 올래는 오래전에 보리 래였다
	- 오다, 돌아오다, 앞으로
	- 미래(未來)
	- 내일(來日)
	- 木은 생명을 대표하고 지구촌은 사람이 살기에 좋은 조건으로 많은 사람들(从) 이 오게되었다. 보리는 중요한 식량 역할을 하게됨.

- 麥(보리 맥, 밀 맥) 
	- 來(올 래(내)) + 夂(뒤쳐져 올 치)
	- 보리는 겨울나기를 하여 봄에 싹을 틔운다. 
	- 보리밟기를 해주어 꽃샘추위를 피해 늦게 틔우도록 해줌
	- 보리는 찬음식으로 여름에 즐겨 먹음. 열 내리는 음식	
	- 맥주(麥酒)

- 麵(국수 면)
	 - 본자는 麪(국수 면)
		- 丏(가늘 면, 뵈지 않을 면)
- 嗇(아낄 색, 인색할 색) 
	- 來(올 래) + 㐭(광 름, 곳집 름)
	- 광(㐭)에 먹거리를 가져오다(來)
	- 광(창고)에 먹거리를 채워넣고 잘 내어놓지 않음
	
	- 吝(아낄 인, 인색할 인)
		- 글로 멋지게 표현하고 입으로 멋지게 말하지만 주지는 않는다
		- [[吝嗇]] : 체면을 돌아보지 않고 재물을 지나치게 아낌
		- 인색한(吝嗇漢) : 아주 인색한 사람을 낮추어 이르는 말
- 墻(담장 장) 
	- 土(흙 토) + 嗇(아낄 색)
	- 광(창고)에 흙(土)을 쌓아 담장(경계)을 만들다 
	
	- 牆(담장 장) 
		- 담장(주변과의 경계를 나누는 담장)
		- 수확한 곡식을 안전하게 보관하다
		- 누가 훔쳐가지 못하도록 나무로 울타리를 만들었다
- 薔(장미 장) 
	- 艹(초두머리 초) + 嗇(아낄 색)
	- 장미(薔薇) 
## 束
	
- 束(묶을 속): 흩어져 있는 나무단을 묶다
- 速(빠를 속)
	-  辶(쉬엄쉬엄 갈 착) + 束(묶을 속)
	- 한쪽에서 오고 다른 쪽에서 와서 묶게 되면 빠르다의 의미
	- 고속(高速)
	- 속도(速度)
	- 속전속결(速戰速決) : 
- 剌(어그러질 랄(날), **물고기 튈 날**)
	- 束(묶을 속) + 刂(선칼도방 도)
	- 묶어놓은 단을 칼로 자를 때 한쪽으로 넘어지는 모습이 어그러졌다(흩어졌다)
	- 발랄하다, 어그러지다		
	- 발랄(潑剌) : 활발하게 약동하는 모양
	- 수라(水剌) : 궁중에서 임금에게 올리는 밥을 높여 이르던 말
	- 수라상(水剌床) : 임금에게 올리는 진지 상(床)
	
- 賴(의뢰할 뢰, 의지할 뢰)
	- 束(묶을 속) + 刀(칼 도) + 貝(조개 패)
	- 묶어놓은 단을 칼로 자를 때 넘어지는 모습의 차이를 장사할 때의 차액에 의지하여 장사를 하다
	- 의뢰(依賴)하다, 의지(依支)하다
	- 信賴 
	- 무뢰한(無賴漢) : 예의가 바르지 못한 사람
	- 依(의지할 의) : 사람은 옷(衣)에 의지한다
## 朿

- 朿(가시 자, 묶을 속, 약속할 속) 
	- 가시있는 나무
	- 찌르다, 가시

- 刺(찌를 자, **찌를 척**, 수라 라) 
	- 刂(선칼도방 도) + 朿(가시 자)
	- **칼로 찌르다**, 칼로 찔러 죽이다
	- 동자 : 㓨(찌를 자) = 刂(선칼도방 도) + 夾(낄 협)
	- 자극(刺戟)
	- 자객(刺客)
	- 척살(刺殺) : 칼로 찔러 죽임
	
- 조율이시(棗栗梨柹)
	- 제사(祭祀)에 흔히 쓰는 대추, 밤, 배, 감 따위의 과실(果實).
	- 제사의 제물을 진설(陳設)할 때, 왼쪽부터 대추ㆍ밤ㆍ배ㆍ감의 차례(次例)로 차리는 격식
	
	- 棗(대추 조)
		- 朿(가시 자) + 朿(가시 자)
		- 대추나무는 가시가 많음으로 가시(朿)를 겹쳤음
		- 조율동서棗東栗西 : 제물을 차릴 때 대추는 동쪽에, 밤은 서쪽에 놓는다 말
		- **우리는 단군의 후손으로 시조를 대표하는 씨가 하나**
		
	- 栗(밤 률(율))
		- 木(나무 목) + 覀(덮을 아)
		- 밤송이 까고 밤껍질 까고 율피 까지 까야 한다. 밤의 특징은 여러번 덮혀 있다
		- 밤 알 3개
		- **삼정승(三政丞) : 영의정, 좌의정, 우의정**
		
	- 柹(감나무 시)
		- 木(나무 목) + 𠂔(그칠 제, 그칠 자)
		- 동자 : 柿(감나무 시) : 市를 사용해서 속자로 쓴다
		- 감나무의 특징은 잘 부러지고 꺽이는 특징을 𠂔(그칠 자)로 표현
		- **씨가 6개**
		- **육조판서(六曹判書)**

	- 梨(배나무 리(이))
		- 木(나무 목) + 利(이로울 리(이))
		- 배는 감기 예방과 소화에 도움을 준다
		- **씨 8개**
		- **팔도현감**		
		
- 策(꽤 책, 채찍 책)
	- 𥫗(대죽 죽: **글**, **대나무**) + 朿(가시 자)
	- 공자 시절에도 책이란 죽간(竹簡)을 의미했다.
	- 가시는 몸에 자극을 주고 통증을 준다. 좋은 계획을 짤 때 **자극**을 준다. 
	- 간책(簡策)

## 東
- 東(동녘 동)  
	- 나뭇가지(木) 사이로 해(日)가 떠오른다
	- 해가 나오는 곳을 부상로 라 함
	- 桑(뽕나무 상)
	- 日出扶桑路 暮入若木枝 (입출부상로 모입약목지)
		- 해는 부상(扶桑 : 동쪽을 상징하는 상상의 나무)의 길로 나오고, 
		- 저물면 약목(若木 : 서쪽을 상징하는 나무)의 가지로 드는 도다.
		- 해는 동쪽으로 뜨고 서쪽으로 진다는 표현을 한시로 나타냈습니다.
	- 부상(扶桑) 
		- 중국 전설에서, 동쪽 바다의 해가 뜨는 곳에 있다는 신성스러운 나무
		- 해가 돋는 동쪽 바다를 빗대어 이르는 말
	- 약목(若木)
		- 해가 지는 곳에 있었다는 상상의 나무
		- 해가 지는 곳을 이르는 말
	
- 凍(꽁꽁 얼 동) 
	- 冫(얼음 빙) + 東(동녘 동)
	- 해가 떠올라 온기가 있는데도 아직도 얼음이 얼어 아직 춥다는 뜻 → 꽁꽁 얼 동
	- 냉동식품(冷凍食品)
	- 동태(凍太)
	
- 棟(용마루 동, **마룻대** 동)
	- 아파트 동(棟)
	
- 陳(묶을 진, 진열할 진, 베풀 진) 
	- 阝(좌부변 부) + 東(동녘 동)
	- 언덕에 나무를 묶어 진열해 놓다. → 묶었다, 진열했다, 벨풀어 놓다.
	- 보석상에 액세서리를 진열해놓다
	- 진열(陳列)
	- 진술(陳述)

## 柬
- 柬(가릴 간, 분별할 간, 나눌 간)
	- 나무단을 큰 것, 작은 것을 나누어 분별하여 묶다

- 練(익힐 련(연))
	- 실을 뽑아 삶아 익혀서 불순물을 빼내다.
	- 좋지 않는 습관은 익혀서 빼내야 한다.
	- 연습(練習)

- 鍊(단련할 연(련))
- 煉(불달구 련(연), 불린 련(연))
	- 연탄(煉炭)
	- 연옥(煉獄) : 세상(世上)에서 죄(罪)를 풀지 못하고 죽은 사람이 천국(天國)으로 들어가기 전(前)에, 불에 의(依)해서 죄(罪)를 정화(淨化)한다고 하는 곳. 천국(天國)과 지옥(地獄)과의 사이에 있다고 함.

- 䦨(가로막을 란(난), 난간 란)
	- 문사이에 난간(베란다)를 만들어 내실과 분별(柬)하다. 

- 欄(난간 란(난), 막을 난)
	- 난간의 재질이 나무(木)이다. 

- 蘭(란초 란(난)
	- 난간에 햇살이 잘 들어오니 난초를 키우다.
