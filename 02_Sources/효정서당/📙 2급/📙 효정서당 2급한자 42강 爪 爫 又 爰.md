# 효정서당 2급 42강 爪 爫 又 爰(늘어질 원, 이에 원)

![](https://www.youtube.com/embed/TxMFkK2LKM8)

## 爫(손톱 조)
- 乳(젖 유) : 아가가 손끝으로 교감하면서 젖을 먹다
	- 어머니 젖을 보고 만듬 
	- 爫(손톱 조) + 子(아가 자) + 乙(굽을 을 변형)

- 孚(알깔 부, 기를 부, 미쁠 부) : 새가 둥지에서 알을 품은 모습 - 

- 浮(뜰 부)
	- 어머니와 아가 사이에 약간 뜨있어 아가가 물위에 뜨있듯 안전하게 있는 모습
	- 부석사(浮石寺)
		- 김삿갓 詩
		- 돌위에 뜨있는 모습으로 산천을 내려다보다
	- 부평초(浮萍草) : 개구리밥
		- 萍(부평초 평)
		- 草(풀 초) 

- 𤔔(다스릴 란(난)) : 
	- 爫(손톱 조) + 糸(실 사) + 匚(상자 방) + 又(오른손 우)
	- **잠박이에 실을 삼해서 상자에 수북히 올려 놓아 누른 모습** → 실을 잘 다스렸다 하여 다스릴 란
	
- 亂(어지러울 란(난))
	- 실이 올라간 모습이 어지럽다(乙)	
	- 난동(亂動)
	- 교란(攪亂)
	
- 辭(말씀 사) : 
	- 𤔔(어지러울 란(난)) + 辛(매울 신, 죄인 신)
	- 섬세한 손으로 실을 잘 다스리 듯 죄인들간의 복잡한 문제를 풀어 심판하다
	- 詞(말 사) : 품사, 부분을 맡았다 하여 말 사
		- 동사(動詞) : 
	- 司(맡을 사)
	
	- 사설(辭說) : 늘어놓거나 기술(記述)한 말이나 글의 내용(內容). 
	- 사양(辭讓) : 
	- 두주불사(斗酒不辭) : 말술도 사양하지 아니한다, 주량이 매우 큼을 일컬음
	- 사양지심(辭讓之心) : 겸손히 마다하며 받지 않거나 남에게 양보하는 마음, 예의 근본
	
## 又(또 우, 오른손 우)
- 又(또 우, 오른손 우)
- 爰(**늘어질 원**, 이에 원) : 손톱 끝을 이용 사물를 끌어당기는 모습 
	- 느즈러질 원
	- 이에 원 (어조사 로 쓸때는) 
- 暖(따뜻할 난) : 해를 많이 받아 따뜻해지면 늘어지다 
	- 난방(暖房) : 
- 援(도울 원) : 여러 사람이 같이 당겨야 한다
	- 救援(구원) 	
- 緩(느릴 완) : 실이 완전히 늘어져 느릴 완
	- 緩行(완행) : 
	- 緩急(완급) : 느림과 빠름. 일의 급(急)함과 급(急)하지 않음.

	
- 受(받을 수) : 손을 펴서 물건을 받다.
- 授(줄 수) : 두 손(扌)으로 공손히 주다

- 桑(뽕나무 상) : 뽕잎을 주기위해 나무중에서 손이 가장 많이 가는 나무
	- 扶桑路(부상로) : 해가 나오는 길
	- 부상(扶桑) : 
		- 해가 돋는 동쪽 바다.
		- 중국(中國) 전설(傳說)에서, 동쪽 바닷속에 해가 뜨는 곳에 있다고 하는 나무.
	
- 圣(힘쓸 골): 쓸데없이 오른손으로 흙을 계속 파다. 흙장난함.
	
- 怪(괴이할 괴)
	- 마음을 쓸데없는 곳에 써, 괴이한 일이 생기다
	- 괴담(怪談) : 괴상한 이야기


- 叉(깍지낄 차, 갈래 차) : 두손으로 깍지를 끼다
	- 爫(손톱 조) + 又(오른 손 우)
	- 교차로(交叉路)
	
- 蚤(벼룩 조) : 깍지를 낀 것처럼 따끔 따끔 꼬집는 벌레가 있다, 벼룩 조
	
- 騷(떠들 소, 소란 소, 근심스러울 소)
	- 馬(말 마) + 蚤(벼룩 조)
	- 말이 벼룩에게 물려서 너무 놀래서 소리 지르다
	- 馬(말 마): 말은 놀래는 성질이 있다
	- 소동(騷動) : 수선거리면서 움직이는 일
	- 소란(騷亂) : 야단스럽고 시끄러움 


- 叔(어릴 숙, 아재비 숙)
	- 숙부님=작은 아버지
	- 논두렁 구멍에 콩을 4-5개 심는다
	- 천신(새), 지신(개미, 지렁이), 썩어서 땅에
	- 콩 하나를 조금 자랐을 때 솎아내다
		- 숙부(叔父) : 아버지의 아우, 작은 아버지
		- 당숙(堂叔) : 종숙의 친근한 일컬음
	
- 尗(콩 숙) : 풀 초를 넣어서 완성해야 한다
- 菽(콩 숙) : 
	- 艹(초두머리 초) + 叔(아재비 숙)
	- 菽麥(숙맥) : 콩인지 보리인지 분별하지 못한다는 뜻
	
- 淑(맑을 숙)
	- 콩을 기를 때는 맑은 물을 줘야한다
	- 신숙주: 둘째임을 알 수 있다.
		- 淑女(숙녀):교양과 예의와 품격을 감춘 점잖은 여자
		- 貞淑(정숙):여자(女子)의 행실(行實)이 곱고 마음씨가 맑음.
		- 靜肅(정숙): 고요하고 엄숙함.

- 寂(고요할 적)
	- 내용설명이 없었음
		- 閑寂(한적):
		- 靜寂(정적)
	
- 督(살필 독)
	- 눈으로 어린 동생을 잘 살피다
	- 監督(감독)
		- 監(살필 감)=臥(누울 와)+皿(그릇 명)
		- 督勵(독려)
	
## 支
- 支(가지 지, 지탱할 지, 가를 지)
	- 오른손으로 나뭇가지를 지고 있다.
	- 땅에 뿌리를 내리고 하늘을 지탱하다
	- 우리의 몸도 팔다리가 있어 균형맞게 땅에 잘 지탱하고 있다.
	
- 肢(사지 지)

- 枝(나뭇가지 지)

- 技(재주 기)
	- 기술자
	- 技術(기술)
		
- 妓(아름다울 기)
	- (기생 기)가 아니다. 
	- 妓女(기녀): 정신이 아름다운 사람 
	- 미나리의 윤여정: 삶의 현장으로 연길할 때 최고의 연기가 나온다
		- 寄生蟲(기생충)
