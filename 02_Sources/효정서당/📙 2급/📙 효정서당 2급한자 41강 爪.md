# 효정서당 2급 41강 爪 (손톱 조)의 유형별 한자

![](https://www.youtube.com/embed/Vzd23HvjBKk)

## 爪
- 爪(손톱 조) = 爫(손톱 조 머리) [87]
	- ★ 부수 11강 4획 부수
	- 손 끝을 보고 만든 글자
	- 섬세한 일
	- 무기(칼귀) [여자에게는 손톱이 무기가 된다]
		- 조선시대 희빈이 손톱으로 용안을 긁다, 궁에서 쫓겨나다
		
- 印(도장 인) = 爫(손톱 조, 왼손 좌) + 卩(병부 절)
	- 屮(싹날 철, 풀 초, 왼손 좌)
	- 왼손 좌는 싹날 철에 포함
	- **손으로 눌러 무릎을 꿇기는 모습**
	- 抑(누를 억) = 扌(재방변 수) + 卬(나 앙)
	
- 妥(타협할 타, 온당할 타)
	- 섬세한 손으로 여자를 만지기전에 타협하다, 만져도 되요? [효정]
	- 여자의 머리채를 붙잡다 → 여자를 복종시키다, 집안이 평안해지려면 → 이러한 행동은 당연하다, 마땅하다
	- 妥協(타협)
	- 妥結(타결) = 두편이 서로 좋도록 협의
		- 結(맺을 결) =糹(가는 실 멱) + 吉(길할 길)
	- 妥當(타당) = 사리에 맞아 마땅함
	- 妥當性(타당성) = 적절하게 들어맞는 성질
	
- 奚(어찌 해, 어찌 혜) = 爫(손톱 조) + 幺(작을 요) + 大(클 대)
	- 爫(손톱 조: 닭벼슬, 왕관 모양자) + 幺(작을 요: 주둥이) + 大(클 대:배가 크다)
	- 벼슬 주둥이 배가 크다 = 새과
	- 음을 빌어 의문사 어찌(육하원칙)
	
- 鷄(닭 계) = 鳥(새 조) + 奚(어찌 해)
	- 酉(닭 유/열째 지지 유)
	- 鷄卵(계란)
	- 鷄肋(계륵) = 닭의 갈빗대, 먹기에는 너무 양이 적고 버리기에는아까워 이러지도 저러지도 못하는 형편
	
- 溪(시내 계) =氵(삼수변 수) + 奚(어찌 해)
	- 시냇가의 오리
	
- 采(캘 채, 풍채 채) = 爫(손톱조 조) + 木(나무 목)
	- 爫(손톱 조 : 손모양, 풀 이파리 ) + 木(나무 목 : 뿌리)
	- 風采(풍채) = 드러나 보이는 사람의 겉모양
	- 캐다의 뜻은 採(캘 채)로 대신
	
- 彩(채색 채) = 彡(터럭 삼: 햇살 모양자) + 采(풍채 채)
	- 文彩(문채)
	- 彩色(채색) = 그림에 색을 칠함
	- 色彩(색채) = 빛깔

- 爲(할 위, 삼을 위, 지을 위) 
	- = 爫(손톱조 조) + 𠃌(-)(다리, 꼬리 모양자) + 灬(연화발 화 : 발)
	- 앉을 원숭이 손으로 머리를 걱적거리는 행위를 하고 있다
	- 원숭이 보고 만들 글자
	- ~을 하다, ~을 위하다, 되다, 이루어지다
	
	- 行爲(행위)
- 僞(거짓 위)
	- 원숭이가 사람 흉내 내는 것은 거짓이다
	
- 㥯(삼가할 은) = 爫(손톱조 조) + 工(장인 공) + 彐(돼지 머리 계)+心(마음 심)
	- 섬세한 손으로 만들다, 마음을 조심해서 만들어야 한다
	- 삼가다 = 몸이나 언행을 조심하다
	
- 隱(숨을 은)
	- 고향 언덕으로 돌아가 농부로 자연과 더불어 다스리며 살다
	- 隱退(은퇴)
	- 隱者(은자)
	- 조선시대 3은 = 정문주, 묵은


	- 문자학 스토리 = 한자는 긴 스토리를 한자로 축소하는 매력이 있다
	- 사랑해 = 愛
	
- ​愛(사랑 애) = 爫(손톱 조) + 冖(덮을 멱) + 心(마음 심) + 夊(뒤저올 치)
	- 덮고 싶은 허물(心)까지도 섬세(爫)하게 뒤(夊)에서 보살펴 주다
	
	- 愛情(애정) = 사랑하는 마음
	- 戀愛(연애) = 남녀 사이에 서로 애틋하게 그리워하고 사랑함
	- 愛憎(애증) = 사랑과 미워함
		
	- 夂(뒤져올 치)[34]
		- 뒷짐을 지고 발을 땅에 끌며 걸어가는 모양, 머뭇거리며 뒤져옴. 
		- 항상 나를 앞지르는 사람, 대상이 있어서 뒤쳐지게 됨
		- 뒤져 오다, 천천히 걷는 모양
		
		- 冬(겨울 동) = = 夂(뒤쳐져 올 치) + ⺀(얼음 빙)
			§ 서서이(夂 ) 얼음(冫)(얼음 빙)이 얼기 시작하는 겨울
		- 冬節(동절) - 겨울철
		- 冬至(동지) - 24절기 하나, 밤이 가장 길고 낮이 가장 짧은 날
		
	- 夊(천천히 걸을 쇠) 
		- 발을 끌며 천천히 걷는 것을 뜻한다. 
		- 夂(뒤져올 치)와 모양이 거의 비슷하여 지금에 와서는 같이 쓰고 있다.
		- 夏(여름 하) = 頁(머리 혈) + 夊(천천히 걸을 쇠)
	

- ​爭(다툴 쟁) = 爫(손톱 조) + 又(오른 손, 또 우) + 亅(갈고리 궐)
	- 손톱으로(爫) 할퀴며 손(又)에 갈고리(亅)를 들고 다투다 [꼬불]
	- 줄을 놓고 서로 팽팽하게 잡아당기는 모습
	- 다투다, 결쟁하다
	
	- 骨肉相爭(골육상쟁) - 뼈와 살이 서로 다툼, 兄弟나 같은 民族끼리 서로 다툼을 뜻함
	- 百家爭鳴(백가쟁명) - 여러 사람이 서로 自己(자기) 主張(주장)을 내세우는 일
	- 犬兎之爭(견토지쟁) - 개와 토끼의 다툼, 兩者 싸움에서 第三者(제3자)가 利益(이익)을 봄
	- 內爭(내쟁) = 집안 싸움
	- 爭取(쟁취) = 투쟁하여 얻음
	- 鬪爭(투쟁) = 상대방을 이기려고 다툼
	- 競爭(경쟁) = 같은 목적을 두고 서로 이기거나 앞서거나 더 큰 이익을 얻으려고 겨루는 것
	- 爭點(쟁점) = 서로 다투는 重要한 點
	- 爭議(쟁의) = 서로 자기의 의견을 주장(主張)하여 다툼, 勞動爭議 준말
		
- ​淨(맑을 정, 깨끗할 정) = 氵(삼수변 수) + 爭(다툴 쟁)
	- 깨끗하다, 맑다, 사념이 없다 
	- 흐르는 물은 다투듯이 흐름
	- 오염되지 않은 말은 물
	- 오염된 것을 걸러내고 깨끗하게 유지하기 위해서는 淨化 과정이 필요
	
	- 淨水(정수) = 깨끗한 물
	- 淨水器(정수기) = 淨(깨끗할 정) + 水(물 수) + 器(그릇 기)
	- 淨淨(정정) = 맑고 깨끗함
	- 淸淨(청정) = 맑고 깨끗함, 더럽거나 속(俗)되지 않음
	- 淨化(정화) = 깨끗하게 함
	- 洗淨(세정) = 깨끗하게 빨거나 씻음
	- 淨書(정서) = 글씨를 깨끗하게 씀, 초잡을 글을 다시 바르게 베낌
