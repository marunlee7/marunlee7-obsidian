# 효정서당 2급 6강 士의 유형별 한자

![](https://www.youtube.com/embed/0eGs6lYbv4E)

## 士
- 士(선비 사)
	- = 조화로운 삶을 살기위해 학문을 하는 사람
	- = 열 십(十)은 음양 + 한 일(一)은 한결
	  	
- 壽(목숨 수, 장수 수)
	- 부수는 선비 사(士)
	- 사일공일구촌
	  
- 疇(밭이랑 주)
	- 늙을 노 + 일앙 주(밭이랑, 밭두둑 주)	
	- 老(늙을 노, 노련할 노, 노인 노)
	  
- 鑄(쇠불릴 주) 
	- 金 + 壽

- 禱(빌 도, 기도 도) 
	- 示(제사 시, 보일 시) + 壽
	- 신이게 자기의 목숨이나 운명을 빈다
	- 할 일이 있을 때 장수 해야 한다
	  	
- 仕(벼슬 사, 받들 사)
	- 공직에 있는 사람은 봉사정신이 투철
	  
- 吉(길할 길) 
	- 선비가 하는 말
	- ↔ 凶(흉할 흉)

- 桔(도라지 길)
	- 도라지는 풀 과인데 나무 목(木)을 넣음
	- 木은 생명, 도라지는 생명에 지장한 영향(호흡기, 폐)을 준다
		
- 結(맺을 결) 
	- 실로 길한 말로 맺다
	- 결혼(結婚)
	- 자매결연(姉妹結緣)
	
- 志(뜻 지)
	- 음양의 조화를 이루는 선비가 **뜻**을 세우다
	
- 誌(기록할 지)
	- 선비가 뜻을 세워 이루어 나가는 것을 기록(言)하다
	- 교지(校誌) 

## 壬

- 壬(천간 임, 짊어질 임, 북방 임)
	- 선비(士)의 에너지 삐침(丿)은 다른 사람의 짐을 짊어지다
	  	
- 任(맡길 임, 맡을 임)  
	- 담임(擔任)
	- 선생님(先生任)
		
- 賃(품삯 임)
	- 일을 맡기면(任) 돈(貝)을 줘야한다
	
- 姙(아이밸 임)
	- 부수는 女
	- 조물주가 여자에게 아이를 맡기다
	- 임산부(姙産婦)

- 淫(음란할 음)
	- 爪(손톱 조)  + 壬(짊어질 임) + 氵(물 수)
	- 남녀가 가까이 하다, 손으로 잡다, 음주가무 → 음란해지다

## 壬으로 보이지만 아래를 길게 쓰야한다.

- 壬=王(줄기 **정**, 곧을 정) 
	- **땅에서 곧게 싹이 올라오는 모습**
	- 맡을 임이 아니라, 줄기 정, 곧을 정
	- 인쇄체에는 王(임금 왕)처럼 나온지만 사실은 곧을 정의 변형이다
	- 呈(드러낼 정) 은 아래를 길게 쓰거나, 王 변형처럼 쓰야 한다
	- **하지만 모든 책에는 壬(천간 임: 아래가 짧은 형태)로 나오고 있다.**

- 呈(드러낼 정, 나타낼 정 )

- 程(과정 정, 헤아릴 정)
	- 禾(벼 화) + 呈(드릴 정)
	- 벼농사의 과정
	
- 廷(조정 정)
	- 조정일을 하는 사람은 곧고 바른사람이 나아가다
	- 청백리(淸白吏) : 청백한 관리
		
- 庭(뜰 정)
	- 정원(庭園)
	
- 聖(성스러울 성, 성인 성)
	- 귀로 들은바를 잘 드러내다(呈)
	- 성인(聖人)
