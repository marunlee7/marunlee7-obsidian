# 효정서당 2급 20강 㕣谷沿鉛船 台 古 豆 曲

![](https://www.youtube.com/embed/mBR3PwTs5hc)

## 㕣

- [[㕣]](숲속 늪 연)
	- 口(구) 산골짜기의 입구를 본뜬 것이고
	- 八(팔) 「시냇물에 접한 깎아지른 듯한 벼랑」
	- 물이 멈추어 있는 곳, 가장자리
	- 늪
	- 푸프딩딩 거무틱틱
- [[谷]](계곡 곡, 골 곡)
- [[沿]](물따라 갈 연)=氵(삼수변 수) + 㕣(산속 늪 연)
	- 㕣 물이 멈추어 있는 곳, 가장자리, 늪
	- 물이 흘러 어느 한 지점에 이르다
	- 沿海岸(연해안) = 바다와 맞닿은 부분의 육지
	- 沿革(연혁) = 변천되어 온 내력, 지나온 경과
	- 沿邊(연변) = 국경, 큰 길가, 강가 따위를 따라 있는 일대의 지방
- [[鉛]](납 연)
	- 鉛筆(연필)
- [[船]](배 선)
## 台
- [[台]](별 태, 기를 이): 별씨가 코와 입을 먼저 생성
	- 口(입 구) + 厶(사사 사)
- [[始]](처음 시): 엄마가 처음으로 아가를 가지다
	- [[始作]](시작)
	- [[始末書]](시말서)
- [[治]](다스릴 치) : 양수에서 잘 다스리다
	- [[治國]](치국)
	- [[治安]](치안)
- [[冶]](불릴 야) : 작은 아기씨가 불어나다
- [[怡]](기쁠 이) 
	- = 忄(심방변 심) + 台(별 태)
	- 기쁘다, 즐거워하다
- [[怠]](게으를 태) : 아버지의 씨(台) 엄마의 씨(台), 입덧이 심할 때 게을르지다
	- [[怠慢]](태만)
	- [[怠業]](태업)
- [[殆]](위태로울 태) : 아가가 충분히 먹지 못해 뼈만 남아 위태로워짐	
	- [[殆半]](태반)
	- [[殆無]](태무)
- [[苔]](이끼 태, 김 태)
	- [[海苔]](해태)
	- [[靑苔]](청태) : 푸른이끼
## 古
- [[古]](오래 고, 옛 고)
	- 십대째 입으로 내려오는
- [[枯]](마른나무 고, 마를 고)
	- 枯渴(고갈)
	- 枯木(고목)
- [[苦]](씀바귀 고, 쓸 고, 괴로울 고)
	- 苦海(고해) = 괴로운 인간세계
	- 苦痛(고통)
	- 苦盡甘來(고진감래)
- [[姑]](시어니 고)
	- 여자중에서 오랜된 사람
	- 姑婦(고부)
	- 姑從(고종)
	- 婦(며느리 부, 아내 부)
		- 姑婦(고부)
- [[故]](고향 고, 연고 고, 원인 고)
	- 故鄕(고향)
	- 故事成語(고사성어) = 옛날 있어썬 일에서 만들어진 語句
- [[固]](굳을 고)
	- 오랫동안 덮어두다
	- 固體(고체)
- [[個]](낱개 개)
	- 사람이 굳어버리다 → 물건 취급하다
	- 個人(개인)
	- 個別(개별)
- [[錮]](땜질할 고, 막을 고)=金(쇠 금) + 固(굳을 고)
	- 막다, 땜질하다
	- 가두다, 가로막다
- [[胡]](늙은소 멱줄 호, 오랑캐 호, 어찌부사 어찌 호)
	- 늙은 소의 미역줄기처럼 늘어진 목
	- 오랑캐 나를 치러 들어오는 적
	- 어찌 나를 치를 오는 것인가?	
	- 蝴蝶(호접)
		- 蝴(나비 호)
	- 丙子胡亂(병자호란)
- [[湖]](호수 호)
	- 江湖(강호)
	- 湖西(호서)
## 豆

- [[豆]](콩 두, 제기 두, 호리병 두)
	- 콩깍지를 보고 만듬
	- 콩까지를 가지고 제기를 만듬
- [[頭]](머리 두)
	- 콩까지 처럼 머리를 받치고 있다
- [[豈]](어찌 기)
	- 의문부사 어찌 기
	- 북을 치면서 오다 + 산을 넘어서
	- 어찌 전쟁 승리하여 오는가?
	- 豈非(기비 : 어찌 ~가 아니랴)
	- 豈唯(기유 : 어찌 다만 그것 뿐이랴)
- [[凱]](개선할 개)
	- 개선장군(凱旋將軍)
- [[鬪]](싸움 투) 
	- 鬥(싸울 두) + 豆(콩 두) + 寸(마디 촌)
	- 鬥(싸울 투, 싸울 두) = 손으로 싸움준비하다
	- 尌(멈출 주, 세울 수, 하인 수)
	- 링에서 법칙안으로 싸우다
	- 鬪志(투지) 
	- 戰鬪(전투)
- [[鼓]](북 고) = 壴(악기 이름 주) + 支(지탱할 지)
	- 북, 북소리, 격력하다
	- 호리병을 나뭇가지로 두드리다
	- 壴(악기 이름 주) = 十(열 십) + 豆(콩 두)
	- 鼓手(고수)
	- 申聞鼓(신문고)
- [[喜]](기쁠 희)= 口(입 구) + 壴(악기 이름 주)
	- 북을 치고 노래를 부르면 즐거워하다
	- 壴(악기 이름 주) = 지지대 위에 올려놓은 북
- [[樹]](나무 수, 세울 수)
	- 樹立(수립)
	- 樹液(수액)
	- 樹林(수림)
	- 樹木園(수목원)
## 曲
- [[曲]](굽을 곡, 바구니 곡, 음악 곡)
	- 바구니를 보고 만듬
	- 辰(별 진, 날 신, 3월 진, 용 진)
- [[農]](농사 농)
	- 3월에 농사짓기 위해 바구니에 씨앗을 가지고 나와 그때가 농사철
- [[濃]](짙을 농)
	- ↔ 淡(맑을 담)
- [[豐]](풍년 풍) = 𠁳(-) + 豆(콩 두)
	- 丰(예쁠 봉, 무성할 봉)
	- 산에 무성하게 있다 
- [[豊]](풍년 풍) = 豆(콩 두) + 曲(굽을 곡)
	- 豊富(풍부)
	- 豊盛(풍성)
