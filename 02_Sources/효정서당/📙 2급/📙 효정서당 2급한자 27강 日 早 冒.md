# 효정서당 2급 27강

![](https://www.youtube.com/embed/VZkOmjdkrAU)

## 日
- 日(해 일, 날 일)
	- = 따뜻하다, 밝다, 빛나다
	- 비교
		- 曰(가로 왈, 말씀 왈)
		- 冃(곁처덮을 모)
	
- 旦(아침 단)
	- 元旦(원단) - 설날 아침
	
- 但(다만 단)
	- = 사람이라면 아침에 일어나야 된다
	- = 다만, 오직
	- = 그러나, 그렇지만
		- 但只(단지)
		- 非但(비단)
		
- 壇(제터 단, 제단 단)
	- = 산보다 높이 제단을 쌓다
		- 敎壇(교단)
		- 花壇(화단)
		
- 亶(높을 단, 믿을 단) = 亠(높을 두) + 㐭(광 름) + 旦(아침 단)
		
	
- 檀(박달나무 단)
	- 檀君(단군)
	- 紫檀木(자단목)
	
- 冥(어두울 명, 저승 명)
	- = 日을 달로 보고 해석
		- 冥福(명복) - 죽은뒤 저승에서 받는 복
		- 冥想(명상)
	
- 得(얻을 득)
	- = 日을 貝(조개 패) 변형으로 본다
		- 所得(소득)
		- 得失(득실)
		
- 亘(펼 선)
- 申(펼 신)
	- = 천 인 지
- 亙(뻗칠 긍, 굳셀 긍)

- 宣(베풀 선)
	- = 文宣王(문선왕) - 공자
	- 宣誓(선서)
	- 宣戰布告(선전포고)
	
- 恒(항상 항)
	- 常(항상 상)
	- 恒時(항시)
	- 恒星(항성)
	
- 早(일찍 조, 이를 조)
- 草(풀 초)
	- = 풀은 아침일찍부터 자란다
- 卓(높을 탁, 뛰어날 탁)
	- = 해가 아침일찍부터 드러나다
		- 卓見(탁견)
		- 卓子(탁자)
		- 卓球(탁구)
- 悼(설퍼할 도) - 2급
	- 哀悼(애도)
	- 追悼(추도)
	
		
- 冒(어린이 복건 모, 무릎쓸 모) = 冃(곁처덮을 모) + 目(눈 목)
	- 冒險(모험)
	
- 帽(모자 모) = 巾(수건 건) + 冒(어린이 복건 모, 무릎쓸 모) 

- 曼(느즈러질 만, 끌 만)
	- = 손으로 눈을 당기듯 아래로 덮어졌다
		- 曼陀羅供(만다라공)
		
- 慢(늘어질 만, 게으럴 만) - 2급
	- = 마음이 느러짐
		- 自慢(자만)
		- 慢性(만성)
		
- 漫(퍼질만, 질펀할 만, 만화 만)
	- 慢畵(만화)
	- 浪漫(낭만)
	
- 鰻(뱀장어 만)
	- 鰻鯉(만리=뱀장어)
		- 鯉(잉어 리)
