# 효정서당 2급 7강 大,高의 유형별 한자

![](https://www.youtube.com/embed/LkCZ0d22Yqg)

## 夭
- 夭(예쁠 요, 일찍 죽을 요) 
	- 大(클 대) + 丿(삐침 별)
	- 예쁜 것은 손을 많이 돼 일찍 죽게되다
	- 생동감있게 움직이는 사람의 모양을 본뜬 글자		
	- 요절(夭折) : 젊은 나이에 죽음
	- 요요(夭夭) : 나이가 젊고 아름답다. 예쁜 복숭아같은 아가씨 [시경]
	- 요요작작(夭夭灼灼)
	
- 添(더할 첨) 
	- 氵(삼수변 수) + 忝(더럽힐 첨)
	- 더하다, 보태다, 덧붙이다
	- 하늘아래 계속 살다보며 자연(하늘)을 훼손하게 된다. 그리고 물방울(氵)처럼 조금씩 더해지다. 
	- 첨삭(添削)
	- 첨가(添加))
		
- 忝(더럽힐 첨) 
	- 天(하늘 천) + 㣺(마음 심)
	- 여기서는 夭(예쁠 요)가 아니라 天(하늘 천)이다.
	- 하늘아래 계속 살다보며 자연(하늘)을 훼손하게 된다. → 욕보이다, 더럽히다
	- 하늘을 대하는 마음을 나타낸 말로 황송하다, 더럽히다
	- 첨도(忝叨)
	- 첨관(忝館)
		
- 笑(웃음 소) 
	- 𥫗(대죽 죽) + 夭(예쁠 요, 일찍 죽을 요)
	- 대나무 이파리가 흔들리는 소리가 웃음 소리같다는 뜻이 합하여 웃음을 뜻함
	- 미소(微笑) : 소리를 내지 않고 빙긋이 웃는 것
	- 담소(談笑)
	- 박장대소(拍掌大笑) : 손뼉을 치면서 크게 웃음 
	
- 送(보낼 송) : 웃으면서(笑) 떠나 보낸다는 뜻이 합하여 보내다
	- 送은 笑를 획줄임 한 것이다.
	-  辶(쉬엄쉬엄 갈 착 : 먼 길) + 笑 关(웃을 소)
	- 송구영신(送舊迎新) : 묵은 해를 보내고 새해를 맞는다
		- 맞이할 때는 무릎꿇고 맞이하다

## 高
- 高(높을 고)
	- 창문있고 아래에 창문이 있다.
	
- 膏(기름질 고) 
	- 月(육달 월) + 高(높을 고)
	- 고량진미(膏粱珍味) : 살진 고기와 좋은 곡식으로 만든 맛있는 음식 			
	
- 稿(볏짚 고, 원고 고) 
	- 禾(벼 화) + 高(높을 고)
	- 볏집이 올라가다
	- 원고(原稿)
	- 원고료(原稿料)
	- 투고(投稿)
					
- 矯(바로잡을 교)
	- 矢(화살 시) : 화살은 길이를 재는 자로 쓰임
	- 교도소(矯導所)
	- 교각살우(矯角殺牛)
	
- 喬(높을 교)
- 橋(다리 교)
- 驕(교만할 교)
	
- 鎬(**냄비 호**, 호경 호)
	- 다리가 긴 큰냄비
	- 이름에 호(鎬)가 들어가면 많은 사람의 밥그릇을 책임지는 리더가 되라는 의미
	- 호경(鎬京)
	
- 蕎(메밀 교)
- 轎(가마 교)
- 嬌(아리따울 교)

- 毫(터럭 호)
	- 高 + 毛
	- 세워진 털
	- 추호(秋毫) : 가을철에 털을 갈아서 가늘어진 짐승의 털이란 뜻으로
		- 몹시 작음을 비유하는 단어

- 豪(**호걸 호**, **굳셀 호**, 호화로울 호)
	- 高 + 豕(돼지 시)
	- 돼지가 털을 세우고 굳세게 나가다
	- 영웅호걸(英雄豪傑)
	- 호화(豪華)
	- 호탕(豪宕)

- 濠(해자 호, 호주 호)
	- = 壕(해자 호)
	- 해자(垓子) : 성 주위에 둘러 판 못
	- 방공호(防空壕)
	- 호주(濠洲)
