# 효정서당 2급 02강 可

![](https://www.youtube.com/embed/cVfcydRfZjE)

## 可
- 可(옳을 가, 가히 가)
	- 내 입에서 나가는 에너지가 갈고리처럼 강하다는 뜻
	- 정확하게 옳다고(丁) 입(口)으로 말하다
	- 부족하지만, 최선을 다하겠다 [ 가히 가]
	- 丁(못 정) 에서 새로운 음인 可(옳을 가)가 나옴
	- 可(옳을 가)에서 음이 결정되면 이제 부수(뜻)만 바뀐다
	- 가능(可能) : 可(옳을 가) + 能(능할 능)
	- 가결(可決)
	- 認可(인가)
	- 可視光線(가시광선)

- 哥(소리 가): 옳습니다, 반복해서 소리를 내다
	- 네. 예
- 歌(노래 가): 소리를 반복해서 내면서 입을 크게 벌리고 행동까지 같이함
	- 哥(소리 가) + 欠(하품 흠)
	- 가수(歌手)
	- 가곡(歌曲)
	- 가요(歌謠)
	- 가사(歌詞)

- 阿(언덕 아, 아첨 아, 아름다울 아) 
	- 阝(언덕 부) + 可(옳을 가)
	- 언덕은 층이 나있어, 올라가려면 고개가 끄떡거려요
	- 정확하지 않은데도 끄덕거릴 경우 아첨하다, 아첨할 아
	- 완벽한 아름다움이 아니라 겉모습이 아름답다 아
	- 아방궁(阿房宮) : 지나치게 크고 화려한 집을 비유적으로 이르는 말
	- 아편(阿片) 
	- 아형(阿兄)
- 河(물 하) 
	- 물이 강력하게 시작하는 곳
	- 대륙의 중원 황하(黃河)를 뜻함
	- 하천(河川)
	- 황하(黃河) : 누른 황토물이 흐르는 강
	- 하해(河海)
- 何(어찌 하) 
	- [의문부사] 육하원칙이 다 들어가 있음
	- 사람이 왜, 무슨 이유로 옳다고 말하는가?
	- 하등(何等)
	- 하여가(何如歌)
- 荷(연꽃 하, 짐 하, 멜 하) 
	- 연꽃은 땅에 살지 않고 물위에 뜨있음
	- 연꽃이 물위에 떠 있듯이 나그네가 짐을 지고 다닌다는 의미로 메다/짊어지다 의 뜻도 됨
	- 荷花(하화) : 연꽃,  수련과의 여러해살이 수초
	- 荷役(하역) : 짐을 싣고 내리는 일
	- 荷役夫(하역부) : 하역(荷役)에 종사(從事)하는 사람
	- 手荷物(수화물) : 여객이 여행에 필요한 것으로 휴대하는 물품
- 奇(기이할 기, 기특할 기) : 大(큰 대) + 可(옳을 가)
	- 엄청 큰 일을 해낼 수 있는 사람	
	- 기적(奇蹟)
	- 기특(奇特)
	- 기이(奇異)
- 寄(부칠 기, 붙을 기, 의지할 기)
	- 宀(집 면, 갓머리) + 奇(기특할 기)
	- 큰 사람은 집위에 붙을 수 있다
	- 붙을 기 일때
		- 寄生蟲(기생충) 
			- : 다른 동물의 양분을 흡수하여 사는 벌레 혹은 남에게 의지하여 사는 사람
		- 寄宿舍(기숙사)
	- 부칠 기로 사용될 때
		- 寄付(기부) 
			- 付(붙을 부)
		- 寄與(기여)
		- 寄贈(기증)
			- 贈 (줄 증)
- 騎(말탈 기) 
	- 馬(말 마) + 奇(기특할 기)
		- 騎士(기사)
		- 騎士道(기사도)
		- 騎馬戰(기마전)
		- 騎手(기수)
