## 刀
- 刀,  刂(칼 도, 선칼도 방)  [p.297]
	- 가로부터 쓰고 세로 삐침을 쓴다
	- 의미 : 생활도구, 자르다(가르마 가른다), 나누다, 분명하다
	
- 初(처음 초) 
	- 衣, 衤(옷 의) + 刀(칼도)
	- 옷을 처음 만들려면 칼로 잘라야 한다
	- 처음 초
	
- 刃(칼날 인)
	- 칼에서 불똥(丶 불똥 주)같이 빛나는 칼날
	
- 忍(참을 인)
	- 칼날로 마음을 베다
	- 인내는 쓰고 그 열매는 달다
	- 忍(참을 인) 세번이면 죽음도 면한다
	
- 認(알 인, 인정할 인) 
	- 말을 끝까지 참고 들어본 후 내용을 알아 인정한다 [꼬불]
		- 인식(認識) :  사물을 분별하고 판단하여 아는 일
		- 인정(認定) : 
		- 인가(認可) : 인정하여 허락함
	
- 刅(상처 창)
	- 칼날을 몸에 살짝 닿기만 해도 상처가 생김
	
- 梁(돌다리 양(량), 다리 량, 들보 량, 대들보 량)
	- 나무를 상처내어 물 위에 걸쳐놓다
		- 양상군자(梁上君子) : 대들보 위에 있는 군자라는 뜻으로, 도둑을 미화하여 젊잖게 부르는 말
		
- 那(어찌 나, 나라 나) 冄(머리 늘어질 염)  + 邑
	- 冄 : 머리를 길게 늘어뜨린 민족들이 사는 고을
		- 의문부사 어찌 나
	
- 召(부를 소): 칼처럼 입으로 말을 정확하게 부르다
	- 소명(召命)
		
- 昭(밝을 소): 내가 부른듯이 해가 나온다
	- 소상(昭詳): 분명하고 자세함.
	
- 照(비칠 조, 비출 조): 해가 부른듯이 나오면 인공불(불화 발) 처럼 더욱 밝아지다
	- 조명등(照明燈) : 
	
- 招(부를 초): 손으로 정중하게 부르다
	- 召(부를 소): 윗사람이 아랫사람을 부르다
	- 招(부를 초): 동등한 입장에서 정중하게 부르다
		
- 超(뛰어넘을 초): 누가(조물주) 부른듯이 달려가서 뛰어넘다
	- 走(달릴 주, 달아날 주) + 召(부를 소)
	- 초월(超越) : 超(뛰어넘을 초) + 越(넘을 월)
			
- 別(가를 별, 다를 별,  나눌 별, 이별할 별) : 살과 뼈를 칼로 가르고 있는 모습
	- 刂(칼 도, 선칼도 방) + 무릎, 뼈(口) + 근육, 살(力)
	- 冎(뼈살 발라낼 과)
		- 별거(別居) : 한 집안 식구로 따로 나가 삶
		- 별명(別名) : 
		- 별미(別味) : 별다른 맛
		- 별세(別世) : 죽음을 높여 이르는 말
		- 특별(特別 : 

- 列(벌릴 열) 
	- (선칼도 방) + 歹(살 바른 뼈 알)
	- 사람이나 물건이 여럿이 죽 벌여 서거나 늘어 선 줄
	- 가로된 것을 횡렬(橫列), 길이로 된 것을 종렬(縱列). 행렬(行列)
	- 벌이다(일을 계획하여 시작하거나 펼쳐 놓다. 놀이판이나 노름판 따위를 차려 놓다. 여러 가지 물건을 늘어놓다.)
	- 늘어서다, 줄짓다
	- 진수열장(辰宿列張): 성좌가 해, 달과 같이 하늘에 넓게 벌려져 있음
