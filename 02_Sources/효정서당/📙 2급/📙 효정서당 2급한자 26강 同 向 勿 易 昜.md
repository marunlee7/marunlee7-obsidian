# 효정서당 2급 26강 同 尙 勿 易

![](https://www.youtube.com/embed/9szQCTxB7h8)

## 同
- 同(한가지 동, 같을 동):凡(무릇 범) + 口
	- 한 가지, 함께, 같다, 무리
	- 凡 무릇, 모두 + 口 사람의 입
	- 모두가 말을 하다, 즉 이야기를 함께 나누다, 여러이 모여 한목소리는 내는 모습
	- 같다 : 여러 사람(冂) 의 말(口)이 하나(一)로 모인다
	- 同生(동생) : 아우나 손아래 누이
	- 同化(동화) : 서로 다른 물건이 닮아서 같게 됨
	- 同僚(동료) : 같은 곳에서 같은 일을 보는 사람
	- 公同(공동) : 여러 사람이 일을 같이 함
	- 同期(동기) : 같은 시기, 동기생
	- 同行(동행) : 길을 같이 감, 또는 그 사람
	- 同時(동시) : 같으 때
	- 同胞(동포) : 같은 어머니로부터 태어나 형제, 자매
	- 同意(동의) : 같은 의견, 같은 意思(의사)
	- 混同(혼동) : 이것과 저것을 구별하지 못하고 뒤섞어서 보거나 생각함
	- 同窓(동창) : 같은 학교에서 공부를 한 관계
	- 同伴(동반) : 데리고 함께 다님, 길을 같이 감
- 洞(골 동, 마을 동, 꿰뚫을 통, 밝을 통)
	- 물이 아주 좋은 곳을 선택하여 마을을 형성하여 살다
	- 산의 지형도 알아야 하고, 먹거리도 잘 알아야 한다
	- 하천을 중심으로 한데 모여 산다는 의미
	- 마을, 동굴
	- 洞察(통찰) : 환히 내다봄
		- 察(살필 찰)
	- 洞察力(통찰력) : 사물을 환히 꿰툻어 보는 能力
	- 洞燭(통촉) : 깊이 헤아려 살핌
		- 燭(촛불 촉)
	- 洞窟(동굴) : 깊고 넓을 굴
	- 洞里(동리) : 마을
	- 洞內(동내) : 동네 안
	- 고을 연관단어
	- 坊(동네 방)
	- 府(관청 부, 마을 부)
	- 村(마을 촌)
	- 穴(구명 혈, 굴 휼)
	- 窟(굴 굴)
	- 署(관청 서, 마을 서)
	- 邨(마을 촌)
	- 郡(고을 군)
- 銅(구리 동)
	- 광물에 관련된 것은 金이 들어간다
	- 구리, 동기(銅器)
	- 銅像(동상) : 구리로 그 사람의 형상을 만들어 세운 기념상의 총칭
	- 銅錢(동전) : 구리로 만든 돈
	- 靑銅(청동)
	- 靑銅器(청동기) : 청동으로 마든 기구(器具)를 두루 이르는 말
	- 金(쇠 금)
	- 銀(은 은)
	- 銅(구리 동)
- 桐(오동나무 동)
	- 가운데가 뚫여있어 물이 잘 빠지고, 재질이 가볍고 공명현상이 있다.
	- 가야금이나 가벼운 장롱 만들때 사용
	- 梧桐一葉(오동일엽)
		- 오동나무 한 잎
		- 오동잎 하나가 떨어지는 것을 보고 가을이 왔음을 안다
		- 한 가지 구실을 보면 일의 전말을 알 수 있다

## 向
- 向
	- 향할 향 
	- 창문 바깥에서 들어오는 북쪽에서 불어오는 바람은 남쪽으로 나간다	
		- 向學熱(향학열)
		- 南向(남향)

- 尙
	- 높힐 상, 오히려 상(부사로)
	- 창문에 더 높이 내다	
	- 高尙(고상)
	
- 常
	- 항상 상
	- 일상을 높이기 위해 옷을 입기 시작, 상대에게 예를 갖추는 행위, 지위를 나타냄	
		- 恒常(항상)
			- 恒(항상 항)
				- 우리 마음이 항상 태양과 같다
		- 恒星(항성)		
- 裳
	- 치마 상, 아랫도리 상
	- 치마는 높혀서 입다
		- 衣裳(의상)
		- 綠衣紅裳(녹의홍상) 
			- 연두저고리와 다홍치마
			- 곱게 차려 입은 젊은 아가씨
		
- 嘗
	- 맛볼 상, 일찍이 상)
	- 旨(맛 지, 뜻 지) + 尙
	- 맛을 높이기 위해 미리 간을 보다
	- 일찍이, 과거에 겪다, 맛을 보다
	
	- 臥薪嘗膽(와신상담)
		- 섭에 눕고 쓸개를 씹는다
		- 원수를 갚으려고 온갖 괴로움을 참고 견딤
	
- 賞(상줄 상, 즐길 상)
	- 그 사람을 높이기 위해 상을 주다
	- 賞春客(상춘객) : 봄 구경을 나온 사람 [즐길 상]
	
- 償(갚을 상)
	- 상을 받거나 돈을 빌리면 반드시 갚아야 한다
	
- 堂(집 당)
	- 흙을 높혀 집을 짓게 되다
	
- 當(마땅 당)
	- 높은 지역땅은 마땅히 밭으로 쓰인다
	
- 黨(무리 당) : 
	- 尙 + 黑
	- 높이는데 창문에 이게 연기가 나가는 거예요. 
	자 불을 뗐을 때 이건 거무르이죠. 거무르 검은 연기가 나간다. 
	불 뗐을 때 창문에 검은 연기가 나가는 거죠. 
	경제가 어려울 때 암울하다고 그래요 그것을 우리가 모여가지고 좋은 지혜로 헤쳐나간다. 그런 뜻을 가진 것이 무리당이에요. 
	지금 현대판으로는 우리가 여당이 있고 야당이 있듯이
	우리가 물에 지어서 좀 더 나은 이 어려움을 헤쳐나갈 그런 좋은 지혜를 계획을 짜는 그런 분들을 이제 무리당이라고 합니다. 
	- 與黨(여당)
	- 野黨(야당)

- 掌(손바닥 장) 
	- 如反掌(여반장)
## 勿
- 勿(말 물, 깃발 물)
	- ~하지 말라
	- 깃발이 휘날리는 모습
	- 非禮勿視, 非禮勿聽(비례물시 비례물청)
	- 非禮勿言, 非禮勿動(비례물언 비례물동)
	- 예가 아닌 것은 보지 말고, 예가 아닌 것은 듣지 말아야 함.
	- 예가 아니면 말하지 말고, 예가 아니면 움직이지 말아야 함.
	
- 物(물건 물)
- 忽(소홀히 홀, 갑자기 홀)
- 惚(황홀할 홀)

## 易
- 易(바꿀 역, 쉬울 이)
	- 周易(주역) : 중국 점에 관한 책, 오경의 하나로 중국 주나라 때의 철학서
	- 바꾸면 쉬울 수 있다 → 변해야 산다
	- 바꾸지 않고 그 자리에서 계속 진흙밭에 있는 것은 어려운 일이다
	- ↔ 難(어려울 난)
	
	- 난이도(難易度) : 어려움과 쉬움의 정도
	
- 賜(줄 사, 사약 사, 윗사람 선물 사)
- 旦(아침 단)

## 昜
- 昜(볕 양)
	- 해가 높이 떠오르면 햇살이 구불구불 퍼져 나가는 모습
	- 중천에 뜬 햇살이 구불구불(勿)하게 펴지는 모양을 본뜸
	- 旦(아침 단) + 勿(말 물)

- 陽(볕 양, 양지 양)
	- 볕(昜)이 언덕(阝)에 비칠 때 
		- 음양(陰陽)
		- 양지(陽地) : 햇볕이 바로 드는 곳.
	- 반대말은 陰(그늘 음, 음지 음)
		- 언덕에 지금까지 구름이 끼었다.


- 場(마당 장) : 마당에 햇살이 잘 들다

- 揚(드날릴 양)
	- 깃발이 휘날리듯 이름을 드날리다

- 楊(버드나무 양, 성씨 양)
	- 수양버들
	- 부처 약사여례상
	- 버드나무에서 진통제가 나옴

- 腸(창자 장) : 
	- 우리 몸에 신체(창자)가 있다
	- 내 몸의 창자는 볕처럼 따뜻하게 해줘야 한다
	
- 暢(화창할 창) : 申(펼 신) + 昜(볕 양)

- 湯(끓일 탕) : 태양이 햇살이 구불구불하듯 탕이 끓는 모습
	
- 傷(상처 상, 다칠 상) : 
	- 아군 + 적군은 아침에 일찍 전쟁을 선포, 깃발을 훼손시킴
	- 똑같이 상처를 입음
	- 昜(볕 양) 태양이 제단 위를 비추는 모습을 그린 것으로 ‘볕’이나 ‘양지’라는 뜻이 있다
	- 중경상(重輕傷)
	- 상이군인(傷痍軍人)
	- 상이(傷痍) : 부상(負傷)당함
	- 상처(傷處) : 몸의 단친 자리
	- 부상(負傷) : 몸에 상처를 입음
	- 손상(損傷) : 물체가 깨지거나 상(傷)함
	- 상해(傷害) : 남의 몸에 상처를 내어 해를 입힘
