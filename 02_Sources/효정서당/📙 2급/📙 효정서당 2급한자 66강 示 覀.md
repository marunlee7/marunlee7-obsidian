# 효정서당 2급 66강 示, 覀(덮을 아)의 유형별 한자

![](https://www.youtube.com/watch?v=j6UqGIKSoqQ&list=PL5NRjbnnZamSxqNm-9qN2VFoD0UwLAJaL&index=65&pp=iAQB)

- 示(보일 시)
	- 礻(보일시변)
	- 보일 시: 제물(祭物)을 차려 놓은 제단의 모양, 祭物을 신에게 보여 준다는 意味로 보이다.
	- 제사 시: 신, 조상의 뜻

- 宗(으뜸 종, 마루 종)
	- 제사를 지내는 집이 으뜸이면서 종가집이다.
	- 마루=최고 높은 것, 용마루
	- 집(宀)에 신(示)을 모셔 두는 곳이 마루요, 종가 댁이다. P280
	- 宗敎(종교): 신의 힘이나 초자연적인 존재에 대한 신앙과 숭배.

- 綜(모을 종)
	- 실을 종가(宗)로 집결시켜 모은다.
	- 綜合(종합): 여러 갈래로 나누어진 부분을 한데 합침.
	
- 崇(높을 숭)
	- 산(山)처럼 종가(宗)댁은 촌수가 높다. P280
	- 高(높을 고)
	- 尙(높일 상)
	- 崇拜(숭배): 높이 우러러 존경함.
	- 崇尙(숭상): 높이어 소중하게 여김
	- 崇高(숭고): 존엄하고 고상함
	
- 嵩(높을 숭)

- 祭(제사 제)
	- 제사 제: 고기를 오른손에 들고 제사상에 올리다. 
	- 고기(⺼)와 또(又) 다른 제물을 제단(示) 위에 올려 놓고 신에게 제사 지낸다. P199
	- 祭祀(제사): 신령에게 음식을 바치어 정성을 표하는 예절.
	- 祭典(제전): 제사의 의식
	
- 際(즈음 제, 사귈 제, 때 제)
	- 즈음 제: 언제쯤 제사를 지내면 많은 사람들이 모일까?
	- 哀(슬플 애): 부모님이 돌아가시면 애사다.
	- 제사는 祝祭(축제), 慶事(경사)
	- 언덕(阝)에서 제사(祭) 지낼 때 친척들을 만난다 하여 사귄다는 뜻이다. P199
	- 交際(교제): 서로 사귐.
	- 交(사귈 교): 정신적인 부분으로 서로 주고 받다
	- 實際(실제): 실지의 경우나 형편
	
- 察(살필 찰)
	- 집(宀)안에 어르신이 제사(祭)지낼 때 제물을 살핀다. P199
	- 考察(고찰): 생각하여 살핌.
	- 視察(시찰): 돌아다니며 실지 사정을 살펴봄.
	- 觀察(관찰):


- 擦(문지를 찰)
	- 손(扌)으로 살펴(察)서 더러운 곳을 문질러 닦는다. P199
	- 擦傷(찰상): 스치거나 문질러서 벗겨진 상처
	- 摩擦(마찰): 서로 맞대어서 비빔.
	
- 缶(장군 부, 질그릇 부)
	- [121]
	- 午(낮 오) + 凵(입 벌릴 감)
	- 항아리에 두껑이 있어 젓갈도 담아둔다.
	- 장군(배가 불룩하고 목 좁은 아가리가 있는 질그릇)
	- 다른 글자와 결합할 때는 缸(항아리 항)자나 陶(질그릇 도)자처럼 토기와 관련된 의미를 전달한다.

- 䍃(질그릇 요, 항아리 요, 노래 요)
	- 질그릇(잿물을 덮지 아니한, 진흙만으로 구워 만든 그릇)
	- 항아리 요: 솥두껑에 고기를 구어먹을 때 사용하는 항아리
	- 소리 요: 항아리 솥두껑에 고기를 구어먹을 때 나는 탁, 탁 소리
	- 항아리 등 악기 두드리는 소리, 소리
	
- 謠(노래가락 요)
	- 가락을 넣어주다
	- 말(言)을 길게 뽑아 고기(⺼)안주를 먹고, 질그릇(缶)으로 술 마시며 노래해요. P199
	- 童謠(동요): 
	- 歌謠(가요): 
	- 民謠(민요): 한 민족의 독특한 노래
	- 䚻(노래 요)
	- 歌(노래 가): 사람이 힘을 크게 벌려서 노래소리를 내다
	
- 搖(흔들 요)
	- 질그릇같은 요람을 묶어서 손으로 흔들다.
	- 손(扌)으로 고기(⺼)를 질그릇(缶)에 담아 흔들어요. P199
	- 搖動(요동): 물체가 흔들리어 움직임.
	- 搖籃(요람): 흔들리도록 만든 물건, 어떠 사물의 발생지.
	
- 遙(멀 요, 멀어질 요)
	- 항아리 소리가 멀리까지 가다
	- 고기(⺼) 담긴 그릇(缶)을 가지고 뛰어(辶) 멀리 달아난다. P199
	- 遙遠(요원): 공간적으로 까마득히 멂.

- 四(넉 사)
	- 四骨(사골): 소의 네 다리의 뼈
	- 四民(사민): 네 가지의 백성, 선비, 농민, 장인, 장사치
	
- 柶(윷 사)
	- 擲柶(척사)
	- 擲(던질 척):扌(재방변 수) + 鄭(정나라 정)
	
- 西(서녘 서)
	- 西, 襾, 覀(덮을 아, 6획)
	- 해가 지면 새가 발을 들여놓은 둥지
	- 해가 지면 새는 둥지로 들어가니 그 모양(西)을 본떠서 서쪽을 뜻한다. P271
	- 西歐(서구): 서부 유럽의 어려 나라
	- 西風(서풍): 서쪽에서 불어오는 바람

- 罰(벌줄 벌, 벌할 벌)
	- 罒, 网, 罓(그물 망)
	- 詈(꾸짖을 리): 법망에 걸려 말씀으로 꾸짖음을 받다
	- 법망(罒)에 위배되는 말(言)한 자를 칼(刂)로 쳐서 벌을 준다. P140
	- 罰金(벌금):
	- 罰則(벌칙): 
	
- 罪(허물 죄): 법망에 어그러져 죄가 되다.
	- 非(아닐 비, 어그러질 비)
	
- 栗(밤나무 률)
	- 覀(덮을 아)+나무 목(木)
	- 가시로 덮여있는 밤나무 3알, 밤껍질, 율피
	- 棗栗梨柹(조율이시)
	- 棗(대추 조)
	- 栗(밤나무 율)
	- 梨(배나무 리)
	- 枾(감나무 시)
	
	- 서쪽(西)에 서 있는 나무(木)가 밤나무이다. P271
	- 栗房(율방): 서밤송이
	- 生栗(생률): 흔히 잔치나 제사때에 나부죽하게 쳐서 깍은 날밤
	
- 粟(조 속)
	- 米(쌀 미) + 覀(덮을 아)
	- 粟米(속미): 조와 쌀
	- 粟粒(속립): 

- 票(표 표, 불똥튈 표)
	- 覀(덮을 아)+잘 보이다. = 작은 티켓
	- 서쪽(西)에서 보이게(示) 달아놓은 쪽지. P271
	- 票決(표결): 투표로 可否(가부)를 결정함
	- 改票(개표): 투표함을 열고, 결과를 조사함.
	
- 標(표할 표)
	- 간판같이 큰 것
	- 나무(木)끝에 쪽지(票)를 매달아서 표시한다. P271
	- 標本(표본): 하나를 가지고 다른 한 종류의 표준을 삼는 물건.
	- 標的(표적): 목표로 삼는 물건.
	
- 漂(뜰 표)
	- 물(氵)위에 쪽지(票)가 떠다닌다. P271
	- 漂流(표류): 둥둥 물에 떠서 흘러감
	- 漂白(표백): 빨아서 하얗게 함. 바래서 희게 함
	
- 遷(옮길 천, 귀양보낼 천)
	- 䙴(가마 천): 가마 덮개를 양손으로 잡고 무릎을 올린 모습.
	- 가마타고 먼길을 옮겨 가다
	- 천서쪽(西)으로 큰(大) 죄지은 몸(己)이 뛰어가는(辶)듯 급히 짐을 옮긴다. P271
	- 遷都(천도): 도읍을 옮김.
	- 遷善(천선): 나쁜 짓을 고쳐 착하게 됨.

	- 孟母三遷之敎(맹모삼천지교): 맹모가 맹자의 교육을 위해 세 번이나 이사를 한 가르침.
