# 효정서당 2급 08강 大의 유형별 한자(2)

![](https://www.youtube.com/embed/10C8Ow01ZZw)

## 太
- 大(큰 대) : 사람의 모습을 보고 만든 글자
- 天(하늘 천) : 큰 대와 같은 모양으로 소우주일때는 **내 머리**가 하늘이다. 
	- 머리에서 밝은 생각을 해야한다. 
	
- 太(**클 태**, **처음 태**)
	- = 大 + 大
	- 세상이 아무리 크더라도 그 중심은 바로 **나**다.
	- 태극기 왼쪽
		- 시작은 바로 나로부터
		- 가는날 까지 하늘과 땅의 에너지를 모두 섭렵하여 생명을 이루는 것 → 極(끝 극, 지극할 극, 다할 극)	
	- 極(끝 극, 지극할 극, 다할 극)
		- 태극(太極) : 우주 만물의 근원
		- 북극(北極)
		- 지극정성(至極精誠)
	
- 奈(어찌 나(내), 능금 내)
	- 제사상에 선조들이 먹던 조율이시중에서 능금(사과)
	- 어찌 이 좋은 과일을 얻었는가?
	- 의문부사 어찌, 어떻게 어지하랴, 어찌 내(나)

- 丈(어른 장, 지팡이 장)
	- 오른손(又)에 지팡이를 잡고 있는 모습
	- 杖(지팡이 장, 어른 장)	
	- 장인(丈人)
	- 장모(丈母)
## 夫
- ReadItLater Inbox/夫(지아비 부, 사내 부, 대저 부)
	- 天 + 大人(하늘과 대인)
	- 3인칭 대명사, **대저**(대체로 보아서)로도 쓰인다. → 대저 부
	- 부부(夫婦)
		- 婦(지어미 부, 아내 부)
		
- 扶(도울 부) 
	- 지어미가 지아비를 돕듯이
	- 扌 + 夫	
	- 부양(扶養) : 스스로의 힘으로 살아 갈 수 없는 사람의 생활을 돌봄
	- 부조(扶助) : 남의 큰 일에 돈이나 물건 등을 도와줌
	- 부조금(扶助金) : 큰일을 치르는 데에 도우려고 내는 돈
	- 상부(相扶) : 서로 도움
	- 상부상조(相扶相助) : 서로서로 도움
	- 억강부약(抑强扶弱) : 강자를 누르고 약자를 조와줌
			
	- 協(도울 협, 화할 협) : 여러 사람이 돕는 것		
		- 협조(協助)
		- 협동(協同)		
	- 助(도울 조) : 지속적으로 한 사람이 돕는 것
		- 且(또 차, 버금 차) + 力 	
	
- 替(바꿀 체) 
	- 지아비가 서로 말을 주고받고 있슴, 일방적이지 않음
	- 夫 + 夫 + 曰
	- Change
	- 세대교체(世代交替)
	- 대체(代替) : 다른 것으로 대신(代身)함.

## 矢
- 矢(화살 시)
	- 화살의 디자인
	- 길이를 잴 때
	- 쏘세요 하면 쏜 화살처럼 빠르다의 느낌
	- 정직하다, 곧다

- 知(알 지)
	- 화살처럼 바로바로 말귀를 알아 들음
	- 하는 말을 바로
		- 지식(知識)
		- 지행일치(知行一致) : 지식과 행동이 일치함
		
- 智(지혜 지, 슬기로울 지) 
	- 知 + 日
	- 세상을 두루 바르게 안다
	- 지혜로운 분은 밝고 따뜻하게 지낸다
		
- 疾(병 질, 빠를 질)
	- 쏜 화살처럼 빠르게 퍼지다
	
- 短(짧을 단, 모자랄 단)
	- 화살은 길이를 잴 때 사용
	- 짧은 대나무를 중심으로 화살을 많이 만들다
	- 단점(短點)
	- 장점(長點)
		- 長(길 장, 넉넉할 장, 장정 장)
	
## 失
- 失(잃을 실)
	- 손에서 잃은 것
	- 失(잃을 실) → 得(얻을 득) → 德(큰 덕)
	- ↔ 得(얻을 득) : 貝(돈) + 寸(법도)
	- 득실(得失)
	- 도덕(道德) : 사람으로서 지켜야 할 도리		
	
- 秩(차례 질) 
	- 낫가래를 잃지 않도록 **차례 차례** 묶어놓다
	- 禾 + 失
	- 질서(秩序)
		- 序(차례 서)
		
- 矣(어조사 의)
	- ~었다, ~리라, ~이다, ~뿐이다, ~느냐	
	- 여의도(汝矣島) : 맑은 물이 한쪽으로 모이는 섬
	- 만사여의(萬事如意) : 모든 일이 뜻하는 대로 잘 됨
	- 만사휴의(萬事休矣) : 만 가지 일이 끝장이라는 뜻. 전혀 가망이 없는 절망의 상태
		- 休 = 勿
