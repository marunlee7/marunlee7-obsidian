## 아기머리 거꾸로 나올 돌

- 㐬(양수 타고 흐를 류, 깃발 류) 
	- 어린아이([![15](https://dic-phinf.pstatic.net/20210331_105/1617188713639qDntD_JPEG/__%28%29.jpg)](https://hanja.dict.naver.com/#)) + 물살
	- 아기머리거꾸로 나올 때 양수 타고 나오다 
	- 또는 아이가 물에 떠내려가는 모습
		- 길을 가는데(疋) 막힘이 없다(㐬)
		- 길을 걷는 것이(疋) 물 흐르듯이 매우 순조롭다(㐬)
	
- 流(흐를 류) 
	- 氵(삼수변 수) + 㐬(깃발 류(유))
	- 水자를 더해 본래의 의미를 더욱 강조
	- 교류(交流) : 서로 주고받음
	- 표류(漂流) : 물에 떠서 흘러감

- 疏(드물 소, 트일 소, 소통할 소) 
	- 疋(발 소, 필 필, 짝 필) + 㐬(깃발 류)
	- 소통하다, 트이다 
	- 疋(어머니 배 속에서 발차기를 하는 모습)
	- 아가가 발차기를 하면서 머리를 돌려서 드물게 흘러 나오다		
	- 소통(疏通)
	
- 蔬(나물 소)
	- 싹이 나올때는 풀 초
	- 아기가 나올때 양수타고 나오듯 풀(艹)도 조금씩 박차고 나오는 모습

	- 나물(사람이 먹을 수 있는 풀이나 나뭇잎 따위, 이것을 양념하여 무친 음식)
	- 푸성귀(식용을 위해 직접 기른 채소)
	- 변변치 못하다(고기반찬에 비해 나물 반찬을 하찮게 여겨)
	- 채소(菜蔬) :  
	- 소반(蔬飯) : 변변치 못한 음식(飮食)
	
## 育
- 育(기를 육) 
	- 아기머리 거꾸로 나올 돌 + ⺼(육달 월 : 기르다)
	- (육체적으로) 기르다, 낳다
	
- 充(채울 충, 가득할 충)
	- 어린아이가 나오면 인문학적으로 어진사람으로 기르다, 채워나가다 → **채울 충**
	- (정신적으로) 기르다
	
- 徹(통할 철, 뚫을 철) 
	- 彳(조금 걸을 척: 행동) + 育(기를 육) + 攵(칠 복: 지도, 매질)
	- 통하다, 관통하다
	- 꿰뚫다(매질하며 힘을 돋우어 주어 가르치면 무엇이든 알 수 있게 된다)
	- 철두철미(徹頭徹尾) : 머리에서 꼬리까지 통한다. 
		- 처음부터 끝까지 방침(方針)을 바꾸지 않고, 생각을 철저히 관철함
	- 비교하기
		- 徹(통할 철)
		- 通(통할 통)
		- 疏(통할 소)
		
- 棄(버릴 기) 
	- 삼태기에 아가를 담은 모습
	- 기아(棄兒) : 어린아이를 내버림, 버림받은 아이
	- 기아(飢兒) : 굶주림에 시달리는 아이
	- 弃(버릴 기) 갑골문을 보면 죽은 아이를 바구니에 담에 버리는 모습
	
- 畢(마칠 필)
	- 田(밭 전) 에 얽기설기 망치는 일을 다 끝내다 → 마칠 필
	- 필경(畢竟) : 마침내, 결국에는
	- 군필(軍畢) : 
	- 미필(未畢) : 아직 다 끝내지 못함
	- 필납(畢納) : 납세나 납품 따위를 끝냄
	- 검사필(檢査畢) : 검사를 마침

## 基
- 3-리소스/Hword1/3급2/其(그 기)
	- 얼기설기 대나무를 엮어 만든 키를 그린 것으로 키 기 였지만 그 기로 쓰이게 됨
	- 그것, 만약, 아마도
	- 어조사 ↔ 是(바를 시, 옳을 시, 이 시)
	
	- 기타(其他) : 그것 外에 또 다른 것
	- 각기(各其) : 각각, 저마다
	- 기간(其間) : 그 사이, 그 동안
	- 급기야(及其也) : 마침내, 마지막에는
	- 부지기수(不知其數) : 그 수를 알지 못한다, 매우 많음
	
- 箕(키 기) 
	- 𥫗(대죽 죽) + 其(그 기)
	- 基(그 기)가 지시대명사로 쓰이자 대나무(竹)을 넣어 키 기로 만듬
	- 키: 곡식을 까부르는 데 쓰는 기구
	- 삼태기: 흙을 담아 나르는 그릇

## 垂
- 垂(드리울 수) 
	- 나무가 아래로 드리워졌다
	- 드리우다 : 한쪽이 위에 고정된 천이나 줄 따위가 아래로 늘어지다
	- 늘어뜨리다  : 식물의 가지와 잎이 늘어져 있는 모습
	
	- 수렴(垂簾) : 발을 늘임, 늘인 발, 수렴 청정
	- 수직(垂直) : 똑바로 드리운 모양
 
- 睡(졸음 수) 
	- 目(눈 목) + 垂(드리울 수)
	- 눈(目)이 늘어지다, 졸음이 쏟아지다 
	- 자다, 졸음 
	
	- 오수(午睡) : 낮잠
	- 수면제(睡眠劑) : 불면증(不眠症)을 진정시켜 잠이 들게 하는 약 
	
- 郵(역마 우, 우편 우) 
	- 垂(드리울 수) + ⻏(우부방 읍)
	- 고을에 말이 가다 쉬다	
	- 우편(郵便)
	- 우정국(郵政局)
	
- 華(빛날 화, 화려할 화) 
	- 艹(풀 초) + 垂(드리울 수)
	- 12획으로 필순에 주의
	
	- 花(꽃 화) : 꽃 하나하나, 꽃 이름	
	- 화갑(華甲) : 61세(華의 획인 6개의 十자와 1개의 一)
	- 영화(榮華) : 권력과 부귀를 마음껏 누리는 일
	- 화려(華麗) : 빛나고 아름다움
	- 승화(昇華) : 사물이 한 단계 고상한 영역을 높아지는 일


## 世
- 世(세대 세, 인간 세)
	- 열 십(十)을 이어서 밑스물 입, 20을 뜻함
	- 서른 삽(卅)
	- 아버지와 나는 30년을 주기로 1세대
	- 세대(世代)

- 枼(엷을 엽, 나뭇잎 엽)
	- 木(나무 목: 생명) + 世(인간 세)
	- 자연도 사람에 비유해 입의 생명이 엷다. → 엷다.
	- 또는 나무(木) 위로 새 잎(世)이 올라오는 모습

- 葉(잎 엽) 
	- 艹(초두머리 초) + 枼(나뭇잎 엽)
	- 지엽(枝葉) : 가지와 잎, 重要하지 않는 部分
		
- 蝶(나비 접)
	- 虫(벌레 훼) + 枼(나뭇잎 엽)
	- 곤충도 생명의 주기가 있음을 비유하여 → 나비 접
	- 나뭇잎을 뜻하는 枼자는 **나비의 날개(날개가 얇다)**를 비유하기 위해 쓰인 것
	
	- 호접지몽(胡蝶之夢) : 莊子가 나비가 되어 날아다닌 꿈
		- 現實과 꿈의 區別이 안되는 것
		- 人生의 덧없음의 比喩
