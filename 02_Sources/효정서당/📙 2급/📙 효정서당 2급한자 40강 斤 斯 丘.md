# 40강 斤 (도끼 근)의 유형별 한자

![](https://www.youtube.com/embed/htcg_gGOQwA)

## 斤

- 斤(도끼 근)
- 近(가까울 근)
- 祈(빌 기)
	- 祈禱(기도)
	- 祈福(기복)
- 折(꺽을 절, 끊을 절)
	- 骨折(골절)
	- 折衷(절충)
- 哲(밝을 철)
	- = 옳고 그름을 
	- 哲學(철학)
	- 明哲(명철)
- 逝(갈 서) = 辶(쉬엄쉬엄갈 착) + 折(꺽을 절, 끊을 절)
	- 逝去(서거)
	- 崩御(붕어) - 천자
	- 흉거
	- 長逝(장서)
- 誓(맹서할 서, 맹세할 세)
	- = 말씀으로 확실하게 맹세하다
	- 盟誓(맹세)
	- 誓約(서약)
- 斯(이 사)
	- 지시대명사
	- 斯盧(사로) = 신라
		- 盧(큰 그릇 노)
	- 斯文(사문)
- 析(끊을 석, 쪼갤 석)
	- = 나무를 도끼로 쪼개다
	- 分析(분석)
	- 解析(해석)
- 新(새 신) = 亲(친할 친) + 斤(근 근)
	- 亲(다보록하게 나올 진)
- 親(친할 친, 어버이 친)
- 薪(떌나무 신)
	- 臥薪嘗膽
- 斬(벨 참)
	- 陵遲處斬(능지처참)
	- 斬新(참신)
	- 處斬(처참)
- 漸(점점 점, 차차 점)
	- 漸入佳境(점입가경)
	- 漸次(점차)
	- 漸進(점진)
- [暫](잠깐 잠)
	- = 형벌이 하루를 넘기지 않으니, 잠깐
	- 暫時(잠시)
	- 暫定的(잠정적)
- 慙(부끄러울 참)
	- = 죄를 졌다고 생각하니 마음이 부끄러움
	- 慙悔(참회)
	- 慙愧(참괴)
- 斥(물리칠 척)
	- = 도끼 날이 엄청 날카로워 모두를 물리 침
	- 排斥(배척)
	- 斥和(척화)
- 斷(끊을 단)
	- = 실을 상자에 넣어 계속 이어준 것을 또끼로 한방에 끊다
	- 斷切(단절)
	- 斷食(단식)
	- 斷續(단속)
- 繼(이을 계) 
	- = 계속 실을 이어간다
	- 繼走(계주)
	- 繼母(계모)

## 丘
- 丘(언덕 구)
	- = 도끼같이 땅이 험한 산
- 岳(큰 산 악)
	- = 언덕진 산
	- 冠岳山(관악산)
	- 雪嶽山(설악산)
		- 嶽(큰산 악)
		- 獄(감옥 옥)
- 兵(병사 병) = 斤 + 廾(두손들 공)
	- = 도끼를 두 손으로 들다
