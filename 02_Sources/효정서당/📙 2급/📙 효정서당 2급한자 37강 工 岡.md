![](https://www.youtube.com/embed/BybOc9afHXc)

## 工
- 工(장인 공)
	- 공구, 잘만드는 사람, 장인 
	- 동사로는 만들 공
	- 하늘 + 뚫을 곤(상하통야) + 땅
	- 자의 역할
	
- 功(일할 공, 공 공) 
	- 잘 만들려면 힘쓰 일하다, 일할 공
	- 일을 하고 나면 공덕이 된다, 공 공
	
- 攻(칠 공) = 칠 복
	- 정치도 만들어 가야한다
	- 좋은 쪽으로 쳐서 만들어 가야한다
- 貢(바칠 공) = 돈
	- 공을 세우면 제물에 대한 보상을 바치다
	
- 恐(두려워할 공) = 工 + 凡(잡을 집) = 心
	- 巩(안을 공) = 손으로 무언가를 확 잡다 
	- 무릇 범이 아니라 잡을 집
	- 마음을 안다, 두려우면 가슴을 부여잡다, 두려울 공
	- 恐怖(공포)
		- 怖(두려워할 포)
		- 布(베 포, 펼 포)
		
- 江(강 강)
	- 물이 흐르다보니까 강이 만들어지다
	- 뜻은 음을 따라가려고 하고, 음은 뜻을 따라가려고 합니다.
	
- 項(항목 항, 목덜미 항) = 頁(머리 혈)
	- 모가지, 머리 혈까지를 항목이라 함
	- 외형에는 머리 혈이 들어감
	- 項羽壯士(항우장사) = 항우와 같이 힘이 쓴 장사

- 紅(붉을 홍, 분홍색 홍)
	- 실에다 색을 입혀 지어가다
	- 적색보다는 밝은 색
	
- 鴻(큰 기러기 홍, 클 홍)=鳥(새 조) + 江(강 강)
	- 雁(작은 기러기 안)
	- 鴻雁(홍안) = 큰 기러기와 작은 기러기
	- 雁行(안항) = 기러기의 행렬(남의 형제를 높혀 부르는 말)
	- 鴻恩(홍은) = 
	
	
- 巧(공교로울 교) = 工(장인 공) + 丂(숨내쉴 고)
	- 丂(숨내쉴 고) = 입안에서 입기운이 나가는 모습
	- 공교하다(工巧--: 솜씨나 꾀 따위가 재치가 있고 교묘하다)
	
	- 妙(묘할 묘)
	- 拙(옹졸할 졸)
	- 丂(공교할 교, 숨 내쉴 고)
	
	- 巧言令色(교언영색) 
		- 말을 교묘하게 하고 얼굴빛을 꾸민다
		- 남의 환심을 사기 위해 교묘히 꾸며서 하는 말
		
- 巫(무당 무) = 工(장인 공) + 从(좇을 종)
	- 하늘과 땅사이에서 일을 이루려고 가교 역할을 사람
	- 제정일치 시대
	
- 靈(신령 령)=霝(비 올 령(영)) + 巫(무당 무)
	- 霝(비 내릴 령)
	- 𠱠(많은 소리 령, 시끄러울 령(영)) = 입구가 3개
	- 靈感(영감) = 심령에 의(依)한 예감(豫感)
		- 感(느낄 감)=心(마음 심) + 咸(다 함)
	- 令監(영감) = 남자 노인을 높이어 이르는 말
		- 令(하여금 령)
		- 領(다스릴 영)
		- 監(살필 감)
