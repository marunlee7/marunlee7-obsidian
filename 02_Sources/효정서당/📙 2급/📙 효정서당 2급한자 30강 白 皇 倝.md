# 효정서당 2급 30강 白의 유형별 한자 2탄!

![](https://www.youtube.com/embed/jJ92zGt6Qo4)

## 皇
- 皇(임금 황) = 白(흰 백) + 王(임금 왕)
	- 임금, 봉황
	- 빛나다, 빛과 같은 역할을 하다가 본래 의미
	
	- 皇帝(황제)
		- 대한제국 때 임금을 높여 부르는 말
		- 중국 청(淸)나라까지 쓰였고, 천자(天子)라 불리기도 했음
	- 皇后(황후) = 皇帝의 正宮
	- 秦始皇(진시황) = 始皇帝
	- 張皇(장황) = 번거롭고 긺, 지루함
	
	
	- 煌(빛날 황) = 火(불 화) + 皇(임금 황)
		- 빛나다
		- 성하다(기운이나 세력이 한창 왕성하다)
	
	- 鳳(봉황새 봉)= 凡(무릇 범) + 鳥(새 조)
		- 봉황(鳳凰 : 예로부터 중국의 전설에 나오는 상서로움을 상징하는 상상의 새
		- 수컷은 鳳(봉) , 암컷은 凰(황)
		- 四靈(사령) = 기린, 거북, 용, 봉황
		
	- 凰(봉황새 황) = 几(안석 궤 : 책상) + 皇(임금 황)
		- 봉황의 암컷

## 倝
- 倝(해 돋을 간, 햇빛이 빛나는 모양 간)
	- 龺(해 돋을 간) = 사방(十)에서 새벽(早 : 새벽 조, 일찍 조)이 오니 해가 돋는다 [꼬불]
	- 풀숲 사이로 해가 떠오르는 모습
	- 햇빛(햇빛이 찬란하게 대지를 비추는 모습)
	
- 乾(하늘 건, 마를 건, 마를 간) = 倝(해 돋을 간) + 乙(아지랑이)
	- 초목 사이에 해가 있으니 대지를 밝게 비추는 모습
	- 하늘이 맑은 날 햇볕이 대지를 갈구게 되면 아지랑이가 피어 오르게 된다
	- 해가 돋고(龺) 사람(人)과 새(乙)와 모든 생물에게 생명을 주는 하늘 [꼬불]
	- 乾(하늘 건) = 하늘에서 바라보는 하늘
	- 天(하늘 천) = 사람이 중심이 되어서 하늘을 볼 때
	
	- 乾草(건초) = 베어서 말린 풀, 말라 죽은 풀
	- 乾命(건명 : 남자)
	- 坤命(곤명 : 여자)
	- 乾坤坎離(건곤감리)
		- 乾(하늘 건)	
		- 坤(땅 곤)
		- 坎(구덩이 감)
		- 離(떠날 리(이))
	- 乾燥(건조) = 습기(濕氣)나 물기가 없음, 마름
	- 乾坤(건곤) = 하늘과 땅을 상징적으로 일컫는 말, 온 세상
	- 乾杯(건배 ) = 잔 비우기
	- 乾達(건달) = 하는 일 없이 빈둥빈둥 놀거나 게으름을 부리는 짓, 그러한 사람
	- 乾電池(건전지)
	- 乾魚物(건어물) = 말린 어물(魚物)
	- 乾石魚(건석어) = 가조기, 배를 갈라 넓적하게 펴서 말린 조기
	
	
- 幹(줄기 간, 중요할 간) = 干(방패 간) + 倝(햇빛 빛나는 모양 간)
	- 줄기, 근본
	- 맡다
	- 倝 풀숲 사이로 해가 떠오르는 모습
	- 干(방패 간, 찌를 간, 막을 간) = 땅을 찔러 준다
	- 해가 돋는(龺: 해 돋을 간) 순간부터 사람들과 방패들을 관리하는 간부 [꼬불]
	
	- 根幹(근간) 
		- 뿌리와 줄기
		- 어떤 사물의 바탕이나 가장 중심 되는 부분(部分)
	- 幹部(간부) = 단체(團體)의 우두머리 되는 사람들
		- 部(떼 부, 무리 부)
	- 幹線(간선) = 도로의 주요 구간 사이를 연결(連結)하는 선(線)
	- 幹事(간사) 
		- = 일을 맡아 처리(處理)함
		- = 단체의 사무를 주장으로 맡아 처리하는 사람
	- 基幹(기간) = 어떤 조직이나 체계를 이룬 것 가운데 중심이 되는 것
	- 基幹産業(기간산업) 
	- 白頭大幹(백두대간) 
		- 한반도를 동서로 갈라 놓은 큰 산줄기의 조선시대의 이름
		- 백두산에서 始作되어 동쪽 海岸線을 끼고 남쪽으로 흐르다가 太白山 附近에서 서쪽으로 기울어 

- 韓(클 한, 나라이름 한) = 倝 + 韋(다룬 가죽 위)
	- 대한민국의 약칭
	- 나라 이름
	- 햇빛이 성을 비추는 모습으로 해석, 아침의 나라
	- 해가 돋을 때부터 일찍부터 가죽을 펴듯 크게 땅(영역)을 넓혀가다 [효정]
	
	- 倝 = 햇빛(햇빛이 찬란하게 대지를 비추는 모습)
	- 韋 = 성을 둘러싸고 있는 모습을 그린 것
	
	- 大韓民國(대한민국)

- 朝(아침 조, 조정 조)
	- 朝鮮(조선) 
		- 鮮(고울 선)
			
		
- 潮(조수 조, 밀물 조)
- 廟(사당 묘)
	- 祠堂(사당)
	- 廟(사당 묘) = 나랏일을 한 관리의 영정 사진이 있는 곳
	- 墓(무덤 묘) = 무덤이 있는 곳
