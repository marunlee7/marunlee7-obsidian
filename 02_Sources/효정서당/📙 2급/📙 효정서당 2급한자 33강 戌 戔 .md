# 효정서당 2급 33강 생활도구 2탄! 戌, 戔 의 유형별 한자

![](https://www.youtube.com/watch?v=P_QANAUCDhg&list=PL5NRjbnnZamSxqNm-9qN2VFoD0UwLAJaL&index=33&t=4s&pp=iAQB)

## 咸
- 弋(주살 익) = 끈을 단 주살
- 戊(창 과) = 강한 무기
- 戌(다 때려부술 술, 개 술[천간], 9월 술[=천간의 무토])

- 咸(다 함) 
	- 戌(개 술) + 입 구(경계) 
	- 도구를 이용해서 리모델링할 때 다 때려부수다, 다 정리가 됐다

- 感(느낄 감) 
	- 心(마음 심) + 咸(다 함)
	- 모두 정리가 되었으니 다같이 느낌이 좋다
	- 感氣(감기)
	- 感性(감성)
	- 感動(감동)

- 減(덜 감) 
	- 咸(다 함) + 水(물 수)
	- 샘물을 다 퍼낸 상태, (- 마이너스)
	- ↔ 加(더할 가): 힘을 쓰는데 입으로 더하다
	- 加減(가감)
	- 減價(감가)
	- 減下(감하)

- 憾(서운할 감, 섭섭할 감) 
	- 忄(심방변 심) + 感(느낄 감)
	- 憾情(감정)
	- 憾恨(감한)
	- 遺憾(유감) = 마음에 남는 섭섭함
	- 有感(유감) = 느끼는 바가 있음

- 國(나라국) 
	- 囗(에울 위) + 或(혹 혹, 어떤이 혹)
	- 或(혹 혹, 혹시 혹, 어떤이 혹)
		- 창을 들고 국토를 지키는 것은 혹시 누가 침범해 오지 않나하는 것이다 [꼬불]

- 惑(의심할 혹, 미혹할 혹)
	- 내가 열심히하듯 다른 사람은 열심히 하지 않는다고 의심하다에서 의심할 혹
	- 혹하는 마음으로 미혹당한다 [꼬불]
	- 迷惑(미혹)
		- 迷(헤맬 미)
	- 不惑(불혹) = 미혹하지 아니한다, 의심하지 않는다는 뜻, 나이 마흔 살

- 域(지경 역) = 土(흙 토) + 或(혹 혹)
	- 區域(구역) = 일정한 기준에 의하여 갈라 놓은 지역

## 成
- 成(이룰 성) = 丁(장정 정) + 戈(창 과)
	- = 건강한 장정이 창(전문 자격증)을 가져 성공을 이루다
	- 成功(성공)
		- 功(공 공, 일할 공)

- 城(재 성, 성 성)
	- 산을 넘어가는 것을 재너머

- 盛(풍성할 성) = 皿(그릇 명) + 成(이룰 성)
	- 그릇에 가득채우다, 풍성하다
	- 豊盛(풍성)
		- 豊(풍성할 풍) = 제기에 바구니 올려놓은 것처럼 많이 쌓여있다

- 誠(정성 성)
	- 말로 반을 이룬다
	- 말을 정성스럽게 하여 이루다

- 歲(해 세, 세월 세, 나이 세) = 止 + 戌 + 步(걸음 보)
	- 止(그칠 지) + 戌(다 때려 부술 술, 9월 술, 숙살) + 𣥂(밟을 달)
	- 步(걸음 보)
		- 바깥에서 하는 행동을 그치고 겨울에 봄을 준비하다
	- 年歲(연세)
		- 年(해 년) = 많이 걸어서 해 년
	- 歲月(세월)

## 戔(잔)

- 戔(해할 잔) 
	- 창을 반복해서 일을 하다
	
- 殘(남을 잔, 잔인할 잔) = 歹(뼈 앙상할 알) + 戔(해할 잔)
	- 뼈를 칼질을 반복해서 발라내어 뼈다귀만 남게 됨
	- 殘金(잔금)
	- 殘酷(잔혹)
	- 殘忍(잔인)
	
- 錢(돈 전)
	- = 金(돈 금)
	- 구리를 헤아여 돈을 만들다
	- 錢臺(전대)
	- 換錢所(환전소)
	- 殘錢(잔전=잔금)
	- 錢價(전가)
	- 錢兩(전냥)
	
- 淺(얕을 천)
	- 걸어갈때 찰박찰박한 정도의 얕은
	- ↔ 深(깊을 심)
	- 淺薄(천박)
	- 淺成(천성)	
	
- 賤(천할 천)
	- 돈가지고 싸우다
	- ↔ 貴(귀할 귀)
	- 貧富貴賤(빈부귀천)
	- 賤待(천대)
		- 待(기다릴 대)		
	
- 踐(밟을 천)
	- 발자국이 남다
	- 實踐(실천)
	- 踐履(천리)

## 哉

- 𢦏(다칠 재, 해할 재) 
	- 창과 + 나무 목 
	- 나무를 창으로 해하다(자르다)

- 哉(어조사 재, 그런가 재) 
	- 口(입 구) + 𢦏(다칠 재)
	- 재난
	- 어조사 ~하다

- 栽(심을 재)
	- 나무를 확실하게 잘라 옮겨 심다
	- 栽培(재배)
	- 植栽(식재)

- 裁(재단 재, 마를 재)
	- 옷을 만들기 위해 옷을 해하다(마름질하다, 재단하다)
	- 制裁(제재)
	- 裁判(재판)

- 載(실을 재)
	- 나무를 해하여 수레에 싣고 가다 
	- 載積(재적)
	- 揭載(게재)
	- 기재

- 程(헤아릴 정)
	- 呈(나타낼 정, 드러낼 정)
		- 구분하기
		- 壬(곧을 정) = 王처럼 오지만 곧을 정이다
		- 壬(맡을 임, 짊어질 임)
		- 聖(성인 성)

- 鐵(쇠 철)
	- 金(쇠 금) + 𢧜(빠를 철, 구렁말 철)
	- 철에 녹인 슨다
	- 鐵筋(철근)
	- 地下鐵(지하철)
