# 효정서당 2급 09강 木(1)

![](https://www.youtube.com/embed/aGy1h0CIM5o)

## 侯
- 侯(제후 후, 벼슬 후) 
	- 人(사람 인) + 厂(기슭 엄, 언덕 엄, 굴바위 엄) + 矢(화살 시)
	- 과녁세운 언덕 엄: 侯의 오른쪽 자형 → **한자 사전에서 보이지 않음**
	- 과녁을 세우고(厂:언덕 엄) 궁술대회(矢)를 하여 사람(亻)이 벼슬을 얻게 되다
	- 활을 쏘는 실력으로 제후의 지위가 정해지다 라는 설에서 → **제후, 임금, 과녁**
		
	- 왕후(王侯) : 帝王(재왕)과 諸侯(제후)
	- 제후(諸侯) : 봉건 시대에 일정한 영토를 가지고 그 영내의 백성을 지배하는 권력을 가지던 사람) 
	- 왕후(王后) : 임금의 아내 后(임금 후, 뒤 후)
	- 왕후장상(王侯將相) : 帝王(재왕), 諸侯(제후), 大將(대장), 宰相(재상)을 통틀어 일컫는 말

- 候(바랄 후, 기다릴 후  기후 후)
	- 최고의 제후를 뽑기위한 것인데 川(내 천: 기후)이 들어가 기후가 좋기를 바라다
	- 川(내 천)을 줄여 하나로 표현함
	- 巜(큰 도랑 괴)
	- 기후(氣候) 


## 木
- 木(나무 목, 목화나무 목)
	- 나무의 **뿌리**를 강조하여 형상화 함
	- 포목점(布木店) 할때는 목화나무 목도 됨
	- 艸 艹(풀 초) 관련 한자 1500여자 로 1위
	- 2위는 木자가 들어가 한자로서 이름 모를 풀, 나무는 없다. 
- 本(근본 본, 뿌리 본, 바탕 본)
		
- 李(오얏나무 이, 성씨 이)
	- **오얏(자두)**
	- 고대에는 자두 새콤한 신맛을 최고 과일
	- 자두는 열매가 나무 열매보다 많이 열린다.

- 床(평상 상, 마루 상) 
	- 옛날에는 마루가 나무로 만들어져 있었다.

## 林
- 林(수풀 림): 나무 2개는 무성하다는 뜻	  

- 霖(장마 림)
	- 우림지역  

- 淋(물 뿌릴 림(임)) 
	- 淋巴腺(임파선) 

- 森(나무 빽빽할 삼)
	- 삼라만상(森羅萬象) : 우주안에 잇는 온갖 사물과 현상

- 禁(금할 금): 
	- 숲(林)에 제단(示)이 있으니 금한다(禁)
	- 示(제사 시, 보일 시): 조상이나 신과 관련된 글자 
	
- 末(끝 말)
	- 나뭇가지 끄트머리를 보고 만든 글자
	- 末年(말년)
	- 말초신경(末梢神經)
	- 沫(거품 말, 물방울 말)
	- 抹(지울 말)

- 未(아닐 미): 
	- 미래(未來)
	- 미안(未安) : 남에게 괴로움을 끼쳐 거북함

- 味(맛 미)
	- 열매가 익었는지 안익었는지 입으로 맛을 보다
	- 味覺(미각)

- 妹(손아랫누이 매) 
	- 女(계집 녀) + 未(아닐 미) 

	- 姉(손윗누이 자) 
		- 女(계집 녀) + 市 
		- 자매(姉妹) : 여자 형제
		- 姉형妹제 雀舌
	
- 茶(차 차, 차 다) 
	- 艹(풀 초) + 余(나 여) 
	- =  艹(초두머리 초) + 𠆢(사람 인) + 木(나무 목)
	- = 차, 차나무
	- 余(나 여, 남을 여) 
	- 茶禮(차례)
	- 중국 황하,  나뭇잎을 끓여 걸러서 차를 마심
	- 우리나라에서는 술을 올리게 됨
	- 중국은 차의 나라이다. 전설에 따르면 기원전 2,737년 중국의 고대 황제인 신농씨가 차를 발견해서 백성들에게 널리 알렸다고 한다. 차는 처음에는 약으로 마셨지만, 지금은 음료로 널리 이용되고 있다. 
	- 茶자에 쓰인 艹자는 찻잎을 뜻한다. 
	- 余자는 작은 집을 그린 것이다. 
	- 그러니 茶자는 소박한 집에서 차를 즐긴다는 뜻으로 만들어졌다 할 수 있다. 
	- 참고로 茶자를 우리말로는 ‘다’라고도 읽는다. 
	- 지금은 많이 사라졌지만 ‘다방(茶房)’이라는 곳은 사실 ‘차방(茶房)’을 우리말로 읽은 것이었다.
	
- 極(빠를 극, 다할 극)
	- 木(나무 목) + 亟(빠를 극)
	- 오행으로 나무는 생명
	- 한 일은 위에서 하늘
	- 삐침 - 천기가 내려움
	- 굽을 을
	- 한 일 - 땅
	- 파임 불 - 직위가 올라감
	- 중간에 사람의 손과 입으로서 우리 우주 자연의 기운이 같이 빠르게 돌아감
	- 그래서 온 힘을 다해서 지극정성을 다해서 지극할 극
	- 생명체를 이루어가는 과정과 이루는 것을 끝에 간다, 끝 극
		- 태극기(太極旗)
		- 지극(至極) : 어떠한 정도나 상태 따위가 극도에 이르러 더할 나위 없음
		- 지극정성(至極精誠)
				
- 漆(옻칠할 칠, 검을 칠) 
	- 氵(삼수변 수) + 桼(옻 칠)
	- 옻은 색이 어둡다 하여 **검다**라는 뜻도 파생
	- 칠판(漆板) : 검을 칠 + 널판지 판
	- 칠흑(漆黑) : 옻칠처럼 검음. 칠흑같은 밤
		
- 黑(검을 흑) 
	- 四(넉 사) + 土(흙 토) + 灬(연화발 화)
	- 토담 집에 불을 때면 굴뚝에 검은 연기가 빠져나가는 모양을 보고 만듬
	- 黑자의 금문을 보면 火(불 화)자 위로 연기가 빠져나가는 굴뚝이 그려져 있었다. 
	- 불을 지피는 용도인 아궁이는 주위가 꺼멓게 거슬리게 된다. 
	- 그래서 黑자는 ‘검다’나 ‘꺼멓게 되다’라는 뜻을 갖게 되었다. 
	- 白(흰 백)자가 순수함을 상징한다면 黑자는 그 반대의 개념을 갖고 있다. 
	- 그래서 黑자는 ‘검다’라는 뜻 외에도 ‘나쁘다’나 ‘악독하다’, ‘횡령하다’와 같은 부정적인 뜻을 전달하기도 한다.

## 朱
- 朱(붉을 주)
	- 적송(赤松) 을 보고 만든 글자
	- 홍적주(紅赤朱)
	- 紅(붉을 홍) < 赤(붉을 적) - 원색 < 朱(붉을 주) - 더 진한 밤색 
		
- 株(그루터기 주)
	- 수주대토(守株待兔)
		- 그루터기를 지켜 토끼를 기다린다
		- 고지식하고 융통성이 없어 구습과 전례만 고집함.
	- 주식회사(株式會社)
	
- 殊(뛰어날 수, 죽을 수, 다를 수) 
	- 歹(살 바른 뼈 알) + 朱(붉을 주)

## 果
- 果(열매 과, 실과 과)

- 菓(과자 과, 과일 과)
	- 과자(菓子)
	
- 課(공부할 과, 매길 과)
	- 공부를 하면 열매가 맺힌다
	- 과정(課程)

- 相(서로 상, 형상 상)
	- 나무는 생명, 사람(손주)의 눈, 할아버지와 손주가 서로 보고 있다
	- 할아버지의 형상을 많이 닮게 된다.
	- 할아버지는 손주가 실력을 갖춰 재상이 되도록 후원한다
	- 想(생각할 상)
	- 霜(서리 상)
	- 상대(相對) : 서로 마주 대(對)함
	- 재상(宰相)
	- 상관(相關)
	- 상사병(相思病) 
	
- 罙(깊을 심, 깊을 담)
	- 동굴(穴 구멍 혈)속으로 나무 햇불(木)을 들고 들어가다.  

- 深(깊을 심)
	- 바다(氵)의 깊은 곳
	- 심해(深海) : 
	- 심야(深夜) : 
	- 심산유곡(深山幽谷) : 깊숙하고 고요한 산과 골짜기.

- 探(더듬을 탐, 찾을 탐)
	- 손에 횟불을 들고 더듬어 찾다
	- 탐구(探究) : 진리(眞理), 학문(學問) 따위를 파고들어 깊이 연구(硏究)함.
	- 탐사(探査) : 더듬어 살펴 조사(調査)함.
