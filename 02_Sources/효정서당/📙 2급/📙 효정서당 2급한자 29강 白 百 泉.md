## 白
- 白(흰 백 / 말할 백)
	- 日 흰백 햇빛이 눈부시게 희다
	- 曰 입에서 나가는 기운
	- 성인은 말씀으로 교화
	- 태양은 빛으로 말없는 성인과 같다
	
- 伯(맏 백)
	- 맏아들, 우두머리
	- 형제를 대표하여 윗사람에게 아뢰는(白)는 사람(亻)
	- 엄지손가락(白)에 해당하는 사람(亻)
	- 큰 아버지를 백부(伯父)
	
	- 백중숙계(伯仲叔季)
		- 남자형제의 차례를 표현
		- 仲(버금 중)
		- 叔(아재비 숙)
		- 季(막내 계 / 철 계 / 계절 계)	
	- 白(흰 백)
		- 엄지 손가락 흰 부위를 본뜸
		- 아뢰다(사람의 머리를 상형하였다는 설이 있어서)		
	- 仲(버금 중)
		- 중앙(가운데(마을(口) 중앙 광장에 부족의 상징인 깃발을 단 장대(丨)를 세웠다는 데서)
		- 둘째 아들
	
- 拍(손뼉칠 박)
	- 拍手(박수)
	- 拍子(박자)
	
- 泊(배댈 박 / 머무를 박)
	- 일박(一泊)
	- 숙박(宿泊)
	- 민박(民泊)
	
- 迫(핍박할 박 / 닥칠 박)
	- 驅迫(구박)
	- 切迫(절박)

- 帛(비단 백)
	- 흰천, 명주
	- 錦(비단 금)
		- 금수강산(錦繡江山)
		- 錦上添花(금상첨화)
	
- 綿(솜 면) 
	- 糹(가는 실 멱) + 帛(비단 백)
	- 솜, 솜옷 
	- 이어지다, 끊어지지 않다(솜을 꼬아 실을 만들기도 했기 때문)
	- 흰 천(帛)에 糹자를 결합해 원료 그대로 사용하던 솜
	
	- 綿(면) = 무명, 무명실, 솜, 목화
	- 면면(綿綿) : 끊임없이 이어짐
	- 면밀(綿密) : 자세하고 빈틈이 없음
	- 면봉(綿棒) : 끝에 솜을 말아 붙인 가느다란 막대
	- 면사(綿絲) : 무명실, 솜을 자아 만든 실
	- 순면(純綿) : 무명실로만 짠 실
	- 면직(綿織) : 무명실로 짠 직물, 면직물
	- 면포(綿布) : 솜을 자아 만든 실로 짠 베, 무명
	
- 百(일백 백 / 모두 백 / 온통 백)
	- 百日祈禱(백일기도)

- 柏(측백나무 백 / 잣나무 백)
	- 울타리에 주로 심는 나무
	
	- 松柏(송백)
	- 冬柏(동백)
	
- 宿(잠잘 숙 / 별자리 수) = 宀(집 면) + 亻 + 百(일백 백 : 모양자 역할)
	- 잠을자다, 宿泊하다
	- 묵다, 오래되다
	- 별자리, 성수(星宿 : 모든 별자리의 별들)
	- 자다 → 숙박하다 → 오래되다
	
	- 宿題(숙제) 
		- 復習이나 豫習을 위하여 집에서 지어 오게 하거나 풀어 오게 하는 問題
		- 두고 생각하여 보거나 解決해야 할 問題
		- 題(제목 제) = 頁 + 是(이 시 / 바를 시 / 옳을 시)
			- 제목, 머리말, 글의 시작 부분, 이마(얼굴의 시작)
			- 이마 = 바른(是) 얼굴(頁)(얼굴의 정면에 있는 이마)
			- 是(옳을 시) = 옳다, 바르다
	- 宿泊(숙박)
	- 宿願(숙원) = 오래도록 지녀온 소원
	- 露宿(노숙) = 한뎃잠, 드러나게 잠을 자다
	- 宿命(숙명) = 날 때부터 타고난 운명
	
- 縮(오그라들 축 / 줄을 축) = 糹(가는 실 멱) + 宿(잠잘 숙)
	- 줄이다, 오그라들다(비단옷은 물세탁하면 줄어 듬)
	- 감축하다, 축소하다
	- 위축(萎縮) : 우그러져 펴지 못함
	- 신축(伸縮) : 늘어남과 줄어듬
	- 축소(縮小) : 줄여서 작아짐
	- 응축(凝縮) : 한데 엉겨 굳어짐
	- 압축(壓縮) : 눌러서 쭈그러뜨림
	- 긴축(緊縮) : 바싹 줄임
	- 축척(縮尺) : 축도(縮圖) 그릴 때 그 축소(縮小)시킬 비례(比例)의 척도(尺度)
	- 단축(短縮) : 시간이나 거리 따위가 짧게 줄어듦
	- 감축(減縮) : 덜리고 줄어서 적어짐. 덜고 줄여서 적게 함
	- 수축(收縮) : 어떤 물건이 오그라들거나 줆
	
- 貌(모양 모 / 모사할 막) 
	- 豸(맹수 치 /벌레 치 / 해태 치) + 皃(모양 모 / 얼굴 모)
	- 모양, 얼굴, 자태, 행동거지
	- 모사하다(模寫 : 사물을 형체 그대로 그리다)
	- 한눈에 들어오는 사람의 용모
	
	- 皃(모양 모 / 얼굴 모) = 무릎을 끓고 앉아있는 사람을 그린 頁의 변형
	- 豸(맹수 치 / 발없는 벌레 치 / 해태 치)
		- 맹수(猛獸)가 발을 모으로 등을 높이 세워 덤벼들려는 모양
		- 다양한 짐승의 이름
		- 고양이괏 몸에 얼룩이나 줄무늬가 있어 쉽게 식별할 수 있다
	
	- 비슷한 한자
	- 像(형상 상 / 모양 상)
	- 姿(모양 자)
	- 態(모습 태) =  心(마음 심) + 能(능할 능)
		- 모습, 형태, 상태
		- 能(능할 능) = 곰의 위풍당당한 모습
	- 能(능할 능)
		- 능하다, 할 수 있다, 능력
		- 곰은 재주가 뛰어나기에 재능을 뜻하게 되었다는 해석
		- 熊(곰 웅 : 能자가 재능으로 가차되면서 熊이 곰으로 쓰임)
		
	- 形(모양 형)
		- 사람이 인위적으로 형태를 만들다(우물을 파서 형틀을 만들어짐)
	- 樣(모양 양 / 상수리나무 상) 
		- 木(나무 목) + 羕(강 길 양)
		- 모양, 견본
		- 羕(강이 길 양 : 강줄기가 양 떼처럼 길게 늘어져 있는 것)
		- 본래 뜻은 나무에 열린 열매
		- 도토리 상수리 나무
		- 양상(樣相) : 생김새나 모습
		- 양식(樣式) : 일정한 모양과 방식
		- 다양(多樣) : 여러 가지 모양 또는 양식
		- 문양(文樣) : 
		- 양태(樣態) : 놓여 있는 모양이나 형편

	- 容(얼굴 용 / 모양 용 / 담을 용)
	
	- 모양((貌樣) 모양(模樣) : 겉으로 나타나는 생김새나 됨됨이.
		- 模(본뜰 모) = 木(나무 목) + 莫(없을 막)
			- 형상, 본뜨다, 본받다
			- 법도, 양식
			- 나무를 깍아 만든 틀을 뜻하기 위해 만들 글자
			- 나무로 없어진(莫) 물건의 모형을 본뜨기 위해 만든 본보기 [꼬불]
			- 莫(없을 막 : 풀 속으로 해가 들어가 큰 형체가 없어진 것이다) [꼬불]
		
	- 용모(容貌) : 사람의 얼굴 모양
	- 외모(外貌) : 겉으로서의 모습
	- 미모(美貌) : 아름다운 얼굴 모습
	- 면모(面貌) : 얼굴의 생김새, 상태나 됨됨이
	- 전모(全貌) : 전체이 모양이나 형편(形便)
	- 변모(變貌) : 달라진 모양이나 모습
	
- 碧(푸를 벽) = 石(돌 석) + 珀(호박 박)
	- 푸르다, 푸른빛, 푸른 옥
	- 상전벽해(桑田碧海)
		- 뽕나무밭이 푸른 바다가 되었다
		- 세상이 몰라 볼 정도(程度)로 바뀐 것
		- 세상의 모든 일이 엄청나게 변(變 : 별할 변)해 버린 것

## 泉
- 泉(샘 천 / 저승 천)
	- 샘(맑은 물), 지하수
	- 황천(黃泉), 저승
	- 암벽에서 물이 흘러나오는 모습
	- 물을 우리 몸에 비유하면 저승 천 
	- 정신은 하늘(白)로 化하고 , 몸은 땅(水)로 돌아간다
		- 승천하다
	- 온천(溫泉) : 
	- 계천(溪泉) : 
	- 구천(九泉) : 죽은 뒤 넑이 돌아간다는 곳
	- 황천(黃泉) : 사람이 죽어서 간다는 곳, 저승
	- 귀천(歸泉) : 황천으로 돌아가다, 즉 죽음
	- 원천(源泉) : 물이 흘러나오는 근원, 사물의 근원
	- 용천(涌泉) : 물이 솟아나오는 샘. 涌(물 솟을 용)
	- 원천징수(源泉徵收)
		
- 線(줄 선) = 絲+泉
	- 줄(무엇을 묶거나 동이는데에 쓸 수 있는 가늘고 긴 물건)
	- 선, 실
	- 노선(路線 : 정치적, 사상적 노선)
	- 물이 끊임없이 흘러 내리는 모습(泉)에 실을 더하여 길게 이어진다는 뜻
	
	- 연관단어
	- 絃(악기줄 현)
	- 絲(실 사)
	- 縷(실 루)
	
	- 선(線) : 그어 놓은 금이나 줄, 철선, 선로(線路)
	- 시선(視線) : 눈이 가는 방향
	- 노선(路線) : 정해 놓고 다니도록 되어 있는 길
	- 직선(直線) : 곧을 줄
	- 탈선(脫線) : 선론에서 벗어남
	- 곡선(曲線) : 구부러진 선(線)
	- 간선(幹線) : 구간(區間)사이를 연결(連結)하는 선
	- 능선(稜線) : 산등성이를 따라 죽 이어진 선
	- 전선(戰線) : 전쟁에서 직접 전투가 벌어지는 지역
	- 접선(接線) : 줄으 댐, 접촉함
	- 혼선(混線) : 말이나 일 따위의 갈래가 얽혀 종잡을 수 없음
	- 지선(支線) : 본선이나 간선에서 갈라져 나간 선
	- 사선(斜線) : 비스듬히 그은 줄
	- 침선(針線) : 바늘과 실
	- 광선(光線) : 빛, 빛의 줄기
	- 복선(伏線) : 만일의 경우 뒤에 생길 일에 대처하려고 남 몰리 미리 베푸는 준비
	- 나선(螺線) : 평면상에 있어서 소용돌이 모양(模樣)의 곡선(曲線)
	- 선로(線路) : 
	- 직선(直線) : 
	- 紫外線(자외선)
