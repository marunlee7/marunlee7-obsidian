## 艮
- [[艮]](그칠 간, 눈동자 그칠 간, 괘 이름 간) [138]
	- 눈을 뒤로 향하고 사람이 서 있는 모양
	- 머무르다, 그치다의 뜻
	- 止(그칠 지): 발을 중지한 것
	- 銀(은 은) 의 약자(略字)
	- 𥃩(어그러질 간, 멀리 볼 요) 이 본자(本字)
	- 사람이 눈을 뒤로 향하게 한 모양으로 외면하다, 원만하다, 배신하다 

	- [[根]](뿌리 근)
		- 나무도 어느정도 자라면 뿌리가 그침
		- 뿌리, 근본, 밑동
		- 艮(그칠 간, 어긋날 간)은 사람의 시선이 땅을 향해 있는 글자
		- 시선이 나무뿌리를 향하고 있다는 의미에서 뿌리를 뜻함
		- 뿌리는 나무를 지탱하는 가장 든든한 근본
		- 사물의 가장 원초적인 근본과 본바탕이라는 뜻
			- 農根(농근): 농사의 근본(根本)이라는 뜻, 농지(農地)을 일컫는 말
			- 事實無根(사실무근): 사실에 근거가 없다.
	
	- [[銀]](은 은, 은빛 은, 화폐 은) 
		- 눈동자를 돌리면 흰자위가 보임
		- 흰자위 같은 색을 은으로 표현
		- 은빛, 은, 화폐
		- 銀(은 은)를 화폐의 대체수단으로 사용
			- 銀行(은행): 돈을 보관하고 빌려 주는 곳
		
	- [[垠]](언덕 은, 지경 은)
		- 부수는 土
		- 조선의 마지막 임금 李垠
		
	- [[限]](한계 한, 한정할 한) 
		- 阝 + 艮
		- 언덕을 만나면 더 이상 갈 수 없음, 한계점
		- 限界(한계)
			
	- [[恨]](한 한, 한할 한) 
		- 心 + 艮
		- 마음 속에 그쳐놓음, 그것이 한이 됨
		
	- [[貇]](간절할 간) 
		- 豸(맹수 치, 해태 치) + 艮
		- 맷돼지가 흔자위를 내고 노려보는 모습
			
	- [[懇]](정성 간) 
		- 貇 + 心
		- 눈을 부라려뜨고 정성을 다 함
			- 懇切(간절)

		- 匿名(익명): 어떤 일을 하면서 자기 신분을 드러내지 않기위해 이름을 밝히지 않는 상태
		- 能士匿謨(능사익모): 재능이 있는 자는 계책(計策)을 숨기고 남에게 알리지 않음

		- 풀 하나, 풀 하나가 시월의 가을 바람에 단풍이 들어 화려하게 빛난다 [꼬불]

## 虍
- [[虍]](범 호, 범무늬 호) [141]

	- 일곱 칠(七) : 무늬를 나타냄
	
	- [[虎]](범 호)
		- 虍(범 호) +(儿) 어진사람 인
	
	- [[虐]](사나울 학, 모질 학) 
		- 범 호 + 왼손 좌(屮), 싹날 철, 풀 초
		- 범이 손톱같은 발톱으로 공격하니 사납다[꼬불]
		- 싹날 철은 왼손 좌에 포함
			- 虐待(학대): 몹시 괴롭히거나 사납게 대우(待遇)함
				- 待(대할 대, 기다릴 대)
			- 虐殺(학살): 참혹하게 마구 무찔러 죽임
			- 虐政(학정): 백성에서 심히 구는 포학(暴虐)한 정치
			- 暴虐(포학): 횡포하고 잔악함
		- 殺(죽일 살, 감할 쇄): 타이밍을 노려 한번에 죽이다
			- 乂(베다) + 木(생명) + 丶(타이밍) 杀(찰나 찰)		
			
	- [[處]](곳 처, 처녀 처): 
		- 범 호(虍) + 천천이걸을 쇠(夊) + 几(안석 궤, 책상 궤)
		- 호랑이가 안석에 어슬렁 가는 모습
		- 범이 천천히(夊) 책상(几)밑 같은 굴을 찾아 들어간 곳. [꼬불]
			- 處所(처소): 사람이 살거나 임시(臨時)로 머물러 있는 곳
			- 處女(처녀): 아직 시집가지 않은 성숙한 여자, 미혼한 여성
			- 處女地(처녀지): 
			- 處理(처리): 일을 다스려 치름
			- 處罰(처벌): 형벌에 처함

## 血
- [[血]](피 혈) [143]
	- 皿(그릇 명)에 피한방울이 떨어진 것
	
	- [[衆]](무리 중) 
		- 血 + 乑(=众 나란히 설 음, 무리 중)
		- 乑(많은 사람 중)
		: 피로 맺어진 많은 사람(무리)
		
		- [[取]](취할 취, 가질 취) 耳(귀 이) + 又(또 우)
			- 오른손으로 귀를 잘라 얻다(취하다)
			- 取捨選擇(취사선택): 취(取)할 것은 취하고, 버릴 것은 버려서 골라잡음
			- 捨生取義(사생취의): 목숨을 버리고 의리(義理)를 좇는다
			
		- [[聚]](모을 취) = 取(가질 취) + 乑(나란히 설 음)
		- [[驟]](빠를 취, 달릴 취) = 馬(말 마) + 聚(모을 취)
		- [[鄹]](추나라 추) = 聚(모을 취) + ⻏(우부방 읍)
		- [[潨]](물들이 총, 물가의 언덕 종) 氵(삼수변 수) + 衆(무리 중)

		- 大衆(대중)
		- 衆生(중생): 많은 사람들
		- 群衆(군중): 한 곳에 무리지어 모여 있는 사람들
		- 聽衆(청중): 강연, 설교 등(等: 등급 등)을 듣는 군중
		- 衆口難防(중구난방): 여러 사람의 입을 막기 어렵다
		- 衆寡不敵(중과부적): 무리가 적으면 대적할 수 없다 [맹자]
		- 出衆(출중): 뭇 사람속에서 뛰어남
		- 下化衆生(하화중생): 아래로 중생을 가르쳐 거둠
		- 博施濟衆(박시제중): 사랑과 恩惠(은혜)를 널리 베풀어 무사람을 구제함
		- 烏合之衆(오합지중): 까마귀 떼와 같이 조직도 훈련도 없이 모인 무리
		- 衆力移山(중력이산): 많은 사람이 서로 힘을 합(合)하면 泰山(태산)도 옮길 수 있다
		- 拈華示衆(염화시중): 꽃을 따서 무리에게 보인다, 말이나 글에 의(依:의지할 의)하지 않고 이심전심(以心傳心)으로 뜻을 전(傳)하는 일.

## 行
- [[行]](다닐 행, 행할 행, 항렬 항) [144]
	- 길에서 다니는 모습을 본뜸 
	- 五行(오행) : 목 화 금 토 수
	- 척촉: 씩씪하게 열심히 다니다 彳(자축거릴 척) + 亍(자축거릴 촉)
	
	- 行列(항렬) 
		- 같은 혈족 사이에서의 대수관계, 현제 자매의 관계가 한 항렬이다
		- 列( 벌릴 렬 )
		
	- 雁行(안항) 
		- 기러기의 행렬(行列), 기러기가 줄지어 날아감
		- 鴻(기러기 홍) : 앞 선두에 있는 기러기 
		- 雁(기러기 안) : 따라가는 기러기 
	
	- [[術]](재주 술) 
		- 行(다닐 행, 부수) + 朮(삽주뿌리 출, 창 출, 밸 출)
		- 조건에 맞춰서(朮) 뿌리를 잘 내리는 약초의 모습
		- 朮(삽주뿌리 출) : 손이 빠르게 움직이고 있다는 뜻을 표현
		- 뜻을 나타내는 다닐행(行 다니다, 길의 모양) + 朮(출→술)
		- 朮(출)은 차조→짝 달라붙다→뒤따라 가는 일
		- 行(행)은 길
		- 사람이 모여서 생긴 마을안의 길→모든 사람이 따르는 길→일을 하는 법

## 襾
- [[襾]], [[覀]], [[西]](덮을 아) [146]
	- 一(한 일) + 卄(스물 입) + 冂(멀 경)
	- 덮다, 가리어 덮다
	- 그릇의 뚜껑을 그린 것
	- 襾 는 실제 쓰임에서는 대부분 모양자 역할만을 하고
	- 부수로 쓰일 때는 覀(아) 자로 표기된다
	- 凵(감)와 冂(경), 一(일)의 합자(合字)
	- 밑에서 덮고( 凵) 위에서 덮고( 冂) 一(일)을 덧대어 또 덮어 가린다는 뜻
	- 冖(덮을 멱): 천으로 위에서 아래로 덮다
	
	- [[西]](서녁 서) 
		- 해가 지면 둥지로 가야 함
		- 西(서녘 서) 부수는 襾(덮을 아)에서 찾아야 함
		
	- [[栖]](깃들일 서) 
		- 새집은 나무에 지음
		- 棲息地(서식지): 동물(動物)이 깃들어 사는 곳
		- [[息]](쉴 식)
		- [[地]](땅 지)
				
	- [[要]](중요할 요, 반드시 요)
		- 구하다, 원하다, 중요하다
		- 원래는 무희가 춤을 추는 모습을 표현: 허리, 허리를 감싸다
		- 후에 허리가 신체에서 가장 중요한 부위라는 의미가 확대되어
		- 중요하다, 요긴하다 뜻으로 바뀜
		- 必要(필요)
		- 不要不急(불요불급) 필요(必要)하지도 않고 급(急)하지도 않음
		- 要領不得(요령부득): 사물의 주요한 부분을 잡을 수 없다.
		
	- [[腰]](허리 요)
		- 月(육달 월) + 要(중요할 요)
		 - 腰折(요절): 허리가 부러진다, 몹시 우스워서 허리가 부러질 듯함
