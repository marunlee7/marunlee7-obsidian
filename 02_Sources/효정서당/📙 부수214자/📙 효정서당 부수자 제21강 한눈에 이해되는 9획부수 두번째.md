## 食
- [[食]]( 밥 식, 먹을 식, 기를 사, 밥 사) [184]
	- = [[𩙿]]= 飠
	- 人(사람 인) + 良(좋을 량, 어질 량)
	- 뚜껑 + 좋은 것만 걸러내는 량
	- 밥, 음식, 먹다
	- 음식을 담는 식기를 그린 것
	- 사람이 살아가기 위해 좋아하며(良) 즐겨먹는 음식물로 밥
		- 𩙿(밥식 변)
		 - 飠약자, 간체자
		 
	- [[飼]](기를 사) 
		- 飠(밥식 식) + 司(맡을 사)
		- 기르다, 먹이다
		- 飼料(사료)
		- 飼育(사육)	
		
	- [[飮]](마실 음) 
		- 𩙿(밥식 식) + 欠(하품 흠)
		- 식기에 담긴 것을 입을 크게 벌리고 마시다
		- 마시다, 음료
			- 欠(하품 흠) = 입을 벌리고 있는 사람
			- 飮食(음식)
			- 飮料水(음료수)
			- 暴飮暴食(폭음폭식): 음식과 술 등을 한꺼번에 많이 먹음
			
	- [[餓]](굶을 아, 주릴 아) 
		- 𩙿 + 我(나 아)
		- 내가 밥을 굶었다
			- 飢餓(기아) = 굶주림
			- 饑餓(기아) = 굶주림
		
		- [[我]](나 아) = 扌(재방변 수) + 戈(창 과)
			- 삼지창을 그린 것
			- 나를 뜻하는 1인칭 대명사
			- 余(나 여)
			- 吾(나 오)
			- 朕(나 짐)
		
	- [[飢]](주릴 기) 
		- 𩙿(밥식 식) + 几(안석 궤)
		- 굶주리다, 흉년이 들다
			- 飢餓(기아)= 굶주림
		
	- [[템플릿 책, 목차, 파일]](주릴 기) ////////
		- 𩙿(밥식 식) + 幾(몇 기)
		- 飢 와 동자(同字)
		- 幾(몇 기) =  𢆶(작을 유) + 戍(수자리 수)
		- 戍(수자리 수) 
			-  戊(천간 무) + 丶(점 주)
			- 人( 사람 인) + 戈(창 과)
			- 수자리(변방을 지키는 일)
			- 사람이 무기를 들고 지키다
			- 戍卒(수졸) = 변경을 지키던 군졸
		- 戊(천간 무)  =  戈(창 과) + 丿(삐침 별)
		
		
	- [[棄]](버릴 기) 
		- 弃(버릴 기) + 木(나무 목)
		- 버리다, 그만두다, 돌보지 않다
		- 죽은 아기를 삼태기에 담아 버리는 모습
		- 돌(아기머리 거꾸로 나올 돌)
		- 厶(마늘 모, 나 모, 사사롭다 모) + 葉(쓰레 받기)와 卄(공 양손)
		-  청소 도구(道具)를 양 손으로 밀고 감을 나타냄. 따라서 널리 버림
			- 棄兒(기아) = 버림받은 아이
			- 暴棄(포기) = 자포자기의 준말
			- 自暴自棄(자포자기) = 자신을 스스로 해(害)치고 버리다
			 
	- [[弃]](버릴 기) 
		- 亠(돼지해머리 두) + 弁(고깔 변)
		- 弁 = 厶(사사로울 사) + 廾(받들 공) 
		- 厶(나 사)
			-팔꿈치를 구부려 자기를 가리키니 나 또는 사사롭다
			- 마늘 모라 하나 글자를 이해하는 데는 도움이 안됨
			- 去(갈 거)
		
## 香
- [[香]](향기 향) =  禾(벼 화) + 曰(가로 왈)  [186]
	- 향기, 향기롭다, 감미롭다
	- 벼가 날이 갈수록 향기가 난다
	- 최고 좋은 향이 밥(禾) 향이다
	- 黍( 기장 서) + 甘( 달감)
	-  맛이 좋은 기장의 뜻. 
	- 기장을 삶을 때 나는 좋은 향기로부터, 轉(구를 전)하여 향기의 뜻
		- 國香(국향): 난초(蘭草)를 달리 이르는 말
		- 香雪(향설): 향내 나는 눈, 흰꽃을 눈에 비유하여 이르는 말
		
	- [[馨]](향기 멀리나갈 형, 꽃다울 형) 
		- 香(향기 향) + 殸(소리 성)
		- 뜻을 나타내는 향기향(香  향기롭다)部
		- 음(音)을 나타내는 글자 殸(성→형)이 합(合)하여 이루어짐.
		
		- 殸(소리 성) =  殳(몽둥이 수) + 声(소리 성)
		- 声(소리 성) =  士(선비 사) + 𠃜(주검 시)
## 音
- [[音]](소리 음) [180] : 서서(立) 입으로 소리를 냄
	- 立 + 曰(가로 왈)
	- 知音(지음) 
		- 거문고 소리를 듣고 안다
		- 자기의 속마음까지 알아주는 친구
	- 不協和音(불협화음)
		- 서로 뜻이 맞지 않아 일어나는 충돌
		- 둘 이상의 음이 같이 울릴 때, 서로 어울리지 않고 탁하게 들리는 음.
		
	- [[暗]](어두울 암) : 해가 없을 때, 어두울 때 소리가 유리함
		- 暗室(암실)
		- 暗記(암기)
		- 暗誦(암송)
	- [[闇]](어두울 암, 숨을 암, 망루 암) 
		- 門 + 音
		-  이 세상보다는 4차원의 소리
## 飛

- [[飛]] = [[飞]](날 비) [183]
	- 隹(새 추) + 羽(깃 우) 
	- 새의 날개와 몸통을 그려 날개를 양쪽으로 피는 모습
	- 날다, 오르다

## 風
- [[風]](바람 풍) [185] : 공기가 들어가면 바로 벌레가 생김
	- 凡(무릇 범) + 虫(벌레 충) 
	- 風俗(풍속)
	- 淸風明月(청풍명월): 맑은 바람은 밝은 달

	- [[凡]](무릇 범, 평범 범, 모든 범, 일반 범)
	- [[几]](안석 궤) 
		-  물질 + 생각 
		-  모양(디자인)을 만들어 냄
	- [[殳]](몽둥이 수, 창 수) =  几(안석 궤) + 又(또 우)
	
	- [[楓]](단풍나무 풍)
		- 단풍(丹楓)
		- 楓嶽(풍악) = 금강산의 가을 명칭(名稱)
	- [[颱]](태풍 태) = 風(바람 풍) + 台(별 태)
		- 颱風(태풍)
		- 颱風警報(태풍경보)
		- 颱風注意報(태풍주의보)
