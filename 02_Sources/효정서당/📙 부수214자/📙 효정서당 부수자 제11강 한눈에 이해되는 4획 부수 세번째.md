### 爪
- [[爪]](손톱 조) : [[爫]](손톱 조 머리) [87]
	- 2~3급한자 41강 爪(손톱 조)의 유형별 한자
	- 손 끝을 보고 만든 글자
	- 섬세한 일
	- 무기(칼귀)
	
	- [[爭]](다툴 쟁):
		-  [[爫]](손톱 조) + [[又]](오른 손, 또 우) + [[亅]](갈고리 궐)
		- 손톱으로(爫) 할퀴며 손(又)에 갈고리(亅)를 들고 다투다.
		- 줄을 놓고 서로 팽팽하게 잡아당기는 모습
		- 다투다, 결쟁하다
			
	- [[淨]](맑을 정, 깨끗할 정)
		- 氵(삼수변 수) + 爭(다툴 쟁)
		- 깨끗하다
		- 맑다
		- 사념이 없다 
		- 흐르는 물은 다투듯이 흐름
		- 오염되지 않은 말은 물
		- 오염된 것을 걸러내고 깨끗하게 유지하기 위해서는 淨化 과정이 필요
	
	- [[愛]](사랑 애)
		- 爫(손톱 조) + 冖(덮을 멱) + 心(마음 심) + 夊(뒤저올 치)
		- 덮고 싶은 허물(心)까지도 섬세(爫)하게 뒤(夊)에서 보살펴 주다
		- 愛情(애정) : 사랑하는 마음
			
	- [[夂]](뒤져올 치)
		- 뒷짐을 지고 발을 땅에 끌며 걸어가는 모양, 머뭇거리며 뒤져옴. 
		- 冬(겨울 동) : 서서이(夂) 얼음(冫 : 얼음 빙)이 얼기 시작하는 겨울
		- 冬節(동절) : 겨울철
		- 冬至(동지) : 24절기 하나, 밤이 가장 길고 낮이 가장 짧은 날
			
	- [[夊]](천천히 걸을 쇠) 
		- 발을 끌며 천천히 걷는 것을 뜻한다. 
		- 夂(뒤져올 치)와 모양이 거의 비슷하여 지금에 와서는 같이 쓰고 있다.
		- 夏(여름 하) : 頁(머리 혈) + 夊(천천히 걸을 쇠)

	- 비슷하게 생긴 한자
	- [[瓜]](오이 과) [97]
		- 오이 덩굴에 열매가 맺혀있는 모습
		- 瓜年(과년) 
			- 15~16세의 여자 
			- 나이가 오이처럼 예쁜 그런 따님
### 爿
- [[爿]](조각널 장, 나뭇조각 장) [90]
	- 큰 조각
	- 큰 송판
	- 널, 널뛰기
	- 책상, 제사상
		
	- [[壯]](씩씩할 장, 장할 장)
		- 爿(조각 장) + 士(선비 사)
		- 이치를 알면 씩씩하게 자를 수 있음
		- 씩씩하다
		- 굳세다
		- 평상과 도끼를 함께 그린 것
		- 壯(장) : 당차고 인품이 훌륭한 남자
		- 壯年(장년):
			- 기운이 씩씩하여 한창 활동이 활발한 시기, 또는 그런 사람
			- 30살 이후의 남자들은 장년이라고 불렀다
	
	- [[莊]](무성할 장, 별장 장)
		- 艹(초두머리 초) + 壯(장할 장)
		- 풀이 씩씩하다, 무성하다
		- 장중하다
		- 영토, 별장
		- 別莊(별장) : 살림집 밖, 경치 좋은 곳에 따로 지어 때때로 묵으면서 쉬는 집
		- 山莊(산장) : 산 속에 있는 別莊
		- 老莊(노장) : 老子와 莊子
		- 莊重(장중) : 장업하고 정중함, 씩씩하고 의젓함
		- 田莊(전장) : 개인(個人)이 소유하는 논밭
				
	- [[裝]](꾸밀 장)
		- 衣(옷 의) + 壯(장할 장)
		- 선비는 옷을 잘 입어야 함
		- 어떠한 것에 임하기 전에 준비한다
		- 꾸미다
		- 장식하다
		- 武裝蜂起(무장봉기) : 학정에 대항하여 무장(武裝)을 하고 일어나는 저항
		- 裝飾(장식) : 겉모양을 아름답게 꾸밈
		- 包裝(포장) : 물건(物件)을 싸서 꾸림
			
	- [[將]](장수 장, 장차 장)
		- 爿(조각널 장) + 肉(육달 월) + 寸( 법칙 촌)
		- 장수는 법도에 맞게 제사를 지냄
		- 장수
		- [[將帥(장수)]]
		- [[將軍(장군)]]
		- [[將官(장관)]]
		- [[日就月將(일취월장)]] 
			- 날마다 달마다 成長하고 發展한다
			- 學業이 날이 가고 달이 갈수록 進步함
		- [[獨不將軍(독불장군)]]  : 남의 의견을 무시하고 혼자 모든 일을 처리하는 사람
		
	- [[獎]](권할 장, 권면할 장, 장려할 장) 
		- 將(장수 장) + 大(큰 대)
		- 크게 될 사람은 권한다
		- 권장하다, 장려하다
		- 勸奬(권장) : 잘하도록 권하여서 장려함
		- 勸(권할 권) : 力(힘 력(역)) + 雚(황새 관)
			
	- [[戕]](죽일 장) : 戈(창 과) + 爿(나뭇조각 장)
	- [[臧]](숨을 장, 착할 장)
		- 臣(신하 신) + 𤕫(병들어 기댈 녁(역)) + 戈(창 과)
		- 장수가 창을 들고 죽일려고 나타나니 신하들이 숨는다
		- 착하다, 좋다
		- 감추다, 숨다, 숨기다
		- 臧否(장부) : 착함과 착하지 못함
		- 善惡(선악) : 착한 것과 악한 것

### 片
- [[片]](조각 편) [91]
	- 조각, 작은조각, 한쪽, 쪼개다
	- 나무의 오른쪽 면을 그린 것, 나무기둥 우측으로 갈라진 나뭇가지
	- 작은 조각의 物件
		
	- 切片(절편) : 끄트러기, 토막, 자투리
	- 片肉(편육) : 앏게 썬 수육
	- 片道(편도) : 가고 오는 길 중 어느 한 쪽
	- 破片(파편) : 깨어진 조각, 부서진 조각
	- 一片丹心(일편단심) : 한 조작의 붉은 마음

### 牙
- [[牙]](어금니 아) [92] 
	- 이빨, 어금니, 숫치(4획)
		
	- [[齒]](이 치):앞니(암)
	- [[牙]](어금니 아):어금미(숫치) 
		- 齒牙(치아) : 앞니와 어금니
		- 象牙塔(상아탑) : 藝術至上主義 사람들이 俗世를 떠나 정적한 藝術만을 즐기는 境地
		
	- [[芽]](싹 아)
		- 艹(초두머리 초) + 牙(어금니 아)
		- 새싹, 싹이 트다, 처음, 시초, 시작
		- [[發]]芽(발아) : 풀, 나무에 눈이 틈, 싹트기
		- [[胚]]芽(배아) : 수정란에 배낭 속에서 분열 증식한 것으로 장차 포자체의 바탕이 됨
		- 麥芽(맥아) : 보리에 물을 부어 싹이 나게 한 다음에 말린 것, 엿기름
			- 麥(보리 맥) : 來(올 래(내)) + 夂(뒤쳐져 올 치)
				- 「보리」의 이삭, 뿌리의 모양을 본뜬 글자
				- 麥酒(맥주) : beer
				- 麥飯(맥반) : 보리밥
		
	- [[雅]](맑을 아, 우아할 아) : 牙(어금니 아) + 隹(새 추)
		- 우아하다, 맑다
		- 어금니 소리가 새처럼 맑다
		- 메 까마귀가 울던 소리에서 뜻이 파생된 것으로 봄
		
		- 淸雅(청아) : 말고 아름다움
		- 雅量(아량) : 너그럽고 깊은 도량
		- 端雅(단아) : 端正하고 아담(雅淡)함
		
		- [[端]](단정할 단, 끝 단) : 立(설 립(입)) + 耑(시초 단, 끝 단)
			- 耑 : 잡초의 뿌리와 이파리를 표현한 것으로 시초라는 뜻
			- 바르다, 단정하다(이파리가 앞으로 곧게 뻗어 나가는 모습에서)
			- 시초, 끝(耑의 의미에서)
			- 多端(다단) : 일이 흐트러져 가닥이 많음
			- 多事多難(다사다단) : 일이 많은데다가 까닭도 많음
			- 複雜多端(복잡다단) : 일이 얽히고 설키다 갈피를 잡기 어려움
			
		- [[邪]](간사할 사, 그런가 야)
			- 어금니 아(牙) + 언덕 부(阝, 阜, 𨸏, 邑)
			- 맑은 소리를 내는 고을 사람들(목소리는 예쁜데, 성정은 그렇지 않음)
			- 간사하다, 사악하다, 바르지 못하다
		
		- 邪惡(사악) : 도리에 어긋나고 악독함
		- 奸邪(간사) : 性質이 刊校(간교)하고 사곡(邪曲)함
		- 妖邪(요사) : 요망(妖妄)하고 간사(奸邪)스러움
		- 邪敎(사교) : 부정하고 요사스러운 종교, 올바르지 못한 가르침
		- 邪氣(사기) : 요망스럽고 간악한 기운

### 羊

		- [[苗]](싹 묘) : 밭에 풀이 나옴같은 모양

			- 명분이 서지 않는 일로 몰골 사납게 싸움
		- 堂狗風月(당구풍월)
			- 서당(書堂) 개 3년에 풍월(風月)을 한다
			- 무식쟁이라도 유식한 사람과 사귀면 견문이 넓어짐
			- 무슨 일 하는 것을 오래오래 보고 듣고 하면 자연히 할 줄 알게 된다는 뜻

### 5획
### 玄
- [[玄]](검을 현, 아득할 현) [95]
	- 검은 실타래 보고 만든 글자
	- 다른 별에서 지구를 보면은 점으로 보여서 거리가 멀어서 아득하게 보인다.
	- 검다, 가물가물하다, 아득하다(거리가 멀면 검은 점으로 보임)
	- 玄(검을 현) : 활과 시위를 그린 것

	- 天地玄黃(천지현황) 
		- 하늘은 위에 있어 그 빛이 검고 아득하고 
		- 땅은 아래 있어서 넓고 그 빛이 누름
		
	- #### 비슷한 글자
		- [[玄]](검을 현) : 검고 굵은 실
		- [[糸]](실 사) : 옷감 짤 때쓰는 실
			- [[紡]](길쌈 방, 실뽑을 방)
		- [[幺]](작을 요) : 가늘고 작은 실
			- 유치원(幼稚園)
			- 유아(幼兒)
	
	- [[絃]](악기줄 현) : 검고 굵은 실, 끈, 현악기
		- 糹(가는 실 멱) + 玄(검을 현)
		- 玄(검을 현) : 활과 시위를 그린 것		
		- 현악기(絃樂器) : 가야금, 거문고, 바이올린, 기타 따위와 같이 줄을 타거나 켜서 소리를 내는 악기
		- 관악기(管樂器) : 긴 통을 이용한 금관 악기와 목관 악기의 총칭
		- 관현악(管絃樂) : 관악기, 현악기, 타악기로 함께 연주(演奏)함
		
	- [[眩]](아찔할 현) : 
		- 目(눈 목) + 玄(검을 현)

### 玉
- [[玉]](구슬 옥) : [[𤣩]](구슬옥 변) [96]
	- 가공된 여러 개의 보석을 끈오 연결해 놓은 모습

	- 玉자가 부수 역할을 할 때는 王자로 표기된다
	- 구슬이 서 말이라도 꿰어야 보배 : 아무리 좋은 것이라도 쓸모 있게 만들어야 값어치가 있다
	- 왕이 되면 옥좌에 오르다
	- 옥은 따뜻함을 준다.

		- 구슬, 진주, 둥근 형태의 보석
		- 朱(붉을 주) : 赤心木(적심목) 이라는 나무
		- 珠稱夜光(주칭야광) : 구슬의 빛이 영롱하므로 야광이라 칭했음
	
	- [[班]](나눌 반)
		- 玨(쌍옥 각) + 刂(선칼도방 도) 
		- 부수는 玉
		- 나누다, 이별하다, 차례, 자리
		- 칼로 옥을 나누는 모습
		- 차례, 순서 : 무언가를 나눈다는 것은 차례대로 분배한다는 뜻에서
		- 자리 : 고려시대 무신과 문신을 무반(武班)과 문반(文班)이라고 하여 이둘을 합쳐 兩班(양반)
		
		- 班長(반장) : 반(班)으로 일컬어지는 조직체의 우두머리
		- 首班(수반) : 반열(班列) 가운데의 수위, 行政府의 우두머리
		- 鄕班(향반) : 시골에 살면서 여러 대 벼슬을 못 한 양반(兩班)		
		
	- [[碧]](푸를 벽)
		- 玉(구슬 옥) + 白(흰 백) + 石(돌 석)

### 瓦
- [[瓦]](기와 와, 질그릇 와) [98]
	- 기와가 서로 맞물려 있는 모습, 
	- ‘기와’나 ‘질그릇’이라는 뜻
	- 瓦자가 부수로 쓰인 글자가 없지만 다른 글자와 결합할 때는 ‘토기’와 관련된 의미
	- 蓋瓦집 → 기와집
	- [[蓋]](덮을 개)
	- 瓦解(와해) : 기와가 깨진다, 사물이 깨져 산산이 흩어짐을 이르는 말
	- 甑(시루 증) : 曾(거듭 증) + 瓦(기와 와)
		- : 시루 - 떡이나 ㅆ깔 따위를 찌는 데 쓰는 둥근 질그릇
		
	- [[曾]](거듭 증, 일찍 증)
		- 구멍이 있는 조리 도구의 일종을 그린 것
		- 지금으로 따지면 떡이나 쌀 따위를 찔 데 쓰는 시루를 그린 것이다. 
		- 갑골문에서의 曾자는 찌다’라는 뜻으로 쓰였었다. 
		- 그러나 후에 曾자가 ‘일찍’이라는 뜻으로 가차(假借)되면서 
		- 지금은 여기에 瓦(기와 와)자를 더한 甑(시루 증)자가 ‘시루’라는 뜻을 대신하고 있다.
