## 口
- [[口]](입 구) [30]
	- 사람의 입모양 = 말하다, 먹다, 맛보다
	- 어떤 모양으로도 쓰임
		
### 사람의 입모양 = 말하다, 먹다, 맛보다의 경우
- [[古]](옛 고, 오래 고)
	- 십 대째 입으로 내려오는 **옛날** 이야기

- [[言]](말씀 언) [149]
	- 제부수 
	- **머리로 두번 생각하고 말해야 한다** [꼬불]
	- 부수로 쓰일 때는 ‘말하다’와 관계된 뜻을 전달하게 된다. 
	- 말이 씨가 되어 자랄 수 있고
	- 성인의 좋은 말씀은 우리 가슴에 좋은 씨가되어 마음에 새기게 된다
	- 공통적인 언어, 좋은 말씀을 할 땐 말씀 언(言)자가 쓰임
	- 비교
		- 口(입 구) : 말하다, 어떤 모양
		- 言(말씀 언) : 말, 좋은말씀, 공통적인 언어
	
- [[吉]](길할 길) 
	- 훌륭한 사람이 하는 말은 모두 훌륭하다
	- 길하다, 상서롭다
	- 士(선비 사) + 口(입 구)
		
### 口(입구) 가 모양으로 쓰일 경우
- [[品]](물건 품, 성품 품)
	- [[性品]] 
	- [[인품(人品)]]
	- 品 대신에 [[稟]](여쭐 품)을 사용해야 맞지만 현대에서는 물건 품으로 사용 
	- 인품이나 성품을 얘기할때도 물건 품자가 쓰인다.
	- [[稟]](여쭐 품) : 禾(벼 화) + 㐭(곳집 름(늠))

- [[哭]](소리내어 울 곡) 
	- [[吅]](부르짖을 훤,현) + 犬(개 견)
	- 개가 울부짖는다는 뜻에서 사람이 슬픔에 겨워 울다의 뜻으로 변화되었음.
	- 부수: 口(입 구)
	- [[대성통곡(大聲痛哭)]] : 큰 소리로 목을 놓아 슬피 욺
	
- [[霝]](비 내릴 령)
	- 雨(비 우) + [[𠱠]](많은 소리 령, 시끄러울 령(영))
	- [[雨]](비 우) : 하늘에서 구름이 덮히면 빗줄기가 주룩주룩 내리다
	
- [[靈]](신령 령,영) 
	- [[霝]](비 내릴 령) + [[巫]](무당 무)
	- 하늘의 소리가 내리다
	- [[영감(靈感)]] : 느낌
	- [[영감(領監)]] : 노인남자를 높여 부르는 말
		- [[領]](거느릴 령, 다스릴 령)
		- [[監]](살필 감)

## 囗
- [[囗]](에울 위, 에둔담 위) [31]
	- 큰 입구 : 별명이다
	- 사방을 둘러싸고 있는 경계선, 울타리 모양으로 에운담을 뜻한다
	- 글 이전에 우리말에서는 울=우리
	- 에울=에워싸다+우리: 에워싸서 보호해주다
	- 돼지우리의 우리도 에워싸다의 의미가 담겼있다.
	
	- [[國]](나라 국) 
		- [[囗]](에울 위) + 戈(창 과) + 口(입 구) + 一(한 일)
		- 에울 위는 외형적으로는 나라를 지켜주는 경계선일 수 있고, 
		- 내면적으로는 생활도구(戈)를 갖춰서 누구나 나라에 일조를 한다는 뜻
		- 둘러싸서 보호한다
	
	- [[因]](의지할 인, 말미암을 인, 인할 인) 
		- 크게 되기 위해 모든 사람들이 도와 줌
		- ~으로 부터, 인하여, 말미암아
		- 부수는: 囗(에울 위)		
			- [[인과(因果)]] : 원인과 결과
			- [[인연(因緣) ]]: 서로의 연분
			
	- [[恩]](은혜 은)
		- 내가 큰사람(大)이 될려고 마음(心)만 먹으면 모두가 나를 돕게 되는데
		- 그것을 우리는 마음속으로 은혜롭게 생각하고, 감사하는 마음을 갖는다
		- 옛날에 의지(因)하고 살았던 그 사람을 마음 속으로 생각하는 은혜. P.250
			- 은사(恩師) : 배움을 받은 은혜로운 스승
			- 은총(恩寵) : 높은 사람에게서 받는 특별한 은혜와 사랑
			
	- [[囚]](**가둘 수, 죄인 수**) 
		- 모든이가 나에게 걸림돌이 될때는 가둔다, 가둘 수
		- 죄인이 되기도 한다, 죄인 수
		- 에워싸서 사람을 가둘 수 있을까? P249
		- [[수인(囚人)]] : 옥에 갇힌 사람
		- [[죄수(罪囚)]] : 
	
	- [[溫]](따뜻할 온) 
		- 氵(물 수) + 囚(가둘 수) + 皿(그릇 명) 
		- 죄수에게 따뜻한 물과 밥을 주듯 따뜻한 마음
		- [[온순(溫順)]] : 성질이 부르럽고 순함
		- [[온상(溫床)]] : 인공으로 열을 가하여 식물을 가꾸는 시설
## 土
- [[土]](흙 토) [32]
	- 一(한 일: 땅) + 丨(뜷을 곤: 심으면) + 一(한 일: 열매가 나온다)
	- 땅에 관련된 것은 모두 土가 나온다.
	- 토극수(土克水) : 흙이 물을 이긴다는 오행(五行)의 운행
	
	- [[吐]](토할 토)
		- 口(입 구) + 土(흙 토)
		- 토하다, 게우다
		
	- [[奎]](별 규, 문운 맡을 규) 
		- 大(클 대) + 圭(서옥 규)
		- 큰 영토를 다스릴 수 있는 큰 인물
		- 현실적으로 보면 부모님이 주신 유전자라고 볼 수 있음
		- 文運(문운): 학문이 발전되는 운
		- 奎章閣(규장각):
		
	- [[佳]](아름다울 가)
		- 아름답다, 좋다, 훌륭하다, 훌륭한 사람
		- 絶世佳人(절세가인): 세상에 비할 데 없이 아름다운 여자
	
	- [[陸]](육지 육, 뭍 육)
		- 육지, 땅, 뭍
		- [[圥]](버섯 록)= [[土]](흙 토) + [[儿]](어진 사람 인)

			- 흙과 산이 결합한 것으로 언덕, 흙덩이
## 士
- [[士]](선비 사) [33]
	- 十 + 一 
	- 十자는 음양관계
	- 세상은 음양관계다. 이긴 사람이 있으면 진 사람이 있고
	- 항상 반대가 존재한다.
	- 그래서 공부를 해서 음양 관계를 알아가는 사람을 선비라 한다.
	- 우리같이 한학을 공부하는 사람을 선비라고 해야한다.
	
	- [[仕]](섬길 사, 벼슬 사) 
		- 사람들을 섬겨야 섬김을 받는다, 섬길 사
		- 사람은 선비가 되어 학식과 덕망을 갖춰 벼슬한다. P.124
		
	- [[志]](뜻 지)
		- 선비가 마음을 먹는 것은 좋은 뜻이다.
		- 자기 뜻을 실천한다는 의지를 표현한 글자
		- 志學(지학): 학문(學問)에 뜻을 둠, 15세를 일컬음

	- [[誌]](기록할 지)
		- 기록하다, 적다
		- 日誌(일지): 직무상의 기록(記錄)를 적은 책
		- 雜誌(잡지): 호를 거듭하며 정기적으로 간행(刊行)되는 출판물(出版物)

	- 뒷짐을 지고 발을 땅에 끌며 걸어가는 모양이나 머뭇거리며 뒤져옴. P.26
	- [[冬]](겨울 동): 
		- 봄여름가을 뒤따라 오는 계절

## 夊
- [[夊]](천천히 걸을 쇠) [35]
	- 대상이 없이 쇠사슬을 찬 듯 혼자 천천히 걸어감.
	- 두 다리를 끌면서 천천히 걸어감. P.26
	- [[夏]](여름 하, 기우제 하)
		- [[頁]](머리 혈) + [[夊]](천천히 걸을 쇠) 
		- 여름에 지내는 기우제 제사 때 천천히 춤추는 데서 유래함.
		- 갑골문에 보면 탈을 쓰고 기우제를 지내는 모습이 나와있음.
		- 현재 한자 인쇄판에는 뒤져올 치로 표기되어 있고,
		- 천천히 걸을 쇠는 인쇄판에 없고 뒤져올 치로 쓰고 있음.

## 大
- [[大]](큰 대) [37]
	- 사람이 큰 일, 역할을 한다는 뜻
	- 하늘+사람+땅: 중간에서 사람이 돕는 역할을 한다, 사람이 큰 일을 한다
	- 큰 일을 하는 사람 大人(대인)으로 생각할 것
	- 반대는 小人(소인)
	  
	- [[天]](하늘 천)
		- 부수는 大(큰 대)
		- 사람의 머리위에 하늘이 있다는 뜻을 표현
		- 하늘은 동그랗고 땅은 네모나다고 생각
	
	- [[ReadItLater Inbox/夫]](지아비 부)
		- 부수는 大(큰 대)
		- 지아비, 남편(男便), 사내

## 女
- [[女]](계집 녀) [38]
	- 系集(계집): 系(이을 계) + 集(모을 집)
	- 계집 녀는 비하말이 아니다.
	- 고대는 모계사회였기 때문에 여자(女)가 모든 가계 문화를 모아서(集) 내려줬음
	- 姓名(성명): 성씨도 여자 성(姓)을 물려줬음
	- 정신문화: 여자라는 뜻도 있지만, 정신과 관련해서도 많이 쓰인다.

	- [[如]](같을 여)
		- 여기서 계집 녀는 정신문화로 해석하여 같이 해보다.
		- 여자는 남자 말을 같이 한다?
		- 여자의 입은 다 같다. P.94
		
	- [[恕]](용서할 서)
		- 마음과 마음을 같게하다(如)에서 평정심을 유지한다는 뜻
		- 평정심을 유지하며 너그러운 마음을 갖는다는 의미에서 용서하다, 인자하다 라는 뜻.
		- 잘못한 사람을 같은 마음으로 용서하다. P.94
		- 容恕(용서): 관용을 베풀어 벌하지 않음, 容(얼굴 용)

	- [[婦]](며느리 부, 지어미 부, 아내 부) 
		- 집에서 어머니가 앞치마를 하고 비질 등 집안일을 하는 모습
		- 夫婦(부부):
