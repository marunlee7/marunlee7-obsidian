## 隹
- [[隹]](새 추) [172]
	- 꽁지가 짧은 새
	- 참새 같은 것
	- 鳥(새 조): 꽁지가 긴 새

- [[雀]](참새 작)
	- 참새는 작음
	- [[작설차(雀舌茶)]] : 갓 눈이 튼 차나무의 새싹을 따서 만든 차

- [[隺]](새 높이 날 학(확), 고상할 각) 
	- 새가 경계를 나와 높이 날아오르려 하다
	- 冖(덮을 멱) + 隹(새 추)

- [[鶴]](학 학, 두루미 학) 
	- 隺(고상할 각) + 鳥(새 조)
	- 학, 두루미(순 우리말)
	- 고대부터 학은 신선이 타고 다니던 새로 알려져 있으며 선비를 상징
	- 고대 중국서도 학은 고상함의 상징

	- [[군계일학(群鷄一鶴)]] : 무리지어 있는 닭 가운데 있는 한 마리의 학
		- 여러 평범(平凡)한 사람들 가운데 있는 뛰어난 한 사람을 이르는 말.
	- [[학수고대(鶴首苦待)]] : 학처럼 목을 길게 빼고 기다린다, 몹시(苦) 기다림

- [[雁]](기러기 안) 
	- 厂(기슭 엄) + 倠(추할 휴)
	- 언덕, 바위에 살면서 사람과 친한 새
	- 안항(雁行) : 줄지어 날아가는 기러기처럼 조금씩 비껴 뒤쳐져 가는 길

- [[鷗]](갈매기 구)

- [[䧹]](매 응) : 집에서 사람이 길들이는 새

- [[應]](응할 응) : 길들인 주인 마음에 응하다

- [[集]](모을 집) 
	- 많은 새들이 나무에 와서 모여있는 것
	- [[집중(集中)]]

- [[雜]](섞일 잡) 
	- 衣 + 木 + 隹
	- 알록달록한 온갖 새들이 모임
	- 섞이다, 뒤섞이다, 어수선하다
	- 여러 가지가 뒤섞여 순수하지 않거나 자질구레한의 뜻의 나타내는 말
	- 잡화(雜貨)
	- 잡지(雜誌)
	- 잡채(雜菜)
	- [[신변잡기(身邊雜記)]] : 자기(自己) 한 몸이 처해 있는 주위(周圍)에서 일상(日常) 일어나는 여러 가지 일을 적은 수필체의 글
	- 복잡다단複雜多端() = 일이 얽히고 설키다 갈피를 잡기 어려움

- [[鳥]](새 조): 꽁지가 긴 색
- [[烏]](까마귀 오)
	- 온 몸이 까만 까마귀 오, 눈 표시가 없음
	- 부수 : 火

- [[焉]](어조사 언/노란 까마귀 언)
	- 황색 까마귀
	- 감탄사
	- 어조사
		- [[於]](어조사 어)
		- [[于]](어조사 우)
		- [[焉]](어조사 언)
## 雨
- [[雨]](비 우) [173]
	- 하늘에서 물방울이 떨어지는 모습을 표현한 것으로, 날씨와 관련된 글자
	- 하늘 + 덮을 멱 + 빗줄기가 뚝뚝

	- [[雲]](구름 운, 말할 운, 어조사 운)
		- [[雨]](비 우) + [[云]](이를 운, 구름 운)
		- 구름, 습기, 덩어리

	- [[云]](이를 운, 구름 운)
		- 뭉게 구름이 피어오른 모습을 그린 것으로 원래는 구름이라는 뜻
		- 일컫다, 말하다
		- 云云(운운) = 글이나 말을 인용(引用)또는 생략(省略)할 때에 이러이러함의 뜻으로 하는 말

	- [[雪]](눈 설) 
		- 雨(비 우) + 彗(비 혜)
		- 내린 눈을 빗자루로 쓰는 모습을 표현한 것이다.
			- 雪上加雪(설상가설)
			- 螢雪之功(형설지공)
			- 雪上加霜(설상가상)
			- 萬古風雪(만고풍설)

	- [[彗]](비 혜, 살별 혜)
		- 손에 빗자루를 쥐고 있는 모습
		- 손에 빗자루를 쥐고 있는 모습을 그린 것으로 ‘빗자루’나 ‘쓸다’
		- 彗星(혜성)
			- 갑자기 나타나 뚜렷하게 뛰어남을 비유하는 말.
		- 彗掃(혜소)
			- 바로 깨끗이 청소(淸掃)함

	- [[慧]](지혜 혜) 
		- 彗(살별 혜) + 心
		- 슬기롭다, 총명하다, 사리에 밝다
		- 빗질하는 모습을 그린 彗 자를 응용해 빗질할 때 햇살에 먼지가 반짝거리듯이 사람의 총명함이 반짝거린다는 뜻을 표현
		- 사리(事理)에 통달하여 사(邪 간사할 사), 정(正 바를 정), 득(得 얻을 득), 실(失 잃을 실)을 판단하여 좋은 것을 취(取 취할 취)하고 나쁜 것을 버리는 슬기
			- 智慧(지혜) = 智(지혜 지) + 慧(지혜 혜)
				- 삶의 경험이 풍부하거나 세상 이치나 도리를 잘 알아 일을 바르고 옮게 처리하는 마음이나 두뇌의 능력, 슬기
			- 慧眼(혜안) = 사물을 밝게 보는 슬기로운 눈

	- [[霧]](안개 무) 
		- 雨 + 務(힘쓸 무)
		- 안개, 안개 자욱하여 어둡다
		- 안개는 공기중의 수증기가 지면에 닿아 부옇게 떠 있는 현상
		- 안개가 짙게 낀 아침은 날씨가 맑은 징조라는 설이 있음

		- 務(힘쓸 무) = 矛(창 모) + 攵(칠 복) + 力(힘 력(역))
		- 敄(힘쓸 무)
			- 창을 들고 훈련하고 힘씀
			- 힘쓰다, 업무
				□ 業務(업무)

	- [[雷]](우뢰 뢰(뇌)) 
		- 雨 + 畾(밭갈피 뢰)
		- 실제로는 田(밭 전)이 3개
		- 눈에 보이지 않는 천둥소리를 표현하기 위해
		- 논밭 사이 밭갈피로 소리가 휘돌아 나가는 모습으로 그려짐
			- 附和雷同(부화뇌동)
				□ 附 붙을 부, 和 화할 화, 雷 우레 뢰(뇌)/ 同 한가지 동
				- 우레 소리에 맞춰 함께한다
				- 자신의 뚜렷한 소신없이 그저 남이 하는 대로 따라가는 것

## 非
- [[非]](아닐 비, 그릇 비) [175]
	- 상대는 아니고 나만 옳다
	- 나는 새의 모양으로 양쪽 날개가 서로 엇갈려 날아간다 하여
	- 어긋나다, 아니다

	- 부정사
		- [[不]](아닐 불)
		- [[勿]](말 물)

	- [[是]](옳을 시, 바를 시) 
		- 日 + [[疋]](바를 정)
		- 해는 누구에게나 정확하고 바르다는 뜻이 합쳐짐
			- 非人間(비인간)
			- 是非(시비) = 옳으니 그르니 하는 말다툼

	- 非禮勿視 非禮勿聽 非禮勿言 非禮勿動(비례물시 비례물청 비례물언 비례물동)
		- 예가 아니면 보지도 말라
		- 예가 아닌 것은 듣지 말아야 하고
		- 예가 아니면 말하지도 말아라.
		- 예가 아니면 움직이지 말아야함.
