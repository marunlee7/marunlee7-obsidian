## 4획
### 方
- [[方]](모서리 모 방, 방향 방, 사 방, 처방 방, 이제 방, 바야흐로 방) [70]
	- 방향, 널리
	- 배의 모서리 또는 밭을 갈때 사용하던 쟁기 모습에서, 모서리 모 방
	- 모서리를 어느 방향으로 설정했느냐에 따라, 방향 방
	- 의사의 처방
	- 필순 주의 할 것

	- [[芳]](향기 방, 꽃다울 방)
		-  艹(초두머리 초) + 方(모 방)
		- 꽃답다, 꽃향기, 꽃향기가 널리 퍼지다
		- 꽃의 향기(艹) 가 사방(方)으로 퍼지는 것 
		- 芳年 十八歲(방년 십팔세)
		- 芳名錄(방명록) 
			- 아름다운 이름을 기록하는
			- 특별히 기념하기 위하여 남의 성명을 기록두는 책.
		
	- [[坊]](동네 방)
		- 土(흙 토) + 方(모 방)
		- 흙(土)으로 사방(方)에 집을 짓고 사는 동네 [꼬불]
		- 坊坊曲曲(방방곡곡): 한군데도 빠짐없이 여러 곳
		 
	- [[彷]](헤맬 방, 배회할 방)
		- 彳(조금 걸을 척) + 方(모 방)
		- 헤매다, 거닐다, 배회하다, 돌아다님
		- 彷徨(방황)
			- 방향이나 위치를 잘 몰라 이리저리 헤매는 것
			- 삶의 분명한 목표를 정하지 못하고 마음의 갈등을 겪거나 갈팡지팡하는 것
	- [[仿]](비슷할 방, 헤맬 방)
		- 亻(사람인변 인) + 方(모 방)
		- 본뜨다, 모방하다
		- 방황하다, 배회하다
		- 다른 사람(人)을 비슷하게 모방하다(方)
	- [[紡]](실 뽑을 방, 길쌈 방)
		- 糹(가는 실 멱) + 方(모 방)
		- 길쌈(실을 내어 옷감을 짜는 모든 일을 통틀어 이르는 말)
		- 실을 뽑다
		- 紡織(방직): 기계를 사용하여 실을 날아서 피륙을 짜는 것
		- 紡績(방적): 동식물의 섬유를 가공하여 실을 만듦
			
	- [[房]](방 방)
		- 戶(외짝문 호) + 方(모 방)
		- 방, 거실, 가옥
		- 외짝문 방문(戶)을 열고 들어가면 네모난 방(方)이 있음
	
	- [[訪]](찾을 방)
		- 言(말씀 언) + 方(모 방)
		- 의견을 물어 찾아가서 답을 찾다, 탐구하다
		- 의견(言)을 얻기 위해 동분서주(方)한다
		- 말이 통하는 사람을 사방(方)에서 찾는다
		- 訪問(방문): 남을 찾아옴 
		
	- [[放]](내놓을 방, 내칠 방, 쫓을 방, 놓을 방): 사방으로(方) 쳐(攵)서 나쁜 놈을 쫓아 버려 안전하게 놀게 한다
		- 방학(放學)
		- 放送局(방송국)
### 无
- [[无]](없을 무) [71]
	- = 無
	- 인지하지는 못하지만 하늘 어딘가에는 있다
		
	- [[無]](없을 무) 
		- 人 + 廾(두손들 공) + 火
		- 양 손에 물건을 든 사람이 춤추는 모습 + 불
		- 불놀이야, 불이 춤춘다
		- 완전히 없음
		
	- [[舞]](춤출 무) 
		- [[舛]](어그러질 천) + 無
		- 춤추다, 날아다니다
		- 鼓舞(고무) 
			- 북을 쳐 춤을 추게함
			- 격력하여 기세를 돋움, 부추겨 용기가 생기게 함
		- 舞踊(무용): 춤
		- 舞臺(무대): 노래, 춤, 연극 따위를 하기 위하여 마련된 곳
		
		- [[舛]](어그러질 천) [136]
			- 양발을 서로 엇갈린 모습을 그린 것
			- 어그러지다, 엇갈려 있다, 어지럽다
			- 止(발 지, 그칠 지)가 서로 엇갈려 모습
			- 舛자가 부수로 쓰이면 발, 발의 동작
			- 舛駁(천박): 뒤섞여서 고르지 못하거나 어수선하여 바르지 못함
			
	- [[旡]](이미기 방, 목맬 기) 
		- =无
		- 몸을 돌려앉은 모습
	
	- [[旣]](이미 기) 
		- 皀(밥 고소할 흡) + 旡(없을 무)
		- 부수는 旡(없을 무), 오른쪽에 쓰인 부수는 방
		- 이미 밥 먹을 타이밍을 놓침
		- 旣望(기망): 음력(陰曆) 16일, 이미 망월(望月:15일)이 지났다
		- 旣往之事(기왕지사): 이미 지나간 일
			
	- [[慨]](슬퍼할 개, 분개할 개)
		- 忄(심방변 심) + 既(이미 기)
		- 밥도 먹지 못할 정도의 슬픔
		- 既(이미 기): 旣 의 속자(俗字)
		- 慨歎(개탄): 슬퍼할 개 + 탄신할 탄(歎)
		- 感慨無量(감개무량): 그지없도록 마음속 깊이 스며들어 느낌
		- 悲憤慷慨(비분강개): 슬프고 분한 느낌이 마음속에 가득 차 있음
		- 憤慨(분개): 몹시 분하게 여김

### 日
- [[日]](날 일) [72] 
	
	- [[昌]](클 창, 창성할 창)
		- 日(날 일) + 曰(가로 왈)
		- 번성하다, 창성하다, 아름답다
		- 해가 지구를 크게 빛춰줌
		- 태양 아래에서 아름다움을 노래하는 모습에서 창성하다, 번성하다
	
	- [[唱]](노래할 창): 
		- 口(입 구) + 昌(창성할 창)
		- 노래부르다, 먼저 부르다
		- 말을 꺼내다
		- 가락을 맞추어 노래를 부름, 가창(歌唱)
		- 夫唱婦隨(부창부수)
			- 남편(男便)이 주장(主將)하고 아내가 이에 따름
			- 가정(家庭)에서의 부부(夫婦) 화합의 도리를 이르는 말
			
- [[曰]](가로 왈) [73]

### 毋
- [[毋]](말 무) [80]
	- 부정사 : ~하지 말라
- [[母]](어미 모) 
	- 젖을 더 이상 주지 말라 
	- 젖을 먹이는 능력이 있는 여인
	- 부수 : 毋(말 무)
	- 名母(명모) 
		- 자식이 어머니의 이름을 직접 부른다는 뜻으로
		- 배운 것을 잘못 이용하여 경솔하게 행동함을 이르는 말
- [[拇]](엄지손가락 무)
	- 扌(재방변 수) + 母(어머니 모)
	- 拇指(무지): 엄지손가락

- [[每]](매양 매, 늘 매, 탐할 매) 
	- 어머니가 비녀를 하고 있는 모습에서 
	- 늘, 항상, 한결같은, 마다
	- 每자가 다른 글자와 결합할 때는 여전히 어머니라는 뜻을 전달
	- 每事不成(매사불성): 하는 일마다 실패(失敗)함
	- 每日每日(매일매일): 하루하루
	
- [[苺]](딸기 매)
- [[海]](바다 해): 氵(삼수변 수) + 每(매양 매) 
	- 每자는 비녀로 머리를 단정하게 묶고 있는 어머니
	- 고대 모계사회는 대지, 바다를 여성에 비유
	- 어머니의 물, 어머니와 같은 마음
	- 海자가 처음 등장한 것은 금문이후=(商)나라 이후
		- 人山人海 = 사람이 헤아일 수 없이 많이 모인 모양(模樣)
		- 海外 = 바다 밖의 다른 나라(外國)
		- 學海 
			- 냇물이 수지 아니하고 흘러서 바다로 들어간다
			- 사람이 學問에 힘쓰는 것의 비유
			- 學問을 넓고 깊은 바다에 비유
### 比
- [[比]](견줄 비) [81]
	- [[匕]](사람 비, 비수 비, 숟가락 비, 변화할 비)
	- 두사람이 나란히 견주고 있는 모습, 서로 나란히 서 있는 사람
	- 견주다-어떠한 차이가 있는지 알기 위하여 서로 대어 보다, 비교(比較)하다

- [[批]](비평할 비) : 손으로 비평함
	- 批判(비판)-비평하여 판정함
	- 批評(비평)-좋고 나쁨, 옮고 그름을 갈라 말함
	- 批准(비준)-조약의 체결에 대한 당사국의 최종적 확인
	- 引繩批根(인승비근) - 새끼줄을 걸어 잡아당겨 뿌리째 뽑아 버린다, 둘이 한 패가 되어 다른 사람을 排斥(배척)함

- [[北]](북녘 북, 달아날 배) : 등을 지고 있는 모습
	- 敗北(패배)
	- 敗(패할 패): 조개 패(貝) + 칠 복(攵, 攴)
		- 회의문자 설 - 뜻+뜻(패가 솥 정일경우, 우리 솥(신성)이 깨졌다, 패배)
		- 형성문자 설 - 뜻+음(칠 攴, 회초리로 치다(뜻)+ 패하다(음)결합

- [[背]](등 배)
	- 背信者(배신자)
	- 敗北者(패배자)

- [[皆]](다 개) 
	- 다, 모두, 함께
	- 여러 사람이 함께 목소리를 내다
	- 白(흰 백)이 曰(가로 왈)로 해석
	- 다른 글자와 결합해 모두라는 뜻을 전달
	- 皆勤賞(개근상)

### 氏
- [[氏]](뿌리 씨, 성씨 시, 각시 씨) [83]
	- 나무 뿌리가 땅속으로 뻗어있는 모습
	- 혈통도 나무뿌리와 마찬가지로 하나의 줄기에서 시작되어 점차 확산		
	- 姓氏(성씨): 성(姓)을 높여 부르는 말
		
- [[氐]](밑 저, 근본 저): 땅속으로 뻗은 나무뿌리에 점을 찍은 것으로 낮다의 뜻
	- 나무 뿌리는 아래에 있음
	- 氏(각시 씨) + 丶(점 주)
- [[低]](낮을 저): 사람의 신분이 낮다 → 단순히 낮다(높다의 반대)
	- 亻(사람인변 인) + 氐(근본 저)
	- 낮다, 숙이다
		- 低頭不答(저두부답): 머리를 푹 숙이고 對答하지 아니함
		- 低頭平身(저두평신) 
			- 머리를 숙이고 몸을 움츠린다
			- 몹시 황송(惶悚)해 하는 모습
		- 低首下心(저수하심)
			- 머리를 낮추고 마음을 아래로 향하게 한다
			- 머리를 숙여 복종(服從)함을 이르는 말
- [[底]](바닥 저)
	- 海底(해저)
- [[邸]](큰집 저)
	- 고을에서 가장 높은 곳에 가장 넓게 지은 집
	- 邸宅(저택)
- [[紙]](종이 지)
	- 氏자는 나무 뿌리가 땅속으로 뻗어있는 모습
	- 竹簡(죽간): 종이가 발명되기 이전에는 대나무를 갈라 글을 썼다
	- 紙 = 부드러운 재질인 천, 비단에 글이나 그림을 그렸다, 이것을 紙
		- 表紙(표지): 책의 겉장
		- 紙面(지면): 종이의 표면, 글이 실린 面(면: 낮 면)
		- 更紙(갱지): 지면이 좀 거칠고 품질이 낮은 종이의 한 가지
		- 製紙(제지): 종이를 만듦
		- 韓紙(한지): 닥나무의 껍질로 만든 종이
		- 壁紙(벽지): 벽에 바르는 종이
### 气
- [[气]](기운기 엄) [84]
	- 사람의 기운이 떠듯이 올라가는 모습
	- 하늘에 감도는 공기의 흐름이나 구름을 표현
	- 구름이나 수증기, 아지랑이의 기체가 떠돌아 다니는 모양을 본뜸
		
	- [[氣]](기운 기): 气(기운 기) + 米(쌀 미)
		- 기운(눈에는 보이지 않으나 오관(五官)으로 느껴지는 현상
		- 기세(氣勢 : 기운차게 뻗치는 형세)
		- 기운, 기세, 날씨
		- 밥을 지을 떄 나는 수증기가 올라가는 모습
		- 숨 술때에 나오는 기운(氣運)
		- 생활, 활동하는 힘
		- 에너지 Energy
		
		- 元氣(원기)
		- 精氣(정기)
		- 生氣(생기)
		- 氣力(기력)
		- 正氣(정기): 바른 기풍(氣風)
		- 蒸氣 = 水蒸氣(수증기): 물이 증발하여 기체(氣體)로 된 것
		
		- 사람은 항상 기운(起運) 이 있어야 하고 하늘의 천기, 땅의 지기를 지닌다
			- 天氣(천기): 하늘에 나타난 조짐(兆朕)
			- 地氣(지기)
			- 氣運(기운) 
		
	- [[汽]](물끊는 김 기)
		- 氵(삼수변 수) + 气(기운 기)
		- 汽車(기차)
		- 汽笛(기적): 기관차, 선박 등의 신호 장치, 그것으로 내는 소리
		- 汽船(기선): 증기(蒸氣)의 힘으로 추진시켜서 다니는 배
