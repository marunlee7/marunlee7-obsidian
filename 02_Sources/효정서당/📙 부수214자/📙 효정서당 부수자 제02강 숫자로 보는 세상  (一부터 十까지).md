## 지사자
- [[지사자(指事字)]]
	- 指(가리킬 지), 事(일 사), 字(글자 자)
	- 무형, 모양이 없는 것으로 점이나 선으로 표현
	- 상형자는 모양이 있는 것을 말하고
	- 지사는 모양이 없는 것을 말한다
	- 지사자는 쓰는 대표적인 한자는 숫자가 있다
	- 숫자는 무형으로 추상적으로 표현한 지사자이다
		- 加(더할 가) : 力 + 口
			- 입으로 말하는 것을 더하다
			- 새로운 音 가(加)를 만들어 낸 지사자
				
	- [[指]](가리킬 지) 
		- 扌 + 旨(맛 지, 뜻 지)
		- 손가락으로 방향을 가리키고 지시를 내린다는 뜻이 확대되어 **가리키다, 지시하다**
		- [[지록위마(指鹿爲馬)]] : 사슴을 가리켜 말이라고 한다 
		- [[진두지휘(陣頭指揮)]] : 직접(直接) 진두(陣頭)에 나서서 지휘(指揮)함
		
	- [[趣]](뜻 취, 재촉할 촉, 벼슬이름 추) : 
		- 走(달릴 주) + 取(취할 취)
		- 달리다, 향하다, 취하다
		- 取자는 귀를 붙잡고 있는 모습을 그린 것으로 ‘얻다’나 ‘취하다’라는 뜻
		- 무언가를 얻기 위해 달려나가는 모습을 표현한 것이다. 
		- 즉,  자신이 성취하고자 하는 것을 향해 열심히 노력하며 달려나가는 모습
		- 주로 ‘마음이 쏠리는 방향’이라는 뜻으로 더 많이 쓰이고 있다.
			- [[취지(趣旨)]] : 어떤 일에 담겨 있는 목적, 의도, 의의
			- [[40. PARA/43. Rereferencess/Hword2/취향(趣向)]] : 하고 싶은 마음이 쏠리는 방향
	- [[取]](취할 취) 
		- 耳 + 又
		- 얻다, 가지다
		- [[취사선택(取捨選擇)]]: 취(取)할 것은 취(取)하고 버릴 것은 버려서 골라잡음
			- 取(가질 취 )
			- 捨(버릴 사 )
			- 選(가릴 선 )
			- 擇(가릴 택 )
## 一
- [[一]](한 일) [1]
	- 시작
	- 첫째
	- 크다
	- 무극(無極) : 동양철학에서 우주의 본체인 태극의 맨 처음 상태를 이르는 말
	- 한결같다 : [[一念(일념)]]으로
	- 근본
	- 하늘 : 한자 위로 올땐 하늘 → [[天]](하늘 천) : 大(큰 대 : 사람) + 一(한 일 : 하늘)
	- 땅: 밑으로 올땐 땅 → [[土]] = 十 + 一(한 일 : 땅)
	- 한 일(一) = 하늘	남자	양(+)
	- 한 일(一) = 땅	여자	음(-)

## 二
- [[二]](두 이) [7]
	- [[于]](어조사 우)
		- 하늘과 땅의 기운의 만남 → 온도차에 의해서 에너지가 달라짐
		- 二(두 이) + 亅(갈고리 궐)
		
	- [[云]](말할 운, 구름 운)
		- 몸에서 나가는 온도로 밖에 온도에 맟춰 모양을 만들어 내다
		- 二(두 이) + 厶(사사 사)
		
	- [[雲]](구름 운) 
		- 雨(비 우) + 云(이를 운, 구름 운)
		- [[운운(云云)]] : 글이나 말을 인용(引用) 또는 생략(省略)할 때에 이러이러함의 뜻으로 하는 말.

- [[三]](석 삼)
	- 부수 : 一(한 일)
	- 이렇듯 '나'는 음양의 조화로 창조가 됩니다.
		- 하늘(아버지)과 땅(어머니) 사이에 있는 나
		- 내 안에는 부모님의 에너지가 들어있다
			- 하늘, 아버지 父
			- 나
			- 땅, 어머니 母
	- [[삼극(三極)]] : 음양설에서 만물을 제재한다는 뜻으로 하늘, 땅, 사람을 가리킵니다.
		- [[천지인(天地人)]] : 삼재(三才)를 이루는 하늘과 땅과 사람을 아울러 이르는 말.  

## 四
- [[四]](넉 사) 
	- 부수 : 囗(에울 위)
	- 에울 : 우리(울타리)
	- 부모님에 의해 창조 될 때 나는 사람(儿)으로 이루어지고 온전한 튼(에울 위)를 만듬
		
- [[國]](나라 국) :  
	- 囗(에울 위) + [[或]](혹 혹)
	- [[或]](혹 혹) : 창을 들고 국토를 지키는 것은 혹시나 누가 쳐들어오지 않을까 하는 것이다. 
		- [[40. PARA/43. Rereferencess/Hword2/전광진 하루한자/혹시(或是)]] : 그러할 리는 없지만 만일(萬一)에.
		
	- [[因]](인할 인) :  
		- 囗(에울 위) + 大(큰 대)
		- 인하다, 말이암다 [원래는 자리였지만 → 현재 茵(자리 인)]
		- 어떠한 원인과 이유
		- 에워싼 영토(領土)를 넓히려고 하는 데에는 반드시 큰 원인, 이유가 있다
		- 에워싼 큰 울타리로 말이암아 의지하면 살아간다 [꼬불]
		- 원인(原因) : 어떤 일의 근본이 되는 까닭
	- [[困]](곤할 곤, 가난할 곤) : 에워싸여 있는 나무는 자라기가 곤란하다 [꼬불]
	- [[囚]](가둘 수) : 예워싸서 사람을 가두다 [꼬불]
	

- [[五]](다섯 오)
	- 부수 : 二
	- 하늘의 기운은 내려오고 丿(삐침 별, 천기)
	- 땅의 기운은 올라감 乙(굽을 을, 지기)
	- 하늘 기운, 땅의 기운이 서로 교차함
	- 五(다섯 오)가 나오면 行(다닐 행)이 따라온다고 생각하시면 됩니다. - 오행
	- 一, 二, 三, 四, 五 까지는 숫자가 기본으로 만들어졌다고 해서 생수
	- 六, 七, 八, 九, 十 은 성수라고 합니다.
	
- [[六]](여섯 륙)
	- 부수 : 八
	- 두 손의 세 손가락을 아래로 편 모양
	- 세포분열, 붙었다 떨어졌다
	
- [[七]](일곱 칠)
	- 부수 : 一(한 일)
	- 칠삭둥이 : 제 달을 다 채우지 못하고 일곱 달 만에 태어난 아이
	- 7이라는 숫자는 하나로 독립 될 수는 있지만 완전한 숫자가 되지 못하기도 합니다.

- [[八]](여덟 팔, 나눌 팔) [11]
	- 단독으로 쓰일 땐, 여덟 팔
	- 八(나눌 팔) : 부수로 쓰일 땐 나누다 라는 뜻
		
	- [[分]](나눌 분) 
		- 칼로 나눔
		- 刀(칼 도) + 八(여덟 팔)
	- [[粉]](가루 분) 
		- 米(쌀 미)을 나누면 가루가 됨
		- 米(쌀 미) + 分(나눌 분)
	- [[忿]](성낼 분)
		- 마음을 나누다, 마음이 산산조각 남, 화내다(火--)
		- 分(나눌 분) + 心(마음 심)
		- [[분노(忿怒)]]
		- [[분쟁(忿爭)]] : 성이 나서 다툼
		- [[분노발작(忿怒發作)]]
		- [[일조지분(一朝之忿)]]

		- [[의사필문(疑思必問)]] : [[의심(疑心)]]이 날 때는 반드시 물을 것을 생각하고, 
		- [[분사필난(忿思必難)]] : 분할 때는 반드시 어려움이 닥칠까 생각함.
			
		- 비슷하지만 전혀 상관 없는 한자
			- 入(들 입) : 사람이 고개 숙이고 들어가는 모습
			- 人(사람 인) : 사람의 옆 모습
## 九
- [[九]](아홉 구, 많을 구) 
	- 부수는 乙 [5]
	- [[乙]](굽을 을, 지기)  + 丿(삐침 별, 천기)

	- 김구(金九) : 九(많다 구)로 쓰임
	- [[십중팔구(十中八九)]] : 열에 여덟이나 아홉. 거의 다됨
		- 거의 예외(例外)없이 그러할 것이라는 추측(推測)을 나타내는 말
	- [[구사일생(九死一生)]] : 아홉 번 죽을 뻔하나 한 번 살아난다. 
		- 여러 차례(次例) 죽을 고비를 겪고 간신히 목숨을 건짐
		
	- [[究]](연구할 구)  
		- 끝까지 찾아내는 것
		- 穴(구멍 혈) + 九(아홉 구)
		- 연구하다, 탐구하다
		- 구멍(穴)에서 구(九)년동안 마지막까지 연구하다 [꼬불]
			- [[구명(究明)]] : 파고들어 밝힘
			- [[30-리소스/Dict/Hword2/다산치학/窮究]] : 속속들이 깊이 연구함
			
	- [[窮]](다할 궁) 
		- 穴(구멍 혈) + [[躬]](몸 궁)
		- 집에 뼈가 앙상한 사람이 있다 → 극에 달하다, 가난하다, 궁하다
			- [[竆]](다할 궁, 궁할 궁)
			- [[呂]](법칙 려(여), 성씨 려(여) , 등뼈 려)
		- [[궁여지책(窮餘之策)]] : 궁한 끝에 나는 한 꾀
		- [[무궁무진(無窮無盡)]] : 끝이 없고 다함이 없음을 형용(形容)해 이르는 말
		
	- [[鳩]](비둘기 구) 
		- 九(아홉 구) +  鳥(새 조) 
		- 비둘기
		- 모이다, 모으다
			- [[구수응의(鳩首凝議)]] : 사람이 모여서 이마를 맞대고 의논하는 모양 
		
	- 부수 : 乙 :  乚(굽을 을 변형, 지기)

- [[十]](열 십) [24]
	- 一(땅) + 丨(뚫을 곤, 하늘) 
	- **다 열렸다, 다 벌어졌다**
	- 양+음 : **완전수**(음양이 갖추어짐)
		
	- 부수로 쓰일 때, **햇빛**이라는 뜻도 있음
	- 햇빛이 있으면 나무가 무성하게 자란다

	- [[南]](남녘 남) :   
		- 十(열 십: 햇빛) + 冂(멀 경) + [[𢆉]](점점 심해질 임, 찌를 임)
		- 𢆉(찌를 임) : 약간 심해지다
		- 잎이 점점 진해지는 모습
