	- [[駐]](말 머무를 주)

	- [[警]](깨우칠 경, 경계할 경) : 깨우치는 말도 순하게 진실된 말로

### 高
- [[高]](높을 고) [189]
	- 亠(높을 두, 머리 두)
	- 鼓手(고수): 북치는 사람
	- 高手(고수): 수가 높은 사람
		
	- [[稿]](볏짚 고, 원고 고)
		- 原稿(원고)
	- [[稾]](볏짚 고)
		- 稾工品(고공품): 짚으로 만든 물건
		
	- [[膏]](기름질 고): 高+月(육달 월)
		- 膏粱珍味(고량진미)
		
	- [[鎬]](냄비 호, 호경 호): 金 + 高
		- 리더가 되라고 이름에 호가 들어감
		- 주나라 수도 호경의 호경 호
	- [[嚆]](울릴 효): 
		- 口(입 구) + 蒿(쑥 호)
		- 嚆矢(효시) 
			- 전쟁터에서 우는 화살을 쏘아 개전의 信號(신호)로 삼다
			- 모든 일의 始初(시초)
		- [[蒿]](쑥 호, 짚 고):  
			- 艹(초두머리 초) + 高(높을 고)
			- 짚(이삭을 떨어낸 줄기와 잎)
			- 볏짚(벼의 낟알을 떨어낸 줄기)

### 髟
- [[髟]](머리늘어질 표, 터럭 발)  [190]
	- 镸 = 長(길다, 자란다, 어른, 우두머리) + 彡(터럭 삼)
	- 머리가 길게 늘어진 모습
	- 髮(터럭 발)에 쓰인다 하여 터럭 발이라고 함
	
	- [[髮]](터럭 발)
		- 髟(머리 늘어질 표) + 犮(개달아날 발, 달릴 발)
		- 개가 달아날 때처럼 생생하게 털이 찰랑거리는 모습
		- 터럭, 머리털, 기르다
		- 길게 드리워진 머리털을 뜻하기 위해 머리털이 드리워진 모습의 髟자를 응용
		-  사람이나 짐승의 몸에 난 긴 털
			- 毛髮(모발)
			- 長髮(장발)
			- 理髮(이발)
			- 危機一髮(위기일발)
				- = 머리털 하나로 천균이나 되는 물건을 끌어당긴다
				- = 당장(當場)에라도 끊어질 듯한 위험(危險)한 순간(瞬間)을 비유
				- = 위험하기가[危機] 한[一] 터럭[髮]으로 무거운 걸 끌듯 함.
				- = 한유의 『與孟尙書書(여맹상서서)』
				- =  其危如一髮引千鈞(기위여일발인천균)
				= 그 위험이 마치 한 터럭으로 천 균 무게를 끄는 것과 같다.
				
			- 間不容髮(간불용발)
				- = 머리털 하나 들어갈 틈도 없다.
				- = 사태가 단단히 급박하여 조그마한 여유도 없음
			- 白髮三千丈(백발삼천장) 
				- = 머리가 몹시 세었다 는 것을 과장
				- = 즉 늙은 몸의 서글픔을 표현(表現)한 것
			- 斷髮美人(단발미인): 단발(斷髮)한 젊은 미인
	
	- 鬚(수염 수, 모름지기 수):  髟(머리 늘어질 표)  + 須(모름지기 수)
		- 須(모름지기 수, 반드시 수): 彡(터럭 삼) +  頁(머리 혈) 
			- 龍鬚鐵(용수철)
			- 虎鬚(호수): 범의 수염
			- 龍鬚(용수): 용의 수염, 임금을 높이어 그의 수염을 이르는 말
		
	- [[髥]](수염 염, 구렛나루 염)
		- 髟(머리 늘어질 표) + 冄(나아갈 염, 머리늘어질 염)
		- 구레나룻 : 귀밑에서 턱까지 잇따라 난 수염
			- 鬚髥(수염)
	
	
### 鬥
- [[鬥]](싸움 투/두) [191]
	- 丮(잡을 극) +(잡을 집)
	- 손을 잡고 싸움
	- 싸우다, 승패를 겨루다
	- 두 사람이 머리를 흩날리며 싸우는 모습을 그린 것
	- 王자는 헝클어진 머리털이 흩날리는 모습을 표현, 왕과는 관련없다
	- 斗(투)의 본자(本字)
	
	- [[斗]](말 두, 싸울 투)
		- 말, 구기
		- 곡식이나 액체를 담는 용도로 사용하던 구기를 그린 것
		- 말 : 용량의 단위
		-  北斗七星(북두칠성): 북극 하늘에 있는 국자 모양의 7개 별자리
		-  국자를 그린 것이기 때문에 양을 재다, 용량
			- 泰山北斗(태산북두)
				- = 중국 제일의 명산인 태산(泰山)과 북두성(北斗星)
				- = 학문, 예술 분야의 대가
			- 斗酒不辭(두주불사)
				- = 말술도 사양하지 아니한다
				- = 주량(酒量)이 매우 큼
			- 三間斗屋(삼간두옥)
				- = 세칸에 한 말들이 밖에 안되는 집
				- = 몇 칸 안되는 오막살이집을 비유
					
		- [[鬪]](싸움 투/두)
			- 鬥(싸울 두) + 豆(콩 두) + 寸(마디 촌)
			- 규칙에 맞게 싸움
			- 주먹쥐고 잡고 싸우다
			- 拳鬪(권투)
			- 鬪爭(투쟁)
		- [[鬧]](시끄러울 요/뇨)
		- 鬥(싸울 두) + 市(저자 시)
			-  시장에서 싸우면 시끄러움

### 鬯
- [[鬯]](울창주 창) [192]
	- 𠚍(창) + 匕(비수 비)
	- 울금향을 넣고 술로 빚음
	- 삐침, 파임, 좌우, 위아래, 입벌릴 감, 비수 비(변화할 비)
	- 울금향 : 여러해살이 풀-카레만들 때
	- 울창주 
		- 울금(鬱金)으로 담근 술
		- 제사를 지내거나 손님을 환대할 때 마시던 술로 향이 독특하여 고급술로 취급되고 있다
	- 10획
	- 그릇(凵）안에  쌀（米）을 넣어  향초（香草）와 함께  술을 빚는 모양
	- 또는 옛날 활집통 모양을 본뜸.
	- 鬯자는 부수로 지정되어 있기는 하지만 결합한 글자가 없고 글자 자체도 거의 쓰이지 않는다.

	- [[鬱]](답답할 울, 울창할 울, 막힐 울)
		-  鬯(울창주 창) + 缶(장군 부) + 林(수풀 림(임)) + 彡(터럭 삼) + 冖(덮을 멱)
		-  울금향을 넣은 질그릇을 숲속에 두면, 발효가 되면서 가스가 참
			- 憂鬱症(우울증)
			- 憂(근심할 우) 
				- = 머리와 마음속에서만 계속 번뇌함, 행동하지 않고
				- = 근심, 걱정
				- = 頁(머리 혈) + 冖(덮을 멱) + 心(마음 심) + 夂(뒤져올 치)
				- = 사람의 머리부터 심장, 발까지가 묘사된 글자라 할 수 있다. 
				- = 머리와 발 사이에 心자가 있어 마치 큰 머리가 심장을 짓눌르는 뜻한 모습을 하고 있다. 
				- = 근심 뜻하기 위해 이렇게 심장이 압박받는 모습으로 그려진 글자이다.
			- 症(병 증세 증)

### 鬼
- [[鬼]](귀신 귀) [194]
	- 甶(귀신 머리 불) + 儿(어진 사람 인) + 厶(사사 사) 
	- 삿댄 곳으로 갔을 땐 귀신 귀
	- 귀신, 혼백(魂魄)
	- 무시무시한 머리를 한 사람의 形像(형상)으로 죽은 사람의 魂(혼)의 뜻을 나타냄. 부수(部首)로 쓰일 경우(境遇)에는 영혼(靈魂)이나 초자연적인 것, 그 작용(作用)에 관한 의미(意味)를 담고 있음.

	- 神出鬼沒(신출귀몰)
		- 귀신(鬼神)처럼 자유자재로 나타나기도 하고, 숨기도 한다
	- 疑心暗鬼(의심암귀)
		- 의심이 생기면 귀신(鬼神)이 생긴다
		- 의심하는 마음이 있으면 대수롭지 않은 일까지 두려워서 불안(不安)해 함
	
	- [[神]](귀신 신) 
		- 囟(숨구멍 신)
		- 잘 풀렸을 땐 귀신 신 
		
	- [[魂]]
		- 넋 혼 
		- 鬼(귀신 귀) + 云(구름 운, 이를 운)
		- 정신(양, 하늘)
		- 넋(정신이나 마음)
		- 사람이 죽으면 하늘로 돌아간다는 영혼
		- 魂(넋 혼)자는 하늘을 떠도는 영혼을 표현하기 위해 云(구름 운)을 씀
		- 땅에 묻게 된다는 魄(넋 백)과는 다른 개념
		- 魂(넋 혼)자는 사람의 정신력이나 마음을 뜻하기도 하는데, 진정한 정신력이란 사람의 영혼에서 나오는 것으로 생각했기 때문이다.
			- 魂飛中天
				- 혼비중천
				- 혼(魂)이 중천에 떴다
				- 정신(精神)없이 허둥거림
				- 죽은 사람의 혼이 공중(空中)에 떠돌아 다닌다
			- 無主孤魂(무주고혼)
				- 거두어 주는 연고자(緣故者)가 없어 떠돌아 다니는 외로운 혼령(魂靈)
	
	- [[魄]]
		- 넋 백 
		- 白(흰 백) + 鬼(귀신 귀)
		- 몸(음, 땅)		
		- 魂魄(혼백)
			- 魂飛魄散(혼비백산)
				- 넋이 날아가고 넋이 흩어지다
				- 몹시 놀라 어찌할 바를 모름				
	- [[魅]](홀릴 매, 매혹할 매)
		- 삿댄 쪽으로 미혹하게 하다		
		- 魅力(매력)
		- 魅惑(매혹)
			- 惑(미혹할 혹)
