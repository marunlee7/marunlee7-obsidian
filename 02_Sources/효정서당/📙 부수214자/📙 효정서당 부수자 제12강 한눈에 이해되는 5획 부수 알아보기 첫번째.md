# 제12강 한눈에 이해되는 5획 부수 알아보기 첫번째

![](https://www.youtube.com/watch?v=rxn6RnwcXZY)

## 用
- 用(쓸 용, 그릇 용) [101]
	- 생활 도구(그릇)을 보고 만듬
	- 물건을 쓰다, 일이 진행되다
- [[甫]](클 보, 큰 부, 남자이름 미칭) 
	- 用 + 父
	- 아버지와 관련이 많이 됨, 가족을 책임지는 자리
	- 아버지는 크게 쓰여지신다
	- 음(音)이 夫(사내 부)와 父(아비 부)와 통하므로 남자의 美稱(미칭)으로 빌어 씀
	- 미칭 : 아름답게 이르는 이름
- [[尃]](넓을 부, 펼 부, 퍼질 포) 
	- 클 보(甫) + 마디 촌(寸)
	- 법도(寸)에 맞게 내 스스로 큰 그릇을 갖춰 남자(甫)의 역할을 함
- [[博]](넓을 박, 도박 박) : 깊고(十) 넓게 알게되면 넓을 박
	- 박사([[30-리소스/Dict/Hword2/최성엽의실용한자/博士]])
	- 도박([[賭博]])
## 甬
- 甬(물 솟을 용, 꽃봉오리 용, 종 용, 골목길 용)
	- 샘 물이 솟아오르는 모양
	- [[涌]](물 솟을 용)
	- [[苚]](꽃봉오리 용) 
	- [[銿]](큰종 종)
	- [[通]](통할 통) 
		- 甬(골목길 용)에서 → 通(통할 통)으로 변함 → 골목길을 벗어나면 道(도)가 있다
		- 도통([[道通]]) : 사물의 오묘한 이치를 깨달아서 통함		
	- [[痛]](아플 통) 
		- 疒(병질 녁/역, 병질 엄, 병상에 드러누운 모양) + 甬(물 솟을 용)
		- 용→통
		- 아프다, 슬프다, 괴롭다
## 田
- 田(밭 전, 사냥할 전) [102]
	- 씨앗을 심어 길러내는 곳
	- [[大田]](대전) : 큰 사람을 길러내는 곳
- 男(사내 남) : 밭에서 힘 쓰는 사람
- 思(생각할 사) 
	- 밭 전을 축소해서 소우주 머리까지 옴
	- 생각은 머리에서 생각하면 마음으로 전이되어 같이 일어남
-  [[恖]](생각 사, 고자) 
	- 囟(숫구멍 신) 
	- 우주의 안테나(기운) + 자꾸 다스려짐
	- 소우주(머리)
- [[甲]](씨앗 갑, 갑옷 갑, 시작 갑, 첫째 갑)
	- 씨앗은 갑옷처럼 딱딱함 → **갑옷 갑**
		- [[鉀]](갑옷 갑) → 갑옷은 쇠로 만듦
	- 천간(天干) → **시작 갑** (갑을병정)
- [[押]](누를 압) → 손으로 압정을 누름
- [[由]](말미암을 유)
	- 열매를 보고 만든 글자
	- 갑을 뒤집으면 ↓ 
	- ~로 부터, ~로 말이암아
	- 하늘과 땅의 수화작용에 의해 ~로부터 라는 뜻이 나옴	
- [[油]](기름 유) : 열매, 과일을 짜면 기름이 나옴
- [[柚]](유자나무 유)

	- 行(다닐 행, 네거리 항)의 생략형
	- 본뜻은 거리지만 걷다의 뜻이 파생
- [[止]](발바닥 지)
	- 발바닥의 모양을 간략히 그린 것으로
 - [[辶]], [[辵]], [[辶]], [[,⻎]](쉬엄쉬엄 갈 착, 걸을 착)
	- 彳(척)과 지(止)가 결합

	- 비교

		- 疋(발 소, 필 필, 짝 필) = 움직이는 발
		- 止(그칠 지) = 발가락으로 발의 모양을 본뜸
		- 走(달릴 주) - 제부수, 달아나는 모양
## 疋 匹
- [[疋]] = [[𤴔]], [[匹]](발 소, 필 필, 짝 필) [103]
	- 발 소 : 발목을 보고 만듬 : 발의 동작이나 상태
	- 필 필 : 길이의 단위
- [[旋]](돌 선) : 사방(方)에서 사람(人)들이 발(疋)을 돌려 돌아선다
	- 㫃(나부낄 언, 깃발 언) + 疋(발 소) 
	- 깃발을 들고 한바퀴 도는 모습
	- 돌다, 회전하다
	- 개선장군([[30-리소스/Dict/Hword3/凱旋將軍]])
	- 검거선풍([[檢擧旋風]])
	- 알선([[斡旋]]) : 남의 일을 잘 되도록 마련하여 줌
	- 선풍([[旋風]])
	- 선회([[旋回]])
## 止
- 止(그칠 지) [77]
	- 발바닥을 보고 만듬
	- 그칠 지 안에는 항상 之(갈 지)가 들어있음
	- 하나는 갔고, 하나는 중지되어있고 그러면서 계속 가고 있는 것
- 正(바를 정) 
	- 止(그칠 지) + 一(한 일)
	- 천도(天道)) : 하늘이 가늘 길
	- 바르다, 옳다, 정당하다
	- 성(口)을 향해(정복하기 위해) 걸어가고 있는 모습을 그린 것
	- 전쟁을 일으키는 데는 정당한 명분이 있어야 한다.
	- 그래서 자신들이 적을 정벌하러 가는 것은 정당하다는 의미에서 바르다
	- 오직 하나(一) 양심에 그쳐(止) 행동하는 것이니 바르고 떳떳한 것이다.[꼬불]
		- 정당([[正當]])
		- 정확([[正確]])
		- 정좌([[正坐]]) : 몸을 똑바로 하고 앉음		
- [[是]](옳을 시, 바를 시, 이 시)
	- 日(날 일) + 正(바를 정)
	- 바르다, 옳다의 뜻
	- 이것(지시대명서)
	- 아래 부분 바를 정(正)으로 발 소(疋)는 아니다. 
	- 시비([[是非]]) :
	- 시시비비([[是是非非]]) : 옳은 것은 옳다, 그른 것은 그르다고 한다. 사리를 공정(公正)하게 판단함을 이르는 말
	- 여시여시([[如是如是]]) : 이러저러하게
- [[定]](정할 정) 
	- 宀(집면, 갓머리) + 正(바를 정)
	- 집(宀)을 지어 아래(下) 사람(人)들이 살 곳을 정해 준다.[꼬불]
		- 定義(정의) - 개념[[槪念]]의 내용을 확정해 보이는 논리적인 규정
		- 確定(확정) - 꽉 결단하여서 틀림없이 정定함
		- 未確定(미확정) - 아직 확정되지 않음
		- 定見(정견) - 일정([[一定]])한 주견
## 癶
- [[癶]](걸을 발, 필 발, 준비된 발) [105]
	- 止(그칠 지)가 변형된 것
	- 사람의 양발을 그린 것(그칠 지 + 그칠 지)
	- 걷다, 가다 [발을 좌우로 벌리고 걸어가는 모양을 본뜸]
	- 가기 전에 모든 걸 준비하고 멈춰있는 상황, 준비된 발로 쏘세요
	- 글자의 상단 부분(머리)에 위치해 발을 뜻하기 위해 만들 글자
	- 止(그칠 지, 발 지)와 足(발 족)자와 같은 뜻을 전달
- [[發]](쏠 발, 출발할 발, 필 발) 
	- 활과 몽둥이를 들고 누군가를 뛰쫓아 가는 모습
	- 화살을 쏜다는 의미에서 누군가를 추격하기 위해 발자국을 따라가는 모습에서
	- 나타나다, 들추다, 밝히다
		- 노발대발([[怒發大發]]) : 몹시 크게 성을 냄
		- 백발백중([[百發百中]]) : 백 번 쏘아 백 번 맞는다
- [[登]](오를 등)
	- 제기(豆)를 준비해서 오르다(癶)
	- 신에게 바칠 음식을 들고 제단 위로 올라가는 모습
	- 오랜 옛날에 쓰던 그릇의 한 가지. 질로 만들면 굽이 높고 모양이 豆(두)와 같음
		- 등교([[登校]])
		- 등산([[登山]])
		- 등재([[登載]]) : 신문, 잡지, 또는 서적 등에 올려 실음.
		- 등장인물([[登場人物]])
		- 등용문([[181 登龍門]]) : 용문에 오른다. 입신출세의 관문을 이르는 말.
		- 등고자비([[02 Booknote/B-점검/登高自卑]])
- [[癸]](열째 천간 계, 북방 계, 헤아릴 계)
	- 걸어(癶)서 하늘(天)의 길이를 헤아린다
	- 지금의 癶(발)자는 유래와 관계없이 천간(天干)의 10번째를 뜻함
	- 겨울, 북방, 천간(天干), 헤아리다
		- 계유([[癸酉]]) : 六十甲子(육십갑자)의 10번째
		- 계미([[癸未]]) : 六十甲子(육십갑자)의 20번째
		- 계사([[癸巳]]) : 六十甲子(육십갑자)의 30번째
		- 계묘([[癸卯]]) : 六十甲子(육십갑자)의 40번째
		- 계축([[癸丑]]) : 六十甲子(육십갑자)의 50번째
		- 계해([[癸亥]]) : 六十甲子(육십갑자)의 마지막인 예순째			
- 비교
	- 止(그칠 지) - ing, 그칠 지 안에는 항상 之(갈 지)가 들어있음
	- 癶(걸을 발) - 가기 전에 모든 걸 준비하고 멈춰있는 상황
## 白
- [[白]](흰 백, 말할 백) [106]
	- 日(해 일) 이라고 보면 햋빛으로 무형, 무색의 → 흰 백
	- 사람의 입(曰 가로 왈)이라고 보면 → 말할 백
	- #### 햇빛은 만물을 생육시키고, 성인은 햇빛처럼 만인을 화육시킨다
	- #### 햇빛은 말없는 성인
	- 화육(化育) : 하늘과 땅의 자연스런 이치로 모든 물건(物件)을 만들어 기름
- [[伯]](맏 백)
	- 伯父(백부): 큰아버지, 둘 이상의 아버지 형가운데 맏이가 되는 형
- [[栢]](잣나무 백, 측백나무 백)
- [[百]](일백 백)
	- 숫자가 많다라는 의미도 있지만 
	- 내가 한 말이 많은 연습을 통해 행동화돼 이루어지는 것을 백 번이라고 해도 됨
		- 백인백색([[百人百色]]) : 많은 사람들이 저마다 다른 특색이 있음
		- 백전백승([[百戰百勝]])
	- 갖은자, 계약서 숫자, 은행 숫자 한자
		- = 한자를 쉽게 고치지 못하도록 만든 한자 
- [[泊]](배 댈 박, 머무를 박)
	- 배를 대다, 머무르다
	- 민박([[民泊]])
	- 숙박([[宿泊]]) : 여관이나 주막에 들어 밤을 자고 머무름
	- 정박([[碇泊]]) : 배가 닻을 내리고 머무름. 碇([[닻 정]])
- [[拍]](손뼉칠 박, 칠 박)
	- 박장대소([[拍掌大笑]]) : 손뼉을 치면서 크게 웃음 
- [[迫]](임박할 박, 핍박할 박) 
	- 辶(쉬엄쉬엄 갈 착) + 白(흰 백)
	- 핍박하다, 다급하다
