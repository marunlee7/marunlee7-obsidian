## 皮
- [[皮]](가죽 피) [107]
	- 厂(바위 엄, 기슭 엄) + 丨(뚫을 곤) + 又(또 우) 
	- 오른손(又)으로 겉 가죽(厂)을 벗기면(丨) 피가나온다
	- 동물의 가죽을 벗겨내는 모습을 그린 것
	- 뜻: 가죽, 껍질, 표면, 겉면
	- 사람이 지치거나 피곤해지면 피부가 먼저 거칠어진다. 
	- 그래서 皮자는 발음과 함께 피부와 관련된 뜻을 전달
		- 革(가죽 혁): 털을 뽑아 만든 가죽
		- 皮(가죽 피): 짐승으로부터 벗긴 채로의 가죽
		- 毛皮(모피)
	
	- [[疲]](피곤할 피) 
		- 疒(병들 녁, 병들어 기댈 녁(역)) + 皮(가죽 피)
		- 지치다, 피곤하다
		- 피곤함(皮)에 힘겨워하는 사람이 병상에 누워있다(疒)는 의미.
		- 疲困(피곤)
	
	- [[波]](물결 파)
		- 가죽을 벗기는 모습을 그린 皮를 응용해 일렁이는 파도를 표현한 글자
		- 일렁이는 물결은 서서히 주위로 흩어져 나가게 된다.
		- 그래서 물결, 주름, 진동하다, 주름지다, 요동치다
		- 波濤(파도) 
		- 一波萬波(일파만파): 한 사건(事件)에 그치지 않고 잇달아 많은 사건으로 번짐
		- 江湖煙波(강호연파): 강이나 호수(湖水)위에 안개처럼 보얗게 이는 잔물결.
		- 平地風波(평지풍파)
			- = 고요한 땅에 바람과 물결을 일으킨다.
			- = 공연한 일을 만들어서 뜻밖에 분쟁을 일으키거나 사태를 어렵고 시끄럽게 만드는 경우
		
	- [[婆]](할미 파) 
		- 老婆(노파)
## 皿
- [[皿]](그릇 명) [108]
	- 그릇 보고 만든 글자

	- [[血]](피 혈) [143] : 그릇에 피 한방울이 떨어지는 모습
		- = 丿(삐침 별) + 皿(그릇 명)
		- 피, 물들이다
		- 고대에는 소, 양의 피를 그릇에 받아 신에게 바쳤다
		- 제사(祭祀)에 희생의 짐성의 피를 그릇에 담아 바친 모양
		- 옛날엔 약속을 할 때 이 피를 서로 빨곤 하였음
		
		- 以血洗血(이혈세혈)
			- = 피로써 피를 씻으면 더욱 더러워진다.
			- = 나쁜 일을 다스리려다 더욱 악(惡)을 범함
		- 血肉(혈육): 피와 살, 자기 소생의 자녀
		- 鳥足之血(조족지혈): 새발의 피, 극히 적은 분량
		- 熱血男兒(열혈남아): 열정의 피가 끓는 사내
	
	- [[盟]](맹세할 맹) : 그릇에 정한수를 떠놓고 천지신명에게 빌다
		- 맹세(盟誓) : 
			- 본딧말 : 盟誓(맹서)
			- 誓(맹세할 서)
		- 선서(宣誓) : 공개적으로 맹세하는 일
	
	- [[益]](더할 익, 많다 익) 
		- 나눠지는데 또 나눠지다, 그릇위에서 
		- 그릇위에 물을 나눠줘서 계속 더해짐
		- [[다다익선(多多益善)]]
	
	- [[溢]](넘칠 일)
		- 바다도 그릇으로 본다면 넘칠 일
		- 海溢(해일) = 바닷물이 넘쳐 육지로 넘어옴

## 目
- [[目]] = [[⺫]](눈 목, 보다 목) [109]
	- 동사로는 보다
	
	- [[看]](볼 간, 살필 간) 
		- 手 + 目
		- 멀리보다 볼 간
		- 자세히 보다 살핀 간
		
		- [[주마간산(走馬看山)]]
			- 말을 타고 달리면서 산을 바라본다는 뜻으로
			- 바빠서 대강 보고 지나감을 이름
		- 간병인(看病人) : 자세히 보다 살필 간으로 쓰임
		- 간호사(看護師) : 자세히 살핀다는 뜻으로 쓰임. [[護]](도울 호)
		
	- [[直]](곧을 직): 
		- 열 개(여러 개)의 눈( 많은 사람)으로 숨어 있는(乚) 것을 바르게 볼 수 있다는 뜻. → 곧다, 바르다
		- 十(열 십) + 目(눈 목) + 乚(숨을 은)
		- 햇빛(十) 보는 각도(目)
		- [[정직(正直)]] : 거짓이나 꾸밈이 없이 성품이 바르고 곧음
		
	- [[眞]](참 진) 
		- 鼎(솥 정) + 匕(숟가락 비)
		- 곧을 직의 변형 모습 [효정]
		- 참되다, 진실되다
		- 참이란, 옳다는 것이 아니라, 그때 그때 일어난 일을 말함
		- 큰 솥 + 수저(= 신에게 바치는 음식은 참되면서도 정성디 담겨야 할 것이다.) [꼬불]
		- 寫眞(사진)

	- [[盲]](맹인 맹)
		- 盲人(맹인): 시각에 이상이 생겨 앞을 보지 못하는 사람

## 示
- [[示]] = [[礻]], 𥘅(제사 시, 보일 시) [113]
	- 제사상을 보고 만듬
	- 고대풍습을 보여주고 있음, 보일 시
	
	- [[社]](토지신 사, 모일 사) 
		- 礻(보일 시) + 土
		- 모이다, 행정단위, 토지 신
		- 示는 신에게 제사를 지내는 제단
		- 社의 본래 의미는 토지의 신
		- 신에게 제사를 지낼 때는 여러 사람이 함께 모여 재물을 바친다
		- 의미가 확대되어 모이다 라는 뜻을 갖게 되었다.
		- 대중국에서는 적은 수의 사람들이 모여 사는 작은 행정단위를 社라 함.
		- [[회사(會社)]]
			- 會(모일 회)
				® 모이다, 만나다
				® 찬합이 결합하는 모습에서
			- 曾(일찍 증)
		
		- 사직(社稷)
			- 토지신과 곡식신 이라는 뜻
			- 사직:  단을 세워 **땅과 곡식의 신에게 제**를 올림
				- 社=땅
				- 稷=곡식
			- [[사직단(社稷壇)]]
			- [[종묘(宗廟)]] : 조선의 역대 왕과 왕비의 혼을 모신 곳
			- [[종묘사직(宗廟社稷)]] : 왕실과 나라를 함께 이르는 말
				- [[稷]](곡식신 직, 피 직, 기울 측): 禾(벼 화) + [[畟]](날카로울 측)
		
	- [[神]](귀신 신, 정신 신) 
		- 示(보일 시) + 申(거듭 신)
		- 뜻을 나타내는 示(제사 시)
		- 음을 나태는 申(펼 신)
	
		- [[申]](거듭 신, 펼 신, 아홉째 지지 신)
			- 번개가 내려치는 모습을 그린 것으로 펴다라는 뜻
			- 거듭, 되풀이 하여
			- 펼쳐지는 모습에서 펴다
			- 아홉째 지지(地支)
			- 방위(方位) 서남서
			- 동물(動物): 원숭이
			- 펴다, 베풀다
			- 부수는 田(밭 전)이지만 부수와는 상관없다.
			- 옛 사람들은 번개가 하늘의 신과 관계된 것으로 생각(=하늘 신)
			- 그래서 神(귀신 신)자가 신이라는 뜻을 대신
			- 申申當付(신신당부): 거듭하여 간곡히 하는 당부
	
		- [[引]](끌 인) 
			- 弓(활 궁) + 丨(뜷을 곤)
			- 끌다, 당기다
			- [[아전인수(我田引水)]]
				- 기의 논에만 물을 끌어넣는다
				- 자기의 이익을 먼저 생각하고 행동함
				- 또는 억지로 자기에게 이롭도록 꾀함을 이르는 말
			- 인수인계(引受引繼) : 업무 따위를 넘겨받고 물려줌
			- 인과자책(引過自責) : 자기의 잘못을 뉘우치고 스스로 꾸짖음
		
		- [[電]](번개 전)
## 衣
- [[衣]] = [[衤]] = [[𧘇]](옷 의, 저고리 의, 입다 의) [145]
	- 윗 옷, 입다
	- 衣(옷 의): 상의
	
	- [[裳]](치마 상): 하의
		- 의상(衣裳) : 상의와 하의를 합친 옷을 의상
		- 의상실(衣裳室)
		- 백의민족(白衣民族) : 예로부터 흰 옷을 숭상하여 즐겨 입은 한민족을 이르는 말
	
	- [[喪]](잃을 상, 죽을 상) 
		- [[吅]](부르짖을 훤) + 十(열 십) + 𧘇(옷의변 의)
		- 관호상제(冠婚喪祭): 관례, 혼례, 상례, 제례
		- 상가지구(喪家之狗) : 초상(初喪)집의 개. 별 대접(待接)을 받지 못하는 사람
	
	- #### 비교: 제사시와 옷의 부수는 다름
	- 礻 = 示(제사 시, 보일 시)
	- 衤 = 衣(옷 의, 저고리 의)

## 禾
- [[禾]](벼 화, 곡물 화) [115]
	- 丿(삐침 별) + 木(나무 목)
	- 뜻 : 벼, 모, 모든 곡식
	- 벼가 여물어 고개를 숙인 모습을 그린 것
	- 부수로 쓰일 때는 곡식의 종류, 가치와 관련된 의미를 전달
	- 과거에는 벼를 세금으로 냈기에 돈이나 세금과 관련된 글자에 영향

	- [[和]](도울 화, 화할 화)
		- 벼가 사람의 입에 들어와 사람의 몸을 도움
		- 화목하다, 온화하다
			- 和睦(화목): 서로 뜻이 맞고 정다움 
			- 睦(화목할 목, 도울 목) = 目(눈 목) + 坴(언덕 륙(육))
				® = 눈가에 주름살이 진 모습
		
	- [[秋]](가을 추)
		- 가을에는 벼(禾), 곡식이 익어(火)간다
	
	- [[鰍]](미꾸라지 추) 
		- 魚(물고기 어) + 秋(가을 추)
		- 가을은 추어탕을 먹는 계절
	
	- [[鞦]](그네 추): 革(가죽 혁) + 秋(가을 추)
	- [[愁]](근심 수): 心(마음 심) + 秋(가을 추)
		- 예전에는 추수가 끝나면 세금을 거둬갔으니 그것이 걱정일 수도 있고
		- 곧 추운 겨울이 올 것이니 그것 또한 걱정일 수도 있다.
		- 가을은 수확의 계절이면서도 근심이 늘어나는 계절이기도 하다.
			- 滿面愁色(만면수색): 얼굴에 가득 찬 수심(愁心)의 빛
			- 憂愁(우수): 우울과 수심, 근심과 걱정
			- 鄕愁(향수): 고향(故鄕)을 그리워하는 마음이나 시름
			- 哀愁(애수): 마음을 서글프게 하는 슬픈 근심

	- 동굴의 모양

	- [[窮]](궁할 궁, 궁구할 궁, 다할 궁)

## 立
- [[立]](설 립) [117]
	- 사람이 서 있는 모습
	- 서다, 똑바로 서다, 임하다

	- 立春大吉(입춘대길): 입춘을 맞이하여 길운을 기원하는 글
	- 立身出世(입신출세): 성공하여 세상에 이름을 드날림
	- 本立道生(본립도생): 기본이 바로 서면, 길 또한 자연스럽게 생긴다
	
	- [[哭]](울 곡): 소리 내어 우는 것
	
	- [[泣]](소리없이 울 읍) 
		- 홀로 서 있는 사람(立) + 눈물(水)
		- 울다, 눈물, 사람이 울고 있음을 표현
		- 소리 내지 않고 눈물을 흘린다는 뜻이다. 
		- 泣訴(읍소): 눈물로써 간절히 하소연함.
		- 訴(호소할 소)
		- 泣斬馬謖(읍참마속)
			- 눈물을 머금고 마속의 목을 벤다
			- 사랑하는 신하를 법대로 처단하여 질서를 바로잡음을 이르는 말
	
	- [[笠]](삿갓 립/입) 
		- 竹(대 죽) + 立
		- 金笠(김입): 김삿갓의 다른 이름
