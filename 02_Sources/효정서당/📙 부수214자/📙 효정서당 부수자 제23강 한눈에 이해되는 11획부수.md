## 魚
- [[魚]](물고기 어) [195]
	- 머리 몸통 지느러미
		
	- [[漁]](고기잡을 어)
		- 고기는 물에서 잡음
		
		- 漁夫(어부)
		- 漁村(어촌) = 고기잡이들이 모여 사는 마을
		
	- [[鯉]](잉어 리) 
		- =  魚(물고기 어) + 里(마을 리(이))
		
		- 鯉魚(이어) = 잉어의 원말
		- 回鯉(회리) = 물음이나 편지(便紙) 따위에 대답(對答)함
		- 叩氷得鯉(고빙득리)
			- = 얼음을 깨뜨려 잉어를 얻음
		- 叩氷得鯉 王祥之孝(고빙득리 왕상지효)
		얼음을 쪼개어 잉어를 얻는 것은 왕상의 효도 이야기 이다. [사자소학]
		- 王祥(왕상) : 고대 중국의 위(魏)나라 서진(西晉)의 효자계모의 심한 학대를 받았지만 계모가 한겨울에 물고기를 먹고싶어하니, 맨 몸으로 얼음을 두드려 잉어를 잡아와 바치니 계모가 감동하여 회개 했다는 지극한 효도 이야기
		- 雪裏求筍 孟宗之孝 剖氷得鯉 王祥之孝 
		- 설리구순 맹종지효 부빙득리 왕상지효
			
	- [[鰍]](미꾸라지 추)
		- 鰍魚湯(추어탕)
	
	- [[鯨]](고래 경) = 魚+ 京(높을 경)
	- [[鰞]](오징어 오) = 魚 + 烏(까마귀 오)
	- [[鰂]](오징어 즉) = 魚 + 則(곧 즉)
		- 烏鰂魚(오징어, 오즉어)
		
	- [[魯]](미련할 노, 둔할 노, 성씨 노, 노나라 노) = 魚 + 曰(가로 왈)
		- = 물고기는 말을 못함 → 미련함, 둔함
			- 鄒魯之鄕(추로지향)
				□ = 공맹의 고향
				□ = 예절을 알고 학문이 왕성한 곳
## 鳥
- [[鳥]](새 조) [196]
	- 꽁지가 긴 새
	
	- [[隹]](새 추): 꽁지가 짧은 새
	- [[鳩]](비둘기 구)
	- [[鷄]](닭 계) 
		- 奚(어찌 해) + 鳥(새 조)
		- 머리는 손톱모양, 부리는 작고, 배는 큼
		- 奚(어찌 해) 
			- 爪(손톱 조) + 幺(작을 요) + 大(큰 대)
			- 어찌, 무슨
			- 원뜻 : 주인에게 종속된 여자 종
			- 어찌, 어디, 무슨과 같이 자신의 신세를 탄식하는 뜻으로 가차(假借)
			- 鷄卵(계란)
			- 卵(알 란)
	- [[鳧]](오리 부)
		- 鳥(새 조) + 几(안석 궤, 물갈퀴 모양)


## 鹵
- [[鹵]](소금밭 로/노) [197]
	- 소금, 소금밭
		
- [[鹹]](짤 함) 
	- 鹵(소금 로(노)) + 咸(다 함)
	- 鹹水(함수) : 바닷물, 짠물
	- 鹹草(함초) : 바다의 수초
	- 鹹泉(함천) = 짠물이 나오는 샘
	
- [[咸]](다 함, 짤 함, 덜 감) 
	- 口(입 구) + 戌(개 술)
	- 다하다, 모두, 남김없이
	- 날이 달린 창을 들고 다 같이 함성을 외치는 모습
	- 누구나 빠짐없이 힘껏 소리를 지른다는 의미에서 모두, 남김없이
	- 口가 더해지면 고함이리는 뜻의 喊(소리 함)
	- 戌(개 술)  = 날이 달린 창

- [[鹽]](소금 염): 신하가 엎드려서 소금밭을 관리함, 그릇에 소금을 채취
	- 臥(엎드릴 와) + 鹵 + 皿(그릇 명)
	- 臥(엎드릴 와) = 臣(신하 신) + 人
	- 鹽田(염전) 


## 鹿
- [[鹿]](사슴 록/녹) [198]
	- 사슴 뿔 + 사슴 다리
	- 동물글자에 올 때는 比(앞 뒷발 비)
	
- [[麗]](아름다울 려, 고울 려) 
	- 丽(짝 려, 고울 려) + 鹿 
	- 곱다, 아름답다, 맑다
	- 丽(짝 려) = 사슴 2마리, 사슴의 뿔
	- =  사슴 2마리가 아름답게 조화를 이룸
		- 高句麗(고구려)
		- 美辭麗句(미사여구) 
			□ = 아름다운 말과 글귀(句), 아름다운 문장
		- 金生麗水(금생여수) = 금은 여수(麗水)에서 남
		- 華麗江山(화려강산) = 빛나고 아름다운 강산

- [[麒]](기린 기) = 鹿 + 其(터 기)
- [[麟]](기린 린) = 鹿 + 粦(도깨비 린)


## 麥
- [[麥]](보리 맥,  밀 맥) [199]
	- 來(올 래) + 夂(뒤쳐올 치) 
	- 보리를 밟아줌
	- 麵(국수 면) : 속자
	- 麪(가릴 면, 뵈지 않을 면) : 정자

## 麻
- [[麻]](삼 마, 참깨 마) [200]
	- 广 + 𣏟(삼실 파) 
	- 麻衣(마의) : 삼베옷
	
- [[魔]](귀신 마, 마귀 마)
- [[痲]](저릴 마) : 마 껍질을 계속 까면 손이 저림
