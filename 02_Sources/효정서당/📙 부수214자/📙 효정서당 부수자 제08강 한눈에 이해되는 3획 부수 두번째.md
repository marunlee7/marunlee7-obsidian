## 3획부수
### 子
- [[子]](자식 자, 아가 자, 아들 자, 씨 자, 열매 자, 불어날 자, 그대 자, 쥐자) [39]
	- 아기가 강보에 쌓여있는 모습
	- 쥐 자(밤11~새벽1시)
	- 동사: 불어날 자로 쓰임 → 字(글자 자)
	
	- [[字]](글자 자) 
		- 마치 사람의 가족이 붇듯이 계속 늘어난다는 뜻을 가지고 있다
		- 집에서 아이를 기른다, 양육하다
		- 글자 전부를 字라 일컬음
		- 집에서 아들을 사랑하는 마음에 가르치기 시작한 글자 [꼬불]
			- 字義(자의): 글자의 뜻
			- 字解(자해): 글자의 풀이, 문자의 풀이, 특히 한자의 풀이
### 宀
- [[宀]](집 면, 갓머리) [40]
	- 집, 건축물과 관련된 뜻을 전달
	- 옛날 움집을 본뜬 자
	- 亠은 지붕 丿 은 두 벽. 
	- 보통 갓머리변이라고 칭하는데, 이 부수가 붙으면 집에 대한 뜻이 된다. P40
	
	- 비교
	- [[突]](갑자기 돌)
		- [[穴]](구멍 혈) + [[犬]](개 견)
	- [[家]](집 가) 
		- 宀(집 면) + 豕(돼지 시)
		- 돼지를 산채로 잡아(豕) 집(宀)으로 매고오는 모습
		- 정착하면서 집에서 돼지를 기르다
		- 두 부수가 합쳐서 만든 家(집 가)는 회의자
		- 出家(출가): 스님이 세속의 연을 버리고 수행을 하러 감
		- 家出(가출): 가정을 버리고 집을 나감
	- [[稼]](농사지을 가, 일할 가,심을 가) 
		- [[禾]](벼 화) + [[家]](집 가)
		- 농사는 벼농사를 지었다
		- 현재는 농사짓는 일이 일하는 거, 일할 가
		- 稼動(가동): 사람이 움직여 일함, 기계 따위를 움직여 일하게 함
	- [[嫁]](시집갈 가) 
		- 여자가 시집와서 집에서 돼지를 기르다
		- 出嫁外人(출가외인): 시집간 딸은 친정 사람이 아니고 남이나 마찬가지라는 뜻
	- [[宅]](집 택)
	- [[宇]](집 우)
	- [[宙]](집 주)
	
### 寸
- [[寸]](손 촌, 마디 촌, 규칙 촌, 법칙 촌, 치 촌)  [41]
	- 오른 손 마디에서 맥박이 뛰는 모습을 보고 만든 글자
	- 씨족집단으로 구성된 농촌사회 (村 : 마을 촌)
	- 寸(치 촌)
	- 손끝에서 맥박(脈搏)이 뛰는 데까지의 사이를 나타내어 「한 치」의 길이
	- 한 치: 척의 십분의 일, 손목에서 맥박이 뛰는 곳.
	- 손을 폈을 때 엄지손가락 끝에서 가운데 있는 손가락까지의 3cm
	- 尺(자 척): 자, 길이, 발바닥의 길이 23~30cm, 한 치의 열배
		- 百尺竿頭(백척간두): 위태로움이 극도(極度)에 달함
		
	- [[村]](마을 촌) 
		- 木(나무 목) + 寸(마디 촌)
		- 내 몸에 맞는 마을에 정착
		- 농막(農幕: 농사짓는 데 편리하도록 논밭 근처에 간단하게 지은 집)
	
	- [[守]](지킬 수) 
		- 宀(집 면) + 寸(마디 촌)
		- 내 집에서 내 법도를 지켜야 지킬 수 있음
		- 지키다에서 다스리다
		- 守株待兔(수주대토): 
			- 그루터기를 지켜 토끼를 기다린다
			- 고지식하고 융통성(融通性)이 없어 구습(舊習)관 전례(前例)만 고집(固執)함
			
	- [[耐]](참을 내, 견딜 내) 
		- 而(말 이을 이) + 寸(마디 촌)
		- 턱수염과 손을 함께 그린 耐
		- 而(말 이을 이): 사람의 턱수염
		- 턱수염을 뽑아 고통을 주는 비교적 가벼운 벌을 아프지만 인내해야 한다, 즉 견디다
			- 忍耐(인내): 참고 견딤
			- 忍耐心(인내심): 참고 견디는 마음
			- 忍耐力(인내력): 참고 견디는 힘
			- 忍耐性(인내성): 참고 견디는 성질(性質)
			- 堪耐(감내): 참고 견딤
			- 耐震(내진): 지진(地震)을 견딤
### 小
- [[小]](작을 소, 젊을 소) [42]
	- 땅속에서 풀싹이 겨우 돋아나온 모양을 본뜸
	- 작고 여리다(크기)
	- [[小]](작을 소) ↔ [[大]](큰 대) 
	
	- [[少]](적을 소, 젊을 소)
		- 小(작을 소) + 丿(삐침 별)
		- 양이 적다, 에너지가 적은 것
		- 丿(삐침 별): 에너지
		- 少年(소년)
		- 少女(소녀)

### 尢
- [[尢]] , 𡯁, ⺎, 尣, 𡯂(절름발이 왕) [43]
	- 한쪽 정강이가 굽은 절름발이의 모양을 본떠 만듬
	-: 大(큰 대)의 변형으로 한 다리는 곧고, 한 다리는 굽은 모양.
	
	- [[尤]](더욱 우) 
		- 尢(절름발이 왕) + 丶(점 주)
		- 절름발이라 균형이 안맞기 때문에 한쪽에 힘을 주면 더 나감, 더욱 우
		
	- [[就]](나아갈 취, 이룰 취) 
		- 京(높을 경) + 尤(더욱 우)
		- 더 높은 곳을 향하여 더욱 나아가서 이루다
		- 成就(성취): 목적(目的)대로 일을 이룸

### 尸
- [[尸]](주검 시, 사람 시) [44]
	- 주검, 시체, 사람, 시동
	- 주검시로는 한번밖에 사용안됨 → 屍(주검 시)
	- 죽은 사람의 제를 관장하던 尸童(시동)이 의자에 앉아있는 모습을 그린 것으로 보기도 한다. 
	- 시동은 옛날에 제사를 지낼 때 임시로 신의 역할을 맡은 사람
	- 그러나 尸자가 부수로 쓰인 글자를 보면 대부분이 ‘죽음’과는 관계없이 
	- ‘사람’이나 또는 ‘앉아있는 사람’과 관계된 의미를 전달
	
	- [[屍]](주검 시)
		- [[尸]](주검 시) + [[死]](죽을 사)
		- 尸童(시동): 祭祀(제사) 지닐 때에 神位(신의) 代身(대신) 交椅(교의)에 앉히던 어린아이 
		- 屍體(시체)
	
	- [[居]](살 거) 
		- 尸(사람 시) + 古(옛 고)
		- 살다, 거주하다
		- 글자의 조합이 마치 사람이 의자에 앉아있는 듯한 모습에서
		- 후에 뜻이 확대되면서 한곳에 정착한다는 의미에서 거주하다 
		- 居住(거주)
		
	- [[住]](살 주)
		- 살다, 거주하다
		- 主(주인 주): 주인, 방안을 밝히던 촛대
	
	- [[屋]](집 옥) 
		- 집에 이르러 옥처럼 충전하다
		- 家屋(가옥): 사람이 들어가 살기 위하여 지은 집


### 屮
- [[屮]](싹날 철) [45]
	- 풀 싹을 보고 만들었음, 풀싹이 하나만 나옴
	- 싹날 철(屮)속에 왼손 좌가 들어가 있음
	- 艸(풀 초): 모든 풀이나 꽃 이름은 풀 초가 들어갑니다.
	
	- [[屯]](진칠 둔, 험할 준) 
		- 屮(싹날 철, 왼손 좌) + 丿(삐침 별)
		- 풀 무더기가 엉켜 있는 거
		- 파랗게 올라오는 초목
		- 屯山(둔산)동
		- 駐屯(주둔): 군대가 한 지역(地域)에 머무는 것
	
	- [[春]](봄 춘)
		- 艸(풀 초) + 屯(진칠 둔) + 日(날 일)
		- 햇빛을 받아 풀 무더기가 자라다

	- [[純]](순수할 순)
		- 풀 무더기가 자라서 실처럼 연결 되는 것
		- 파랗게 올라오는 초목, 풀은 가공하기 전의 순수한 모습
		- 糸(실 사) 누에고치에서 뽑은 희고 반짝이는 비단도 순수함을 연상케 
			- 至高至順(지고지순): 더할 수 없이 높고 순수(純粹)함
			- 純眞無垢(순진무구): 마음과 몸이 아주 깨끗하여 조금도 더러운 때가 없음
			- 純粹(순수): 다른 것이 조금도 섞이지 않음
			- 純潔(순결): 몸과 마음이 아주 깨끗함
			- 單純(단순): 복잡하지 않고 간단(簡單)함
			- 純眞(순진): 마음이 꾸밈이 없고 참됨
			- 淸純(청순): 맑고 순박(純朴)함
	- [[鈍]](무딜 둔, 둔할 둔)
		- 무디다, 섬세하지 못하다, 우둔하다
			- 愚鈍(우둔): 어리석도 둔함
			- 鈍濁(둔탁): 성질이 둔하고 혼탁함
### 山
- [[山]](메 산, 뫼 산) [46]
	- 메(산을 예스럽게 이르는 말), 뫼, 무덤
	- 부수로 쓰일 때 : 산의 이름, 산의 기세, 높다
	
	- 宣也(선야): 산은 우리에게 늘 베푼다
		- [[宣]](베풀 선)
			- 宀(집 면) + 亘(베풀 선)
			- 뻗치다, 베풀다, 넢리 펴다
		- [[亘]](베풀 선): 하늘과 땅 사이에 태양을 그린 것
		- [[也]](잇기 야, 어조사 야)
			- 어조사 : ~이다, ~구나, 또한, 역시
		
	- [[仙]](신선 선)
	- [[嶺]](고개 령(영)): 山(뫼 산) + 領(거느릴 령)
		- 고개, 산봉우리
		- 거느리다(領) + 山
		- 여러 산을 거느리고 있다
		
	- [[領]](거느릴 령): 令(하여금 령) + 頁(머리 혈)
		- 사람을 거느리며(頁) 명령을 내린다(令)
		- 거느리다, 통솔하다
		- 頁(머리 혈) - 얼굴이나 머리, 우두머리와 관계있음	
		- [[要領不得(요령부득)]]: 사물의 주요한 부분을 잡을 수 없다. 말이나 글의 요령(要領)을 잡을 수 없음
		- 要領(요령): 사물(事物)의 요긴(要緊)하고 으뜸 되는 줄거리.
			- 미립. 경험(經驗)을 통하여 얻은 묘한 이치(理致)나 요령(要領).
			- 적당(適當)히 꾀를 부려 하는 짓.

### 川
- [[川]](내 천) [47]
	- 巛: 개미허리(川 의 古字)
	
	- [[巠]](지하수 경, 물줄기 경): 하늘에서 물이 땅으로, 지하로 필요한 곳까지 흐르다
		
	- [[經]](벼리 경(별), 날줄 경, 글 경, 지날 경, 다스릴 경) 
		- 糸(실사) + 巠(물줄기 경)
		- 지나다
		- 날실(베틀 사이로 날실이 지나가는 모습):(날실이)지나다
		- 다스리다, 경영하다(비단 실을 엮어 베를 짜듯이 기초를 닦고 일을 해나간다는 의미에서)

	- [[州]](고을 주) 
		- 물이 잘 빠지는 곳에 마을을 형성하고 살았다
		- 큰 고을, 마을
		- 晉州市(진주시)
### 工
- [[工]](장인 공, 만들 공, 구기 공) [48]
	- 명사: 잘 만드는 사람
	- 동사: 만들다.
	- 구기=도구=자
	
	- [[功]](일할 공, 공 공) 
		- 力(힘 력(역)) + 工(장인 공)
		- 힘써 일하다.
		- 힘써 일해 공을 세우다
		- 공로(功勞), 업적, 사업
		- 땅을 다지는 도구를 들고 힘을 쓰다
		- 나랏일에 힘써 준다는 의미에서 공로, 업적, 사업

	- [[空]](빌 공, 하늘 공)
		- 穴(구멍 혈) + 工(장인 공)
		- 비다, 구멍, 헛되다, 공허하다
		- 도구(工)로 구멍(穴)을 만들었다 → 공간, 비다, 구멍
		- 空軍(공군)
		- 空手來空手去(공수래공수거): 빈손으로 왔다가 빈손으로 간다
		- 空理空論(공리공론): 사실에 맞지 않은 이론과 실제와 동떨어진 논의
		- 無主空山(무주공산): 임자가 없는 빈 산
		
	- [[虛]](빌 허)
		 - 虎(범 호) + 丘(언덕 구)
		- 호랑이를 잡으려고 땅을 파놓은 곳
		- 비다, 없다
		- 虛空(허공): 텅빈 공중

### 己
- [[己]](몸 기, 자기 기) [49]
	- 몸 기: 나 자신, 자기
	- 유연한 내 몸
	- 人(사람 인): 문장에서 타인으로 쓰인다
	
	- [[改]](고칠 개) 
		- 己(자기 기) + 攵(칠 복)
		- 고치다, 고쳐서 새롭게 하다, 바꾸다
		
	- [[紀]](벼리 기) 
		- 糹(가는 실 멱) + 己(몸 기)
		- 벼리(그물 코를 꿴 굵을 줄, 일이나 글의 뼈대가 되는 줄거리)
		- 벼리, 밑바탕, 단서
		- 여러 개의 실 가닥에서 하나를 뽑아낸다는 뜻을 표현하기 위해 만든 글자
		- 후에 어떤한 것의 시작점이라는 의미가 확대되어 실마리, 단서
		- 世紀(세기): 시대
		- 紀元前(기원전): 서력 기원(紀元) 이전 B.C
	
	- [[忌]](꺼릴 기) 
		- 心(마음 심) + 己(몸 기)
		- 꺼리다, 질투하다, 시기하다
		- 忌日(기일): 조상이 죽은 전날
	
	- [[記]](기록할 기)
		- 말(言)을 나(己)의 머릿속에 저장, 보관하다, 기억하다
