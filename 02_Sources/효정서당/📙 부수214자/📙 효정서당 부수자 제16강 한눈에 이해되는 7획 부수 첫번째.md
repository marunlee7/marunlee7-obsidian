## 7획
## 見

- [[見]](볼 견, 뵐 현) [147]
	- [[目]](눈 목) + [[儿]](어진사람 인)
	- [[目]](눈 목): 눈과 관련된 뜻
	- [[見]](볼 견): 보다, 보는 것, 보이는 것, 보는 행위에 주로 쓰임
	- [[見]](뵐 현): 나타나다, 보이다
		- 見牛未見羊(견우미견양)
			- 소는 보고 양은 보지 않았다,
			- 무엇이나 보지 않은 것보다는 직접 눈으로 보고 들은 것에 對하여 한층 더 생각하게 된다는 말
		- 見物生心(견물생심) - 물건(物件)을 보면 욕심(慾心)이 생긴다
		- 見利思義(견리사의) - 눈앞의 이익을 보면 의리(義理)를 먼저 생각함
		- 先入見(선입견) - 사람, 사물, 주의 따위에 對하여 실제의 경험보다 앞서, 미리 마음 속에 부정확(不正確)하게 알고 있는 것으로 해서 형성된 고정적인 관념이나 견해(見解)
		- 謁見(알현) - 지체 높은 사람을 찾아 뵙는 일
			- 謁(뵐 알)
		- 見學(견학)
	- [[現]](나타날 현) : 玉(구슬 옥) + 見(볼 견)
		- 사물의 모습이 드러나다
		- 눈 앞에 실제 있는 일을 뜻함
			- 發現(발현) - 숨겨져 있던 것이 바깥으로 드러나 보임
	- [[覺]](깨달을 각)
		- 見(볼 견) + 𦥯(배울 학)
		- 견학을 가면은 깨닫다
		- 覺悟(각오)
		- 覺醒(각성)
		- 𦥯(배울 학) = 𦥑(깍지 낄 국) + 爻(사귈 효) + 冖(덮을 멱)

## 角
- [[角]](뿔 각, 모서리 각, 닿을 각) [148]
	- [[⺈(]]칼도 도) + [[用]](쓸 용) 
	- 짐승의 뿔을 그린 것
	- 뿔, 모퉁이의 뜻에서 모나다, 각지다, 겨루다, 경쟁하다
		
	- 角度器(각도기)
	- 四角形(사각형)
	- 四角帽(사각모)
	- 矯角殺牛(교각살우) - 쇠뿔을 바로 잡으려다 소를 죽인다, 결점이나 흠을 고치려다 수단이 지나쳐 도리어 일을 그르침.

	- [[觸]](닿을 촉, 찌를 촉)
		- 角(뿔 각) + 蜀(벌레 촉, 자벌레 촉, 촉나라 촉, 나라이름 촉)
		- 머리에 난 뿔(角)로 애벌레(蜀)는 자기한테 닿는 촉감을 안다 [꼬불]
		- 찌르다, 받다, 무언가에 의해 자극을 받는다는 것
			- 觸角(촉각): 절지동물의 머리에 있는 감각기, 더듬이
			- 觸感(촉감): 무엇에 닿았을 때의 느낌
			- 接觸(접촉): 맞붙어서 닿음
			- 抵觸(저촉): 서로 충돌함, 서로 부딪침
			- 觸覺(촉각): 피부의 겉에 다른 物件(물건)이 닿을 때 느끼는 感覺(감각)
			- 感觸(감촉):
			- 觸發(촉발): 일을 당(當)하여 충동, 감정 따위를 유발함
			- 一觸卽發(일촉즉발)
				- 한 번 닿기만 하여도 곧 폭발한다
				- 조그만 刺戟(자극)에도 큰 일이 벌어질 것 같은 아슬아슬한 狀態(상태).

	- [[蜀]](벌레 촉, 자벌레 촉, 촉나라 촉, 나라이름 촉)
		- 벌레 눈(目) 강조
		- 그물에 쌓여 있는 벌레이니 애벌레다 [꼬불]

	- [[燭]](촛불 촉)
		- 火(불 화) + 蜀(자벌레 촉, 나라이름 촉)
		- 불은 불인데, 애벌레 꿈틀거리듯 넘실넘실 거리는 촛불

	- [[濁]](흐릴 탁)
		- 물 속에 애벌레가 있으니 가스(gas)가 생겨 물이 흐려진다
		- 濁水(탁수)
		- 濁流(탁류) = 흘러가는 흐린 물, 결백하지 못한 무리들
		- 混濁(혼탁) = 맑지 아니 함

	- [[屬]](붙을 속)
		- = 尾(꼬리 미) + 蜀(벌레 촉)
		- = 애벌레는 꼬리가 계속해서 연결되어 있다, 붙을 속
		- = 지붕(尸) 밑에 물을 피해 애벌레가 붙어산다 [꼬불]
			- 屬國(속국) = 정치적으로다른 나라에 매여 있는 나라
			- 屬性(속성) = 사물의 특징, 또는 그 성질

	- [[囑]](부탁할 촉)
		- 口 + 屬(붙을 속)
		- 委囑(위촉) =

	- [[獨]](홀로 독)
		- 개처럼 애벌레가 홀로 앉아 있다
		- 獨立(독립) = 남에게 의지하지 않고 제 힘으로 따로 섬

## 言
- [[言]](말씀 언, 다스릴 언) [149]
	- 亠(머리 두)
	- 머리(亠)로 두번 생각하고 말(口)해야 한다 [꼬불]
	- 머리(亠)로 두(二)번 생각하고 입(口)으로 말하는 말씀이니,  말씀 언
	- 머리로 2번 조화롭게 생각하고 입으로 하는 말
	- 갑골에서는
		- 辛(매울 신) + 口(입 구, 혀)
		- 말이 씨가 된다
		- 입은 말로써 복을 지을 수 있고, 병은 입을 통해 들어가서 병을 얻을 수 있다

	- [[語]](말씀 어)
		- 말(言) 로 나(吾)의 뜻을 알리는 말씀이니, 말씀 어 p256 
		- 言(말씀 언) + 吾(나 오)
		- 國語(국어)
		- [[吾]](나 오)
					
	- [[話]](말씀 화)
	- [[訓]](가르칠 훈)
	- [[罰]](벌줄 벌)
	- [[訛]](그릇될 와)
		- 化(변화할 화)가 와로 바뀜
		- 訛傳(와전) = 訛(그릇될 와) + 傳(전할 전)

## 谷
- [[谷]](계곡 곡) [150]
	- 골짜기의 뜻 - 계곡 사이로 물이 흐르는 모습

	- [[浴]](목욕할 욕): 계곡에서 목욕하다
	- [[俗]](풍속 속): 산을 넘으면 풍속이 달라짐
	- [[欲]](바랄 욕): 계곡 물을 마을 사람들과 같이 마시고 싶은 마음
		- 欲望(욕망)
	- [[慾]](욕심 욕): 바랄 욕이 컨트롤이 안되면 욕심이 됨

	- [[㕣]](숲속 늪 연, 산속 늪 연)
	- [[鉛]](납 연)
		- 鉛筆(연필)
	- [[沿]](물따라갈 연)
	- [[船]](배 선): 배를 타고 마을을 이동


## 豆
- [[豆]](콩 두, 제기 두, 호리병 두) [151]
	- [[荳]](콩 두)
	- [[登]](오를 등) : 재기를 준비해서 올라감
		- 登山(등산)
		- 登校(등교)

	- [[壹]](병 하나 일, 갖은자 한일)
		- 士(선비 사) + 冖(덮을 멱) + 豆(콩 두)
		- 호리병이란 뜻으로 쓰임
		- 壹百萬元

	- [[樹]](나무 수, 세울 수) = - 木(나무 목) + 尌(멈출 주)

	- [[逐]](쫓을 축)
		- 角逐(각축) : 겨루고 쫓음

	- [[豖]](돼지발걸음 얽을 축, 못걸을 축) = 豕(돼지 시) + 丶(점 주)
		- 돼지 잡을 때 다리를 묶어놓음

	- [[琢]](쫄 탁, 다듬을 탁)

	- 切磋琢磨(절차탁마)

		- 옥돌을 자르고 줄로 쓸고 끌로 쪼고 갈아 빛을 내다
		- 학문이나 인격(人格)을 갈고 닦음

	- [[咳]](기침 해)

		- 百日咳(백일해) : 어린아이의 기침

	- [[核]](씨 핵)
			- 나무의 핵
			- 核家族(핵가족)

	- [[亠]](머리 두, 돼지해머리 두)
		- 亥(돼지 해)에 많이 쓰인다하여 돼지해머리 두

	- [[嬰]](예쁠 영, 어린아이 영)

		- 돈 문제는 칼처럼 즉시 해결

## 赤
- [[赤]](붉을 적, 없을 적) [155]
	- 大 + 火 
		- 붉다, 멸하다의 뜻
		- 없다, 비다, 벌거벗다
		- 불이 크게 나서 붉게 보임, 갓난아기가 태어날 때의 색
	- [[紅]](붉을 홍)
	- [[朱]](붉을 주) : 붉은밤색, 소나무의 자연색
	- [[赫]](빛날 혁) :
		- 커다라 불을 뜻하는 赤(붉을 적)을 두 개 나란히 늘어놓아 환하게 불이 빛나는 뜻을 나타냄
		- 태양에 가까움
		- 오행의 丙火
	- *비슷한 한자
		- 亦(또한 역) = 조사
			- 머리 + 몸통 + 양쪽 똑같은 겨드랑이가 또 있다

- [[走]](달릴 주, 달아날 주) [156]

	- [[起]](일어날 기)
		- = 달릴려고 내 몸(己)을 일으켜 세우다
	- [[越]](뛰어넘을 월) = 走 + 戉(도끼 월)
		- 도끼가 있으면 개울도 뛰어넘을 수 있음
		- 超越(초월)
		- 超(뛰어넘을 초)
- [[足]](발 족, 만족할 족) [157]
	- 부수로 쓸 땐 획이 하나 줄음(𧾷, 6획)

## 身
- [[身]](몸 신) [158]
	- 아이를 가진 모습

	- 父牲我身(부생아신) 아버지는 내 몸을 낳으시고
	- 母鞠吾身(모국오신) 어머니는 내 몸을 기르셨네
	- [[我]] - 실력을 갖춘 나
	- [[吾]] - 말을 잘하는 나

	- [[殷]](은나라 은, 음악 은) 
		- 㐆(돌아갈 은) + 殳(몽둥이 수, 칠 복)
		- 身(몸 신)의 반대는 㐆(돌아갈 은)
		- 身 뒤집은 거 + 殳(몽둥이 수, 칠 복)
		- 몸을 뒤집으며 몽둥이로 악기연주하는 모습

	- [[殳]](몽둥이 수, 칠 복) = 几(안석 궤) + 又(또 우)
		- 몽둥이, 창
		- 치다, 때리다
