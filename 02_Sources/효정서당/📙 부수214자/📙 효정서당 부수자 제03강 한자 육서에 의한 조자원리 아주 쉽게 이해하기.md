## 조자원리
- 조자원리(造字源理) : 육서에 의해 한자 구성 원리를 6가지로 분류한 것
	- [[造]](지을 조)
	- [[字]](글자 자)
	- [[源]](근원 원)
	- [[理]](다스릴 리)
  
## 상형자, 지사자, 회의자, 형성자, 전주차, 가차자
- [[상형자(象形字)]] (4%) : 사물의 모양을 본뜸
- [[지사자(指事字)]] (1%) : 추상적인 사물 또는 관념을 나타냄(점, 선, 기호, 숫자)
- [[회의자(會意字)]] (13%) 
	- 會(모을 회) : 떡시루이 김이 모락모락 나면 사람이 모임
	- 意(뜻 의) : 사람이 서서(설 립) 마음속의 소리를 냄
	- 상형자 + 지사자 = 새로운 의미
	- 두개의 부수가 만나서 뜻만 취하고 새로운 음이 탄생되는 것
	- 明(밝을 명): 회의자	
- [[40. PARA/43. Rereferencess/Hword2/전광진 하루한자/형성자(形聲字)]] (80% 이상) 
	- = 한 부분은 뜻, 한 부분은 소리를 나타냄
	- = 회의자에서 음(音)은 그대로 가고, 뜻만 바뀌는 것(訓)을 형성자		
	- 萌(싹 맹) : 형성자
	- 沐(목욕할 목, 머리 감을 목)
	- 吐(토할 토)
	- 形(모양 형)
	- 聲(소리 성)		
- [[전주자(轉注字)]] (1% 미만) 
	- 글자 본래의 의미에서 파생, 확장 되는 것
	- 음이 계속해서 바뀌는 것을 전주자 라고한다
		- [[專]](오로지 전)
		- [[傳]](전할 전)
		- [[轉]](구를 전, 옮길 전 )
		- [[注]](물댈 주)			
	- [[樂]](악기 악, 풍류 악, 즐거울 락, 좋아할 요)
		- 전통악기가 나무 지지대 위에 올라간 모습
		- 나무에 받쳐놓은 악기 모양			
- [[가차자(假借字)]]
	- 비슷한 소리를 가진 글자를 빌려쓰는 것
	- 실제 뜻과는 상관없이 소리만 빌어씀
		- [[假]](거짓 가)
		- [[借]](빌릴 차)
		- [[叚]](빌릴 가)
	- 美國 - 
	- 英國 - 
	- 印度(인도)

## 1획부수
- [[乙]](새 을) [05]
	- [[乚]], [[乛]], [[⺄]](굽을 을, 새 을) 
	- 새 을 이란 의미로는 거의 안 씀 
	- 한자는 동그라미가 없다
	- 태양이나 어머지 젖도 동그라미이다.
		
	- [[乳]](젖 유) 
		- [[爪]](손톱 조) + 子 + [[乚]](굽을 을 : 동그라미 처럼)
		- [[孚]]자가 아이를 손톱으로 잡은 모습
		- [[乙]]자는 어미의 가슴을 표현한 것으로 풀이된다.	
		  	
	- [[直]](곧을 직)
		 
- [[亅]](갈고리 궐) [06]
	- 사람이 만들어서 에너지를 끌어오는 것	
	  
	- [[于]](어조사 우)
		- 二(두 이) + 亅(갈고리 궐)
		- 二(두 이) 하늘과 땅 사이의 에너지를 끌어오는 것(丨)
		-  ~에, ~에서, ~부터, ~까지, ~에게,  ~을, ~를	
		
	- [[事]](일 사, 섬길 사)
		- 亅(갈고리 궐) + 又(오른손 우)
		- 오른손으로 깃대를 들고 앞장 서는 모습
		- 사친(事親) : 어버이를 섬김
		- 만사(萬事) : 만가지 일

## 2획부수

- [[亠]](머리 두, 높을 두) [08]
	- **땅(一) + 높다(丨)**
		
### 亠(머리 두)의 경우
- [[交]](사귈 교) 
	- 머리가 비슷한 사람끼리 **사귀다.**
	- 사귀어 친구가 되면 **서로의 것을 배워가게 된다**.
	- [[乂]](다스릴 예, 벨 예)
		- 베어지지만, 나무가  더 잘 자랄 수 있게 가지치기를 해주면 나무를 다스릴 수 있게 됩니다.
	- [[爻]](본받을 효) 
		- 반복되어 본받게 된다고 하여 본받을 효라는 뜻이 있음
- [[3-리소스/Hword1/5급2/效]](본받을 효)
	- 사귐이 좋지 않을 때는 쳐서(攵) 좋은 것을 본받게 한다
- [[30-리소스/Dict/Hword1/8급/校]](학교 교)
- [[較]](비교할 교)
	- 예나 지금이나 차를 비교한다.
	
### 亠(높을 두)의 경우
- [[高]](높을 고)
	- 2층집, 높다, 크다, 고상하다, 뛰어나다
	- 고산지대(高山地帶)
	- 고상(高尙) : 몸가짐과 품은 뜻이 깨끗하고 높아 世俗된 비천한 것에 굽히지 아니함
	
- [[稿]](볏짚 고, 원고 고)
	- [[禾]](벼 화): 고개숙인 모습
	- [[원고(原稿)]] : 인쇄물의 본보기를 삼기 위하여 쓴 글이나 그림, 사진
	- [[기고(寄稿)]] : 신문, 잡지 따위에 싣기 위하여 原稿(원고)를 보냄
	- [[유고(遺稿)]] : 죽은 사람이 남긴 원고(原稿)
	- [[초고(草稿)]] : 시문의 초벌로 쓴 원고
	
- [[稾]](볏짚 고)
	- 아래에 禾(벼 화)를 쓰기도 함
	- 고공품(藁工品) : 짚이나 풀줄기로 만든 가마니, 새끼, 멍석과 같은 수공품
- [[膏]](기름 고)	
- [[京]](높을 경, 서울 경) 
	- 높은 성문이 있다
- [[景]](볕 경, 경치 경)
	- 日(해 일) + 京(높을 경)
	- 해가 뜨면 경치가 좋다하여 → 경치 경
- [[鯨]](고래 경)  


## 人
- [[人]](사람 인) [09]
	-  亻(사람인 변)
	- 사람의 옆 모습
	
- [[仙]](신선 선) 
	- 사람(人) + 베푸는(山)
	- 山(뫼 산) : [[선야(宣也)]] 라고 해서 항상 베푼다는 의미를 지닌다.
	- 그래서 仙의 신선은 놀고 먹고 베푸(山)는 사람(人)이다.
		  
- [[休]](쉴 휴)

## 匕
- [[匕]](비수 비, 사람 비, 변할 비, 거꾸러질 비, 숟가락 비) [21]
	- 수저, 숟가락을 본뜬 것으로 해석하기도 함
	
- [[化]](될 화, 변화할 ) 
	- 사람이 변하여 되다
- [[花]](꽃 화)
- [[貨]](재물 화)
- [[靴]](가죽 신 화)
	- 革(가죽 혁) + 化
