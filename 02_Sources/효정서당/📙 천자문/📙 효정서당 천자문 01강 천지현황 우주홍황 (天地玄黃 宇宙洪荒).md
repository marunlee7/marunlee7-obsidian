## 天地玄黃
- [[天]]
	- [[昊]](여름하늘 호, 클 호)
		- 태양이 있는 하늘, 태양의 온기가 가장 크다
		- 호천망극

- [[地]](땅 지) : 
	- 土가 부수
	- [[也]](**어조사 야**, **잇기 야**, 뱀 야) : 뱀이 또라이를 튼 모습
	- [[巳]](뱀 사, 자식 사) : 생명을 대표
	- [[語助辭]] : 
		- 문장을 도와주는 말 150여개
		- 종결사로 쓰임. 
		- 문장연결의 접속사로 쓰여 **잇기 야** : 그러나, 또, 그리고
		- 문장 앞에서는 **너 야**
		- [[他]](다를 타)
			- [[타인(他人)]]
			- [[타국(他國)]]
	- [[地]](땅 지)
	- [[池]](연못 지) : 물에서도 생명이 살 수 있다

- [[玄]](검을 현) : 
	- 부수, 검은 실타래 보고 만든 글자
	- 아득할 현, **가물 현**, 검을 현
	- **아득하다, 고요하다, 오묘하다, 깊다, 검다, 크다**

	- [[絃]](악기줄 현) : 실타래 중 검은실로 악기를 만들다.
	- [[弦]](활시위 현) : 
		- 상현(上弦) : 
		- 하현(下弦) : 

- [[黃]](누를 황) : 
	- 땅의 색, 지지색야
	- 누를 황, 늙을 황, 저승 황, 넓을 황
	- 땅의 색은 누렇다. 누를 황
	- 늙으면 황천길 간다해서 늙을 황, 저승 황
	- 땅은 넓고 넓어 황금빛 벌판 넓을 황
	- [[十]] : 십
	- [[廿]] : 밑 스물 입
	- 하늘에서 꽉 차(廿) 땅(一)으로 떨어져 분류(八)되다 - [[黃]](누를 황)

## 宇宙洪荒
- [[于]](어조사 우, 말할 우)
	- 부수는 두 이(二)
	- ~에, ~에서(를, 을)
	- = [[於]](어조사 어)

- [[宇]](집 우, 공간 우) : 하늘을 큰 지붕으로 보면 끝없는 공간으로 보아 **공간 우**

- [[宙]](집 주, 끝 없는 시간 주) : 
	- 우주 : 공간과 시간을 일컬어 우주라 함

- [[洪]](넓을 홍) : 
	- 氵(물 수) + [[共]](함께 공) → 共을 그림으로 차용함
	- 물이 사통팔달 터질 수 있는 길로 보아 넓을 홍
	- [[巷]](거리 항) : 

- [[荒]](거칠 황) : 
	- [[亡]](달아날 망, 없을 망, **망할 망**) : 사람 머리(亠)가 숨어 [[乚]](숨을 은)
	- [[流]](흐를 유)
	- [[蕪]](거칠 무)
