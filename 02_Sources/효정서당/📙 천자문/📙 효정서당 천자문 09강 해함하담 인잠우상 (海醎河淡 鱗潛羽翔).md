# 천자문 09강 해함하담 인잠우상 (海醎河淡 鱗潛羽翔)

![](https://www.youtube.com/watch?v=PGU7bER35lY)  

## 4자 풀이
- [[해함하담(海鹹河淡)]] : 바닷물은 짜며 강물은 담백하고
- [[인잠우상(鱗潛羽翔)]] : 비늘 있는 고기는 물속에 잠기고, 날개 있는 새는 공중(空中)에 낢.

## 海醎河淡

- [[海]](바다 해)
	- [[母]](어미 모)
		- [[毋]](말 무)
	- [[每]](매양 매, 늘 매)
		- 어머니는 늘 정갈한 모습이다. 
	- [[梅]](매화나무 매)
		- 매화나무 열매는 어머지 젖꼭지처럼 탐스럽다. 
	- [[苺]](딸기 매)
	- [[海]](바다 해)
		- 하해와 같은 마음이 어머니의 마음
		- 바다 → 바다드리다 에서 왔다는 유래. 정화한다.
		- [[파라밀다(波羅蜜多)]] : 바라밀다
			- 피안에 도달하다, 깨달음의 언덕으로 건너간다는 뜻.

- [[鹹]](짤 함)
	- 북방지미(北方之味) : 북쪽의 맛이 짠맛이다.
		- 동-木 간,담 신맛 [[酸]](실 산)
		- 서-金 폐, 대장 매운맛 辛(매울 신)
		- 남-火 심장, 소장 쓴맛 苦(쓸 고)
		- 북-水 신장, 방관 짠맛 鹹(짤 함)
		- 중-土 단맛 甘(달 감)
	- [[醎]] (짤 함)은 속자이다
	- [[鹵]](소금 로(노))
		- [[소금(素金)]]
	- [[咸]](다 함, 성씨 함)
		- [[感]](느낄 감)
		- [[減]](뺄 감, 덜 감)
			- 빚도 재산이다. 
			- [[가감(加減)]]
		- [[喊]](소리칠 함)
			- [[고함(高喊)]] : 크게 외치는 소리
- [[河]](물 하)
	- [[황하(黃河)]] : 누른 황토물이 흐르는 강. 곤륜산에서 발원 [[돈황(敦煌)]]하다
		- 물하 → 황하 → 양자강 → 바다 → [[발해(渤海)]]
	- [[可]](옳을 가, 가희 가)

- [[淡]](맑을 담, 싱거울 담)
	- [[薄]](엷을 박) : 담박하다 
	- ↔ [[濃]](짙을 농)
	- [[담수어(淡水魚)]]
	- [[炎]](불꽃 염, 염증 염)

## 鱗潛羽翔

- [[鱗]](비늘 린)
	- 魚(물고기 어) + [[粦]](도깨비불 린(인))
	- 설문해자 : 魚甲也
	- [[역린(逆鱗)]] : 용의 가슴에 거꾸로 난 비늘
	- [[燐]](도깨비 불 린)
		- 米(쌀 미)가 아니라 火(불 화)이다. 
		- 도깨비가 불꽃을 들고 여기저기서 번쩍번쩍 거리다.
	- [[隣]](이웃 린, 가까울 린)
	- [[潾]](맑을 린)
		- 소주 상표
	- [[憐]](불쌍히 여길 련)

- [[潛]](잠길 잠)
	- [[兓]](날카로운 참) : 입에서 나오는 강한 입김이 계속 나가다
	- [[朁]](일찍이 참, **입김낼 참**)
	- [[潛]](잠길 잠)
		- = [[藏]](감출 장)과 같은 뜻이다. 
		- 풀초는 덮다라는 뜻이 있다.
		- [[잠수(潛水)]]
	- [[蠶]](누에 잠)
		- [[蚕]](누에 잠) : 하늘 벌레. 간체자
	- [[簪]](비녀 잠)
		- 누에고치처럼 디자인된 대나무로 만든 비녀
	- [[僭]](외람될 참, 주제넘을 참)
	- [[譖]](참소할 참)

- [[羽]]
	- 새 깃털 보고 만든 글자
	- 설문해자 : 鳥長毛也
- [[翔]](날 상, 날아오를 상)
	- **飛也**
	- [[羊]]과 같이 순한 짐승은 무리지어 다닌다. 무리짓다, 순하다
	- 새들 또한 무리지어 다닌다의 뜻
	- 설문해자 : **回飛也**

	- 羊(양) → 翔(상) : 음이 바뀐 이유는?
	- [[祥]](상서로울 상)
		- 신에게 양을 제사 희생물로 올리는게 상서로운 일이 일어난다
	- [[詳]](자세할 상)
		- 상서로운 일이 일어나게 신께 자세히 말하다.
	- [[庠]](학교 상)
		- 큰집인데 순한 양떼같은 어린아이부터 어른까지

## 노자 天地合一
- [[法]] : 법 법, **본받을 법**
- 人法地 사람은 땅을 본받고, → 후덕해야 한다. 
- 地法天 땅은 하늘을 본받고, → 밤에는 휴식을 취하고, 동물등 세상에 내어주다
- 天法道 하늘은 도를 본받고, → 성인들이 글을 지어 道를 알게하다
- 道法自然 도는 스스로 그러함([[자연(自然)|自然]])를 본받는다. →
