# 천자문 11강 시제문자 내복의상 (始制文字 乃服衣裳)

![](https://www.youtube.com/watch?v=4mhAfSeHONk)  


## 4자 풀이
[[시제문자(始制文字)]] : 비로소 문자(文字)를 창제하였고
[[내복의상(乃服衣裳)]] : 이에 웃옷과 치마(아래 옷)를 입게 하셨다.

文은 설명할 수 있어야 하고 
<u>字는 분해할 수 있어야 한다.</u> 
## 始制文字

- [[始]](처음 시, 비로소 시)
	- 부인이 아가씨를 처음(台) 가지다
	- 女(계집 녀)
	- 계집(系集) : 
	- 설문해자: 初也
		- [[初]](처음 초)
			- 옷을 만들기 위해 처음 칼을 대다
		- [[創]](비로살 창, 처음 창)
			- 비로소 창고부터 짓게 되다.
	- [[台]](별 태, 기르 이)
		- [[厶]](사사 사: 코) + [[口]](입 구: 입)
	- [[跆]](밟을 태)
		- [[태권도(跆拳道)]]
	- [[笞]](볼기 칠 태)
		- [[태형(笞刑)]]
	- [[怡]](기쁠 이)
		- 엄마가 아빠의 정자를 받게되면 이보다 기쁠 수가 없다. 

- [[制]](절제할 제, 마를 제, 지을 제)
	- 설문해자 : 裁也
	- 未(아닐 미, 무성할 미 : 무성한 나무)와 옷감(巾)을 칼(刂)로 자르다. → 마를 제, 지을 제
	- [[제약(制約)]] : 어떤 조건(條件)을 붙여 제한(制限)함.

	- [[裁]](마를 재) : 80%는 옷 준비가 되다. 

	- [[製]](지을 제) : 완성된 옷. 완제품
		- [[제약(製藥)]]
		- [[제품(製品)]]
		- [[제조(製造)]]

- [[文]](글월 문, **무늬 문**, 꾸밀 문, 엇갈릴 문)
	- 얼굴에 엇갈리게 무늬를 새기다.
	- 설문해자 : 錯畵也
	- ### 契越文 → 문을 활용하여 지혜로 건너가다
		- [[此]](이 차)
		- [[彼]](저 피)
		- 불교에서 [[반야심경(般若心經)]]

	- [[丰]](예쁠 봉, 무성할 봉)
	- [[㓞]](새길 갈)
		- 끌로 새기다
	- [[絜]](헤아릴 혈)
		- 무성한 삼을 벤후 가는실, 굵은실을 잘 헤아리다. 
		- 반구 혈구지도
	- [[契]](계약 계, 약속 계, 맺을 계)
		- **끌**의 음이 있다. = 곧 **글**이라 하여 후에 의식혁명이 발전함
	- [[越]](뛰어넘을 월)
		- 도끼 월 + 달릴 주
		- 혼자 도끼 준비하여 달리면 뛰어넘을 수 있다.
		- [[월남(越南)]]
		- [[월반지사(越畔之思)]]
		- 월반

- [[字]](글자 자, 씨 자, 아가 자, 불어날 자) 
	- 宀 + 子
	- 설문해자 : 乳也 → 지혜가 불어나다
	- 글자는 필요하면 불어난다


## 乃服衣裳

- [[乃]](이에 내)
	- 자기 입에서 나오는 입김, 말을 이어 연결해주다.
	- 설문해자: 曵詞之難也
		- [[曳]](끌 예)
- [[服]](옷 복)

- [[衣]](옷 의)

- [[裳]](치마 상)
