# 천자문 08강 과진이내 채중개강 (果珍李柰 菜重芥薑)

![](https://www.youtube.com/watch?v=UPZSUgODSvU)  


## 4자 풀이
[[과진이내(果珍李柰)]] : 과일은 오얏과 벚(능금)을 보배로 여기고
[[채중개강(菜重芥薑)]] : 채소는 겨자와 생강을 중히 여기도다.

## 果珍李柰
- [[果]](실과 과, 열매 과, **결단할 과**)
	- 나무에 열매 열리는 모습
	- 필순은 위에서부터 한번에 내린다.
	- 설문해자 : 實也
	- 동자: [[敢]](감히 감, 용감할 감)

	- [[裹]](쌀 과, 보자기 과)
		- [[과량(裹糧)]] : 먼길을 다닐 때에 양식(糧食)을 싸가지고 가거나 옴. 
		- [[과량이송(裹糧以送)]] : 
	- [[課]](공부할 과)
		- 공부를 과정별로 나다보면 결과가 나오다.
		- [[과정(課程)]]

- [[珍]](보배 진)
	- [[珠]](구슬 주) : 붉은색을 뛴 구슬
	- [[璧]](구슬 벽) : 
	- [[珍]](구슬 진, **보배 진**) : 아롱진 무늬가 있는 검은 진주 

- [[李]](오얏 리, 성씨 리)
	- 자도(紫桃) = 자두
	- 신맛이 나서 미용, 장수에 도움
	- 씨가 많아서 열매가 많이 열림. 이씨는 자두처럼 후손이 많다해서 성씨 리

- [[柰]](능금나무 내, 사과 내, 어찌 나)
	- 제삿상에 올라가는 붉은 능금(사과)
	- [[사과(沙果)]], [[사과(砂果)]]
	- 어찌 나 : 어찌 이렇게 좋은 사과를 구했는가
	- [[奈]](어찌 내, 어찌 나)
	- [[那]](어찌 나)

- 싯달타(석가)는 
	- 태어난 곳은 무화과(무수)
	- 깨달음은 보리수 나무에서
	- 열반은 사라쌍수
## 菜重芥薑

- [[菜]](나물 채)
	- [[蔬]](나물 소)
		- 아가가 엄마 뱃속에 있을 때 맨발로 있다. [[疋]](발 소, 맨발 소)
		- 뱃속에서 양수([[流]])타고 나오다
		- 식물도 수를 저장하고 태양의 수화작용으로 땅에서 움트 싹이 나오는 모습
		- [[채소(菜蔬)]]
		- [[疏]](소통할 소, 드물 소)
			- 아가가 처음에는 드물게 발차기 한다 하여 → 드물 소
	- [[采]](풍채 채)
	- [[採]](캘 채)
	- [[彩]](채색 채, 빛날 채, 문채 채) : 식물에 햇빛이 잘 비추면 빛나게 되는 모습

- [[重]](거듭 중, 무거울 중, 중요할 중)
	- 사**람이 등에 무거운 짐을 거듭 실었다.**  → 무거울 중, 중요할 중, 거듭 중
	- 쓰기암기는 **車에 아래, 위에 짐을 실다.**
	- 설문해자 : 厚也
		- [[厚]](두터울 후)
	- = [[再]](거듭 재, 두번 재)
	- [[임중도원(任重道遠)]])

	- 鍾(쇠북 종, 술잔 중)

- [[芥]](겨자 개)
	- 설문해자 : 辛也
	- 찬음식에 열을 올리는 겨자를 끼어 먹는다.

- [[薑]](생강 강)
	- [[畺]](지경 강) 
	- [[疆]](지경 강)
	- 땅의 따뜻한 기운을 모아서 가지고 있다. → 잡내, 잡념 제거 효능
	- 우리 몸의 9교를 열어준다
	- 五辛菜 : 자극이 강하고 냄새가 많은 것이 특징이다.
		- 마늘 [[대산(大蒜)]]
		- 파 [[혁총(革蔥)]]
		- 부추 [[난총(蘭蔥)]]
		- 달래 [[자총(慈蔥)]]
		- 아위 [[흥거(興蕖)]]
