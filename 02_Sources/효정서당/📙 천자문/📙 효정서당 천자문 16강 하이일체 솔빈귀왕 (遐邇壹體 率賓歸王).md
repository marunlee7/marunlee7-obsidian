# 천자문 16강 하이일체 솔빈귀왕 (遐邇壹體 率賓歸王)

![](https://www.youtube.com/watch?v=-h32KPIz7VA)  


## 4자 풀이

- [[하이일체(遐邇壹體)]] : 멀고 가까운 곳이 일체가 되어
- [[솔빈귀왕(率賓歸王)]] : 거느리고 와서 복종하여 임금에게 귀의한다. 

## 遐邇壹體
- [[遐]](멀 하)
	- 설문해자 : 遠也
	- [[승하(昇遐)]]

- [[邇]](가까울 이)
	- 설문해자 : 近也

	- [[爾]](너 이, 끝날 이, 마칠 이, 뿐 이)

- [[壹]](한 일, 갖은자 한일)
	- 專一 한결같다

- [[體]](몸 체)
	- 骨(뼈 골) + 豊(제기 예, 풍년 풍)
	- 내 몸은 뼈로 구성되어 있지만 안에는 풍성하게 쌓여있다.
	- 總一二屬也
		- 머리: 정수리, 안면, 턱
		- 몸: 어깨, 등, 엉덩이
		- 손, 팔, 팔뚝
		- 발, 넓적다리, 정강이

## 率賓歸王
- [[率]](거느릴 솔, 비율 율)
	- 捕鳥畢也포조필야 사냥터 밭에 놓은 그물이다. 
	- 밭 작물을 해치는 동물을 위해 쳐놓은 검고 굵은 실의 망 

- [[賓]](귀한손님 빈, 맞이할 빈, **복종할 빈**)
	- 설문해자 : 所敬也
	- [[丏]](뵈지않을 면, 가늘 면) 
	- [[국빈(國賓)]]

	- 嬪(궁녀 빈)
	- 殯(빈소 빈)

- [[歸]](돌아갈 귀)
	- 설문해자: 女嫁也 여자가 시집을 가다
	- [[𠂤]](쌓을 퇴) = [[堆]](언덕 퇴: 쌍이다)자가 생략된 것이다.
	- [[귀가(歸家)]]
	- [[귀국(歸國)]]

- [[王]](임금 왕)
	- 설문해자 : 天下所歸也
	- [[主]](임금 주, 주인 주)
