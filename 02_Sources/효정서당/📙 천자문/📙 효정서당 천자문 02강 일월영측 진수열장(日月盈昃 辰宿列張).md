# 천자문 02강 일월영측 진수열장(日月盈昃 辰宿列張)

![](https://www.youtube.com/watch?v=GKvSJ3Q9BHY)  


## 4자 풀이

[[일월영측(日月盈昃)]] : 해는 서산에 기울며 달은 이지러졌다가 차고
[[진수열장(辰宿列張)]] : 12진과 28수가 벌여서 분장되었도다.

## 日月盈昃

- [[日]](해 일) : 
	- 태양에 흑점이 있다고 생각
	- 삼족오
	- 소전에는 實也 : 알곡을 꽉 채우는 역할

- [[月]](달 월) 
	- 초승달 - 보름달 - 그믐달 - 6일간의 암흑 - 초승달 

- [[盈]](찰 영)
	- 皿(그릇 명) + [[夃]](이문 얻을 고, 이문 얻을 고, 쌓일 영)
	- 발품을 팔아 남은 이문으로 인해 계속 쌓이다.
	- [[盈]]의 반대말은 ↔ [[虧]](이지러질 휴)
	- [[携]](가질 휴, 맬 휴, 이끌 휴)
		- 휴대(携帶) : 

- [[昃]](해기울 측) : 
	- 부수는 日
	- 해가 사람 밑으로 떨어지는 모습이다
	- [[仄]](기울 측) → [[측성(仄聲)]]
	- **상성 ↗ 입성↘ 거성 υ** 
	- [[평성(平聲)]] : 
	- 천자문의 평측 - 시어로서 평측을 맞추다, 글자가 뒤바뀌다.

- [[辰]](별 진, 조개 진, 3월 진, 때 신, 날 신, 용 진, 다섯째 지지 진)
	- 제부수 [161]
	- 바닷가 바위밑 조개가 입벌리고 나오는 모습
	- 이때가 자연으로 보면 3월 진
	- 생할 수 있는 시간 진시 진(7~9시)
	- 나올 수 있는 날 신(=生) [[생신(生辰)]]
	- 띠로는 용 진
	- 봄하늘에는 3월 전갈좌가 나오는 별 진 → 12辰

## 辰宿列張

- [[宿]](잘 숙, 별자리 수)
	- 집에서 많은 사람들이 잠을 자다
	- 28宿(수)(28개의 별자리) : 7개씩 28개의 별자리
		- 동방 창룡칠수: 각(角) 항(亢) 저(氐) 방(房) 심(心) 미(尾) 기(箕)
		- 북방 현무칠수: 두(斗) 우(牛) 여(女) 허(虛) 위(危) 실(室) 벽(壁)
		- 서방 백호칠수: 규(奎) 누(婁) 위(胃) 묘(昴) 필(畢) 자(觜) 삼(參)
		- 남방 주조칠수: 정(井) 귀(鬼) 유(柳) 성(星) 장(張) 익(翼) 진(軫)

	- [[佰]](우두머리 백)
	- [[伯]](맏 백)
		- 여인숙
		- 숙박료
	- [[縮]](오그라들 축, 줄일 축)
		- 실크옷(糸)이 밤이슬을 맞아 줄어들다.
		- [[축소(縮小)]] : 줄여서 작아짐. 또는 작게 함. 

- [[列]](벌릴 렬) 
	- [[列]] → 뼈를 발라 칼로 벌리다.
	- [[歺]] [[歹]] (뼈 앙상할 알 : 해골 보고 만듬 ) 
	- [[𣦼]](후빌 잔) : 손으로 뼈에 있는 살을 후벼내다.
	- [[餐]](밥 차, 반찬 찬) : 고기살에 뼈가 있듯 밥식이 붙어 밥 찬, 반찬 찬
	- [[粲]](정미 찬) : 쌀을 후벼 [[정미(精米)]]하다
	- [[燦]](밝을 찬, 빛날 찬) 
		- 쌀을 정미하듯 쌀알까지도 분석해 드러나게 되는 **밝을 찬, 빛날 찬**
	- [[裂]](찢어질 렬) 
		- 옷을 계속 벌리게 되면 찢어지다.

- [[張]](베풀 장, 어른 장)
	- 활을 벌리듯 멀리까지 베풀다 
	- [[출장(出張)]] : 
	- [[弓]](활 궁)
		- 활에는 항상 화살이 있어야 한다. 
		- [[궁시(弓矢)]] : 활과 화살
		- 제주 관덕중 - 인의예지를 갖춘 사람이 화살을 잡을 수 있다.
