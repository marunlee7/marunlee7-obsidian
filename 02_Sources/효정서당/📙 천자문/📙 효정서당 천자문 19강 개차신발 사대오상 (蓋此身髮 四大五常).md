# 천자문 19강 개차신발 사대오상 (蓋此身髮 四大五常)

![](https://www.youtube.com/watch?v=xmZtqCnUg08)  


## 4자 풀이
- [[개차신발(蓋此身髮)]] : 대개 이 몸과 터럭은
- [[사대오상(四大五常)]] : 네 가지 큰 것과 다섯 가지 떳떳함이 있다.
	- 하늘, 땅, 스승, 부모
	- 오상(五常) : 인의예지신(仁義禮智信)
## 개별한자
- [[蓋]](덮을 개, **대개/대강 개**)
	- 풀로 그릇을 덮고 가다
	- 艹(풀 초)자와 [[盍]](덮을 합)자가 결합
	- 대개 개, 대강 개 → 부사 역할
	- [[去]](갈 거, 없앨 거)
		- 가다, 없대다, 버리다, 내몰다, 덜다, 피하다 죽이다
	- [[苫]](이엉 점,덮을 점)
	- [[覆]](덮을 복)
		- [[복개(覆蓋)]] : 

- [[此]](이 차)
	- 此의 부수는 그칠 지(止)
	- 허신은 止也 : 이곳에 와서 그친다 로 표현함
	- 此(이 차) : 사람이 거꾸로와 이곳에서 그치다 
	- 彼(저 피) : 
	- [[피차(彼此)]] : 이것과 저것
	- 匕(사람 비, 비수 비, 변화할 비)
	- [[紫]](보라색 자, 자줏빛 자)
	- [[雌]](암컷 자)
		- 이곳에 힘이 센곳에 와서 머물다
	
- [[身]](몸 신)
	- 제몸부수
	- 임신한 사람의 모습
	- 허신 : 
		- 미미(微微) → 창대(昌大)
		- [[姙]](아이밸 임)
		- [[娠]](아이벨 신)
		- [[孕]](아이밸 잉)
			- [[잉태(孕胎)]] : 
		- [[躬]](몽 궁) : 

- [[髮]](터럭 발)
	- [[髟]](머리늘어질 표, 터럭 발)
	- [[彡]](무늬 삼) = 文(무늬 문)
	- [[犮]](개 달아날 발, 달릴 발)
	- 개가 달려나갈 때 털이 날리듯 생동감있게 표현함
	- 허신 : 根也 라 함
	- [[毛]](터럭 모, 풀 모)

- [[四]]
	- 부수는 에울 위
	- 허신 : 음수야 陰數也
	- 일 양수
	- 이 음수
	- 삼 양수
	- 사 음수
	- 오 양수
	- ==== 生數
	- ==== 成數
	- 지수화풍
	- [[사대(四大)]]
		- 하늘 天
		- 땅 地
		- 임금 君
		- 부모님 親
		- 四大恩中

- [[大]] 
- [[五]]
	- 부수는 [[二]](두 이)
	- 하늘에서 삐침, 땅에서 파임불
	- 허신 : 五行也 
	- 乂(다스릴 예, 벨 예)

- [[常]](떳떳할 상)
	- 부수는 [[巾]](헝겊 건, 수건 건: 천을 의미)
	- [[尙]](높을 상, 오히려 상)
	- 고상(高尙)
	- [[裳]](치마 상)
	- 군(치마 군)
	- [[오상(五常)]] : 인의예지신(仁義禮智信)
