## 4자 풀이
- [[금생여수(金生麗水)]] : 금은 여수에서 나오고
- [[옥출곤강(玉出崑岡)]] : 혹은 곤강에서 나도다.

## 金生麗水
- [[金]](쇠 금)
	- 깊은 곳에서 있으면서 반짝거리다
	- 철-흑색
	- 구리-청색
	- 동-붉은색
	- 은-흰색
	- 금-황색 長也

- [[生]](날 생)
	- 흙에 심으면 싹이 나오다 ↔ 死
	- [[先]](먼저 선)
		- 生 + 儿
	- 나아갈 신 : 먼저 나아가는 사람은 신이다.
	- [[贊]](도울 찬) : 나아갈 때 도와주는 사람이 있다. 돈으로 도와주다
	- [[讚]](기릴 찬) : 나아갈 때 말, 칭찬으로 도와주다 
		- [[稱讚]]
	- [[牲]](희생 생)
	- [[性]](성품 성) : 마음에서 일어나는 성품
	- 姓(성씨 성) : 모계사회 일때의 어머니의 성을 따르다. 
	- 星(별 성) : 별에도 각 일생이 있다. 

- [[麗]](고울 려, 아름다울 려)
	- [[丽]](짝 려) 두 마리의 사슴([[鹿]])이 함께 있을 때 가장 아름답다.
	- [[鹿]](사슴 록)
	- [[고구려(高句麗)]] : 만물의 조화를 이루었다는 의미가 있는 국호

- [[水]](물 수)
	- 겨울 북쪽
	- 冫(얼음 빙) : 이수변은 별칭
	- 氵(삼수 변)

## 玉出崑岡
- [[玉]](구슬 옥)
	- [[王]](구슬 옥 변)
	- [[옥편(玉篇)]]
	- [[옥좌(玉座)]]
	- [[옥동자(玉童子)]]
	- [[금지옥엽(金枝玉葉)]]

	- **玉 5덕**
		- 仁 : 
		- 義 : 숨김없이 자신을 드러내다
		- 智 : 멀리서도 간파하다
		- 勇(날랠 용)
		- [[絜]](헤아릴 혈)
			- [[혈구지도(絜矩之道)]]

	- [[屋]](집 옥)
		- 해가 지면 집에 들어가 에너지를 얻다. (인 의 지 용 혈)
		- 내가 5덕을 갖추는 집

	- [[握]](쥘 악)
		- [[악수(握手)]] : 악수를 하면 상대의 집(屋)을 알 수 있다.

	- [[辟]](임금 벽, 피할 벽, 비유할 비)
		- 임금이 죄인을 다스릴 때 피하고 싶은 마음이다.

	- [[璧]](구슬 벽)
		- [[완벽(完璧)|완벽(完璧)]]

	- [[壁]](벽 벽)
	- [[避]](피할 피)

		- [[곤충(昆蟲)]]
		- [[곤손(昆孫)]] : 여섯 째 대의 손자
	- [[混]](흐릴 혼, 섞을 혼)
