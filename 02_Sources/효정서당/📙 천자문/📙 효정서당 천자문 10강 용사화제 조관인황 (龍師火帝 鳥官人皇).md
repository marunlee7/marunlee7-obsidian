# 천자문 10강 용사화제 조관인황 (龍師火帝 鳥官人皇)

![](https://www.youtube.com/watch?v=pqCNvfHqiD0)  



## 4자 풀이
[[용사화제(龍師火帝)]] : 용(龍)으로 관사(官師)의 이름을 붙인 복희씨(伏羲氏)요, 불로 관사(官師)의 이름을 붙인 신농씨(神農氏)요,
[[조관인황(鳥官人皇)]] : 새(鳥)로 관사(官師)의 이름을 붙인 소호씨(少昊氏)요, 복희씨(伏羲氏) 이전의 인황씨(人皇씨)

## 龍師火帝

- [[龍]]
	- [[䏍]](작은벌레 연) → [[能]](재능 능, 능할 능) : 할수있다의 최상급 [[熊]](곰 웅)
	- [[肙]](장구벌레 연) → [[絹]](비단 견, 그물 견, 누에고치 견)
	- [[蜎]](장구벌레 연) →
	- 용은 변신하는 정자, 아버지의 씨를 말한다. 용=미르
	- 용문(龍汶) : 용은 물가에 사는 것이 좋다. [[汶]](물가 문)
	- 용두(龍頭) : 
	- 용꿈, 개천에서 용나다, 용상에 오르다 → 실제 이루어 나아가는 과정으로 龍을 본다.

	- [[寵]](사랑할 총)
		- 수억개의 정자중에서 정착할 때는 가장 사랑 받은 상태
		- [[한자이야기 298 총애(寵愛)|총애(寵愛)]]
	- [[籠]](대바구니 롱) 
		- 용은 머리를 곳추세우고 올라가려는 성질
	- [[襲]](세습할 습, 엄습할 습, 공습할 습, 염습할 습)

- [[鱗]](비늘 린(인))
	- [[역린(逆鱗)]] : 용의 가슴에 거꾸로 난 비늘. 鱗甲也 비늘껍질이다.
	- 문사(文師)의 스승 : 龍
	- 武의 스승 : 虎
	- [[삼황(三皇)]] : 중국 고대 전서에 나타난 세 임금. 
		- 천왕 **복희씨 : 용사(龍師)**
		- 지황 **신농씨 : 화제(火帝)** 수인(燧人)
		- 인황 헌원씨(황제)
		- **치우(蚩尤)천황** : 중국의 전설상의 인물 = 단군왕검
			- 단기 2333 + 2024 = 4357년
			- 제정일치의 단군래
			- 巫(무당 무)
		- **탁록**에서 헌원과 치우의 싸움 → 華夏族이 시조가 됨 = 漢族
			- 요 순 우 탕 문 무 주 공
	- [[오제(五帝)]]
		- 소호금천씨 소호(小昊)
		- 제곡고신씨
		- 전욱고양씨
		- 단당요임금(제요방훈)
		- 단당순임금(제순중화)

- [[師]](스승 사)
	- [[巾]](헝겊 건)
- [[火]]
- [[帝]](임금 제)
	- 면류관을 쓴 모습으로 천을 늘어뜨리다 → 제후
	- [[帀]](두를 잡)
	- [[巿]](슬갑 불, 앞치마 불)
