# 천자문 07강 검호거궐 주칭야광 (劒號巨闕 珠稱夜光)

![](https://www.youtube.com/watch?v=9AQdBuAqklM)  


## 4자 풀이
- [[검호거궐(劒號巨闕)]] : 칼에는 거궐을 이름하고
- [[주칭야광(珠稱夜光)]] : 구슬에는 야광주를 일컫도다

- 거궐은 칼이름이니 구야자가 만들었다. 월왕 구천이 오나라를 멸하고 보검 여섯 개를 얻으니 오구, 담노, 간장, 막야, 어장, 거궐이 그 하나다
## 劒號巨闕

- [[劒]](칼 검)
	- 많은([[僉]] : 다 첨) 사람들이 칼을 가지고 다님


- [[號]](이름 호, 부르짖을 호)
	- [[号]] : 입에서 나오는 기운
	- [[호령(號令)]]
	- [[아호(雅號)]] : 예술가(藝術家), 학자(學者)들이 본명(本名) 외(外)에 가지는 별호(別號).
	- [[呼]](부를 호)

- [[巨]](클 거)
	- 손으로 도구(工) 잡고 재는 모습이다.
	- [[工]](장인 공) : 자를 보고 만듦
	- [[거인(巨人)]]
	- 설문해자 : 
		- [[規]](잴 규, 법 규, 동그라미 규, 컴퍼스 규)
			- 캠퍼스 둘레에서 지키다
		- [[矩]](곱자 구, 모날 구, 법도 구)
		- [[準]](평평할 준, 준할 준, 수증기 준)
			- [[준승(準繩)]]
			- [[규구준승(規矩準繩)]]

- [[闕]](대궐 궐, 큰집 궐) 
	- 門(문 문) + [[欮]](상기 궐, 숨내쉴 궐)
	- [[屰]](거스를 역)
	- 설문해자 : 문관야(門觀也)
	- [[噘]](돌 뜰 궐)


## 珠稱夜光

- [[珠]](구슬 주)
	- [[여의주(如意珠)]] : 용의 턱 아래에 있다는 영묘한 구슬
	- 설문해자: 蚌蛤也
		- [[방합(蚌蛤)]] : 

- [[稱]](일컫을 칭, 저울대 칭)
	- 양을 재어 공표하다.
	- [[爯]](들 칭) : 손으로 저울을 들어올리다. → 헤아리다
	- [[冉]](늘어질 염, 나아갈 염) : 저울대가 늘어져있다
	- [[대칭(對稱)]] : → 저울대 칭으로 쓰이는 경우

	- [[衡]](저울대 형)
		- [[전형(銓衡)]] 

- [[夜]](밤 야)

- [[光]](빛 광)
	- 불을 머리에 지고 있는 사람의 모양이다. 
	- 설문해자 : 明也
	- [[광명시(光明市)]]
	- [[胱]](오줌통 광)
		- [[방광(膀胱)]] : 오줌통.
	- [[輝]](빛날 휘)
		- 촛불시위대 모양처럼 촛불이 군사대형처럼 보이다.
		- [[軍]](군사 군)
		- [[運]](움직일 운)
		- [[揮]](지휘할 휘)
