# 가을 버들

바람도 없이  
떠난 잎이  
'철렁!'  
땅에 떨어지니  
야윈 가지  
한 올 한 올  
저녁 안개 속에  
걸려있다.  
부러진 갈대  
마른 연잎이랑  
서로 기대 서 있을 때  
원앙새는 옷이 추워  
잠도 채 못 이룬다.

後秋柳詩 후추류시

無風脫葉下鏘然 무풍탈엽하장연   
瘦影絲絲掛暮煙 수영사사괘모연   
折葦枯荷相伴住 절위고하상반주   
鴛鴦衣冷不成眠 원앙의랭불성면 

―신위(申緯·1769~1845)

![일러스트|464](https://www.chosun.com/resizer/v2/4JPBV3T5OP3XX5S7HPKE3366OU.jpg?auth=b3873a5f83095429476799a0ea940240009ff0f83efd0779f3c409e35c0c1351&width=464)

19세기 전반기의 시인 자하(紫霞) 신위가 1818년 춘천에 머물 때 지었다. 
조선조 제일이라는 평을 듣는 시인의 시답다. 
가을이 깊어가 잎이 스스로 떨어진다. 
지는 잎은 지상의 모든 조락(凋落)을 뜻하니 그처럼 떨어지는 소리가 크다. 
잎을 떨구고 야윈 가지에는 부러진 갈대와 말라버린 연잎이 
동병상련에 어울리는 친구라, 서로 추운 몸을 비빈다. 
초목 모두 입었던 옷을 벗은 저녁, 짙은 녹음 속에서 햇볕을 즐기던 원앙이도 추위에 몸을 떨고 있다. 
시를 읽으려니 낙엽이 져버린 야윈 버드나무인 양, 
어느새 잠 못 드는 원앙이인 양 몸과 마음이 오싹해져 따뜻한 것을 그립게 한다.


## 📗

後秋柳詩 : 가을이 깊은 뒤의 버드나무를 읊은 한시로, 대표적인 칠언절구(七言絶句) 형식

- 無風脫葉下鏘然 → 무풍 / 탈엽 / 하 / 쟁연
	- 무풍(無風): 바람이 없고
	- 탈엽(脫葉): 잎이 떨어지고
	- 하(下): 아래로
	- 쟁연(鏘然): 쟁그랑 소리가 난다
- 瘦影絲絲掛暮煙 → 수영 / 사사 / 괘 / 모연
    - 수영(瘦影): 마른 그림자
    - 사사(絲絲): 실오라기처럼 가늘게
    - 괘(掛): 걸려 있고
    - 모연(暮煙): 저녁 연기(저녁 안개)
- 折葦枯荷相伴住  → 절위 / 고하 / 상반 / 주
	- 折葦(절위): 부러진 갈대
	- 枯荷(고하): 마른 연잎
	- 相伴(상반): 서로 짝하여
	- 住(주): 머물다, 함께 있다
- 鴛鴦衣冷不成眠 → 원앙 / 의랭 / 불 / 성면
    - 원앙(鴛鴦): 원앙새
    - 의랭(衣冷): 옷이 차갑고
    - 불(不): ~하지 못하고
    - 성면(成眠): 잠을 이루다

 :: 아래 형태로 정리하게끔 퍼플렉시티에서 작업한다.  :: 詩 구절분석하여 어휘별로 읽고 외우면 <br>개별한자 이해도가 쑥쑥. 실망치말고 계속 할 것<br>원문-음독-분석문에서 분석문이 이해도가 빠르니<br>분석문을 애써 무시하지말고 적극 이용해서 외울것
