# 7언시 읽는법
 :: 7언시 읽는법을 참조해서 시읽고<br>나중에도 이 노트를 수정 보완할 것<br>기억: 2/2/2/1로 네 부분(4율어구, 4음보)로 끊어서 읽습니다.

## 검색 키워드

- ""折葦枯荷相伴住 구절 읽는 법""
- "아래 형태로 정리해줘"
## 7언시 한 구절 읽는 법

- 7언시는 한 구절이 7글자로 이루어져 있습니다.
- 읽을 때는 보통 2/2/2/1로 네 부분(4율어구, 4음보)로 끊어서 읽습니다.
- 각 의미 단위별로 호흡을 나누고, 운율감을 살리며 읽으면 자연스럽습니다.
- 일반적으로 한 구를 읽을 때는 2/2/3 또는 4/3으로 끊어 읽는 것이 통념입니다. 즉, 의미 단위나 호흡 단위에 따라 다음과 같이 읽을 수 있습니다.

## 적용 예시

- 無風脫葉下鏘然
	- → 무풍 / 탈엽 / 하 / 쟁연
	- (또는)
	- → 무풍 / 탈엽 / 하 / 쟁연

	- 무풍(無風): 바람이 없고
	- 탈엽(脫葉): 잎이 떨어지고
	- 하(下): 아래로
	- 쟁연(鏘然): 쟁그랑 소리가 난다

- 鴛鴦衣冷不成眠
	- 鴛鴦/衣冷/不成眠
	- (원앙 / 의랭 / 불성면)
	- 또는
	- 鴛鴦衣冷 / 不成眠
	- (원앙의랭 / 불성면)


실제 낭송이나 해석에서는 의미에 따라 자연스럽게 끊어 읽으면 됩니다. 학술적으로는 7언구를 2/2/3으로 분절하거나 4/3으로 나누는 방식이 많이 쓰입니다
## 실제 읽는 흐름 : 운율 살려 읽기

무풍 / 탈엽 / 하 / 쟁연
- 의미 단위별로 자연스럽게 멈추며, 마지막 한 글자는 여운을 두고 읽는 것이 전통적인 방식입니다.
- 각 의미 단위(2글자씩)마다 가볍게 멈추어 읽고, 마지막 한 글자는 여운을 두고 읽습니다.
