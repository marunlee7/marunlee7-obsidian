# 도깨비

무비_대사 :: 공유편 누구의 인생이건 신이 머물다 가는 순간이 있다. </br>당신이 세상에서 멀어지고 있을 때</br>누군가 세상 쪽으로 등을 떠밀어 주었다면,</br>그건 신이 당신 곁에 머물다 간 순간이다.</br>너의 삶은 너의 선택만이 정답이다.


## 기본정보

불멸의 삶을 끝내기 위해 인간 신부가 필요한 도깨비, 그와 기묘한 동거를 시작한 기억상실증 저승사자. 그런 그들 앞에 '도깨비 신부'라 주장하는 '죽었어야 할 운명'의 소녀가 나타나며 벌어지는 신비로운 낭만 설화
