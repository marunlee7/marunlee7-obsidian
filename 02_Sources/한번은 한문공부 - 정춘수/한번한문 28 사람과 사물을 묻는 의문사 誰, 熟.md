# 한번한문 28 사람과 사물을 묻는 의문사 誰, 熟
- 누구 수 誰, 누구 숙 熟은 사람과 사물을 가리키는 대표적인 의문사
- '누구' '무엇' '어느 것'
- 熟이 대명사로 주로 주어나 목적어에 쓰인다면,
- 誰는 주어, 목적어, 관형어, 서술어로 두루 쓰임
- 다른 의문문과 같이 誰, 熟은 목적어로 쓰일 때 동사 앞에 오고, 전시사의 목적어로 쓰여도 전치사 앞에 옮.
- 吾誰欺 欺天乎
- 오수기 기천호
