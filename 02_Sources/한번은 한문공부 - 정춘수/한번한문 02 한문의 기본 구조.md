# 한번한문 02 한문의 기본 구조
- **한문의 기본 구조는 주어 + 서술어**
  
  天地玄黃 宇宙洪荒 日月盈昃 辰宿列張
  
  천지현황 우주홍황 일월영측 진수열장
  
  하늘과 땅은 까맣고 누르니 우주란 넓고도 거칠다
  
  해와 달은 찼다 기울고 별과 별자리는 줄지어 펼쳐 있다 - 천자문
  
  ​
  
  **한문의 어순은 주어 + 서술어 + 목적어/보어**
  
  人法地 地法天 天法道 道法自然
  
  인법지 지법천 천법도 도법자연
  
  사람은 땅을 따르고 땅은 하늘을 따르고 하늘은 도를 따르고 도는 스스로를 따른다. - 노자 25장
  
  法 : 본받을(따를) 법, 법으로 삼다, 본받다, 따르다
  
  한문 어순의 간단한 3가지 원칙
  
  ① 주어는 서술어 앞에 온다 : 주어 + 서술어 (한국말과 같음)
  
  ② 꾸미는 말은 꾸밈을 받는 말 앞에 온다 : 수식어 + 피수식어 (한국말과 같음)
  
  ③ **목적어나 보어는 서술어 뒤에 온다 : 서술어 + 목적어/보어 (한국말과 다름)**
  
       人             法           地
  
     주어        서술어     목적어
  
  (사람은)   (따른다)   (땅을)
  
  한문에서는목적어와 보어의 양상이 우리말보다 복잡. 서술어 뒤어의 문장 성분을 우리말로 옮겼을 때 주어나 부사어처럼 해석되는 사례가 흔하여 이견이 있음
  
  亂解  이해가 어려움   多情 정이 많음   下車 차에서 내림   登山 산을 오름
  
  따라서 '서술어+목적어/보어' 관계 해석 시, 목적어냐 보어냐를 세밀하게 따지기보다 **서술어 자리에 오는 동사와 목적어/보어 자리에 오는 명사 사이의 의미관계를 살피는** 것이 핵심.
  
  그런 다음 그 의미 관계에 따라 뒤에 오는 명사에 '～을/를'이나 '～이/가', '～에/에서' 같은 우리말 조사를 붙여 풀이하는 게 좋음
  
  ​
  
  **한문은 고립어**
  
  한문은 고립어 : 낱말이 저마다 고립되어 있어 다른 낱말의 영향으로 형태가 변하지 않음
  
  한국어는 교착어 : 조사나 어미 등을 단어의 기본형에 붙여(교착하여) 써야 온전하게 말이 됨
  
  我 : 주어(나는), 목적어(나를), 관형어(나의), 서술어(나이다)
  
  食 : 종결(평서형) (먹다), 연결(종속적) (먹으면), 전성(관형사형) (먹는), 전성(명사형) (먹기, 밥)
  
  한문은 교착어가 조사나 어미로 구별되는** 문법적 관계나 의미 차이를 어순으로 나타낸다**
  
  같은 단어라도 주어 자리에 있으면 명사 구실을 하며 명사적인 뜻을 나타내고, 서술어 자리에 있으면 동사나 형용사 구실을 하면서 동사나 형용사적인 뜻을 나타낸다. 관형어 자리에 가면 형용사 구실을 하고 부사어 자리에 가면 부사 구실을 한다.
  
  信信信也 疑疑亦信也
  
  신신신야 의의역신야
  
  믿을 것을 믿는 것이 믿음이다. 의심할 것을 의심하는 것도 역시 믿음이다. - 순자 비십이자

  시시비비위지지 비시시비위지우

  **한문은 단음절어**
  
  한 단어가 1음절 한자로 이루어진 언어
  
  단어와 문장의 기본구조가 유사함 → 단어의 짜임을 확장하면 문장구조가 됨
  
  **문장구조**
  
  순서대로 풀이   - 주술관계 : 주어 + 서술어 /～이/가 ～하다 (명사+ 형용사), ～이/가 ～한다 (명사+동사), 
  
                                                ～이/가 ～이다  (명사 + 명사) / 天知, 天長
	- 병렬관계 : 자격이 나란한 말이 나란히 놓임 / ～와/과/하고 ～ / 天地, 洪荒
	- 수식관계 : 꾸미는 말과 꾸밈을 받는 말 관계 / ～의 ～, ～하는/한 ～, ～하게 ～한다  / 人道, 相利
	  
	  역순으로 풀이   - 술목관계 : 서술어 + 목적어 / ～을/를 ～ 한다 (동사+명사) / 假道
	- 술보관계 : 서술어 + 보어 / ～가/이 ～한다, ～에 ～한다 (동사+명사) / 無窮 
	  
	  兼相愛 交相利
	  
	  겸상애 교상리
	  
	  다같이 서로 사랑하고 서로 이롭게 한다 - 묵자 겸애 중편

	  **한문은 성조 언어**.
	  
	  한문은 우리말과 다르게 한자마다 음의 높낮이 변화에 따른 성조를 지니고 있음
	  
	  평성    낮고    평평한 짧은 소리                            知, 之, 爲(하다)
	  
	  측성    상성    처음에 낮았다 높아지는 긴 소리    是. 也
	  
	  거성    높고 긴 소리                                    地
	  
	  입성    급하게 그치는 소리                         列
	  
	  **허사가 중요한 이유**
	  
	  실사 : 문장에서 뼈대 구실을 하는 명사, 동사, 형용사 등
	  
	  허사 : 어구와 어구의 관계를 보여주고 말의 느낌인 어기를 나타내는 말
	  
	  대명사와 부사는 실사와 허사의 성격을 동시에 지니고 있음
	  
	  허사는 한문을 읽을때 한문이 지닌 음조와 호흡을 추정하도록 도와주고, 문장을 어디서 끊고 이어야 하는지 알려줘서 의미의 덩어리를 구분하기 쉽게 해주며, 말의 성격이나 느낌을 표시하여 문장에 배인 감정이나 정조를 파악할 수 있게 해줌
