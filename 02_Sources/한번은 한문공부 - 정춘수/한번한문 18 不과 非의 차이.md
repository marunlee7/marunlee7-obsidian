# 한번한문 18 不과 非의 차이
alias:: [[不]], [[非]]

- 不과 非는 부정사를 대표하는 글자이나 쓰임에 차이가 있음
- ​
- 不은 주로 동사나 형용사 앞에서 동작이나 행위, 상태를 부정함
- 弗은 不과 비슷한 기능을 하나, 형용사 앞에서는 쓰이지 않고 부정의 어기가 不보다 셈
- 否는 不+동사   可否 = 可不可
- ​
- 非는 명사 앞에서 판단을 부정함
- 논어와 맹자 이전 시기에는 匪가 非처럼 쓰임
- ​
- 王之不王 不爲也 非不能也
- 왕지불왕 불위야 비불능야
  왕이 왕 노릇을 하지 못하는 것은 하지 않아서이지 할 수 없어서가 아니다. - 맹자 양혜왕 상
