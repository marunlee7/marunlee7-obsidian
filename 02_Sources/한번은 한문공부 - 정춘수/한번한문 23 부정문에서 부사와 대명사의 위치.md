# 한번한문 23 부정문에서 부사와 대명사의 위치
alias:: 부사, 대명사

- 한문에서 부사의 위치는 정해져 있으며, 특히 부정문에서 부사는 부정사 앞에 놓이느냐 뒤에 놓이느냐에 따라 의미가 상이함
- 不必然 반드시 그렇게 되는 것은 아니다 (부분 부정)
- 必不然 반드시 그렇게 되지 않는다 (전체 부정)
- 小故 有之不必然 無之必不然 體也 若有端 大故 有之必然 無之必不然 若見之成見也
- 소고 유지불필연 무지필불연 체야 약유단 대고 유지필연 무지필부연 약견지성견야
- 작은 원인은 그것이 있더라도 반드시 그렇게 되는 것은 아니지만 그것이 없으면 반드시 그렇게 되지 않는다
- 구체젹인 예로 단초가 있는 것과 같다
- 큰 원인은 그것이 있으면 반드시 그렇게 되고 없다면 반드시 그렇게 되지 않게 된다
  무언가를 보면 보게되는 것과 같다 - 묵자 경 상
- ​
- 부사의 의미에 따라 전체 부정과 부분 부정의 차이가 모호해지기도 함
- 不復와 復不 사이에 큰 차이가 없음
- 이중 부정은 강한 긍정이므로 부사의 위치에 따른 의미 변화가 없음
- ​
- 부정문에서 대명사의 위치도 중요함
- 평서문 : 동사 + 대명사 (목적어)
- 부정문 : 부정사 + 대명사 (목적어) + 동사
- 我無爾詐 爾無我O
- 아무이사 이무아O
  우리는 너희를 속이지 않았고 너희는 우리를 근심시키지 않는다 - 좌전 선공 15년
- 不患莫己知 求爲可知
- 불환막기지 구환가지
  자기를 알아주지 않는다고 근심하지 말고 알아줄 만하게 되록록 애쓴다 - 논어 이인
