# 한번한문 26 시간을 표현하는 부사
alias:: 시간 부사

- 한문은 과거, 현재, 미래 같은 시제나 진행, 완료, 예정 같은 동작의 양상 표현이 발달되지 않은 언어
- 그런 개념에 들어맞는 문법적 장치가 없으나, 다만 시간이나 시간의 흐름을 강조해야 할 때 부사를 사용하여 대략의 의미를 표현
- ​
- '장차 장' 將, 미래를 표시
- 흔히 '장수' '장군'으로 쓰이는 명사이나,
- 부사로 쓰여 미래를 표시하면 '장차 ～하려 한다' '아마 ～일(할) 것이다'
- 부사로 쓰여 시간을 표시하지 않고 '오직' '또한' '거의' 같은 뜻을 나타낼 때도 있음
- 國將興 聽於民 將亡 聽於神
- 국장흥 청어민 장망 청어신
  나라가 장차 흥하려 할 때에는 사람에게서 듣고 장차 망하려 할 때에는 신에게서 듣는다 - 좌전 장공 32년
- ​
- '이미 기' 旣, 완료를 표시
- 주로 부사로 사용하여 '이미' '여전히' 등으로 해석합
- 동작이나 상태의 완료를 나타내므로 '～한 후에' '곧' 등으로 풀이되기도 함
- '～어 놓다' '～어 두다' 등으로 번역하면 완료의 효과를 더 강조할 수 있음
- ​
- 歸居來兮 田圓將蕪 胡不歸 旣自以心爲形役 奚惆愴獨悲
- 귀거래혜 전원장무 호불귀 기자이심위형역 해추장이독비
- 돌아가자! 시골이 장차 황무지가 되려 하는데 어찌 돌아가지 않는가?
  이미 스스로 미음을 육체에 부역시켜 놓고서 어찌 가슴 아파하며 홀로 슬퍼하는가? - 도연명 귀거래사
