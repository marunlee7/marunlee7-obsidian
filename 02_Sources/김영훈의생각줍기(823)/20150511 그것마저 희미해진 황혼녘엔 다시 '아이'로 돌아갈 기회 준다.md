# 20150511 그것마저 희미해진 황혼녘엔 다시 '아이'로 돌아갈 기회 준다
![480](https://img.hani.co.kr/imgdb/resize/2015/0511/143125182506_20150511.JPG)

## 📘 CartoonAllegory

하늘은...
청춘의 
텅 빈 기억 속에 
'호기심'을 채워주고
나이들면
잃어버린 기억의 한편에
'통찰의 지혜'를 채워준다. 

**그것마저 희미해진 황혼녘엔 다시 '아이'로 돌아갈 기회 준다.** 

내게 청춘의 호기심을 채운 것은 친구였다.
