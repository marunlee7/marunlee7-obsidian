# 여수댁 같은 보통명사 말고, '고유명사'인 내 이름을 불러주세요

![480](https://img.hani.co.kr/imgdb/resize/2013/0529/136973651640_20130529.JPG)

## 📘 CartoonAllegory

시집온 뒤에 누구의 엄마로,
나이 들어선 
여수댁이나 1008호 아줌마로,
이 나이엔 그냥
할머니로 불리며 살아온 내 생애
여수댁 같은 보통명사 말고, '고유명사'인 내 이름을 불러 주세요.
