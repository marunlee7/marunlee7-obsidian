# 20130729 나도 너처럼 짧아도 빛나는 '새날'을 노래하고 싶다

![480](https://img.hani.co.kr/imgdb/resize/2013/0729/137500378597_20130729.JPG)

## 📘 CartoonAllegory

칠 년이라는
기나긴 어둠고 침묵의
껍질을 깨고 
환골탈태하여
짧은 칠 일 동안 
'새 삶'을 노래하는 너

나도 너처럼 짧아도 빛나는 '새 날'을 노래하고 싶다.
