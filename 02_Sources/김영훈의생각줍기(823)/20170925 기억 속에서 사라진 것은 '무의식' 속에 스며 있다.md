# 20170925 기억 속에서 사라진 것은 '무의식' 속에 스며 있다

![480](https://img.hani.co.kr/imgdb/resize/2017/0924/00501287_20170924.JPG)

## 📘 CartoonAllegory

불 속에서
사라진 것은 '재' 속에 남아 있고

기억 속에서 사라진 것은 '무의식' 속에 스며 있다.

## 📓 청상
> [!note]+


기억속에 묻혀버린 것도
내 몸이 기억한다. 
잊고 또 잊어도 
내 몸이 기억한다.
계속 정진할 것.
