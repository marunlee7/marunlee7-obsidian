# 20130703 어제의 '잣대'로 오늘의 나를 재려 하지 마십시오

![480](https://img.hani.co.kr/imgdb/resize/2013/0703/137276057080_20130703.JPG)

## 📘 CartoonAllegory

내세울 것 없는 학벌과
드러낼 것 하나 없는 나
분명 과거에는
그런 사람이었을지 모르겠습니다.
하지만 지금도 그럴 것이라
예단하지 마십시오.
어제와 다른 오늘, 
오늘과 다른 내일을 꿈꾸는 난,
과거의 내가 아니므로.

어제의 '잣대'로 오늘의 나를 재려 하지 마십시오
