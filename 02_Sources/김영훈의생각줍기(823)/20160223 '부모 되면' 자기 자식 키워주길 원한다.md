# 20160223 '부모 되면' 자기 자식 키워주길 원한다

![480](https://img.hani.co.kr/imgdb/resize/2016/0222/00551333902_20160222.JPG)

## 📘 CartoonAllegory

'자식'은 어릴 땐 같이 놀아주길 원하고,
철들면 방에서 나가주길 원하고,
어른되면 간섭 않길 원하고.
'부모 되면' 자기 자식 키워주길 원한다.
