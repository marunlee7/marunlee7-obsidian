# 20181003 증오는 사랑의 '소각'이다


## 📘 CartoonAllegory
![480](https://img.hani.co.kr/imgdb/resize/2018/1002/00502186_20181002.JPG)

미움은 사랑의 '소진'이고
**증오는 사랑의 '소각'이다.**

## 📓 청상
> [!note]+

- 증오(憎惡) : 아주 사무치게 미워함. 또는 그런 마음.
	- [[憎]](미울 증)
	- [[惡]](악할 악)

- 소진(消盡) : 아주 사라져 다 없어짐.
	- [[消]](사라질 소)
	- [[盡]](다할 진)
- 소각(燒却) : 불에 태워 없애 버림.
	- [[燒]](불사를 소)
	- [[却]](물리칠 각)
