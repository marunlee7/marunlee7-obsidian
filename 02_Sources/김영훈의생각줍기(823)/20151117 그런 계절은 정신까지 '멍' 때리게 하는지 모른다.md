# 20151117 그런 계절은 정신까지 '멍' 때리게 하는지 모른다

![480](https://img.hani.co.kr/imgdb/resize/2015/1117/144766597636_20151117.JPG)

## 📘 CartoonAllegory

가을이 '멍'들면 단풍 들고,
곱게 물든 마음이 '멍'들면 먹먹해지는 거다.

그런 계절은 정신까지 '멍' 때리게 하는지 모른다.
