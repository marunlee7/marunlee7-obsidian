# 20130702 삶은 교통표지판과 많이 닮았나 봅니다

![480](https://img.hani.co.kr/imgdb/resize/2013/0702/137267365193_20130702.JPG)

## 📘 CartoonAllegory

때론 천천히 갈 때도, 
다른 길을 선택할 때도,
남에게 양보할 때도,
자신을 세우지 못하고 계속 앞으로 갈 때도, 
==다시 돌아가야 할 때도 있는게 삶인가 봅니다. 
삶은 '교통표지판'과 많이 닮았다 봅니다. ==
