# 20170726 부평초처럼 '밑바닥'을 모르는 공감능력 부족 때문이다

![480](https://img.hani.co.kr/imgdb/resize/2017/0725/00502374_20170725.JPG)

## 📘 CartoonAllegory

명선수가
'명감독'이 되지 못하는 이유는

부평초처럼 '밑바닥'을 모르는 공감능력 부족 때문이다.


## 📓 청상
> [!note]+

공감 능력이 부족한 이유는 부평초처럼 밑바닥을 모르기 때문이다.
공감 능력은 바닥을 알 때 생긴다. 
바닥은 어려움이고 고통이고 잊혀진 시간이다.
