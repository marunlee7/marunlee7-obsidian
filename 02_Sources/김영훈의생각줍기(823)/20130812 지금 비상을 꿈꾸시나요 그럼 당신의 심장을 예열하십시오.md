# 20130812 지금 비상을 꿈꾸시나요? 그럼 당신의 심장을 예열하십시오

![540](https://img.hani.co.kr/imgdb/resize/2013/0812/137621390605_20130812.JPG)

## 📘 CartoonAllegory

창공을 날기위해 
모든 에너지를 쏟아 이륙하는 비행기처럼 

당신도 지금 비상을 꿈꾸시나요? 
그럼 당신의 심장을 예열하십시오.

  :: ==If 제목 = 청상 = 비유, <br>역으로 주제, 메세지(요약)을 예측해보기==

- 청상, 비유 : 비상을 꿈꾸는 당신, 심장을 예열하라.
- 주제 : 마음 다짐
- 요약 : 창공을 나는 비행기, 이륙하기 위해 에너지 쏟듯
