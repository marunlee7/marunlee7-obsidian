# Custom Theme 적용하기

인터넷에서 원하는 logseq theme을 찾았다면, 쉽게 적용해 볼 수 있다. 본인이 작업하는 logseq 디렉토리를 살펴보면 custom.css 파일이 있는데, 이 파일의 내용을 custom theme code repository에서 지원하는 custom.css 파일의 내용으로 덮어쓰기만 하면 곧바로 적용된다. 즉, 다운로드 > custom.css 파일 내용 덮어쓰기 > 적용
