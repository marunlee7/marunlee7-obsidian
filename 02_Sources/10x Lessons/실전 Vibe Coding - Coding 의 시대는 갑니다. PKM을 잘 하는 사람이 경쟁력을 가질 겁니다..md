# 실전 Vibe Coding - Coding 의 시대는 갑니다. PKM을 잘 하는 사람이 경쟁력을 가질 겁니다.



![](https://www.youtube.com/watch?v=kDdPf4cVHTs)  

✨ 오늘의 영상 소개  
오늘 영상은 2025년 9월 14일 진행된 ‘재미로 하는 Vibe Coding’ 라이브 방송 중  
📚 PKM (Personal Knowledge Management) 에 대해 이야기한 부분을 편집한 영상입니다.  
👉 IT를 잘 모르시는 분들도 AI 시대를 준비하기 위해 꼭 보셔야 할 영상이에요!  
  
💻 제 경험에서 나온 이야기  
저는 30년간 Coding을 하며 살아남을 수 있었습니다.  
그 시절에는 회사들이 많은 Coder를 필요로 했고, 코딩 자체도 쉽지 않았습니다.  
  
하지만 이제는 상황이 달라졌습니다.  
🤖 AI를 이용하면 더 빠르고, 저렴한 비용으로 코딩이 가능한 시대가 왔습니다.  
즉, 더 이상 Coding 기술만으로는 미래 경쟁력이 될 수 없습니다.  
  
🧠 미래의 경쟁력 = 나만의 PKM  
앞으로는 AI가 갖지 못한 나만의 경험과 노하우가 경쟁력이 됩니다.  
그리고 그것들이 사라지지 않도록 잘 관리하는 것이 중요합니다.  
  
그 관리 기술이 바로…  
📚 PKM (Personal Knowledge Management) 입니다!  
  
오늘 영상에서는  
🚀 AI 시대의 경쟁력으로서 PKM의 중요성과  
🗂️ 제가 직접 활용하고 있는 간단한 PKM 방법을 소개합니다.  
  
🙌 앞으로의 계획  
저는 계속해서 PKM을 공부하고 있으며,  
기회가 되는 대로 영상을 통해 공유하겠습니다.  
✨ ‘재미로 하는 Vibe Coding’ 라이브 방송에도 많은 관심 부탁드립니다!  
  
👍 구독 & 좋아요는 영상 제작에 큰 힘이 됩니다.  
감사합니다! 🙏
