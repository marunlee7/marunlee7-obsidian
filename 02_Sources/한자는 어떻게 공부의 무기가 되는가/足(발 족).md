# 足(발 족)
***
- moc: [[MOC B-한자는 어떻게 공부의 무기가 되는가]]

- 語彙: [足(발 족)](http://hanja.naver.com/hanja?q=足(발 족))
- 意味: 
- 單語: 
	- 足(발 족):  **무릎에서 발까지의 모양**
	- 손과 발을 합해 수족(手足)이라 부른다.
	- 수족같은 존재 - 꼭 필요한 존재
	- 만족할 때 쓰는 표현
		- 풍족(豊足)
		- 만족(滿足)
		- 부족(不足)
	- 홀로 쓰이기보다는 외쪽의 변으로 많이 사용된다. 
		- 도로(道路)
		- 路(길 로)
			- 𧾷(발족 족) + 各(각각 각)
			- 각자의 발로 걷는 길
	- 발과 관련한 단어
		- 실천(實踐)
			- 踐을 파자하자하면 𧾷(발족 족) + 戔(해칠 잔, 나머지 잔)
			- 발로 밟아가면서 예정대로 이행하다는 뜻
			- 실천은 말이 아니라 발로 한다는 뜻이다.
		- 답사(踏査): 발로 밟아가면서 직접 조사하다
			- 踏(밟을 답)
			- 査(조사할 사)
		- 도약(跳躍)
	
	- 발과 관련된 부정적인 단어
		- 파행(跛行)
			- 파행을 겪다 → 뭔가 일이 제대로 되지 않는 걸 의미
			- [[跛]](절름발이 파): 절름발이를 의미한다.
				- 한쪽 발을 가죽으로 묶어 걸음걸이가 파도 같다는 뜻
				- 절룩거리며 걸으니 당연히 진도가 느리다.
		- 유린(蹂躪)
			- 국토가 적에게 유린됐다고 할 때의 유린은 짓밟혔다는 의미다.
			- 蹂(밟을 유)=𧾷(발족 족) + 柔(부드러울 유), 굳은 물체를 발로 짓밟아 부드럽게 만든다는 뜻이다.
			- 躪(짓밟을 린)=𧾷(발족 족) + 藺(골풀 린(인)), 골풀은 창포와 비슷한 식물이다. 풀을 발로 짓밟는 걸 의미한다. 
		- 주저(躊躇)
			- 躊(머뭇거릴 주)
				- 𧾷(발족 족) + 壽(목숨 수)
				- 목숨이 걸린 일에 발길이 떨어지지 않아 머뭇거리는 모양이다. 
			- 躇(머뭇거릴 저)
				- 𧾷(발족 족) + 著(나타날 저)
				- 자신을 드러내는 발걸음을 해야 할 지 머뭇거린다는 뜻이다.
		- 차질(蹉跌): 
			- 발을 헛디디어 넘어지다
			- → 일이 실패(失敗)로 돌아감.
			- 蹉(미끄러질 차)
				- 𧾷(발족 족) + 差(, 어긋날 차, 다를 차)
				- 발길이 엇갈려서 넘어지는 것이다.
			- 跌(거꾸러질 질)
				- 𧾷(발족 족) + 失(잃을 실)
				- 실수로 발을 잘못 디뎌 넘어지는 것이다.
	- 발에서 파생한 글자
	- 走(달릴 주)
	- 疋(발 소, 짝 필)
