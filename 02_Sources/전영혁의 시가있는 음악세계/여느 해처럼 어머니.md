# 여느 해처럼 어머니

어머니  
햇쑥도 캐고  
어린 보릿잎도 뜯으시어  
여느 해처럼  
된장국 만들어 주세요  

어머니  
댓님도 싸매 주시고  
옷고름도 매 주시어  
여느 해처럼  
색동옷 곱게 입혀 주세요  

어머니  
봉선화도 심으시고  
맨드라미도 가꾸시어  
여느 해처럼  
뜨락에 쓸쓸함을 달래 주세요  

어머니  
속 빈 강정처럼 살아온  
속만 썩이는 못난 자식을  
여느 해처럼  
꾸짖어 주세요  

한 세상을  
사랑으로 내어주신  
어머니.
