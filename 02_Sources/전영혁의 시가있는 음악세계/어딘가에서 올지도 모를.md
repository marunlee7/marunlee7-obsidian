# 어딘가에서 올지도 모를

누군가가 몹시 보고 싶을 때가 있다  
옥빛 하늘에 빠진  
바람결처럼  
누군가를 지독히도  
느끼고 싶을 때가 있다  
조금씩 붉어져 가는 잎새나  
어쩌다 가을에 홀로 핀  
장미같이  
부끄러움도 잊고 싶을 때가 있다  
어느 날 갑자기 날아든 당혹한 고백을  
사랑했었다는 지금은 완료된  
과거분사로라도  
내 가당찮은 희망을  
그려보고 싶을 때가 있다  
문 열면 가슴이 저린 날  
문 닫은 우체국 소인이 찍힌  
투두둑  
봉투 뜯는 소리를 듣고 싶을 때가 있다
