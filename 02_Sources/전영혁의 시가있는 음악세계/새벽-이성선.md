# 새벽-이성선
새벽-이성선 / 이성선  
  
  
五更의 하늘 속에서 산이  
천천히 문을 열고  
나오는 모습을 보았다  
  
닭이 울었다  
  
닭울음은 마을 너머 동트는 곳보다 먼  
곳에서 들려왔다  
  
나의 호흡  
나의 생명은  
내 곁에 있다  
  
그러나 산은 산너머에서 오고  
나도 나 너머에서 왔다
