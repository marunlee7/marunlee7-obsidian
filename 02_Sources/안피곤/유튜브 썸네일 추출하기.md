# 유튜브 썸네일 추출하기

**목차**

## 사용법

1. 유튜브 링크 복사하여 입력박스에 붙여넣기 합니다.
2. 엔터키 입력 또는 "Export" 버튼을 클릭합니다.
3. 추출된 이미지 위에서 마우스 우클릭하여, "다른 이름으로 이미지 저장"을 선택하여 파일로 저장합니다.
