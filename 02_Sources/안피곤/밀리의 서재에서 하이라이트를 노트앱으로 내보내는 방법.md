# 밀리의 서재에서 하이라이트를 노트앱으로 내보내는 방법

이 글에서는 밀리의 서재 앱에서 하이라이트한 내용을 마크다운 형식으로 추출하여 노트앱(옵시디언)으로 내보내는 방법을 안내합니다.

1\. 밀리의 서재 앱에서 “내 서재” 아이콘 메뉴를 클릭합니다.

![blob](https://blog.kakaocdn.net/dn/EiqYV/btsEFTunjx3/L6WxbCpRIlCt9kjvP43kJk/img.png)

2\. 그리고 독서 노트를 클릭한 다음, 하이라이트를 선택합니다. 그다음 원하는 책에서 공유 아이콘 버튼을 눌러 웹링크를 복사합니다.

![blob](https://blog.kakaocdn.net/dn/dHSpdf/btsEE1Nc0yf/jB05jAJS0Q6CdSlgIrzQWK/img.png)

3\. 웹링크를 브라우저에 붙여넣기하면 아래 페이지가 나타납니다.

![blob](https://blog.kakaocdn.net/dn/wUKnt/btsEKDRqQaH/4FduGIx5Vcj51A469Q1cW1/img.png)

4\. 이 페이지에서는 마우스 드래그 복사 기능이 막혀 있습니다. [드래그프리(마우스 드래그 & 마우스 우클릭 해제)](https://chrome.google.com/webstore/detail/drag-freemouse-drag-mouse/cnfngpgfjllafbghaimjcmailafcdhod?hl=ko) 익스텐션을 설치하여 해제할 수 있지만, 아래 방법을 추천합니다.

## Markdown Web Clipper 크롬 익스텐션 사용하기

현재 보고 있는 페이지를 마크다운 형식으로 추출하려면 [MarkDownload](https://chrome.google.com/webstore/detail/markdownload-markdown-web/pcmpcfapbekmbjjkdalcgopdkipoggdi) 라는 크롬 익스텐션을 사용할 수 있습니다.

![blob](https://blog.kakaocdn.net/dn/cxKYoA/btsEHFB9CNl/sn5G5YTs7SUqFkr8vinSSk/img.png)

## 옵시디언 북마크틀릿 사용하기

옵시디언 북마크틀릿을 사용하여 하이라이트한 내용을 옵시디언으로 내보내면, 아래와 같이 추출됩니다. 사용 방법은 [즐겨찾기 버튼으로 만드는 옵시디언 웹 클리퍼(Obsidian Web Clipper) - tistory](https://olait.tistory.com/43) 블로그 글을 참고해주세요.

![blob](https://blog.kakaocdn.net/dn/wAj89/btsEE2SSARR/UKK03bwyhEnHlNGoQ6Blb0/img.png)

## 결론

밀리의 서재 앱과 웹을 둘 다 실행해야 하는 번거로움이 있습니다. 하지만 이 방법으로 밀리의 서재에서 읽은 책으로 부터 하이라이트한 내용을 쉽게 마크다운 형식으로 추출하여 노트앱(옵시디언)으로 내보내는 편리한 방법입니다. 이를 통해 사용자는 효과적으로 독서 노트를 작성하고 관리할 수 있습니다.

그리고 만약 가능하다면 옵시디언 플러그인으로 제작해 볼 생각입니다.
