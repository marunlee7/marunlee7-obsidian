# Zoottelkeeper Obsidian MOC 인덱스 파일 자동 생성(개선된 버전)

Zoottelkeeper 플러그인은 자동으로 모든 파일과 폴더를 인덱싱하여 MOC(Master of Control) 파일을 자동으로 생성하는 플러그인입니다. 이를 통해 모든 노트와 문서를 쉽게 찾아볼 수 있는 인덱스 페이지를 자동으로 만들어 줍니다.

## 주요 기능

Zoottelkeeper는 다음과 같은 주요 기능을 통해 Obsidian 사용자의 경험을 향상시킵니다:

- **자동 인덱스 생성**: 폴더와 파일을 자동으로 인덱싱하여, 각 폴더에 대한 명확한 개요를 제공합니다.
- **포함/제외 폴더 설정**: 사용자가 지정한 폴더만 인덱싱에 포함시키거나 특정 폴더를 제외할 수 있습니다.
- **템플릿 설정**: 사용자는 인덱스 파일의 형식을 자유롭게 정의할 수 있습니다, 이는 템플릿을 통해 가능합니다.
- **메타 태그 사용**: 인덱스 파일에 사용자 정의 메타 태그를 추가하여, 더욱 세분화된 분류와 검색이 가능해집니다.

## 사용자 이점

Zoottelkeeper를 사용하면 다음과 같은 이점을 누릴 수 있습니다:

- **시간 절약**: 파일과 폴더의 이동, 생성 및 삭제를 자동으로 추적하여, 수동으로 인덱스를 업데이트할 필요가 없습니다.
- **구조화된 지식 관리**: 모든 파일과 폴더에 대한 명확한 개요를 제공하여, 지식과 아이디어 사이의 연결을 강화합니다.
- **유연한 조직**: 폴더 구조의 변경이 필요할 때, Zoottelkeeper는 자동으로 모든 인덱스를 업데이트하여 최신 상태를 유지합니다.

## 사용 예와 팁

- **포트폴리오 관리**: 자신의 작업을 카테고리별로 구분하여, 각 카테고리에 대한 인덱스 파일을 생성하면 포트폴리오를 쉽게 관리하고 공유할 수 있습니다.
- **연구 프로젝트**: 연구 관련 파일과 노트를 주제별로 구분하고, Zoottelkeeper를 사용하여 각 주제에 대한 인덱스를 자동 생성함으로써 연구 자료를 효과적으로 관리할 수 있습니다.

## Zoottelkeeper 플러그인 설치 및 설정 방법

구조화된 지식 관리를 위해 Zoottelkeeper 플러그인을 설치하는 방법은 두 가지입니다. 하나는 Obsidian의 커뮤니티 플러그인을 통한 기본 버전 설치이고, 다른 하나는 BRAT 플러그인을 사용하여 개선된 버전을 설치하는 방법입니다.

### 기본 버전 설치

1. **Obsidian 열기**: Obsidian 앱을 열고 설정으로 이동합니다.
2. **플러그인 탭 접근**: 설정 내에서 ‘커뮤니티 플러그인’ 섹션으로 이동합니다.
3. **Zoottelkeeper 검색 및 설치**: ‘플러그인 검색’ 필드에 'Zoottelkeeper’를 입력하고 검색 결과에서 플러그인을 찾은 후 설치 버튼을 클릭합니다.
4. **플러그인 활성화**: 설치가 완료되면, 플러그인 리스트에서 Zoottelkeeper를 찾아 활성화시킵니다.

### 개선된 버전 설치 (BRAT 플러그인 사용)

개선된 버전의 Zoottelkeeper 플러그인을 사용하고자 하는 경우, BRAT(Beta Reviewer’s Auto-update Tool) 플러그인을 통해 설치할 수 있습니다. 이 버전에는 다음과 같은 개선 사항이 포함됩니다:

- 루트 폴더에 인덱스 파일을 생성하지 않는 기능
- 한국어 번역 추가

BRAT 플러그인을 사용하면 베타 버전 플러그인을 쉽게 설치하고 자동으로 업데이트할 수 있습니다. 아래 단계를 따라 Zoottelkeeper 플러그인을 설치해 보세요:

1. **BRAT 플러그인 설치**: Obsidian 설정의 ‘커뮤니티 플러그인’ 섹션에서 'BRAT’를 검색하여 설치합니다. [↗ BART 설치 링크](https://obsidian.md/plugins?id=obsidian42-brat)
2. **BRAT 플러그 설정으로 이동**: BRAT 플러그인을 활성화한 후, 설정 아이콘을 클릭하여 BRAT 설정에 접근합니다.
3. **Beta Plugin List 설정**: 'Beta Plugin List’에 `anpigon/zoottelkeeper-obsidian-plugin` 를 추가하여 개선된 Zoottelkeeper 플러그인을 설치합니다.
4. **Zoottelkeeper 플러그인 설치**: Beta Plugin List에 추가한 후, BRAT 플러그인이 자동으로 Zoottelkeeper 플러그인을 설치하고, 향후 업데이트가 있을 때마다 자동으로 업데이트를 진행합니다.
![blob](https://blog.kakaocdn.net/dn/cuYmME/btsEKDDTCc4/lR8QkaKC28txfzRkMNgnK1/img.png)

이 두 가지 방법을 통해, 사용자는 자신의 필요에 가장 잘 맞는 Zoottelkeeper 플러그인 버전을 선택하여 설치할 수 있습니다. 기본 버전은 빠르고 쉽게 설치할 수 있는 반면, 개선된 버전은 추가 기능과 한국어 지원의 이점을 제공합니다. 이제 Obsidian에서 더욱 향상된 Zoottelkeeper 플러그인을 통해 파일과 폴더를 체계적으로 관리하고, 지식 관리 경험을 한층 더 향상시킬 준비가 되었습니다.
