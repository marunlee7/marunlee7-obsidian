# 옵시디언 웹 스크랩핑 가이드의 모든 것

## 1\. 브라우저 북마크릿을 활용한 스크랩핑

브라우저 북마크에 자바스크립트 스크립트(Javascript)를 추가함으로써, 웹 페이지를 옵시디언으로 쉽게 가져올 수 있는 방법입니다. 별도 설치가 필요 없어 편리하지만, 자바스크립트에 대한 기본적인 이해가 필요합니다. 초보자에게는 설정이 어려울 수 있습니다.

- 북마크릿 스크립트 코드: [여기서 확인](https://gist.github.com/kepano/90c05f162c37cf730abb8ff027987ca3)
	- 북마크릿 스크립트 제작 도구: [Obsidian Clipper Maker](https://obsidian-clipper-maker.ganesshkumar.com/)
- 제작자 설명: [Obsidian Web Clipper by Steph Ango](https://stephango.com/obsidian-web-clipper)
- 한글 가이드: [옵시디언 웹 클리퍼 만들기](https://olait.tistory.com/43)

## 2\. Markdown Web Clipper 브라우저 익스텐션

웹 페이지를 마크다운 파일(.md) 형식으로 저장할 수 있게 해주는 브라우저 확장 프로그램입니다. 저장 위치는 사용자가 지정해야 합니다. 구글 크롬, 마이크로소프트 엣지, 파이어폭스, 사파리에서 사용 가능합니다.

- [Google Chrome](https://chrome.google.com/webstore/detail/markdownload-markdown-web/pcmpcfapbekmbjjkdalcgopdkipoggdi)
- [Mozilla Firefox](https://addons.mozilla.org/en-GB/firefox/addon/markdownload/)
- [Microsoft Edge](https://microsoftedge.microsoft.com/addons/detail/hajanaajapkhaabfcofdjgjnlgkdkknm)
- [Apple Safari](https://apple.co/3tcU0pD)

### 추가 크롬 확장 프로그램

- [Obsidian Clipper by faumeier02](https://chromewebstore.google.com/detail/hkfbbbooclabgjedfpeihhjjlgeclgnf): 사용하기 간편합니다. 크롬 브라우저를 사용한다면 이 익스텐션을 추천합니다. 🌟
- [Obsidian clipper by Ruoyu](https://chromewebstore.google.com/detail/obsidian-clipper/kglbgdfofhdginkekaacmajnhpoplnpj?hl=ko): 사용하기 쉽고 기능이 단순합니다. 오픈 소스 프로젝트로 소스코드는 [GitHub에서 확인 가능](https://github.com/Ruoyu-Klaus/obsidian-clipper).
- [markClipper - obsidian web clipper](https://chromewebstore.google.com/detail/markclipper-obsidian-web/mcbhalpamcihagflkpllacdcfmmnjemn?hl=ko): 스크랩 기능 외에 노트 작성 및 스크린샷 캡처 가능. 오픈 소스 프로젝트로 소스코드는 [GitHub에서 확인 가능](https://github.com/anilkumarum/mark-clipper).
- [Obsidian Web](https://chromewebstore.google.com/detail/obsidian-web/edoacekkjanmingkbkgjndndibhkegad): [Local REST API 플러그인](https://obsidian.md/plugins?id=obsidian-local-rest-api) 이 필요하며, 설정이 복잡할 수 있습니다. 추천하지 않습니다.

## 3\. 옵시디언 커뮤니티 플러그인: Extract Url Content

옵시디언 노트에 URL을 입력하면 해당 웹페이지 스크랩하여 마크다운 형식으로 변환합니다. 옵시디언 내에서 간편하게 사용할 수 있습니다.

- Extract URL content 플러그인 설치하기: [여기서 설치](https://obsidian.md/plugins?id=extract-url)

저의 경우에는 옵시디언 내에서 [Surfing 플러그인](https://obsidian.md/plugins?id=surfing) 를 사용하여 웹페이지를 탐색하다가, 웹페에지 스크랩이 필요할 때 아주 가끔 사용하는 유용한 플러그인입니다. 추가로 [Templater 플러그인](https://obsidian.md/plugins?id=templater-obsidian) 과 [obsidian-scrapers 스크립트](https://github.com/basilioss/obsidian-scrapers?tab=readme-ov-file) 를 사용해 스크랩하는 방법도 있습니다.

## 4\. Omnivore 사용 🌟추천

[Omnivore](https://omnivore.app/) 와 [옵시디언 Omnivore 플러그인](https://obsidian.md/plugins?id=obsidian-omnivore) 을 사용하는 스크랩 방법입니다. 옴니보어 플러그인을 사용하기 위해서는 [Omnivore](https://omnivore.app/settings/api) 에 가입 후 API Key를 발급받아야 합니다. 현재 가장 자주 사용하는 방법이며 옵니보어 스크랩 기능에 매우 만족합니다. 옴니보어에서 하이라이트한 부분만 가져오는 기능도 가능합니다. 옴니보어는 [오픈소스](https://github.com/omnivore-app/omnivore) 이며 가격은 무료입니다.
