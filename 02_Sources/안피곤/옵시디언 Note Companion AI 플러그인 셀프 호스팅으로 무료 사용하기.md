# 옵시디언 Note Companion AI 플러그인 셀프 호스팅으로 무료 사용하기

## 옵시디언을 혁신하는 AI 도구: Note Companion AI 플러그인

디지털 노트의 세계에서 효율성과 체계성은 성공의 열쇠입니다.

수많은 정보를 관리하며 생산성을 극대화하고 싶은 연구자, 학생, 지식 근로자라면 주목하세요.

오늘 소개할 **Note Companion AI** 는 옵시디언(Obsidian) 사용자들에게 혁신적인 경험을 선사하는 강력한 AI 플러그인입니다.

단순한 노트 정리를 넘어, 손글씨 디지털화부터 회의 기록 자동화까지,

이 도구는 여러분의 워크플로우를 완전히 바꿔놓을 것입니다.

자, 함께 이 놀라운 플러그인의 기능과 셀프 호스팅 방법을 탐험해보시죠!

## Note Companion AI란 무엇인가?

**Note Companion AI** 는 옵시디언 환경에 최적화된 AI 기반 플러그인으로,

노트 관리의 번거로움을 줄이고 생산성을 극대화하는 데 초점을 맞춘 도구입니다.

이 플러그인은 단순한 자동화를 넘어, 지능형 추천과 콘텐츠 통합을 통해 사용자의 지식 관리 경험을 한 단계 업그레이드합니다.

연구자부터 학생, 전문가까지, 옵시디언을 사용하는 모든 이들에게 필수적인 파트너로 자리 잡고 있습니다.

## 주요 기능: 생산성을 극대화하는 혁신적 도구

Note Companion AI는 다양한 기능을 통해 사용자의 워크플로우를 혁신적으로 개선합니다.

아래에서 핵심 기능을 하나씩 살펴보겠습니다.

### 1\. 자동화된 노트 정리

AI가 폴더 구조, 태그, 파일 이름을 분석하여 최적의 정리 방안을 추천합니다.

단 한 번의 클릭으로 적용하거나, 반복 작업을 자동화할 수 있어 시간을 절약할 수 있습니다.

특히 **Inbox 폴더** 를 활용하면 매일 쌓이는 노트를 체계적으로 분류하는 데 큰 도움이 됩니다.

### 2\. 핸드노트 디지털화

손으로 쓴 노트나 다이어그램을 디지털화하고 싶으신가요?

OCR 기술을 통해 손글씨를 검색 가능한 텍스트로 변환하여, 더 이상 종이 노트를 따로 관리할 필요가 없습니다.

### 3\. 오디오 및 비디오 처리

음성 파일을 텍스트로 변환하거나 유튜브 영상의 핵심 내용을 요약하는 기능은 바쁜 현대인에게 필수적입니다.

예를 들어, 긴 강의 영상을 5분 만에 핵심 내용으로 정리할 수 있다면 얼마나 유용할까요?

### 4\. AI 채팅 및 스마트 자동완성

작성 중인 노트와 기존 노트의 문맥을 이해하는 AI 채팅 기능은 작업 효율을 극대화합니다.

또한, 텍스트 입력 시 스마트 자동완성 기능을 통해 글쓰기 속도를 놀라울 만큼 높여줍니다.

### 5\. 회의 노트 강화

회의 녹음 내용을 기존 노트와 통합하여 구조화된 기록을 생성합니다.

더 이상 회의 후 별도의 정리 작업에 시간을 낭비하지 않아도 됩니다.

## 활용 사례: 일상 속 혁신

Note Companion AI는 다양한 상황에서 강력한 도움을 제공합니다.

- **지능형 Vault 관리**: 자연어 검색과 AI 추천 기능을 통해 옵시디언 Vault를 체계적인 지식 베이스로 탈바꿈시킵니다.
- **효율적인 노트 작성**: Ollama AI를 활용해 중복 콘텐츠를 방지하고 관련 내용을 제안받아 빠르고 정확한 노트 작성이 가능합니다.

또한, 사용자 정의 템플릿으로 일관된 포맷을 유지할 수 있으며,

iOS Shortcut을 통한 모바일 통합으로 클라우드 기반 작업도 문제없습니다.

더 많은 활용 사례는 [공식 홈페이지](https://www.fileorganizer2000.com/) 에서 확인해보세요.

## Note Companion AI 셀프 호스팅 방법

Note Companion AI의 강력한 기능을 온전히 활용하기 위해,

로컬 또는 클라우드 환경에서 셀프 호스팅을 설정할 수 있습니다.

아래에서 로컬 호스팅과 Vercel을 통한 무료 호스팅 방법을 단계별로 안내합니다.

### 로컬 호스팅: 내 컴퓨터에서 직접 실행하기

로컬 환경에서 셀프 호스팅을 설정하는 방법은 다음과 같습니다.

#### 1단계: 리포지토리 다운로드

리포지토리를 로컬에 클론합니다.

```
git clone https://github.com/different-ai/note-companion.git
cd note-companion
```

#### 2단계: 패키지 설치

필요한 종속성을 설치합니다.

```cmake
npm install
```

#### 3단계: 환경 변수 설정

**환경 변수 설정**  
`.env` 파일을 생성하세요.

```
cp ./packages/web/.env.example ./packages/web/.env
```

`.env` 파일을 열어 사용하려는 AI의 API 키를 입력합니다.

```
### The fastest way to get started is just to add your OpenAI API key to the .env file.
OPENAI_API_KEY="sk-xxxx"
OPENAI_API_BASE=""  # The base URL for OpenAI API
GROQ_API_KEY="gsk-xxxx"
GROQ_API_BASE=""  # The base URL for Groq API
ANTHROPIC_API_KEY="sk-xxxx"
ANTHROPIC_API_BASE=""  # The base URL for Anthropic API
GOOGLE_API_KEY="sk-xxxx"
GOOGLE_API_BASE=""  # The base URL for Google API
DEEPSEEK_API_KEY="sk-xxxx"
DEEPSEEK_API_BASE=""  # The base URL for DeepSeek API
MISTRAL_API_KEY="sk-xxxx"  # The API key for Mistral AI OCR
```

#### 3단계: 실행

다음 명령어로 앱을 빌드하고 실행합니다.

```
cd packages/web && pnpm build:self-host && pnpm start
```

또는 Docker를 사용하는 경우:

```
docker compose up -d
```

#### 4단계: 옵시디언 플러그인 설정

옵시디언 내 Note Companion 플러그인 설정에서 "Enable Self-hosting" 옵션을 활성화하고, 서버 주소를 입력한 후 옵시디언을 재시작합니다.
