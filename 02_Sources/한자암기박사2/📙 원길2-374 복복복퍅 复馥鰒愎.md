# 📙 원길2-374 복복복퍅 复馥鰒愎

- [[复]] 
	- 거듭 복, 돌아올 복
		- 사람(𠂉)들은 해(日)가 지면 천천히 걸어(夊) 거듭 돌아오니 **거듭 복, 돌아올 복**
- [[馥]] 
	- 향기 복
		- 향기(香)가 거듭(复) 풍겨 향기로우니 **향기로울 복**
	- [[馥馥]]
	    - 복복
	        - 향기가 가득 풍기는 모양.
	        - 예시: 꽃밭에서 복복한 향기가 일었다.
	        - 馥 향기로울 복, 馥 향기로울 복
	- [[馥郁]]
	    - 복욱
	        - 향기가 진하고 넘침.
	        - 예시: 복육한 향기가 방에 가득하다.
	        - 馥 향기로울 복, 郁 무성할 욱
	- [[郁馥]]
	    - 욱복
	        - 향기가 무성히 진동하는 모양.
	        - 예시: 봄꽃이 육복하게 피었다.
	        - 郁 무성할 욱, 馥 향기로울 복
- [[鰒]] 
	- 전복 복
		- 물고기(魚)와 함께 바다에서 거듭(复) 엉겨 붙어 자라는 전복이니 **전복 복**
		- 전복은 무엇에 엉겨 붙어 자라지요
	- [[全鰒]]
	    - 전복
	        - 껍질째로 온전한 전복. 식재료.
	        - 예시: 잔칫상에 전복이 올랐다.
	        - 全 온전할 전, 鰒 전복 복
	- [[鰒卵]]
	    - 복란
	        - 전복의 알.
	        - 예시: 복란을 채취하다.
	        - 鰒 전복 복, 卵 알 란
	- [[乾鰒]]
	    - 건복
	        - 말린 전복.
	        - 예시: 건복은 귀한 약재로 쓰였다.
	        - 乾 마를 건, 鰒 전복 복
- [[愎]] 
	- 강퍅할 팍
		- 마음(忄)이 거듭(复) 꼬여 괴팍하니 **괴팍할 퍅**
	- [[乖愎]]
	    - 괴복
	        - 고집 세고 거스르는 성질.
	        - 예시: 괴복한 성격 때문에 충돌이 잦았다.
	        - 乖 어길 괴, 愎 고집할 복
	- [[愎性]]
	    - 복성
	        - 고집스러운 성질.
	        - 예시: 그는 복성이 강해 타협을 모른다.
	        - 愎 고집할 복, 性 성품 성
