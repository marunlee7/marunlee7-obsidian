# 📙 원길2-303 후후구 后逅垢

- [[后]] 
	- 임금/왕후 후
		- 몸(尸)이나 입(口)으로 지시하는 임금이나 황후니 **임금 후, 왕후 후**
		- 尸 주검 시, 몸 시 변형
		- 원래는 사람의 몸 뒤에 있는 똥구멍의 뜻이었는데, 음을 빌려 임금의 뜻(侯,皇)으로 쓰다가, 요즘에는 侯(과녁 후, 제후 후)를 남자 임금, 后를 왕후(王后)의 뜻으로 씁니다.
	- [[后妃]]
	    - 후비
	        - 임금의 부인. 왕비와 빈(嬪)을 아울러 이르는 말.
	        - 后 임금의 부인 후, 妃 임금의 아내 비
	- [[王后]]
	    - 왕후
	        - 임금(왕)의 정실 부인, 곧 왕비.
	        - 王 임금 왕, 后 임금의 부인 후
	- [[后蜂]]
	    - 후봉
	        - 꿀벌 무리에서 알을 낳는 역할을 하는 벌, 여왕벌.
	        - 后 임금의 부인 후, 蜂 벌 봉
- [[逅]] 
	- 만날 후
		- 만나기 어려운 왕후(后)를 만나듯 가다가(辶) 우연히 만나니 **우연히 만날 후**
		- 辶 뛸 착, 갈 착 = 辵
		- 만날 확률이 적은 만남에 쓰입니다.
	- [[邂逅]]
	    - 해후
	        - 우연히 만남. 예기치 않게 서로 마주침.
	        - 邂 우연히 만날 해, 逅 만날 후
	- [[邂逅相逢]]
	    - 해후상봉
	        - 우연히 서로 만나서 대면함.
	        - 邂 우연히 만날 해, 逅 만날 후, 相 서로 상, 逢 맞이할 봉
- [[垢]] 
	- 때구
		- 흙(土) 같은 때가 임금(候)처럼 좋은 것에 끼어 더럽히니 **때 구, 더럽힐 구**
	- [[垢面]]
	    - 구면
	        - 때 묻은 얼굴, 더러운 얼굴.
	        - 垢 때 구, 面 얼굴 면
	- [[垢衣]]
	    - 구의
	        - 때 묻은 옷, 낡고 더러운 옷.
	        - 垢 때 구, 衣 옷 의
	- [[純眞無垢]]
	    - 순진무구
	        - 아주 순수하고 때가 없음(마음이나 행실에 아무런 티나 욕심이 없음).
	        - 純 순수할 순, 眞 참 진, 無 없을 무, 垢 때 구
