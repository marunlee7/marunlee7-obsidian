# 📙 원길2-147

- 旨 
	- 뜻 지 ← 맛 지
		- 비수(匕)로 햇(日)빛에 익은 과일을 잘라 먹어 보는 맛이니 **맛 지**
		- 말이나 글에 담긴 맛은 뜻이니 **뜻 지**
	- 甘旨
		- 감지
	- 論旨
		- 논지
	- 要旨
		- 요지
	- 趣旨
		- 취지
- [[脂]] 
	- 기름 지
		- 고기에서 맛을 내는 기름이니 **기름 지**
	- [[脂肪]]
		- 지방
	- [[脂肪肝]]
		- 지방간
	- [[脫脂綿]]
		- 탈지면
- 指 
	- 가리킬 지 → 손가락 지 
		- 손( 扌)으로 맛(旨)볼 때 쓰는 손가락이니 **손가락 지**, 손가락으로 무엇을 가리키니 **가리킬 지**
		- ==指揮者==
		- 지휘자
			- 음악이나 군대를 지휘하는 사람.
			- 그는 국립 교향악단에서 가장 뛰어난 **지휘자**로 손꼽힌다.
				- 指 가리킬 지 揮 휘두를 휘 者 놈 자
	- ==指鹿爲馬==
		- 지록위마
			- 윗사람을 농락하여 권세를 마음대로함.
				- 指 가리킬 지 鹿 사슴 록(녹) 爲 할 위 馬 말 마
	- 指壓
		- 지압
	- 指南
		- 지남
	- 指導
		- 지도
	- 指示
		- 지시
- [[詣]] 
	- 이를[至] 예
		- 말(言)로 뜻(旨)을 전하려고 나아가 이르니 **나아갈 예, 이를 예**
	- [[造詣]]
		- 조예
			- 학문(學問)이나 예술(藝術), 기술(技術) 따위의 분야(分野)에 대(對)한 지식(知識)이나 경험(經驗)이 깊은 경지(境地)에 이른 정도(程度).
	- [[精詣]]
		- 정예
			- 정수(精粹)한 학술(學術)에 대(對)한 조예(造詣)가 깊은 정도(程度).
				- 精 정할 정  詣 이를 예
	- [[詣闕]]
		- 예궐
			- 궁궐(宮闕)에 이름
