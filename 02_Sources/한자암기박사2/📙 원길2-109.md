# 📙 원길2-109 노노노나나 奴弩駑拏拿

- [[奴]] 
	- 종 노
		- 여자(女)의 손(又)처럼 힘들게 일하는 종이니 **종 노, 남을 흉하게 부르는 접미사 노**
			- 奴 주로 남자 종
			- 婢 여자 종 비
			- 賣國奴 매국노 처럼 남을 흉하게 부르는 접미사로도 쓰입니다.
	- 奴名
	- 奴婢
	- 奴隷
	- 賣國奴
	- 守錢奴
- [[弩]] 
	- 쇠뇌 노
		- 시키는 일만 하는 종(奴)처럼 조준해 놓은 대로만 쏘아지는 황(弓)이니 **쇠뇌 노**
			- 쇠뇌 - 여러 개의 화살이 잇달아 나가게 만든 활의 한 가지
	- 弩手
	- 弩砲
	- 弓弩
- [[駑]] 
	- 둔한말 노
		- 종(奴)처럼 시키는 일만 하는 둔한 말(馬)이니 **둔한말 노**
	- 駑鈍
	- 駑馬
	- 駑馬十駕
- [[拏]] 
	- 잡을 나
		- 종(奴)처럼 직접 손(手)으로 잡으니 **잡을 나**
		- 拿 잡을[拏同] 나 本資
	- 漢拏山
- [[拿]] 
	- 잡을[拏同] 나
		- 합하여(合) 손(手)으로 잡으니 **잡을 나**
			- 拏 잡을 나의 속자
	- 拿捕
	- 拿來
	- 拿囚
	- 拿引
	- 拿致
