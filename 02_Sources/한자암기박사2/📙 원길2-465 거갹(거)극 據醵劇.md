# 📙 원길2-465 거갹(거)극 據醵劇

- [[據]] 
	- 근거 거 → 의지할 거
		- 손( 扌)으로 범(虍)이나 돼지(豕)를 잡으려고 도구에 의지하니 **의지할 거**
		- `약` : ==拠== - 손( 扌)으로 어느 곳(処)을 잡고 의지하니 **의지할 거**
- [[醵]] 
	- 추렴할 거 | 추렴할 갹
		- 술(酉)도 빚고 범(虍)이니 돼지(豕)도 잡아 잔치하기 위하여 추렴하니 **추렴할 갹, 거**
	- [[出斂]]
	    - 출렴
	        - ‘출(出)’은 ‘나가다, 내다’의 뜻이고, ‘렴(斂)’은 ‘모으다, 거두다’의 뜻이다.
	        - 출렴은 ‘돈이나 물건을 거두어 내다’, 즉 징수하거나 모금함을 의미한다.
	        - 예시: 세금을 출렴하다 (세금을 거두어 들이다).
	        - 出 날 출, 斂 거둘 렴
	- [[醵出]]
	    - 거출
	        - 여러 사람이 함께 돈이나 물건을 내놓아 공동으로 부담하거나 조달하는 것을 뜻한다.
	        - 예시: 행사 비용을 거출하다.
	        - 醵 보태다 거, 出 내다 출
	- [[醵出制年金]]
	    - 거출제연금
	        - 여러 사람이 함께 자금을 모아 일정한 금액을 정기적으로 지급하는 연금 제도이다.
	        - 예시: 회사는 거출제연금 제도를 도입했다.
	        - 醵 보태다 거, 出 내다 출, 制 만들 제, 年 해 년, 金 쇠 금
- [[劇]] 
	- 심할 극 → 연극 극
		- 범(虍)과 돼지(豕)를 잡으려고 칼(刂)로 찌르는 것이 심하니 **심할 극**
		- 심하게 실제와 똑같이 하는 연극이니 **연극 극**
- 물이악소 이위지 물이선소 이불위 
	- 물이악소 이위지 물이선소 이불위
		- 악이 작다고 해서 그것을 하지 말고(아무리 작은 악이라도 하지 말고)
		- 선이 작다고 해서 그것을 아니하지 말라(비록 작은 선일지라도 하라)
