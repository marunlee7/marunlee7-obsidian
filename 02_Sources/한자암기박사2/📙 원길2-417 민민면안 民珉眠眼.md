# 📙 원길2-417 민민면안 民珉眠眼

- [[民]] 
	- 백성 민
		- 모인(冖) 여러 씨(氏)족들로 된 백성이니 **백성 민**
		- 冖 덮을 멱 → 여기서는 모여있는 모습
- [[珉]] 
	- 옥돌 민
		- 구슬(王)처럼 백성(民)들이 좋아하는 옥돌이니 **옥돌 민**(=[[玟]])
- [[眠]] 
	- 잘 면
		- 눈(目) 감고 백성(民)들은 자니 **잘 면**
- [[眼]] 
	- 눈 안
		- 눈(目)동자를 멈추고(艮) 바라보는 눈이니 **눈 안**
			- 艮 멈출 간, 어긋날 간
