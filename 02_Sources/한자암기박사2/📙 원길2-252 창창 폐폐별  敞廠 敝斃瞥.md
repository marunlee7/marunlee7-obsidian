# 📙 원길2-252 창창 폐폐별 敞廠 敝斃瞥

- [[敞]] 
	- 시원할 창
		- 높은(尙) 것을 쳐(攵) 버리면 시원하고 넓으니 **시원할 창, 넓을 창**
		- 尙 오히려 상, 높을 상, 숭상할 상
	- [[敞然]]
	    - 창연
	        - 탁 트여 시원하고 밝은 모양
	        - 敞 시원할 창, 然 그러할 연
	- [[高敞]]
	    - 고창
	        - 지명이름(전라북도 고창군), 또는 높고 탁 트인 곳
	        - 高 높을 고, 敞 시원할 창
	- [[寬敞]]
	    - 관창
	        - 넓고 시원함, 공간이 탁 트여 넓음
	        - 寬 너그러울 관, 敞 시원할 창
	- [[敞開]]
	    - 창개
	        - 활짝 열다, 넓게 개방함
	        - 敞 시원할 창, 開 열 개
	- [[敞亮]]
	    - 창량
	        - 밝고 시원함, 공간이 환하고 넓음
	        - 敞 시원할 창, 亮 밝을 량
	- [[敞屋]]
	    - 창옥
	        - 탁 트인 집, 넓고 시원한 집
	        - 敞 시원할 창, 屋 집 옥
	- [[敞篷]]
	    - 창봉
	        - (자동차 등에서) 지붕이 없는, 오픈카
	        - 敞 시원할 창, 篷 덮개 봉
	- [[敞廳]]
	    - 창정
	        - 넓고 시원한 홀(거실)
	        - 敞 시원할 창, 廳 관청 정
- [[廠]] 
	- 공장 창
		- 집(广)에서 넓게(敞) 터진 헛간이니 **헛간 창**
		- 헛간에 기계를 놓고 물건을 만드는 공장이니 **공장 창**
			- 广 집 엄, 바위집 엄
			- 헛간 - 문짝이 없는 광
	- [[工廠]]
	    - 공창
	        - 물건을 대량으로 만드는 곳
	        - 工 장인 공, 廠 공장 창
	- [[軍工廠]]
		- 군공창
			- 군에서 필요(必要)한 장비(裝備)ㆍ피복 등(等)을 생산(生產)하고 재생(再生)ㆍ가공(加工)ㆍ정비(整備) 등(等)의 작업(作業)을 하는 곳.
			  軍 군사 군 工 장인 공 廠 공장 창
	- [[兵器廠]]
	    - 병기창
	        - 무기와 군수품을 제조·보관하는 곳
	        - 兵 군사 병, 器 그릇 기, 廠 공장 창
	- [[車廠]]
	    - 차창
	        - 자동차나 차량을 만드는 공장
	        - 車 수레 차, 廠 공장 창
	- [[造船廠]]
	    - 조선창
	        - 배를 만드는 공장(조선소)
	        - 造 지을 조, 船 배 선, 廠 공장 창
	- [[修理廠]]
	    - 수리창
	        - 기계나 차량 등을 수리하는 공장
	        - 修 닦을 수, 理 다스릴 리, 廠 공장 창
	- [[飛機廠]]
	    - 비기창
	        - 비행기를 만드는 공장
	        - 飛 날 비, 機 틀 기, 廠 공장 창
	- [[印刷廠]]
	    - 인쇄창
	        - 인쇄물을 제작하는 공장
	        - 印 도장 인, 刷 인쇄할 쇄, 廠 공장 창
	- [[製糖廠]]
	    - 제당창
	        - 설탕을 만드는 공장
	        - 製 만들 제, 糖 사탕 당, 廠 공장 창
- [[敝]] 
	- 해질 폐 → 깨질 패
		- 작은(小) 성(冂)은 조금(小)만 쳐도(攵) 해지고 깨지니 **해질 폐, 깨질 폐**
			- 해지다 - 닳아서 떨어지다
- [[斃]] 
	- 죽을 폐
		- 깨져(敝) 죽으니(死) **죽을 폐**
	- [[斃死]]
	    - 폐사
	        - 주로 짐승이나 어패류가 갑자기 죽음. 급격한 기온 변화나 병균 감염 등으로 소, 오리, 물고기, 조개 따위가 죽는 것을 이름
	        - 사람에게는 쓰지 않고, 동물이나 물고기 등에 한정해 사용하는 표현
	        - 斃 자빠질 폐, 死 죽을 사
	- [[斃死率]]
	    - 폐사율
	        - 전체 개체수 중에서 폐사(斃死)한 개체의 비율
	        - 斃 자빠질 폐, 死 죽을 사, 率 비율 률
- [[瞥]] 
	- 눈깜짝할 별
		- 깨진(敝) 틈으로 슬쩍 보니(目) **슬쩍 볼 별**
	- [[瞥見]]
	    - 별견
	        - 언뜻 봄, 잠깐 스쳐 보는 것
	        - 瞥 언뜻 볼 별, 見 볼 견
	- [[瞥觀]]
	    - 별관
	        - 언뜻 봄, 순간적으로 관찰함
	        - 瞥 언뜻 볼 별, 觀 볼 관
	- [[瞥眼間]]
	    - 별안간
	        - 눈 깜짝할 사이, 아주 짧은 순간
	        - 瞥 언뜻 볼 별, 眼 눈 안, 間 사이 간
	- [[一瞥]]
	    - 일별
	        - 한 번 언뜻 봄
        - 一 한 일, 瞥 언뜻 볼 별
