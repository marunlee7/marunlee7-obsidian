# 📙 원길2-450 우모모뢰견 牛牟牡牢牽

- [[牛]] 
	- 소 우
		- 뿔 있는 소를 본떠서 **소 우**
		- `유` 半 반 반
- [[牟]] 
	- 성(姓)/보리[大麥] 모 → 탐낼 모
		- 사사로이(厶) 소(牛)가 보리를 탐내니 **보리 모, 탐낼 모, 성씨 모**
		- 보리 같은 곡식을 소가 좋아하지요.
	- [[牟麥]]
	    - 무맥
	        - 무라고도 하는 식물의 한 종류, 또는 이와 관련된 약재
	        - 牟 소 무, 麥 보리 맥
	- [[牟利]]
	    - 무리
	        - 이익을 탐함, 이윤 추구
	        - 牟 소 무, 利 이로울 리
	- [[釋迦牟尼]]
	    - 석가모니
	        - 불교의 창시자, 붓다의 존칭
	        - 釋 풀 석, 迦 깔 자, 牟 소 무, 尼 비구니 니
- [[牡]]  
	- 수컷 모
		- 소(牛)가 흙(土)을 힘차게 갈면 수컷이니 **수컷 모**
	- [[牡牛]]
	    - 모우
	        - 수소, 수컷 소를 의미
	        - 牡 수컷 모, 牛 소 우
	- [[牝牛]]
	    - 빈우
	        - 암소, 암컷 소를 의미
	        - 牝 암컷 빈, 牛 소 우
	- [[牡瓦]]
	    - 모와
	        - 수컷 기와, 용의 수컷과 암컷을 구분할 때 수컷을 뜻함
	        - 牡 수컷 모, 瓦 기와 와
	- [[牡丹]]
	    - 모단
	        - 작약과에 속하는 화초, 흔히 ‘모란’이라 부름
	        - 牡 수컷 모, 丹 붉을 단
- [[牢]] 
	- 우리[畜舍] 뢰
		- 집(宀) 중 소(牛)를 굳게 가두는 우리니 **우리 뢰, 굳을 뢰**
		- 우리처럼 가두는 가옥이니 **감옥 뢰**
	- [[亡羊補牢]]
	    - 망양보뢰
	        - 양을 잃은 뒤에 우리를 고친다는 뜻으로, 일이 잘못된 뒤에 뉘우쳐도 늦다는 의미
	        - 亡 잃을 망, 羊 양 양, 補 기울 보, 牢 우리 뢰
	- [[牢却]]
	    - 뇌각
	        - 우리를 없애거나 허물어 버림
	        - 牢 우리 뢰, 却 물리칠 각
	- [[牢獄]]
	    - 뇌옥
	        - 감옥, 죄수를 가두는 곳
	        - 牢 우리 뢰, 獄 옥 옥
- [[牽]] 
	- ==이끌/끌 견==
		- 검은(玄) 고삐로 묶어(冖) 소(牛)를 끄니 **끌 견**
		- 玄 검을 현, 아득할 현
		- 冖 덮을 역 → 여기서는 묶는 모습
		- 부수 : ==牛== [⺧,牜] (소우, 4획)
