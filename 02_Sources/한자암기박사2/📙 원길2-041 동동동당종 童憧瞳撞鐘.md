# 📙 원길2-041 동동동당종 童憧瞳撞鐘

- [[童]] 
	- 아이 동
		- (어른은 일터에 나가고) 서서(立) 마을(里)에서 노는 사람은 주로 아이니 **아이 동**
		- `유` 重 무거울 중, 귀줄할 중, 거듭 중
- [[憧]] 
	- 동경할 동
		- 마음(忄)은 항상 어린(童) 시절을 그리워하니 **그리워할 동**
	- [[憧憬]]
	    - 동경
	        - 마음속으로 바라고 그리워하는 감정을 뜻한다. 희망이나 미래의 어떤 상황을 간절히 원하며 생각하는 상태이다.
	        - 예시: 그는 성공을 간절히 동경했다.
	        - 출전 : 문학, 심리학
	        - 憧 동경할 동, 憬 깨달을 경
	- [[憧憬心]]
	    - 동경심
	        - 동경하는 마음, 즉 어떤 대상을 향한 그리움이나 희망의 감정이다.
	        - 예시: 동경심이 그를 움직였다.
	        - 출전 : 심리학, 문학
	        - 憧 동경할 동, 憬 깨달을 경, 心 마음 심
- [[瞳]] 
	- 눈동자 동
		- 눈(目)에서 아이(童)처럼 작은 눈동자니 **눈동자 동**
			- 目 눈 목, 볼 목, 항목 목
	- [[瞳孔]]
	    - 동공
	        - 눈의 가운데 검은 부분으로 빛이 들어가는 구멍이다.
	        - 예시: 동공이 확대되었다.
	        - 출전 : 의학, 해부학
	        - 瞳 동공 동, 孔 구멍 공
	- [[瞳子]]
	    - 동자
	        - 눈동자를 뜻하는 말이나 아이를 일컫는 용법도 있다.
	        - 예시: 동자가 맑은 눈동자를 가지고 있다.
	        - 출전 : 문학, 일상
	        - 瞳 동공 동, 子 아들 자
- [[撞]] 
	- 칠 당
		- 손( 扌)으로  아이(童)처럼 무엇을 치니 **칠 당**
		- 아이들은 손을 가만 놓아두지 않고 무엇을 하거나 침을 생각하고 만든 글자.
	- [[撞球]]
	    - 당구
	        - 큐를 이용해 공을 치는 구기 스포츠이다.
	        - 예시: 그는 당구를 즐긴다.
	        - 출전 : 스포츠
	        - 撞 칠 당, 球 공 구
	- [[撞棒]]
	    - 당봉
	        - 당구 큐대를 뜻한다.
	        - 예시: 당봉을 새로 샀다.
	        - 출전 : 스포츠 장비
	        - 撞 칠 당, 棒 몽둥이 봉
	- [[自家撞着]]
	    - 자가당착
	        - 자기 자신의 말이나 행동이 앞뒤가 맞지 않는 모순 상태를 뜻한다.
	        - 예시: 그의 주장은 자가당착에 빠졌다.
	        - 출전 : 철학, 논리학
	        - 自 스스로 자, 家 집 가, 撞 칠 당, 着 붙을 착
- [[鐘]] 
	- 쇠북 종
		- 쇠(金) 소리가 아이(童) 소리처럼 맑은 쇠북이니 **쇠북 종**
		- 쇠북처럼 종치는 시계니 **종치는 시계 종**
			- `유` 鍾 쇠북 종, 술잔 종
