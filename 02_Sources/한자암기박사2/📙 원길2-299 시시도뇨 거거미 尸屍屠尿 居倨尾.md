# 📙 원길2-299 시시도뇨 거거미 尸屍屠尿 居倨尾

- [[尸]] 
	- 주검 시
		- 사람이 누워 있는 몸을 본떠서 **주검 시, 몸 시**
		- ==사람이나 집과 관련된 글자에 부수로==도 쓰입니다.
		- `유` [[戶]] 문 호, 집 호
- [[屍]] 
	- 주검 시
		- 몸(尸)이 죽은(死) 주검이니 **주검 시**
	- [[屍身]]
	    - 시신
	        - 죽은 사람의 몸
	        - 屍體(시체)
	        - 屍 주검 시, 身 몸 신
	- [[屍體]]
	    - 시체
	        - 사람이나 동물이 죽은 몸.
	        - 屍 주검 시, 體 몸 체
	- [[剖棺斬屍]]
	    - 부관참시
	        - 무덤을 파서 관을 꺼내 시체를 다시 참수하는 극형 또는 그 형벌.
	        - 죽은 뒤에도 죄인을 벌함.
	        - 剖 가를 부, 棺 널 관, 斬 벨 참, 屍 주검 시
- [[屠]] 
	- 죽일 도
		- (짐승을) 주검(尸)처럼 사람(者)이 죽여 잡으니 **죽일 도, 잡을 도**
	- [[屠戮]]
	    - 도륙
	        - 여러 사람을 무참히 죽임, 살육.
	        - 屠 무리 죽일 도, 戮 죽일 륙
	- [[屠殺]]
	    - 도살
	        - 동물을 잡아 죽이거나, 여러 사람을 무참히 죽임.
	        - 屠 무리 죽일 도, 殺 죽일 살
	- [[浮屠]]
	    - 부도
	        - 탑 또는 부처를 이르는 말, 불탑.
	        - 屠에 가까운 소리이나 뜻은 다름.
	        - 浮 뜰 부, 屠 무리 죽일 도
- [[尿]] 
	- 오줌 뇨
		- 주검(尸)으로 소화되어 나오는 물(水)이 오줌이니 **오줌 뇨**
	- [[糖尿]]
	    - 당뇨
	        - 소변에 당분이 섞여 나오는 병, 당뇨병.
	        - 糖 사탕 당, 尿 오줌 뇨
	- [[放尿]]
	    - 방뇨
	        - 소변을 봄.
	        - 放 놓을 방, 尿 오줌 뇨
	- [[糞尿]]
	    - 분뇨
	        - 똥과 오줌.
	        - 糞 똥 분, 尿 오줌 뇨
	- [[泌尿器]]
	    - 비뇨기
	        - 소변을 생성·배설하는 기관(신장, 요관, 방광, 요도 등 포함).
	        - 泌 스밀 비, 尿 오줌 뇨, 器 그릇 기
- [[居]] 
	- 살 거
		- 몸(尸)이 오래(古) 머물러 사니 **살 거**
		- 몸(尸)이 오래(古) 머무르려고 앉으니 **앉을 거**
- [[倨]] 
	- 거만할 거
		- 사람(人)이 앉아서(居) 맞이하면 거만하니 **거만할 거**
		- 일어서거나 나가지 않고 앉아서 대함은 거만한 것이지요.
	- [[倨慢]]
	    - 거만
	        - 남을 업신여기고 뽐냄.
	        - 倨 거만할 거, 慢 게으를 만
		- ↔ [[謙遜]]
		    - 겸손
		        - 남을 존중하며 자기를 낮춤.
		        - 謙 겸손할 겸, 遜 겸손할 손
	- [[倨侮]]
	    - 거모
	        - 거만하고 업신여김.
	        - 倨 거만할 거, 侮 모욕할 모
	- [[倨傲]]
	    - 거오
	        - 거만하고 오만함.
	        - 倨 거만할 거, 傲 거만할 오
- [[尾]] 
	- 꼬리 미
		- 짐승의 몸(尸)에서 털(毛)이 난 꼬리니 **꼬리 미**
		- 꼬리처럼 무엇의 끝이니 **끝 미**
