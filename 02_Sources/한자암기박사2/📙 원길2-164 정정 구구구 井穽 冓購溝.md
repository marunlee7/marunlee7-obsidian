# 📙 원길2-164 정정 구구구 井穽 冓購溝

- [[井]] 
	- 우물 정
		- 나무로 엇갈리게 쌓아 만든 우물의 우물틀이니 **우물 정, 우물틀 정**
		- 옛날에는 우물을 파고 흙이 메워지지 않도록 통나무를 井자 모습으로 짜서 쌓아 올렸지요. 
	- 井華水(정화수)
	- 油井(유정)
	- 坐井觀天(좌정관천)
- [[穽]] 
	- 함정 정
		- 구멍(穴)을 우물(井)처럼 깊게 파놓은 함정이니 **함정 정**
	- [[30-리소스/Dict/Hword2/한자암기박사/陷穽]](함정) - 짐승을 잡기 위하여 파놓은 구멍이, 빠져나올 수 없는 곤경이나 계략을 비유하여 이르는 말
- [[冓]] 
	- 쌓을 구, 짤 구
		- 우물틀(井)처럼 거듭(再) 쌓으면 짜니 **쌓을 구, 짤 구**
		- 再 다시 재, 두번 재
- [[購]] 
	- 살 구
		- 돈(貝)을 쌓아(冓) 모아 물건을 사니 **살 구**
		- 貝 조개 패, 재물 패, 돈 패
	- [[購讀]](구독)
	- [[購買]](구매)
	- [[購入]](구입)
	- [[購販場]](구판장)
- [[溝]] 
	- 도랑 구 → 개울 구
		- 물(氵)이 쌓여(冓) 흐흐는 개울이니 **개울 구**
		- 개울 - 골짜기나 들에 흐르는 작은 물줄기.
	- [[溝瀆]](구독) : 개천과 수렁.
	- [[溝池]](구지)
	- [[溝壑]](구학)
	- [[怨溝]](원구)
	- [[畝溝]]
		- 묘구
			- 고랑
			- 두둑한 땅과 땅 사이에 깊고 좁게 들어간 곳을 이랑에 상대하여 이르는 말
			- 畝 이랑 묘 溝 도랑 구
