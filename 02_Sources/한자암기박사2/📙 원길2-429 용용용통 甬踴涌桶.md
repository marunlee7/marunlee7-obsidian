# 📙 원길2-429 용용용통 甬踴涌桶

- [[甬]] 
	- 종꼭지 용 → 솟을 용
		- 꽃봉오리가 부풀어 오르는 모양을 본떠서 **솟을 용**
- [[踴]] 
	- 뛸 용
		- 발(足)을 솟게(甬) 뛰니 **뛸 용**
	- [[踊躍]]
	    - 용약
	        - 뛰어오르고 뛰놀며 큰 소리로 기뻐함
	        - 예시: 사람들은 축제에서 옹약하며 즐겼다.
	        - 踊 종꼭지 용 → 솟을 용, 躍 뛸 약
	- [[舞踊]]
	    - 무용
	        - 음악에 맞추어 몸을 움직이며 춤추는 일
	        - 예시: 무용 공연이 아름다웠다.
	        - 舞 춤 출 무, 踊 종꼭지 용 → 솟을 용
- [[涌]] 
	- 물 솟을 용
		- 물(氵)이 솟아(甬) 오르니 **샘솟을 용, 끓어오를 용**
		- [[湧]] 물솟을 용의  원자 
	- [[涌起]]
	    - 용기
	        - 물 등이 솟아오르거나 어떤 기운이 일어남
	        - 예시: 분노가 가슴 속에서 용기쳤다.
	        - 涌 솟을 용, 起 일어날 기
	- [[涌沫]]
	    - 용말
	        - 물이 넘쳐 일어나는 거품
	        - 예시: 파도의 용말이 하늘을 향해 솟았다.
	        - 涌 솟을 용, 沫 거품 말
	- [[湧溢]]
	    - 용일
	        - 물이 흘러 넘침
	        - 예시: 샘물이 넘쳐 용일했다.
	        - 湧 솟을 용, 溢 넘칠 일
- [[桶]] 
	- 통(桶) 통
		- 나무(木) 둘레가 솟게(甬) 가운데를 파서 만든 통이니 **통 통**
	- [[洋鐵桶]]
	    - 양철통
	        - 얇은 철판으로 만든 통
	        - 예시: 양철통에 물을 담았다.
	        - 洋 바다 양, 鐵 쇠 철, 桶 통 통
	- [[休紙桶]]
	    - 휴지통
	        - 종이를 버리는 통, 쓰레기통
	        - 예시: 휴지통에 쓰레기를 버렸다.
	        - 休 쉴 휴, 紙 종이 지, 桶 통 통
