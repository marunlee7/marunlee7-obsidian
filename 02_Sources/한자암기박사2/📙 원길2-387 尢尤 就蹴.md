# 📙 원길2-387 왕우 취축 尢尤 就蹴

- [[尢]] 
	- 굽을 왕, 절름발이 왕
		- [양팔 벌리고(一) 다린 벌린 사람(人)을 본떠서 만든] 큰 대(大)의 한 획을 구부려 절름발이를 나타내어 **굽을 왕, 절름발이 왕**
- [[尤]] 
	- 더욱 우 → 허물 우
		- 절름발이(尢)에 점(丶)까지 있어 더욱 허물이니 **더욱 우, 허물 우**
- [[就]] 
	- 나아갈 취
		- (벼슬자리가 많은) 서울(京)로 더욱(尤) 나아가 꿈을 이루니 **나아갈 취, 이룰 취**
- [[蹴]] 
	- 찰 축
		- 발(足)을 앞으로 나아가게(就) 뻗어 차니 **찰 축**
	- [[蹴球]]
	    - 축구
	        - 발로 공을 차서 경기하는 스포츠, 축구.
	        - 예시: 그는 학교에서 蹴球를 즐긴다.
	        - 蹴 찰 축, 球 공 구
	- [[蹴踏]]
	    - 축답
	        - 발로 밟음, 짓밟음.
	        - 예시: 군중이 몰려 蹴踏 사고가 일어났다.
	        - 蹴 찰 축, 踏 밟을 답
	- [[蹴殺]]
	    - 축살
	        - 발로 차서 죽임.
	        - 예시: 신화에서 괴물을 蹴殺했다는 전설이 전해진다.
	        - 蹴 찰 축, 殺 죽일 살
	- [[一蹴]]
	    - 일축
	        - 한 번의 발길질, 단호히 거절하거나 물리침.
	        - 예시: 그는 제안을 一蹴했다.
	        - 一 한 일, 蹴 찰 축
 :: ==글자와 글자의 훈음과 단어 선정==   
	- 한자에는 같은 뜻으로 여러 형태의 글자가 있는 경우도 있고,
	- 한 글자에 여러 훈(뜻)과 음이 있는경우도 있습니다.
	- 이런 경우에는 일상생활에서 많이 쓰이는 글자와 훈과 음을 선택하였고,
	- 단어도 활용빈도가 높은 것 위주로 선정하여 책에 실었습니다.
	- 간혹 급수시험을 주관하는 단체에 따라서 훈과 음이 다른 경우도 있는데,
	- 당연히 일상생활에서 많이 쓰이는 것 위주로 수정해야 하고,
	- 시험에 나온다면 이것도 정답으로 인정해야 합니다.
