# 📙 원길2-339 시의애 후후후 矢矣埃 侯喉候

- [[矢]] 
	- 화살 시
		- 화살을 본떠서 **화살 시**
- [[矣]]  
	- 어조사 의
		- 내(厶)가 쏜 화살(矢)이 목표에 다다랐다는 데서, 문장의 끝에 쓰여 완료를 나타내는 어조사니 **어조사 의
- [[埃]] 
	- 티끌 애
		- 흙(土)으로 사사롭게(厶) 화살(矢)처럼 떨어지는 티끌이니 **티끌 애**
		- 공중으로 올라간 화살이 땅에 떨어지듯이 그렇게 떨어지는 것을 티끌이라 했네요.
	- [[埃滅]]
	    - 애멸
	        - 티끌 애(埃), 멸할 멸(滅)로, 티끌이나 먼지를 없애거나 제거하는 뜻임.
	        - 일상적으로는 아주 작은 먼지를 말하거나 그것을 없애는 의미로 쓰임.
	        - 예시: 방 안의埃滅에 신경 썼다.
	        - 埃 티끌 애, 滅 멸할 멸
	- [[埃塵]]
	    - 애진
	        - 티끌 애(埃), 티끌 진(塵)으로, 먼지나 티끌, 흙먼지를 통틀어 일컫는 말임.
	        - 흔히 매우 작은 먼지를 의미하며, 일상에서 먼지나 오염된 작은 알갱이를 가리킬 때 사용됨.
	        - 예시: 埃塵이 공기 중에 떠다닌다.
	        - 埃 티끌 애, 塵 티끌 진
	- [[塵埃]]
	    - 진애
	        - 티끌 진(塵), 티끌 애(埃)로, 먼지나 아주 작은 티끌을 뜻함.
	        - 埃塵과 같은 의미로 쓰이며, 먼지 혹은 티끌의 총칭임.
	        - 예시: 塵埃가 쌓여 방을 청소했다.
	        - 塵 티끌 진, 埃 티끌 애
- [[侯]]  
	- ==제후 후==
		- 사람(亻)이 만들어(工) 화살(矢)을 쏘는 과녁이니 **과녁 후**
		- 과녁을 잘 맞히는 사람이 되었던 제후니 **제후 후**
			- 諸侯 제후 - 봉건시대에 제왕으로부터 일정한 領土를 받아 영내의 백성을 다스리는 영주.
				- 諸 모든 제, 여러 제 領 거느닐 령, 우두머리 령
- [[喉]] 
	- 목구멍 후
		- 입(口)안에 과녁(侯)처럼 둥근 목구멍이니 **목구멍 후**
		- 과녁의 둥근 원처럼 생긴 목구멍을 생각하고 만든 글자.
	- [[喉頭]]
	    - 후두
	        - 목구멍 후(喉), 머리 두(頭)로, 인체에서 목의 윗부분, 숨과 음식물의 통로이자 소리를 내는 기관인 후두를 뜻함.
	        - 호흡기와 소리 생성에 중요한 역할을 하며, 성대가 위치함.
	        - 예시: 후두염으로 목소리가 변했다.
	        - 喉 목구멍 후, 頭 머리 두
	- [[咽喉]]
	    - 인후
	        - 삼킬 인(咽), 목구멍 후(喉)로, 목구멍 부분 전체를 가리키며 음식물과 공기가 지나가는 인체 부위를 뜻함.
	        - 일반적으로 후두를 포함하는 더 넓은 개념이며, 삼키는 기능까지 포함함.
	        - 동양 의학에서는 喉와 咽을 구분하여 喉는 숨 쉬는 곳, 咽은 음식물이 지나가는 곳으로 설명하기도 함.
	        - 예시: 인후염으로 목이 아프다.
	        - 咽 삼킬 인, 喉 목구멍 후
	- [[耳鼻咽喉科]]
	    - 이비인후과
	        - 귀 이(耳), 코 비(鼻), 인후 인(咽), 목구멍 후(喉), 과(科)는 과목이나 과(과)로, 귀, 코, 목에 관련된 질병을 진료하는 의학 분야임.
	        - 흔히 '이비인후과'라고 부르며, 감기나 중이염, 비염, 후두염 등을 치료함.
	        - 예시: 목이 아파 이비인후과를 방문했다.
	        - 耳 귀 이, 鼻 코 비, 咽 삼킬 인, 喉 목구멍 후, 科 과목 과
- [[候]] 
	- 기후 후 → 염탐할 후
		- 바람에 날릴까봐 과녁(侯)에 화살(丨)을 쏠 때는 기후를 염탐하니 **기후 후, 염탐할 후**
