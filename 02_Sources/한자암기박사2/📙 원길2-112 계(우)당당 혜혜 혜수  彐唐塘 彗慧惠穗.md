# 📙 원길2-112 계(우)당당 혜혜 혜수 彐唐塘 彗慧惠穗

- [[彐]] 
	- 고슴도치 머리 계
		- 고슴도치 머리 모양을 본떠서 **고슴도치 머리 계**
		- 오른손의 손가락을 편 모양을 본떠서 **오른손 우**
- [[唐]] 
	- 당나라/당황할 당  → 갑자기 당
		- 집(广)에서라도 손(彐)에 회초리(丨) 들고 입(口)으로 갑자기 소리치면 황당하니 **갑자기 당, 황당할 당**
		- 갑자기 세력을 떨쳤던 당나라니 **당나라 당**
		- 갑자기 : 미처 생각할 겨를도 없이 급히
		- 丨뚫을 곤 → 회초리로 봄
- [[塘]] 
	- 못[池] 당
		- 흙(土)으로 갑자기(唐) 막혀 물이 고인 연못이니 **연못 당**
	- 池塘
		- 지당
			- 못.
			- 넓고 오목하게 팬 땅에 물이 괴어 있는 곳.
				- 池 못 지 塘 못 당
	- 春塘臺
		- 춘당대
			- 창경궁(昌慶宮) 안에 있는 대(臺).
				- 春 봄 춘 塘 못 당 臺 대 대
- [[彗]] 
	- 살별 혜 → 비/혜성 혜
		- 풀 무성한 가지 두 개(丰丰)를 손(彐)으로 묶어 만든 비니 **비 혜**
		- 빗자루 모양으로 꼬리를 끌며 날아가는 혜성이니 **혜성 혜**
			- 彗星 혜성
				- ㉠ 빛나는 긴 꼬리를 끌고 태양을 도는 별. 꼬리별.
				- ㉡ (밤하늘에 혜성이 갑자기 나타나듯) 어떤 분야에서 갑자기 뛰어나 드러남을 비유한 말.
			- 丰 풀 무성할 봉, 예쁠 봉, 풍성할 풍
			- 살별 = 혜성 = 꼬리별
- [[慧]] 
	- 슬기로울 혜 → 밝을/지혜 혜
		- 잡념을 비(彗)로 쓸어버린 마음(心)처럼 밝은 지혜니 **밝을 혜, 지혜 헤**
		- 智慧(지혜) : 슬기 
- [[惠]] 
	- 은혜 혜
		- 언행을 삼가고(叀) 어진 마음(心)을 베푸는 은혜니 **은혜 혜**
			- 叀 - 車에 점(丶)찍는 일은 삼가니 **삼갈 전** - 실제 쓰이는 글자는 아닙니다.
			- 恩惠(은혜) : 자연(自然)이나 남에게서 받는 고마운 혜택(惠澤).
- [[穗]] 
	- 이삭 수
		- 벼(禾)에서 은혜로운(惠) 이삭이니 **이삭 수**
			- 이삭 ㉠ 여기서는 ㉠의 뜻
				- 곡식에서 열매가 많이 달린 부분
				- 곡식을 거둘 때 흘린 낟알
	- [[落穗]]
		- 낙수
			- 추수한 후 땅에 떨어져 있는 이삭
			- 어떤 일의 뒷이야기를 비유
				- 落 떨어질 락(낙) 穗 이삭 수
	- [[穗肥]]
		- 수비
			- 벼ㆍ보리 따위의 이삭이 줄기 속에서 자라나기 시작(始作)할 무렵에 효과(效果)를 보기 위(爲)하여 주는 추비(追肥).
				- 穗 이삭 수 肥 살찔 비
			- 追肥
				- 추비
					- 농작물(農作物)에 첫 번 거름을 준 다음, 파종(播種)ㆍ이식(移植)한 뒤에, 다시 주는 거름.
						- 追 쫓을 추 肥 살찔 비
	- [[穗狀]]
		- 수상
			- 이삭과 같은 모양(模樣).
				- 穗 이삭 수 狀 형상 상
	- [[揷穗]]
		- 삽수
			- 꺾꽂이를 하려고 일정(一定)한 길이로 잘라 낸 식물체(植物體)의 순.
				- 插 꽂을 삽 穗 이삭 수
