# 📙 원길2-223 차자자 사시 此雌疵 些柴

- [[此]] 
	- 이 차
		- 멈추어(止) 비수(匕)로도 찌를 만한 가까운 이것이니 **이 차**
		- 止 그칠 지
		- 匕 비수 비
		- 비수 - 짧고 날카로운 칼
- [[雌]] 
	- 암컷 자
		- 수컷 옆에 그쳐(止) 비수(匕) 같은 부리로 먹이를 먹는 새(隹)는 암컷이니 **암컷 자**
	- [[雌犬]]
	    - 자견(암개)
	        - 암컷 개. 즉, 암컷인 개를 의미한다. 일본어로는 '메스이누(めすいぬ)'라 하며, 영어로는 female dog 또는 bitch로 번역된다
	        - 예시 : 우리 집 강아지는 아직 어린 자견이다.
	        - 雌 암컷 자, 犬 개 견
	- [[雌雄]]
	    - 자웅
	        - 암컷과 수컷. 동물이나 식물에서 암수의 구별을 이르는 말.
	        - 예시 : 이 종은 자웅이 뚜렷하게 구분된다.
	        - 雌 암컷 자, 雄 수컷 웅
	- [[雌性]]
	    - 자성
	        - 암컷의 성. 생물학적으로 암컷의 성질이나 특징을 뜻한다.
	        - 예시 : 자성 개체만이 알을 낳는다.
	        - 雌 암컷 자, 性 성품 성
- [[疵]] 
	- 허물 자
		- 병(疒)든 이(此) 부분이 흠이고 허물이니 **흠자, 허물 자**
		- `유` [[庇]] - 안전하도록 집(广)안에 나란히(比) 숨겨 덮으니 **덮을 비**
		- 疒 병들 녁
		- 흠 ① 어떤 물건의 이지러지거나 깨어지거나 상한 자국 ② 어떤 사물의 모자라거나 잘못된 부분. 
	- [[疵痕]]
	    - 차흔
	        - 흠집과 흔적. 물건이나 피부 등에 남은 흠이나 상처의 자국을 이른다.
	        - 예시 : 도자기 표면에 작은 차흔이 남아 있다.
	        - 疵 흠 차, 痕 흔적 흔
	- [[隱疵]]
	    - 은차
	        - 겉으로 드러나지 않는 흠. 쉽게 발견하기 어려운 결점이나 하자를 뜻한다.
	        - 예시 : 중고차 거래 시 은차가 있는지 꼼꼼히 확인해야 한다.
	        - 隱 숨을 은, 疵 흠 차
	- [[瑕疵]]
	    - 하자
	        - 물건이나 일에 있는 흠이나 결점. 법률적으로는 계약 대상물에 있는 결함을 뜻하기도 한다.
	        - 예시 : 하자가 있는 상품은 교환 또는 환불이 가능하다.
	        - 瑕 흠 하, 疵 흠 차
	        - [[瑕疵]]
- [[些]] 
	- 적을 사
		- 단지 이(此) 것 두(二) 개뿐이라 적으니 **적을 사**
	- [[些略]]
	    - 사략
	        - 아주 적거나 대수롭지 않음. 사소하고 중요하지 않은 일이나 내용.
	        - 예시 : 사략한 부분은 생략해도 무방하다.
	        - 些 조금 사, 略 간략할 략
	- [[些末]]
	    - 사말
	        - 아주 사소한 것. 중요하지 않고 하찮은 일이나 부분.
	        - 예시 : 사말에 신경 쓰지 말고 중요한 일에 집중해라.
	        - 些 조금 사, 末 끝 말
	- [[些事]]
	    - 사사
	        - 사소한 일. 대수롭지 않거나 별로 중요하지 않은 일.
	        - 예시 : 그런 사사에 너무 신경 쓰지 마라.
	        - 些 조금 사, 事 일 사
	- [[些少]]
	    - 사소
	        - 양이 아주 적음. 또는 아주 적은 분량.
	        - 예시 : 사소한 양의 음식만 남았다.
	        - 些 조금 사, 少 적을 소
	        - [[些少]]
- [[柴]] 
	- 섶[薪] 시
		- 그쳐(止) 비수(匕) 같은 낫으로 자른 땔나무(木)니 **땔나무 시, 섶 시**
		- 섶 - 잎나무, 풋나무, 물거리 따위의 땔나무를 통틀어 이르는 말.
	- [[柴奴]]
	    - 시노
	        - 땔나무를 하는 머슴. 땔나무를 마련하거나 운반하는 일을 맡은 하인 또는 머슴을 이른다.
	        - 柴 섶 시, 奴 종 노
	- [[柴木]]
	    - 시목
	        - 땔나무. 불을 때는 데에 쓰는 나무로, 주로 연료나 난방용으로 사용된다.
	        - 柴 섶 시, 木 나무 목
	- [[柴糧]]
	    - 시량
	        - 땔나무와 식량. 생활에 꼭 필요한 기본 자원인 땔감과 양식을 아울러 이르는 말.
	        - 柴 섶 시, 糧 양식 량
	        - [[柴糧]]
	- [[柴扉]]
	    - 시비
	        - 땔나무로 만든 문. 주로 시골집이나 초가집 등에 쓰던 소박하고 투박한 문을 뜻한다.
	        - 柴 섶 시, 扉 문 비
	        - [[柴扉]]
