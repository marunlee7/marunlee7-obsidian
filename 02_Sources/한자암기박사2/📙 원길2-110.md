# 📙 원길2-110 피피 파파(피)파 皮披 坡跛婆

- [[皮]] 
	- 가죽 피
		- 가죽(厂)을 칼(丨)들고 손(又)으로 벗기는 모양에서 **가죽 피, 성씨 피**
			- 厂 굴바위 엄, 언덕 엄 → 가죽으로 봄
	- 皮骨
		- 피골
			- 살가죽과 뼈.
			- 그는 너무 말라서 皮( )이 상접한 것 같았다.
				- 皮 가죽 피 骨 뼈 골
	- 皮膚
		- 피부
			- 皮 가죽 피 膚 살갗 부
	- 皮革
		- 피혁
	- 毛皮
		- 모피
	- 鐵面皮
		- 철면피
- [[披]] 
	- 헤칠 피
		- 손( 扌)으로 가죽(皮)을 뒤집어 헤치니 **헤칠 피**
	- 披見
		- 피견
			- (편지(便紙)ㆍ책 따위를)해쳐 봄. 열어 봄.
				- 披 헤칠 피 見 볼 견
	- 披瀝 
		- 피력
			- 평소(平素)에 숨겨둔 생각을 모조리 털어내어 말함.
				- 披 헤칠 피 瀝 스밀 력
	- 披露宴
		- 피로연
			- 파헤쳐 드러내는 잔치
			- 들춰서 널리 펴고 드러내어 알리는 잔치
			- 떳떳하게 결혼한 부부임을 세상에 알림으로써 서로가 서로를 책임지는 부부가 되는 약속을 굳건히 하기 위해서이다.
			- ref [김병기 전북대 중문과 교수](https://blog.naver.com/shin0641/223221111721)
			- 結婚ㆍ出生 等과 같은 기쁜 일을 널리 알리기 爲하여 베푸는 잔치
				- 披 헤칠 피 露 이슬/**드러낼** 로 宴 잔치 연
	- 猖披 
		- 창피
			- 체면(體面)이 깎일 일을 당(當)하여 부끄러움.
				- 猖 미쳐 날뛸 창 披 헤칠 피.
- [[坡]]   
	- 언덕 파
		- 흙(土)을 가죽(皮)처럼 단단히 쌓아 만든 언덕이니 **언덕 파**
	- 坡州
		- 파주
	- 洪蘭坡
		- 홍난파
			- 우리나라 근대 음악의 선구자이며 작곡자
- [[跛]] 
	- 절름발이 파 | 비스듬히설 피
		- 한 발(足)이 가죽(皮)만 있는 듯 힘을 못 쓰는 절름발이니 **절름발이 파**
		- 절름발이처럼 비스듬히 서니 **비스듬히 설 피**
			- 절름발이 - 한쪽 발이 없거나 역할을 못하여 양쪽의 균형이 맞지 않은 경우
	- 跛行
		- 파행
	- 跛行的
		- 파행적
	- 跛立
		- 피립
