# 하루 종일 맥북과 씨름하다보면 하루가 얼마나 짧은지 모른다

하루 종일 맥북과 씨름하다보면 하루가 얼마나 짧은지 모른다. 
그러다가 시간 부족에 조급해지고
늘 그렇지만 시간은 유한하고 누구에게나 공평한데
내게 주어진 시간이 왜 이러 짧은지는
내 집중과 몰입에 달려있는 것이다. 
좀 더 생산적인 작업과
15분, 단 15분이라도 책을 읽을 수 있는 **마음청소** 시작하자.
