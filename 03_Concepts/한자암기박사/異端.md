# 異端(이단)

정통(正統)이나 일반적인 사상, 종교, 학설에서 벗어난 주장이나 교파. 
기존의 권위나 교리와 다른 새로운 사상이나 신앙을 의미함. 
異 다를 이 : 다르다, 서로 다르다, 이상하다
端 끝 단 : 끝, 시작, 단서, 근원
직역 : 다른 끝, 다른 근원

유사어 : 사교([[邪敎]]), 변종([[變種]])
반의어: 정통([[正統]])
