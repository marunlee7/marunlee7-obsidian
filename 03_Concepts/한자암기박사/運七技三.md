# 運七技三(운칠기삼)

|     |     |
| --- | --- |
|     |     |

일의 성패는 70%는 운(運)에, 30%는 기술(技)이나 노력에 달려 있다"는 의미의 사자성어입니다.
즉, 인생이나 사업, 시험, 투자 등에서 아무리 실력과 노력이 있어도 
결국 운이 더 큰 비중을 차지한다는 현실을 표현한 말

## 유래와 일화

이 말의 유래로는 청나라 포송령의 『요재지이』에 관련 이야기가 등장하며, 
고전적으로는 과거시험에 낙방한 선비가 "운칠기삼"이라는 말을 듣게 되는 설화가 전해집니다.

## 현대적 의미

오늘날에는 "노력만으로 모든 것이 결정되지 않는다"는 현실 인식을 담아, 
==겸손과 위로, 또는 냉철한 자기성찰==의 의미로 널리 쓰이고 있습니다
