# 合倂症(합병증)

어떤 질병에 걸렸을 때 그와 관련하여 추가로 나타나는 다른 질병이나 증상. 주로 의학에서 사용되며, 원래의 질환에 덧붙어 생기는 부수적인 병을 의미함. 合(합할 합) 倂(아울러 병) 症(증세 증).
