# 踏破(답파)

걸어서 다 돌아다님. 길이나 산을 밟아 지나감, 정복함. 踏 밟을 답, 破 깨뜨릴 파
험한 길이나 먼길을 끝까지 걸어 나감.
너른 지역(地域)을 종횡(縱橫)으로 두루 걸어서 돌아다님.

- **한자 풀이**
    - 踏(밟을 답): 밟다, 디디다
    - 破(깨뜨릴 파): 깨뜨리다, 허물다
- **뜻풀이**  
    “답파(踏破)”는 ‘밟아서 깨뜨리다’, ‘밟으며 지나가다’, ‘어떤 곳을 샅샅이 다니다’라는 뜻의 한자어입니다.  
    주로 산이나 들, 혹은 어떤 지역을 직접 발로 밟으며 두루 돌아다니는 행위를 가리킵니다.
- **용례**
    - “그는 전국의 산천을 답파하며 풍광을 감상했다.”
    - “답파하다”라는 동사로도 쓰여, ‘현장을 직접 밟아 조사하다’의 의미로 사용됩니다.
- **비슷한 말**
    - 답사([[踏査]]): 직접 밟아 조사함
    - 주유([[周遊]]): 두루 돌아다님
