# 語彙(어휘)

어떤 일정한 범위 안에서 쓰이는 단어의 수효. 또는 단어의 전체.
어떤 종류의 말을 간단한 설명을 붙여 순서대로 모아 적어 놓은 글.
語 말씀 어 [[彙]] 무리 휘
[[辭彙]](사휘) : 어떤 종류(種類)의 말을 간단(簡單)한 설명(說明)을 붙여 순서(順序)대로 모아 적어 놓은 글.

- 語彙의構造 
	- 무한한 어휘의 느슨한 집합이 지니는 유동적이고 가변적인 구조. 
	- 의미 변화를 촉진하는 요인이다. 
	- 특정 언어의 음운 체계나 문법 체계는 수가 한정된 요소로 구성되지만, 어휘는 무한하기 때문에 개별 단어는 쉽게 생성되거나 다른 뜻을 지니게 되기 때문이다.
