# 花崗石(화강석)

화강암(花崗岩)이라고도 하며, 주로 석영, 장석, 운모 등으로 이루어진 조립질(顆粒狀) 심성 화성암. 색상과 무늬가 다양하며, 질감이 단단하고 내구성이 뛰어나 건축 및 조각, 외장재 등에 널리 쓰임. 花(꽃 화) 崗(언덕 강) 石(돌 석).
