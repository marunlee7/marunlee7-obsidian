# 憶念(억념)

마음속에 단단히 기억하여 잊지 아니함. 또는 그런 기억.
憶 생각할 억 念 생각 념(염)

憶念輪(억념륜 )
삼륜(三輪)의 하나. 부처가 설법(說法)하기 전에 먼저 청중(聽衆)의 영리(怜悧ㆍ伶俐)하고 둔함을 살피는 일을 이른다.

憶念彌陀佛(억념미타불 )
아미타불(阿彌陀佛)을 마음속으로 생각해서 잊지 아니함.
