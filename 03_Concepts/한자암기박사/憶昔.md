# 憶昔(억석)

지난 시절을 떠올림. 예전에 있었던 일이나 과거를 회상하거나 그리워하는 마음을 나타내는 말. 주로 시문(詩文) 등에서 옛 추억이나 지난날을 회상할 때 사용됨. 憶(생각할 억) 昔(예 석).
