# 或曰(혹왈)

어떤 사람이 말했다, 또는 누군가 말했다는 뜻. 의견이나 이야기를 간접적으로 전할 때 사용.
예시) 혹왈(或曰), "그의 말이 옳다."
或 혹, 曰 가로 왈
