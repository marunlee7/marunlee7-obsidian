# 記名捺印(기명날인)

- 자기 이름을 쓰고 도장을 찍는 것.
- 다른 사람이 이름을 대신 써줘도 무방하나, 도장은 반드시 날인해야 함.
- 예시: 계약서에는 기명날인이 필요하다.
- 記 기록할 기, 名 이름 명, 捺 누를 날, 印 도장 인
