# 泰然自若

어떤 상황에서도 침착하고 평온함. 
어떤 곤란하거나 위급한 상황에서도 침착하고 태연하며 평온함을 이르는 말.
예문: "그는 위기 속에서도 태연자약하게 대처했다."
[[泰然]](태연) : 매우 편안하고 침착함을,  → 모두 편안함과 침착함을 나타내며,
[[自若]](자약) : 스스로 변함없이 한결같음을 뜻합니다.
泰 클 태 然 그러할 연 自 스스로 자 若 같을 약
