# 口蜜腹劍(구밀복검)

입에는 꿀이 있고 배 속에는 칼이 있다는 뜻. 
말은 친근하고 달콤하지만 속으로는 해칠 생각이 있음을 이르는 고사성어.
입에서는 달콤한 말을 하지만 뱃속에는 칼을 품고 있음.
口(입 구), 蜜(꿀 밀), 腹(배 복), 劍(칼 검)
