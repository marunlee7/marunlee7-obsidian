# 擬(비길 의)


> [!summary]+ 한자정리
> 擬 (비길 의)
> 
> *   기본의미 및 자형
>     *   **바림, 견주다, 본뜨다** 등의 뜻을 가진 글자입니다.
>     *   모양자 : 扌(손 수) + 疑(의심할 의)
>     *   구성원리 : 손(扌)으로 의심스러운 것을 견주어 본다는 의미에서 '견주다', '본뜨다'의 뜻을 나타냅니다.
> 
> *   관련 용법
>     *   **견주다**: 다른 것과 비교하여 비슷하거나 같은지 알아보다.
>     *   **본뜨다**: 다른 것과 똑같이 만들거나 흉내 내다.
>     *   **의정(擬定)**: 초안을 만들다.
> 
> *   관련 어휘
>     *   [[擬態]](의태): 생물이 다른 사물과 비슷하게 보이도록 하는 것.
>     *   [[模擬]](모의): 실제와 비슷하게 흉내 내어 실시하는 것.
>     *   [[假擬]](가의): 가상으로 설정함.
>     *   [[擬古]](의고): 옛것을 본받음.
>     *   [[比擬]](비의): 빗대어 비교함.
> 
> *   고전 활용
>     *   사서삼경에는 직접적으로 활용된 예는 찾기 어렵습니다.
>
