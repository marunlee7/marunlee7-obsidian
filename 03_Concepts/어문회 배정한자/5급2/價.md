# 價(값 가)

- [[價]]   
	- 값 가 → 가치 가
		- 사람(人)이 장사(賈)할 때 부르는 값이니 **값 가**
		- 값을 매기는 가치니 **가치 가**
			- `약` 価
				- 사람(人) 물건을 덮어(覀) 놓고 파는 값이니 **값 가**
				- 값을 매기는 가치니 **가치 가**
			- 賈 덮어(覀) 쌓아놓고 재물(貝)을 파는 장사니 **장사 고, 성씨 가**
	- 油價
		- ==유가==
			- 석유(石油)의 가격(價格).
				- 油 기름 유 價 값 가
	- ① ==代價== ② 臺架
		- 대:가
			- 물건(物件)을 산 대신(代身)의 값
				- 代 대신할 대 價 값 가
	- 價格
		- 가격
			- 물건(物件)이 지니고 있는 가치(價値)를 돈으로 나타낸 것. 값.
				- 價 값 가 格 격식 격
	- 價値
		- 가치
			- 사물(事物)이 지니고 있는 쓸모.
				- 價 값 가 値 값 치
	- 單價
		- 단가
			- 각 단위(單位)마다의 값.
				- 單 홑 단 價 값 가
	- 原價
		- 원가
			- 상품(商品)을 완성(完成)시킬 때까지 소비(消費)한 재화(財貨), 용역(用役)을 단위(單位)에 따라 계산(計算)한 가격(價格). 생산비(生產費).
				- 原 언덕 원 價 값 가
	- 定價
		- 정가
			- 정해진 값.
				- 定 정할 정 價 값 가
	- 廉價
		- 염가
			- 時勢보다 헐한 값. 싼 값.
				- 廉 청렴할 렴 價 값 가
	- 公示地價
		- 공시지가
			- 국토교통부 장관이 조사･ 평가하여 공시한 표준지의 단위 면적(m)당 가격
				- 公 공평할 공 示 보일 시 地 따 지 價 값 가
