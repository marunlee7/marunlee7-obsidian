# 孼(서자 얼)
 

> [!summary]+ 한자정리
> 孼(서자 얼)
> 
> *   기본의미: 서자, 재앙, 사특함
> *   자형: 
>     *   모양자: 孽(싹 얼)의 본자
>     *   구성원리: 자(子, 아들 자) + 蘖(황벽나무 얼)
> *   관련 용법:
>     *   정실 부인이 아닌 다른 여자에게서 태어난 자식을 의미합니다.
>     *   재앙이나 사특한 기운을 비유적으로 나타내기도 합니다.
> *   관련 어휘:
>     *   [[孽子]](얼자): 서자
>     *   [[孽腹]](얼복): 서자를 낳은 배
>     *   [[孽種]](얼종): 서자, 비하하는 표현으로 사용됨
>
