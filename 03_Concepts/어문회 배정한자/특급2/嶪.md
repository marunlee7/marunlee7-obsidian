# 嶪(산높을 업)
 

> [!summary]+ 한자정리
> 嶪 (산높을 업)
> 
> *   기본 의미: 산이 높고 험준함을 나타내는 한자입니다.
> *   자형 설명:
>     *   모양자 : 山(뫼 산) + 業(업)
>     *   구성원리 : 산(山)이 겹쳐 있는 모습으로, 험준하고 높다는 의미를 강조합니다. '業(업)'은 소리 요소로 작용합니다.
> *   관련 용법: 일상적인 단어로는 잘 사용되지 않지만, 산세의 험준함을 묘사할 때 쓰일 수 있습니다.
> *   관련 어휘:
>     *   嶪嶪(업업): 산이 험준하고 높이 솟아 있는 모양.
> *   고전 활용: 천자문, 사서삼경 등에서는 찾아보기 어렵습니다.
