# 剡(날카로울 염)
 

> [!summary]+ 한자정리
> 剡(날카로울 염)
> 
> *   기본 의미: 날카롭다, 깎다, 뾰족하다.
> *   자형: 炎(불꽃 염) + 刀(칼 도)
>     *   구성 원리: 불꽃처럼 타오르는 칼날의 형상에서 '날카롭다'는 의미를 나타낸다.
> *   관련 용법:
>     *   날카로운 칼로 무언가를 깎아내는 모습에서 유래하여 '깎다', '다듬다'의 의미로 확장되었다.
>     *   '뾰족하다'는 의미도 내포한다.
> *   관련 어휘:
>     *   [[剡溪]](염계): 지명. 중국 저장성에 있는 계곡.
>     *   [[剡注]](염주): 깎아서 덧붙임. 수정하여 첨가함.
>
