# 嶸(산가파를 영)
 

> [!summary]+ 한자정리
> 嶸(산 가파를 영)
> 
> *   기본의미: 산 모양이 험하고 가파르다.
> 
> *   자형: 山(뫼 산) + 榮(영화 영)
> 
>     *   모양자: 山(산), 榮(꽃 영)
>     *   구성원리: 산(山)이 영화(榮)롭게 솟아오른 모양을 나타냅니다. 산세가 험준하고 빼어남을 의미합니다.
> 
> *   관련 용법: 주로 산의 험준함, 기이함, 빼어남을 묘사할 때 사용됩니다.
> 
> *   관련 어휘:
> 
>     *   [[崢嶸]](쟁영): 산세가 높고 험준함, 재능이 뛰어나 두각을 나타냄.
>     *   [[嶸峻]](영준): 산이 높고 험함.
>     *   [[嶸崎]](영기): 산길이 험하고 가파름.
> 
> *   문장 고시 고전 활용: 해당 글자는 천자문이나 사서삼경에서 직접적으로 자주 사용되지는 않지만, 산세의 험준함을 묘사하는 시나 문학 작품에서 찾아볼 수 있습니다. 구체적인 예시는 작품에 따라 다릅니다.
>
