# 肄(익힐 이)
 

> [!summary]+ 한자정리
> 肄 (익힐 이)
> 
> *   기본 의미: 익히다, 배우다
> *   자형: 나무를 심는 모양을 본뜬 글자
> *   구성 원리: 나무를 심는 모습에서 '익히다', '배우다'의 의미로 확장
> *   관련 용법:
>     *   학업이나 기술을 익히는 것을 의미
>     *   예의범절을 익히는 것을 의미
> *   관련 어휘:
>     *   [[肄業]](이업): 학업을 중도에 그만둠. (예: 대학교 2학년을 肄業하다)
>     *   [[肄習]](이습): 반복하여 익힘.
