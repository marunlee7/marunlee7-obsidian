# 渲(바림 선)


> [!summary]+ 한자정리
> ## 渲
> 
> *   **음**: 바림 선
> *   **기본 의미**:
>     *   바림 (수묵화 기법에서 색을 자연스럽게 번지게 하는 것, 그러데이션)
>     *   작은 흐름, 물이 잔잔히 흐르는 모양
> *   **자형**: 물 수(氵) + 宣(베풀 선)으로 구성. 물을 사용하여 색을 펼쳐 바림하는 의미를 나타냄.
> *   **관련 용법**: 주로 수묵화에서 색을 번지게 하여 깊이와 입체감을 표현하는 기법에 사용됨.
> *   **관련 어휘**:
>     *   **渲染(현염)**: 물감을 풀어 번지게 함. 분위기를 고조시키거나 과장하여 표현할 때도 사용.
>     *   **渲染法(현염법)**: 수묵화에서 색을 번지게 하는 바림 기법.
>     *   **渲流(현류)**: 잔잔히 흐르는 물, 작은 물줄기가 흐르는 모양.
>
