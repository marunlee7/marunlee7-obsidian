# 堧(빈땅 연)
 

> [!summary]+ 한자정리
> 堧(빈땅 연)
> 
> *   기본의미 및 자형:
>     *   비어 있는 땅, 집터의 빈터, 묘지 주변의 땅을 의미합니다.
>     *   모양자 : 土(흙 토) + 耎(อ่อน할 연)
>     *   구성원리 : 형성자. 흙(土)이 부드럽고(耎) 비어 있는 땅이라는 의미를 나타냅니다.
> *   관련 용법:
>     *   일반적으로 잘 쓰이지 않는 한자입니다.
> *   관련 어휘:
>     *   壖地(연지): 집터의 빈 땅.
>     *   封壖(봉연): 묘지 주변의 땅.
