# 丵(풀 무성할 착)

[[辛]] 매울 신 =  立(설 립(입)) + 十(열 십)

[[丵]] 풀 무성할 착

[[菐]] 무더기 복 = 풀 무성하듯(丵) 큰(大) 무더기니 **무더기 복**

[[僕]] 종 복 = 사람(人)중 무더기(菐)로 많은 일을 처리하는 종이니 **종 복**
