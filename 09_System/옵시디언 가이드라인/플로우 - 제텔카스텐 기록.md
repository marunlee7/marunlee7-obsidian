# 제텔카스텐 기록 준비 발췌 실제

- [[2024-12-23]]
- 책을 읽으면서 옵시디언 또는 록식을 기반으로 정리하는 전체 과정은 어떤 흐름으로 진행되는지 정리해보자.
- 제텔카스텐 기록준비 - 독서형태
	- 종이책, 전자책, 오디오북 장단점 정리
	- 밀리의 서재 이용하기
- 제텔카스텐 문장 발췌 - 메모방법
	- 종이책
		- 1) 마음껏 밑줄 그으면서 읽고 나중에 타이팅 - 기화펜으로 밑줄 긋고 30분 내에 타이핑
		- 2) 핸드폰 카메라 이용 OCR 기능 이용
		- 3) 메모할 부분을 녹음해서 음성인식
- 제텔카스텐 실제 예시
	- 작가의방 :  https://youtu.be/sFGFCTGxscI?si=CzoZd39rhgGkgIeZ
	- 북 캡쳐 메모한 내용 이용 문헌노트 만들기 방법 - 템플릿 양식이 된다
		- 1) 캡쳐한 내용을 나의 언어로 제목만들기
		- 2) 캡쳐한 내용은 아래 내용으로 넣기
		- 3) 태그는 키워드로 넣는다.
		- 4) 여기까지는 이제껏 내가 해온 방식인데 템플릿의 형태가 중요하다.
			- 작가의방 템플릿 구성
				- 나만의 언어 제목
				- 날짜
				- 토픽 : 태그
				- 내용
		- 5) 캡쳐 내용에 대한 내느낌, 태그 넣고
		- 6) 출처 기록
		- 7) 제목은 나만의 메타언어로 넣기가 중요
		- 8) 기존 북 목차는 유지하되 내용만을 나만의 메타언어 변경제목으로 넣는게 중요
		- 9) 연결 - 관련 메모는 아래에 링크해준다.
		- 0) 기호 구분
