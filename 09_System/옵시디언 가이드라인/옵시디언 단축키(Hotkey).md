## 옵시디언 기본 단축키

### 파일관리

| **동작**           | **Mac 단축키**                            | **설명**                  |
| ---------------- | -------------------------------------- | ----------------------- |
| 명령어 팔레트 열기       | $\text{Cmd} + \text{P}$                | 옵시디언의 거의 모든 기능을 검색하여 실행 |
| 빠른 파일 열기 (퀵 스위처) | $\text{Cmd} + \text{O}$                | 파일 이름을 검색하여 빠르게 노트 이동   |
| 새 노트 생성          | $\text{Cmd} + \text{N}$                | 현재 Vault에 새 빈 노트 생성     |
| 활성 창(탭) 닫기       | $\text{Cmd} + \text{W}$                | 현재 보고 있는 노트/탭 닫기        |
| 모든 노트에서 검색       | $\text{Cmd} + \text{Shift} + \text{F}$ | Vault 내 모든 파일에서 텍스트 검색  |
| 현재 노트 안에서 검색     | $\text{Cmd} + \text{F}$                | 현재 파일 내에서 텍스트 검색        |

### 편집 및 서식

| **동작**        | **Mac 단축키**                                                    | **설명**                           |
| ------------- | -------------------------------------------------------------- | -------------------------------- |
| 편집/읽기 모드 전환   | $\text{Cmd} + \text{E}$                                        | 라이브 프리뷰(편집) 모드와 읽기 모드 전환         |
| 실행 취소 / 다시 실행 | $\text{Cmd} + \text{Z} / \text{Cmd} + \text{Shift} + \text{Z}$ | 일반적인 실행 취소 및 다시 실행               |
| 서식 없이 붙여넣기    | $\text{Cmd} + \text{Shift} + \text{V}$                         | 복사한 텍스트의 스타일 정보를 제거하고 붙여넣기       |
| 현재 줄 삭제       | $\text{Cmd} + \text{Shift} + \text{K}$                         | 커서가 있는 줄 전체 삭제 (텍스트 선택 없어도 작동)   |
| 외부 링크 삽입      | $\text{Cmd} + \text{K}$                                        | $\text{[텍스트](링크 주소)}$ 형식으로 링크 삽입 |
| 들여쓰기 / 내어쓰기   | $\text{Tab} / \text{Shift} + \text{Tab}$                       | 리스트 등에서 들여쓰기/내어쓰기                |

### 코어 플러그인

| **기능 영역**  | **플러그인 (명령어)** | **Mac 기본 단축키**                          | **설명**                                                    |
| ---------- | -------------- | --------------------------------------- | --------------------------------------------------------- |
| **데일리 노트** | 오늘 데일리 노트 열기   | $\text{Cmd} + \text{D}$                 | 오늘의 데일리 노트를 열거나 생성합니다.                                    |
| **퀵 스위처**  | 빠른 전환 열기       | $\text{Cmd} + \text{O}$                 | 파일을 검색하여 이동합니다. (일반 단축키와 동일)                              |
| **명령 팔레트** | 명령어 팔레트 열기     | $\text{Cmd} + \text{P}$                 | 모든 명령어에 접근합니다. (일반 단축키와 동일)                               |
| **그래프 뷰**  | 로컬 그래프 열기      | $\text{Cmd} + \text{Option} + \text{G}$ | 현재 노트의 로컬 그래프 뷰를 엽니다.                                     |
| **파일 탐색기** | 새 파일 생성        | $\text{Cmd} + \text{N}$                 | 새 노트를 생성합니다. (일반 단축키와 동일)                                 |
| **파일 탐색기** | 현재 파일 제목 편집    | $\text{F2}$                             | 현재 파일의 이름을 변경합니다.                                         |
| **백링크**    | 활성 파일의 백링크 토글  | Ctrl + Shift + B                        | (기본 설정 없음, 사용자 지정 권장)                                     |
| **템플릿**    | 템플릿 삽입         | $\text{Cmd} + \text{2}$                 | (기본 설정 없음, 사용자 지정 권장. $\text{Cmd} + \text{T}$ 등으로 많이 사용됨) |
|            |                |                                         |                                                           |
|            |                |                                         |                                                           |
