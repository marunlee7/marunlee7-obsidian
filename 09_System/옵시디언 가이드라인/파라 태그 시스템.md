# 아카이브


옵시디언에서 효율적인 분류와 검색을 위해 많이 사용되는 **상태(Status), 중요도(Priority), 그리고 주제(Topic) 태그 일람**을 제안하며, 계층적 태그(`parent/child`)를 사용하여 체계를 잡는 방법을 제시합니다.

## 🏷️ 옵시디언 태그 시스템 제안 (체계화)

태그를 단순히 나열하는 대신, 
용도에 따라 **접두사**를 붙여 **계층적 태그**로 분류하면 검색이 훨씬 용이해집니다.

### 1. ⚙️ 상태 및 실행 태그 (Action & Status Tags)

노트의 **실행 상태**와 **후속 조치 필요 여부**를 나타냅니다.

| **태그 (예시)**            | **사용 목적**                           | **기존 태그 매칭**            |
| ---------------------- | ----------------------------------- | ----------------------- |
| **`#status/todo`**     | 아직 처리하거나 완료해야 할 사항이 남아 있는 노트.       | `#작업노트` (후속 작업 필요 시)    |
| **`#status/review`**   | 주기적으로 검토하거나 재확인해야 하는 노트.            | `#사색` (깊은 성찰이 필요할 때)    |
| **`#status/ongoing`**  | 현재 진행 중인 프로젝트나 작업과 관련된 노트.          | `#작업노트` (진행 중인 작업 기록 시) |
| **`#status/archived`** | 완료되거나 비활성화되어 `4. 기록 보관소`로 옮겨야 할 노트. | (신규 추가)                 |

### 2. ⭐ 중요도 및 유형 태그 (Priority & Type Tags)

노트의 **중요도, 가치, 그리고 학습 유형**을 나타냅니다.

|**태그 (예시)**|**사용 목적**|**기존 태그 매칭**|
|---|---|---|
|**`#p/high`**|당장 실행하거나 기억해야 할 **최우선 정보**.|`#중요`, `#일침`|
|**`#type/insight`**|개인적인 **깨달음, 통찰, 사색**이 담긴 노트.|`#사색`, `#인생기억`|
|**`#type/quote`**|외부 자료에서 발췌하거나 **필사(인용)**한 내용.|`#필사`, `#베껴쓰기`|
|**`#type/concept`**|핵심 개념, 이론, 정의 등 **지식의 원천**이 되는 노트.|(신규 추가)|

### 3. 📝 주제 및 영역 태그 (Area & Topic Tags)

노트가 PARA 시스템의 **어떤 영역**에 속하는지 명시합니다.
(`20-에어리어` 폴더 구조와 일치해야 합니다.)

| **태그 (예시)**          | **사용 목적**             |
| -------------------- | --------------------- |
| **`#area/health`**   | 건강 및 운동 루틴, 식단 관련 노트  |
| **`#area/finance`**  | 재정, 예산 관리 관련 노트       |
| **`#area/learning`** | 특정 기술 학습, 강좌 진행 노트    |
| **`#topic/PARA`**    | PARA 시스템 자체에 대한 지식 노트 |

## 🛠️ 기존 태그를 체계화하는 방법

사용자님의 기존 태그들을 위 시스템에 맞춰 재분류하면 다음과 같습니다.

|**기존 태그**|**제안된 계층 태그**|**설명**|
|---|---|---|
|`#사색`|`#type/insight`, `#status/review`|사색은 깨달음 유형(`type`)이자, 주기적 검토가 필요한(`status`) 노트로 분류합니다.|
|`#작업노트`|`#status/todo`, `#status/ongoing`|작업의 상태에 따라 `todo` 또는 `ongoing`으로 나눕니다.|
|`#중요`, `#일침`|`#p/high` (Priority)|중요도 계층 태그로 통합하여 관리합니다.|
|`#필사`, `#베껴쓰기`|`#type/quote`|내용의 유형을 나타내는 태그로 통합합니다.|
|`#인생기억`|`#type/insight` 또는 `#area/journal`|개인의 깊은 통찰 또는 저널 영역의 핵심 기록으로 분류합니다.|

이러한 계층적 태그 시스템을 사용하면, 예를 들어 `#p/high` 태그로 모든 중요한 노트를 모으면서, `tag:#p/high tag:#area/health` 와 같은 조합으로 **'건강 영역에서 가장 중요한 내용'**만 필터링할 수 있어 매우 강력해집니다.

이 가이드라인은 옵시디언의 **계층적 태그** 기능을 활용하여 복잡도를 낮추고 검색 효율을 높이는 데 초점을 맞춥니다.

## 🏷️ 옵시디언 태그 시스템 통합 관리 가이드

이 시스템은 태그를 **상태/실행력**, **중요도/유형**, **영역/주제**의 세 가지 주요 계층으로 분류합니다.

|**태그 구조 원칙**|**설명**|**예시**|
|---|---|---|
|**계층적 태그 사용**|태그 이름에 `/` (슬래시)를 사용하여 그룹화하고 검색 효율을 높입니다.|`#status/todo`|
|**소문자/단수형**|태그의 일관성을 위해 소문자 또는 캐멀 케이스(CamelCase)를 사용하고, 일반적으로 단수형을 권장합니다.|`#area/health` (O), `#Area/Healths` (X)|
|**폴더와의 분리**|물리적 폴더(`20-에어리어`)와 태그(`area/`)는 별개로 사용하며, 필요에 따라 태그가 유연성을 제공합니다.|`#area/health`는 `20-에어리어/건강` 폴더 안에 있지 않은 노트에도 적용 가능합니다.|

### 1. ⚙️ 상태 및 실행 태그 (Status Tags)

노트의 **후속 조치 필요 여부**나 **실행 단계**를 표시합니다. 
기존의 `#작업노트` 역할을 대체합니다.

|**태그**|**역할**|**기존 태그 매칭**|
|---|---|---|
|**`#status/todo`**|즉시 처리해야 할 **할 일**이나 작업이 남은 노트.|`#작업노트` (후속 작업 필요 시)|
|**`#status/ongoing`**|현재 진행 중인 프로젝트나 지속적인 작업 기록.|`#작업노트` (진행 중)|
|**`#status/review`**|주기적으로 **검토**하거나 성찰이 필요한 노트. (예: 주간 회고 전)|`#사색` (깊은 성찰이 필요할 때)|
|**`#status/archived`**|완료되었거나 비활성화되어 `40-아카이브`로 이동 예정인 노트.|(신규 추가)|

### 2. ⭐ 중요도 및 유형 태그 (Priority & Type Tags)

노트가 가진 **정보의 가치**나 **학습의 유형**을 분류합니다.

|**태그**|**역할**|**기존 태그 매칭**|
|---|---|---|
|**`#p/high`**|당장 실행하거나 **최우선으로 기억**해야 할 중요한 통찰/정보.|`#중요`, `#일침`|
|**`#type/insight`**|개인적인 **깨달음, 통찰**, 또는 감정 기록.|`#사색`, `#인생기억`|
|**`#type/quote`**|외부 자료에서 발췌하거나 **필사(인용)**한 내용.|`#필사`, `#베껴쓰기`|
|**`#type/concept`**|핵심 개념, 이론, 정의 등 지식의 원천이 되는 노트.|(신규 추가)|

### 3. 🌐 영역 태그 (Area Tags)

노트가 속한 **PARA 영역**을 명시합니다. 폴더 구조와 병행하여 유연한 분류를 가능하게 합니다.

|**태그**|**역할**|**매칭 폴더 예시 (20-에어리어 내)**|
|---|---|---|
|**`#area/journal`**|저널, 일일/주간/월간 노트 등 주기적 기록.|`20-에어리어/Journals`|
|**`#area/health`**|건강, 식단, 운동 등 자기 관리 영역.|`20-에어리어/건강`|
|**`#area/career`**|경력 개발, 업무 루틴 등 직업적 책임 영역.|`20-에어리어/커리어`|
|**`#area/learning`**|특정 기술 학습, 지식 연마 등.|`20-에어리어/개인 개발`|

---

이 통합된 태그 시스템을 사용하면, 예를 들어 `tag:#p/high tag:#area/health`와 같은 검색 쿼리를 통해 **"건강 영역에서 가장 중요한 내용"**만 즉시 찾아볼 수 있습니다.
