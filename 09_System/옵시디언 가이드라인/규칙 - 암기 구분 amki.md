# 규칙 - 암기 구분 Amki


개인적인 평가 : 특정 책에 대해 자신의 점수를 매기는 것
다시 시도하면 어려운 일이 좋게 변하고 쉬워진다.
어려운 일을 다시 시도하면 결국 좋은 결과를 얻고 쉬워진다.
예시) 한자 적용, 안키 앱 이용시

## 안키 4단계

- 다시, 어려움, 좋음, 쉬움
- [[Again]], [[Hard]], [[Good]], [[Easy]] 

## 이해도 평가

- Again 
	- 다시
	- 아직 잘 기억되지 않거나, 어려운 경우. 
	- 이 카드는 다시 보아야 한다는 의미입니다.
- Hard
	- 어려움
	- 기억하기는 하지만 여전히 어렵거나, 
	- 조금 더 연습이 필요한 경우.
- Good
	- 좋음
	- ==잘 기억되지만, 조금 더 연습하면 더 좋을 것 같은 경우.==
	- 기준점
- Easy
	- 쉬움
	- 매우 잘 기억되며, 더 이상 반복할 필요가 없는 경우.
