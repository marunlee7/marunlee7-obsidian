# 태그 규칙

- 구성이유: 높은 수준의 논리와 문장력 → 글쓰기의 토대를 만든다.
- 태그를 꼭 프론트매터에 모아서 써야 할 필요는 없어요. 
- 하지만 그렇게 하고 싶으시면 Linter나 Frontmatter Tag Suggest 같은 플러그인 사용하시는 것도 괜찮을 것 같아요.

## 메모 성격
- 대화 : 대화에 필요한 말들, 논리 정연한 말들
- 명언 : 삶의 통찰력이 돋보이는 글들, 영화, 책에서 보이는 멋진 말들
- 공감 : 글쓴이의 문장이 여러모로 공감될 때
- 추천: 모든 것의 추천
- 청상: 문장에 대한 나의 감상 일때
- 단상: 청상과 차이점은 무언가
