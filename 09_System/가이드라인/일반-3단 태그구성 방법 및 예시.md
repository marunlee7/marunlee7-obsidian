# 일반-3단 태그구성 방법 및 예시


## 🛠️ 구성 방법 및 예시

태그를 구성할 때는 각 단계가 독립적이면서도 상호 보완적으로 작동하도록 설계하는 것이 중요합니다.

### 1. 📚 예시 1: 소프트웨어 개발 프로젝트 관련 지식

|태그 단계|구성 방법 (원칙)|태그 예시|설명|
|---|---|---|---|
|1단: 개념|지식의 핵심 내용/주제|`#개발언어/Python`, `#알고리즘/정렬`, `#클라우드/AWS`|가장 상위의 지식 유형을 분류합니다.|
|2단: 영역|해당 정보가 사용되는 프로젝트/업무|`#프로젝트/데이터분석`, `#업무/백엔드`, `#팀/마케팅`|정보의 사용 맥락을 분류합니다.|
|3단: 출처|정보가 생성된 경로|`#출처/Udemy강의`, `#문서/DRD`, `#유형/회의록`|정보의 원본을 분류합니다.|

👉 최종 태그 세트 예시:

- `#개발언어/Python` / `#프로젝트/데이터분석` / `#출처/Udemy강의`
- `#클라우드/AWS` / `#업무/백엔드` / `#문서/DRD`
    

### 2. 📰 예시 2: 뉴스/아티클 스크랩

|태그 단계|구성 방법 (원칙)|태그 예시|설명|
|---|---|---|---|
|1단: 개념|아티클의 핵심 주제/기술|`#주제/AI반도체`, `#기술/LLM`, `#정책/EU`|기사 내용의 핵심을 분류합니다.|
|2단: 영역|개인적인 관심사/검토 필요 분야|`#검토/투자`, `#스터디/기술동향`, `#관심/미래산업`|정보를 활용하려는 목적을 분류합니다.|
|3단: 출처|매체 유형 또는 발행 기관|`#매체/WSJ`, `#매체/TechCrunch`, `#유형/논문`|정보의 신뢰도 판단에 도움을 줍니다.|

👉 최종 태그 세트 예시:

- `#주제/AI반도체` / `#스터디/기술동향` / `#매체/WSJ`
- `#기술/LLM` / `#검토/투자` / `#유형/논문`


## 🏷️ 예시 3: 개별 한자 및 한자 어휘 공부

한자/어휘 학습 노트는 '개념'을 언어 자체로, '영역'을 학습 수준이나 목표로 분류하여 체계적인 복습을 가능하게 합니다.

| 태그 단계            | 구성 방법 (원칙)                 | 태그 예시                               | 설명                 |
| ---------------- | -------------------------- | ----------------------------------- | ------------------ |
| 1단: 개념 (Concept) | 무엇을 배우는가? (한자의 종류, 어휘의 주제) | `#한자/부수`, `#어휘/의료`, `#문법/사자성어`      | 지식의 가장 기본적인 내용 분류  |
| 2단: 영역 (Domain)  | 왜 배우는가? (학습 목표, 시험 등)      | `#시험/JLPT_N1`, `#목표/법률용어`, `#수준/고급` | 학습의 목적 또는 적용 분야 분류 |
| 3단: 출처 (Source)  | 어디서 얻었는가? (교재, 앱, 웹사이트)    | `#교재/능률보카`, `#앱/네이버사전`, `#유형/오답노트`  | 정보의 원본 또는 생성 경로 분류 |

- 개념/어휘/법률, 개념/한자/법
- 영역/목표/업무용어, 영역/수준/중급
- 출처/교재/법률문서해설


## 🏷️ 예시 4: 언론사 칼럼 시리즈를 모아서 볼 때

칼럼이나 사설 시리즈는 시간이 지나면서 그 내용이 변화하거나 연속성을 갖습니다. 이때 '영역'을 시리즈 이름으로 두어 관리하면 편리합니다.

|태그 단계|구성 방법 (원칙)|태그 예시|설명|
|---|---|---|---|
|1단: 개념 (Concept)|칼럼의 핵심 주제 또는 내용|`#주제/경제`, `#주제/정치개혁`, `#기술/IT정책`|기사 내용의 핵심 주제|
|2단: 영역 (Domain)|칼럼 시리즈의 이름 또는 프로젝트|`#시리즈/이코노미스트`, `#프로젝트/데이터활용`, `#개인/필사주제`|시리즈 또는 정보의 사용 맥락|
|3단: 출처 (Source)|집필한 사람/기관 또는 유형|`#저자/김OO`, `#언론사/한겨레`, `#유형/사설`|정보의 출처 및 작성자|

- 개념/기술/IT정책, 개념/주제/산업규제
- 영역/시리즈/기술과미래, 영역/프로젝트/법규검토
- 출처/저자/김정운, 출처/언론사/매일경제
