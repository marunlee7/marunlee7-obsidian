# 3단 출처 (Source) 태그 최종 표준 목록
## 📚 3단 출처 (Source) 태그 최종 표준 목록


이 표는 앞으로 사용자님의 모든 노트에 **'3단 출처'** 를 부여할 때 참고하실 수 있는 사전 역할을 합니다.


| **원본 분류 (Prefix)** | **Prefix 역할 (정보 근원)**        | **세부 형식 (Value 예시)**                 | **기타 (적용 기준)**                 |
| ------------------ | ---------------------------- | ------------------------------------ | ------------------------------ |
| **`외부`/**          | **정제된 외부 콘텐츠** (전문적인 지식 근원)  | `도서_단행본`, `매체_칼럼_단건`, `아티클_웹`, `보고서` | 신뢰도가 높은, 단발성/정규 발행 콘텐츠         |
| **`웹_매체`/**        | **웹 기반 플랫폼**을 통한 정보 근원       | `웹_블로그`, `웹_영상/유튜브`, `커뮤니티_글`        | 개인/비정규적인 웹 플랫폼 콘텐츠             |
| **`협업`/**          | **타인과의 상호작용**을 통해 얻은 정보      | `회의록`, `이메일`, `메신저`, `구두_전달`, `인터뷰`  | 팀워크, 업무 관련 지식의 출처              |
| **`경험`/**          | **사용자의 능동적인 활동**을 통해 얻은 정보   | `실습/테스트`, `관찰`, `시행착오`, `직접_경험`      | 지식 획득 경로가 '행동'일 때              |
| **`내부`/**          | **사용자 자신의 사고 과정**을 통해 생성된 정보 | `개인_성찰`, `사색`, `통찰`, `아이디어_구상`       | 정보 근원이 노트 작성자의 '머릿속'일 때        |
| **`수집`/**          | **단순 저장/링크** 형태의 데이터         | `북마크`, `스크린샷`, `데이터_파일`              | 2차 가공 없이 수집된 원본 데이터            |
| **`저널_형식`/**       | **문서의 형식**이 일지/저널 형태일 때      | `일반_저널`, `활동_일지`, `스터디_기록`           | 기록물의 형식 자체를 분류                 |
| **`시스템`/**         | **지식 관리 시스템 자체**에 대한 문서      | `시스템_문서`, `주제_MOC`, `태그_정의`          | T-시스템의 구조를 다룰 때 (2단 영역과 교차 가능) |

  - 경험/실습/테스트   # 직접 해보고 얻은 결과 (경험)
  - 웹_매체/기술블로그  # 테스트를 위해 참고한 외부 문서 (외부)
