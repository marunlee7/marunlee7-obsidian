# T03 한자 어휘 템플릿 양식 Ver1


> [!note]
> 향후 일관성이 깨지거나 오류가 발생할 경우, 이 지시서를 복사하여 제게 제시해 주시면 됩니다.


이 지시서는 **Obsidian Templater 플러그인 에러 방지** 및 **한자어휘 학습의 효율성**을 극대화하도록 설계된 템플릿 양식의 최종 규칙을 정의합니다.

## 1. 전반적인 레이아웃 및 아웃라인 규칙

| **항목**                                      | **아웃라인 기호 (-)** | **들여쓰기** | **백틱 (``)** |
| ------------------------------------------- | --------------- | -------- | ----------- |
| **최상위 단어 태그** (`[[단어]]`)                    | 사용 안 함          | 0단       | 사용 안 함      |
| **중간 머리말** (`한자 분석`, `연관 어휘`)               | **사용**          | 1단       | **사용** (필수) |
| **모든 내용 항목** (독음, 의미, 한자 풀이, 예시, 훈음, 하위 항목) | **사용**          | 1단 또는 2단 | 사용 안 함      |
|                                             |                 |          |             |

## 2. 세부 항목 구성 및 작성 규칙 (필수 준수 사항)a

| **순서** | **항목 구성**   | **내용 구성 원칙**                                                                             |
| ------ | ----------- | ---------------------------------------------------------------------------------------- |
| **1.** | `[[단어]]`    | **Obsidian 문서 제목 태그.** (예: `[[記錄]]`)                                                     |
| **2.** | `- 독음`      | 독음 표기 (예: `- 기록`)                                                                        |
| **3.** | `- 상세 의미`   | **순수 한국어 정의만 작성**하며, 한자 풀이를 혼합하지 않음. **볼드체(******) 사용 가능.**                              |
| **4.** | `- 한자 풀이`   | **한자 구성 요소만 풀이**하며, 정의를 혼합하지 않음. (예: `기록할(記) + 기록할(錄)이라는 뜻.`)                            |
| **5.** | `- 예시`      | 활용 예시 문장.                                                                                |
| **6.** | `- 훈음`      | 구성 한자의 훈음 나열. **슬래시(`/`) 사용 허용** (예: `定 정할 정/정할 정, 稿 볏짚 고/원고 고`)                         |
| **7.** | `- 최종 링크`   | **Obsidian 내부 링크 생성 필드.** 슬래시(`/`)는 **절대 사용하지 않으며**, 훈음 등은 **콤마(`,`) 또는 공백**으로 구분하여 작성함. |
| **8.** | `-` 한자 분석`` | **백틱(`)**과 **아웃라인(`-`)**을 사용하여 머리말 생성.                                                   |

## 3. 최종 출력 예시

```
[[記錄]]
- 기록
	- **사물이나 사실을 글이나 그림, 영상 등으로 남기는 것.** 또는 그 남긴 내용.
	- 기록할(記) + 기록할(錄)이라는 뜻.
	- 예시) 그는 매일의 일과를 빠짐없이 수첩에 기록하는 습관이 있다.
	- 記 기록할 기, 錄 기록할 록
	- [[記錄_기록_사물이나 사실을 글이나 그림, 영상 등으로 남기는 것._기록할(記) + 기록할(錄)이라는 뜻._記 기록할 기, 錄 기록할 록]]
- `한자 분석`
	- 記 (기록할 기)
		- 말씀 언(言)과 몸 기(己)가 합쳐져, 말(言)을 자신의 것(己)으로 삼아 잊지 않도록 남긴다는 뜻.
	- 錄 (기록할 록)
		- 쇠 금(金)과 물줄기 록(彔)이 합쳐져, 옛날에 쇠(金)로 된 그릇 등에 무언가를 새겨 남긴다는 뜻.
- `연관 어휘`
	- 문서([[文書]]) - `유사 개념` : 기록한 내용을 적어 놓은 서류.
	- 문헌([[文獻]]) - `상위 개념` : 글이나 문자로 기록되어 전해 내려오는 책이나 자료.
	- 보존([[保存]]) - `유사 개념` : 사물이나 상태가 변하지 않도록 잘 지켜 남김.
```
